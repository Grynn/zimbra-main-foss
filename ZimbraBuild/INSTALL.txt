To build and install the entire pile:

cvs co LiquidAdmin  LiquidArchive  LiquidConsole  LiquidMon  ThirdParty

cd LiquidMon

make clean; make

Results in:

./i386/liquid-store-1.0-YYYYMMDDHHMM.i386.rpm
./i386/liquid-store-1.0-YYYYMMDDHHMM.tgz
./i386/liquid-mta-1.0-YYYYMMDDHHMM.i386.rpm
./i386/liquid-mta-1.0-YYYYMMDDHHMM.tgz

Build a linux box, FC2 minimal install, plus the following packages:
	binutils
	cpp
	cvs
	cyrus-sasl-devel
	e2fsprogs-devel
	expat
	expat-devel
	gcc
	gcc-c++
	glibc-devel
	glibc-headers
	glibc-kernheaders
	kernel-smp
	krb5-devel
	libstdc++-devel
	openssl-devel
	zlib-devel

Read Anand's yum docs for a new yum.conf smb://kenny/public/software/linux/fedora/core/2/HOWTO.liquid

/etc/init.d/sendmail stop
chkconfig sendmail off

RPM INSTALLATION:

rpm -i liquid-store-1.0-YYYYMMDDHHMM.i386.rpm
rpm -i liquid-mta-1.0-YYYYMMDDHHMM.i386.rpm

su - liquid
lqmyinit
lqcontrol startup

Logs go to syslog, local0.  This goes to /var/log/messages by default.

TAR FILE INSTALLATION:

Create liquid, postfix and postdrop groups.
Create liquid user in liquid, postfix AND tty groups
create postfix user in postfix group (NOT IN postdrop group)

cd <BASEDIR>
tar xzf liquid-store*tgz
tar xzf liquid-mta*tgz

sh bin/setperms.sh <BASEDIR>
