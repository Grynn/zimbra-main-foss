Creazione di un nuovo documento

Puoi creare un documento nella Valigetta, progettare la disposizione del contenuto con strumenti di formattazione di tipo Rich Text, controllare l'ortografia e usare parole chiave per creare con facilità link ad altre pagine.

  1. Clicca sulla cartella della Valigetta da utilizzare. Nel riquadro del contenuto verranno visualizzati i file esistenti.

  2. Nell'elencoNuovo sulla barra degli strumenti, seleziona Documento. Verrà visualizzata una pagina vuota. Nell'intestazione, immetti il nome del documento.

  3. Crea il contenuto della pagina. Puoi utilizzare l'editor Rich Text per aggiungere stili, colore e tabelle alla pagina.

  1. Clicca su Salva per salvare il file. Per chiudere il documento, clicca sulla X nell'angolo in alto a destra.

Nota – Se il documento è stato creato nella Valigetta errata, è possibile spostare il file selezionandolo e trascinandolo nella cartella desiderata.
