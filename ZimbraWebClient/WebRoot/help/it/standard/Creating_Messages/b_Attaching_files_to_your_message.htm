xxxxAllegare file al messaggioxxxx

xxxxI messaggi email possono includere allegati.xxxx xxxxPuoi allegare documenti, fogli di calcolo, immagini, presentazioni e altri tipi di file.xxxx

xxxxPer allegare un file a un messaggio:xxxx

  1. xxxxScrivi il messaggio come descritto in xxxxxxxxComposizione di un nuovo messaggio emailxxxxxxxx.xxxx

  2. xxxxClicca su xxxxxxxxAggiungi allegatixxxxxxxx.xxxx xxxxViene visualizzato un nuovo pannello che ti permette di caricare fino a 10 allegati.xxxx

  3. xxxxPer caricare un file, clicca su xxxxxxxxSfoglia... xxxxxxxxper individuare il file.xxxx

  4. xxxxSeleziona il file e clicca su xxxxxxxxAprixxxxxxxx.xxxx

  5. xxxxCarica altri allegati nello stesso modo.xxxx

  6. xxxxPer mostrare le immagini caricate nel corpo del messaggio, seleziona la casella di controllo xxxxxxxxMostra immagini nel corpo del messaggioxxxxxxxx.xxxx 

  7. Dopo aver aggiunto tutti gli allegati, clicca su Allega. xxxxGli allegati vengono elencati al di sotto della riga dell'oggetto.xxxx

  8. xxxxQuando il messaggio è pronto per l'invio, clicca su xxxxxxxxInviaxxxxxxxx per inviarlo insieme agli allegati.xxxx

xxxxNota –xxxxxxxx I destinatari del tuo messaggio email devono avere il software appropriato per aprire e leggere il file.xxxx xxxxI file creati in software di uso comune come Microsoft Office, potranno essere aperti anche da altri utenti sui sistemi desktop più diffusi.xxxx I formati di file comuni, come i file di testo o HTML e i file di immagini, ad esempio GIF o JPG, possono essere aperti con diversi programmi. xxxxDipende comunque sempre dal tipo di sistema usato dall'utente.xxxx

Per eliminare un allegato, deseleziona la casella corrispondente prima di inviare il messaggio email.
