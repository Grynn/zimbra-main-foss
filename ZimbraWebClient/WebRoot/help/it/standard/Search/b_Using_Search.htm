Uso della funzione di ricerca

La funzione di ricerca Zimbra è uno strumento molto potente. Sebbene sia possibile imparare a creare query di testo complesse, come viene descritto in Descrizione del linguaggio di query, un metodo più rapido consiste nel digitare un nome o una parola, senza parole chiave. Ad esempio, per trovare tutti i messaggi contenenti la parola "ora" in qualsiasi posizione all'interno dell'oggetto, nella riga a:, da:, cc: o ccn:, nel corpo del messaggio o in un file allegato, è sufficiente digitare ora nel boxino di ricerca e cliccare su Cerca.

Puoi quindi scegliere dove eseguire la ricerca: nei messaggi, nei contatti personali, nelle Agende personali e negli elenchi di impegni personali.

Puoi inserire qualsiasi valore, ad esempio una parola, il nome, il cognome, il numero di telefono o il nome di dominio.   Per altre opzioni, puoi usare l' asterisco * come carattere jolly.  

Uso di * come carattere jolly nelle ricerche

L'asterisco * può essere usato come carattere jolly  nelle ricerche per trovare contenuto che comprende parole simili fra loro.  

Puoi usare l'asterisco come carattere jolly dopo un prefisso. Ad esempio, la ricerca di do* restituisce parole come dono, donatore, domanda, ecc.

Per cercare contatti presenti nelle tue Rubriche, la query deve essere una parola intera. Ad esempio, per eseguire una ricerca nei contatti in base al numero di telefono, è necessario inserire il numero per intero, così come appare nel campo del contatto. L'inserimento del solo prefisso non consente di ottenere risultati.

Per eseguire una ricerca di contatti nella tua Rubrica:

  1. Nella sezione di ricerca Trova, seleziona Contatti personali dall'elenco a tendina. La ricerca verrà eseguita in tutte le tue Rubriche.

  2. Inserisci un nome o altri dati sui contatti nel boxino di ricerca.

  3. Clicca su Cerca per eseguire la ricerca.

Per eseguire una ricerca rapida di messaggi email:

  1. Nella sezione di ricerca, seleziona Messaggi email dall'elenco a tendina.

  2. Inserisci un nome o altri dati relativi alla mail nel boxino di ricerca.

  3. Clicca su Cerca per eseguire la ricerca.

Creazione di una query di ricerca complessa

Se ti capita di eseguire la stessa query di ricerca con frequenza, o desideri cercare informazioni dettagliate specifiche, può risultare utile crearla e salvarla nella cartella Ricerche. Per ripetere la ricerca, non dovrai fare altro che cliccare sulla ricerca salvata.

Per creare e salvare una query di ricerca

  1. Nel riquadro di navigazione della ricerca, clicca su modifica. Si apre un riquadro Crea nuova ricerca.

  2. Scegli dove creare la cartella di ricerca.  Se decidi di salvarla nella cartella principale, la ricerca verrà visualizzata nella sezione Ricerche. Se selezioni un'altra cartella, la ricerca verrà salvata nella cartella scelta.

  3. Nel campo Nome ricerca, assegna un nome alla ricerca.

  4. Nel campo Query di ricerca, digita la tua query usando il linguaggio di query descritto in Descrizione del linguaggio di query. La query può includere anche gli operatori And o Or.

    Uso di And o OR nella creazione di una query di ricerca

    Per le ricerche che usano più criteri, è possibile trovare elementi che soddisfano solo uno dei criteri specificati o tutti i criteri. :

    • La ricerca di messaggi che soddisfano uno qualsiasi dei criteri specificati viene chiamata ricerca OR, in quanto il messaggio viene considerato una corrispondenza se contiene o X o Y.

    • Una ricerca di messaggi contenenti sia X che Y viene definita ricerca AND in quanto il messaggio deve soddisfare tutti i criteri specificati per poter essere considerato una corrispondenza.

  5. Clicca su Crea ricerca per creare una ricerca e salvarla nella cartella specificata.

Per eseguire quella ricerca, clicca sul nome che le hai assegnato nel riquadro di navigazione. I risultati verranno visualizzati nel riquadro del contenuto.
