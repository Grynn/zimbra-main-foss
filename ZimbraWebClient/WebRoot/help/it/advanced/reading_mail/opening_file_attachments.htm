xxxxApertura dei file allegatixxxx

xxxxI messaggi email possono includere file allegati come documenti di elaborazione testi, fogli di calcolo, file di testo, file ZIP, immagini, applicazioni eseguibili o qualsiasi altro tipo di file che risieda nella rete di computer del mittente.xxxx xxxxI messaggi che contengono allegati sono contraddistinti da una piccola graffetta accanto all'oggetto.xxxx

Alcuni tipi di file, ad esempio le immagini JPEG o i file di testo, possono essere aperti sulla maggior parte dei sistemi. Altri, ad esempio i documenti Microsoft Word, devono essere aperti nella stessa applicazione in cui sono stati creati.

xxxxQuando ricevi un messaggio con allegati, puoi cliccare due volte sul nome del file nella riga Allegati dell'intestazione del messaggio.xxxx xxxxViene visualizzata una finestra di dialogo in cui viene chiesto se vuoi aprire o salvare prima il file.xxxx

  • xxxxSe scegli di visualizzare l'allegato, è necessario che nel tuo computer sia installata l'applicazione che consente di leggere quel tipo di file.xxxx

  • xxxxSe scegli di scaricarlo, viene visualizzata una finestra con il pulsante Sfoglia che mostra il tuo file system corrente.xxxx xxxxScegli un percorso appropriato e salva il file.xxxx xxxxPotrai aprirlo in un secondo momento anche al di fuori del tuo client di posta.xxxx

Nota: Gli allegati possono essere eliminati dai messaggi e-mail. La selezione di Rimuovi tutti gli allegati consente di eliminare i file dalla casella di posta. Quando si rimuove un allegato, questo viene eliminato immediatamente dalla propria casella di posta. 

xxxxVisualizzazione di allegati come HTMLxxxx

xxxxÈ possibile che nel tuo sistema sia disponibile la funzione che consente di visualizzare gli allegati come HTML.xxxx xxxxCiò consente di aprire un allegato anche se l'applicazione corrispondente non è installata.xxxx Clicca su Visualizza comehtml per visualizzare il contenuto del file. xxxxQuesta funzione viene impostata dal tuo amministratore del sistema.xxxx

xxxxProtezione dai virusxxxx

xxxxLa diffusione dei virus informatici può avvenire anche attraverso gli allegati email.xxxx xxxxGli amministratori della posta seguono la prassi ormai consolidata di implementare i filtri anti-virus nella posta in arrivo e di eseguire la scansione anti-virus nei personal computer.xxxx xxxxTuttavia, esiste sempre la possibilità che qualche virus riesca ad aggirare tutte le difese impostate.xxxx

xxxxNel caso dei virus della posta, l'attivazione avviene quasi sempre al momento dell'apertura o della visualizzazione del file allegato che li contiene.xxxx xxxxMolti virus si diffondono attraverso le voci delle Rubriche degli utenti.xxxx xxxxPer questo motivo, è importante aprire sempre con grande cautela i file allegati, anche quando provengono da mittenti conosciuti e attendibili.xxxx
