Uso dell'editor Rich Text

Per creare le tue pagine, puoi usare l'editor Rich Text per formattare il contenuto oppure  Origine HTML per formattare la pagina con i tag HTML. Inoltre puoi passare da un formato all'altro per sfruttare le funzioni dell'editor Rich Text e inserire tabelle e fogli di calcolo.

Quando usi l'editor Rich Text per apportare modifiche, la barra degli strumenti a due righe ti consente di formattare con facilità la pagina. Seleziona le dimensioni, i tipi e i colori dei caratteri, gli stili di testo e i colori dello sfondo. Il funzionamento di questi strumenti di formattazione è analogo a quello di molti strumenti di elaborazione testi.

Inoltre, puoi aggiungere tramite clic alla tua pagina tabelle, fogli di calcolo, allegati e immagini.

Aggiunta di stili di paragrafo

L'icona    nella barra degli strumenti mostra gli stili di paragrafo disponibili. La funzione di questi stili è di rendere più semplice  la formattazione del testo e mantenere uno stile uniforme nelle pagine del Blocco Note. Gli stili più comuni sono i seguenti: Normale per la maggior parte dei paragrafi; sei livelli di Intestazioni per identificare i titoli di sezione;  Indirizzo per formattare il testo in corsivo e Preformattato da usare quando desideri visualizzare il testo nel carattere Courier. Questi stili non possono essere modificati, ma le opzioni di modifica ti consentono di applicare una formattazione diversa, cambiando il carattere, le dimensioni, il colore, ecc.

Aggiunta di tabelle alla pagina

L'aggiunta di tabelle alla tua pagina è una procedura in tre fasi: innanzitutto viene aggiunta una tabella, quindi ne vengono configurate le proprietà, infine viene aggiunto il contenuto. Per aggiungere una tabella, clicca sull'icona che la raffigura nella barra degli strumenti, scegli Inserisci tabella e quindi seleziona il numero di righe e colonne da includere.

Aggiunta di un foglio di calcolo alla tua pagina  

Per aggiungere un foglio di calcolo, clicca sull'icona che lo raffigura, nella barra degli strumenti. Un foglio di calcolo a sei colonne viene aggiunto alla pagina. I fogli di calcolo possono essere usati per creare  database per registrare elenchi di attività, orari, impegni, ecc. In aggiunta alle funzionalità base, includono una serie di funzioni predefinite.

Aggiunta di allegati e immagini alla tua pagina

È possibile caricare file collegati dalla pagina nella scheda Documenti. Clicca sull'icona a forma di graffetta, , o sulle immagini,  e scegli Sfoglia per selezionare il file da caricare. Se carichi un file già incluso nei tuoi Documenti o nella Valigetta, verrà visualizzato un messaggio di notifica.

Collegamento a una pagina del Blocco Note o a una pagina web

Puoi collegarti ad altre pagine del Blocco Note in Documenti o a un URL web esterno e  assegnare al link un titolo descrittivo per far sì che l'indirizzo URL non venga visualizzato.
