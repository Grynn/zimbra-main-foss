xxxxApertura dei file nella Valigettaxxxx

I file contenuti nella Valigetta possono essere aperti da qualsiasi computer. xxxxÈ necessario disporre dell'applicazione software appropriata per aprirli, leggerli o aggiornarli.xxxx I formati di file comuni, come i file di testo o HTML e i file di immagini come . gif o . jpg possono essere aperti con diversi programmi.

Per aprire un file, clicca su di esso con il tasto destro del mouse e scegli Apri. xxxxSe il tipo è sconosciuto, viene visualizzata una finestra di dialogo in cui viene richiesto se salvare il file su disco o aprirlo con un'applicazione specifica.xxxx
