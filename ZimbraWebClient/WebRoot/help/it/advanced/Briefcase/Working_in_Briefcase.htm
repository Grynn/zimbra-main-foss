xxxxUso della Valigettaxxxx

La Valigetta consente di salvare file nell'account in modo da potervi accedere ogni volta che si entra nell'account da qualsiasi computer.

Puoi caricare documenti, fogli di calcolo, presentazioni, immagini e file pdf dal tuo computer o dalla tua rete di computer e salvare allegati inviati con i tuoi messaggi email nella cartella Valigetta. È possibile creare diverse cartelle Valigetta per organizzare i file e condividerle con altri utenti.

xxxxI file presenti nella Valigetta sono copie del file originale.xxxx Pertanto, le modifiche apportate ai file nella Valigetta non si riflettono sul file originale.

Caricamento dei file

Il caricamento di un file comporta il trasferimento di file dal tuo PC all'account email. xxxxIn questo modo il file è disponibile ogni volta che entri nel tuo account.xxxx La quantità di file che è possibile caricare nella Valigetta dipende dallo spazio disponibile nell'account.

xxxxPer caricare i filexxxx

  1. Apri la Valigetta e clicca su Carica file nella barra degli strumenti.

  2. Nella finestra di dialogo Carica il nuovo file nella Valigetta clicca su Sfoglia per individuare i file da caricare. Puoi selezionare più file da caricare contemporaneamente se si trovano nella stessa directory. Per aggiungere altri file, clicca su Aggiungi per individuarli.

xxxxNon puoi caricare due file con lo stesso nome nella stessa Valigetta.xxxx Puoi specificare nella finestra di dialogo Carica come trattare i file con lo stesso nome quando tenti di caricare il file. xxxxQuando modifichi questa opzione, la modifica viene salvata nella finestra di dialogo.xxxx

    • xxxxSe desideri che venga segnalato il caricamento di un file con lo stesso nome, seleziona xxxxxxxxChiedimixxxxxxxx.xxxx Se un file ha lo stesso nome di un file già caricato, viene visualizzata la finestra di dialogo Conflitto di caricamento. Puoi mantenere il file esistente o sostituirlo con il file caricato. xxxxPuoi anche visualizzare il nuovo file.xxxx xxxxQuesta è l'impostazione predefinita.xxxx

    • Se desideri sempre sostituire il file esistente con il file caricato che ha lo stesso nome, seleziona Sostituisci con il mio file. xxxxIl nuovo file viene caricato e sostituisce quello esistente.xxxx xxxxL'operazione viene segnalata con un messaggio di notifica.xxxx

    • xxxxSe non vuoi che i file presenti nella Valigetta vengano sostituiti, seleziona xxxxxxxxTieni il file esistentexxxxxxxx.xxxx xxxxIl file non viene caricato e appare un messaggio di notifica.xxxx

  1. xxxxDopo aver caricato tutti i file, clicca su xxxxxxxxOKxxxxxxxx.xxxx xxxxIl file viene visualizzato nella Valigetta.xxxx 

  2. Seleziona Visualizza per impostare la modalità di visualizzazione dei file nella Valigetta.

    • Vista Gestione elementi. I file sono visualizzati per tipo (PDF, Word, PowerPoint e così via)

    • xxxxVista dettagliataxxxxxxxx.xxxx I file sono visualizzati come elenco con l'indicazione del nome, del tipo, della dimensione, della data e del proprietario. Nella vista dettagliata, clicca su un'intestazione per ordinare i file in base a nome, dimensione o data.

    • xxxxVista a colonnexxxxxxxx.xxxx I nomi file sono visualizzati in colonne.

Nota: xxxxÈ possibile che nel tuo account sia stata impostata una dimensione massima dei file da caricare.xxxx Se un file è troppo grande, viene visualizzato un avviso.
Le dimensioni dei file caricati influiscono sullo spazio disponibile nel tuo account.

xxxxSono disponibili diverse opzioni per visualizzare i tuoi file nelle cartelle Valigetta.xxxx xxxxScegli quella che preferisci dal menu xxxxxxxxVisualizzaxxxxxxxx sulla barra degli strumenti Valigetta.xxxx

Ordinamento dei file

Quando la Valigetta è visualizzata con la Vista dettagliata, i file possono essere ordinati in base a nome, dimensione o data. L'ordine è mantenuto quando si passa a un'altra vista.

Condivisione della Valigetta

Le cartelle Valigetta possono essere condivise con altre persone sia all'interno che all'esterno della tua organizzazione. È possibile concedere autorizzazioni ad altre persone per creare e modificare i contenuti di una Valigetta o semplicemente per visualizzare le pagine.
