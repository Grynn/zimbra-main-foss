Gli allegati ai messaggi email possono essere file nel computer, file della Valigetta di @ClientNameShort@, elenchi di contatti o altri messaggi email.

xxxxNota –xxxxxxxx I destinatari del tuo messaggio email devono avere il software appropriato per aprire e leggere il file.xxxx xxxxI file creati in software di uso comune come Microsoft Office, potranno essere aperti anche da altri utenti sui sistemi desktop più diffusi.xxxx I formati di file comuni, come i file di testo o HTML e i file di immagini, ad esempio GIF o JPG, possono essere aperti con diversi programmi. xxxxDipende comunque sempre dal tipo di sistema usato dall'utente.xxxx

Per aggiungere allegati mediante il link Aggiungi allegato:

  1. xxxxScrivi il messaggio nel modo descritto in xxxxxxxxComposizione di un nuovo messaggio emailxxxxxxxx.xxxx

  2. Clicca su . Nella finestra di dialogo Allega file vengono visualizzate le schede Il mio computer, Valigetta, Allega email e Contatti. Seleziona la scheda relativa al tipo di allegato.

    • Il mio computer. Individua una cartella nel computer e seleziona il file. Seleziona il file e clicca su Apri, oppure trascina i file da una cartella sull'intestazione del messaggio.

    • Valigetta. Seleziona la cartella desiderata. I relativi file verranno visualizzati a destra. Seleziona i file da allegare.

    • Allega email. Se il messaggio email da allegare non si trova in In arrivo, seleziona la cartella. Puoi cercare un messaggio nella cartella selezionata. I messaggi email sono visualizzati a destra. Seleziona il messaggio da allegare.

    • Contatti. Seleziona l'elenco dei contatti. I nomi in elenco sono visualizzati a destra. È possibile cercare un nome nell'elenco dei contatti. Seleziona i nomi da allegare. Vengono allegati come file .vcf.

Nota: Utilizza CTRL-clic o MAIUSC-clic per aggiungere più allegati contemporaneamente.

  2. Dopo aver aggiunto tutti gli allegati, clicca su Allega. xxxxGli allegati vengono elencati al di sotto della riga dell'oggetto.xxxx

  3. xxxxQuando il messaggio è pronto per l'invio, clicca su xxxxxxxxInviaxxxxxxxx per inviarlo insieme agli allegati.xxxx

Come aggiungere allegati dal computer mediante trascinamento

  1. Seleziona uno o più file e tieni premuto il cursore sui file file da allegare.

  2. Trascina i file file nell'area di intestazione del messaggio e rilascia il cursore. I nomi dei file verranno visualizzati nell'intestazione.
