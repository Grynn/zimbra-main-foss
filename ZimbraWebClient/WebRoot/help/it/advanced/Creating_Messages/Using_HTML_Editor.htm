Uso dell'editor HTML

Per comporre i tuoi messaggi, puoi usare l'editor HTML.  I messaggi creati in linguaggio HTML sono più semplici da leggere perché possono includere la formattazione del testo, la numerazione, gli elenchi puntati, uno sfondo colorato, tabelle e link.

  • Alcuni client di posta non accettano messaggi formattati in HTML.

Per impostare l'editor HTML come editor predefinito e selezionare le impostazioni predefinite dei caratteri, incluso lo stile, le dimensioni e il colore dal client web, apri la scheda Preferenze, Generale.
