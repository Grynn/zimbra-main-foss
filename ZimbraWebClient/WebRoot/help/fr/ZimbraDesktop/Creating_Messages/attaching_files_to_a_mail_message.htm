Joindre des fichiers à votre mail

Lorsque vous composez un message, vous pouvez joindre des pièces jointes depuis votre ordinateur, les fichiers du Porte-documents, les contacts de votre carnet d'adresses et les mails.

Remarque : Les destinataires de votre mail doivent disposer du logiciel adéquat pour pouvoir ouvrir et lire le fichier. Pour les fichiers créés avec des logiciels d'utilisation courante comme Microsoft Office, les utilisateurs d'un système bureautique classique pourront les ouvrir. Des formats de fichiers courants comme les fichiers texte et HTML et les fichiers images comme les .GIF ou .JPG peuvent être ouverts à partir de différents programmes. Néanmoins, cela dépend du type de système de l'utilisateur.

Pour joindre les fichiers en utilisant le lien Ajouter pièce jointe :

  1. Rédigez le mail comme décrit dans la section Création d’un mail.

  2. Cliquez sur   . La boîte de dialogue Joindre un ou plusieurs fichiers affiche les onglets de Mon ordinateur, Porte-documents, Joindre mail et Contacts. Sélectionnez un onglet suivant le type de pièce à joindre.

    • Mon ordinateur. Parcourez n'importe quel dossier sur votre ordinateur, puis sélectionnez le fichier. Sélectionnez le fichier, puis cliquez sur Ouvrir.  Vous pouvez également faire glisser des fichiers depuis le dossier pour saisir l'en-tête du mail.

    • Porte-documents. Sélectionnez le dossier Porte-documents. Les fichiers contenus dans ce Porte-documents s'affichent à droite. Choisissez les fichiers que vous souhaitez joindre.

    • Joindre mail. Si ce mail n'est pas contenu dans votre Boîte de réception, sélectionnez le dossier. Vous pouvez rechercher un mail dans le dossier que vous avez sélectionné. Les mails s'affichent à droite. Sélectionnez le mail devant être joint.

    • Contacts. Sélectionnez le carnet d'adresses. Les noms du carnet d'adresses s'affichent sur la droite. Vous pouvez parcourir votre carnet d'adresses pour rechercher un nom. Choisissez les noms à joindre. Ils sont joints au message en tant que fichiers .vcf.

Remarque : Utilisez les combinaisons de touches Ctrl-clic ou Shift-clic pour ajouter plusieurs pièces jointes à la fois. Plusieurs pièces jointes sont ajoutées à une ligne dans la boîte de dialogue Joindre les fichiers.

  1. Pour afficher les images que vous avez téléchargées dans le corps du mail, cochez la case Afficher les images dans le corps des mails.  

  2. Une fois toutes les pièces jointes ajoutées dans cet onglet, cliquez sur Joindre.

  3. Cliquez sur Envoyer pour envoyer le message et les pièces jointes.

Pour ajouter des pièces jointes à partir de votre ordinateur par glisser-déposer

  1. Sélectionnez un ou plusieurs fichiers et maintenez le curseur sur le ou les fichiers à joindre.

  2. Faites glisser le ou les fichiers vers la zone de l'en-tête du mail et relâchez le curseur. Les noms de fichiers s'affichent au niveau de l'en-tête.

 

 

 

Suppression d'une pièce jointe

Pour supprimer une pièce jointe, décochez la case à côté de la pièce jointe avant d’envoyer le mail.
