Ajout d’adresses Cc : et Cci : .

Les abréviations Cc : et Cci : signifient respectivement copie carbone et copie carbone invisible.

  • Cc : vous permet d’envoyer une copie d’un mail à une personne intéressée par ce message, mais cette personne n'est pas le destinataire principal. Tous les destinataires mis en Cc peuvent voir la liste complète des adresses lorsqu’ils lisent le mail.

  • Cci : vous permet d’envoyer une copie d’un mail à une personne mais son adresse n’apparaît pas dans les copies du mail envoyées aux autres destinataires.

Pour saisir des adresses Cci : dans un nouveau mail :

  1. Dans la fenêtre de création, sélectionnez Afficher Cci.

  2. Cliquez sur À :, Cc : ou Cci : qui s’affiche à côté des champs d’adresse pour faire apparaître la boîte de dialogue de sélection d’adresses. Sinon, saisissez vos adresses directement dans les champs concernés.

Vous pouvez envoyer un mail sans insérer la moindre adresse dans le champ À : à condition qu'il existe au moins une adresse dans les champs Cc : ou Cci : .
