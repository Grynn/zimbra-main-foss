Création d'un mail

Pour écrire un mail :

  1. Cliquez sur dans la barre d'outils. La page de création de mail s'affiche.

  2. Dans le champ À : entrez l'adresse de l'utilisateur ou cliquez sur le lien À : pour rechercher une adresse depuis vos carnets d'adresses ou l'annuaire de la société (liste globale d'adresses). Si vous ajoutez une liste de distribution, vous pouvez cliquer sur Développer pour voir qui se trouvent sur la liste de distribution. Vous pouvez sélectionner les adresses une à une ou tous les membres de la liste de distribution.

Remarque :  Si votre prénom et nom de famille ne forment pas une adresse mail correcte (nom@domaine.com), une boîte de dialogue de confirmation s'affichera pour vous avertir que cette adresse ne paraît pas correcte. Vous pouvez quand même choisir d'envoyer le mail, même s'il est probable que sa livraison ne s'effectuera pas.

  1. Remplissez la ligne Objet et entrez votre message.

  2. Si vous souhaitez recevoir un accusé de lecture, cliquez sur Options, puis sélectionnez Demander un accusé de lecture.

  3. Saisissez le corps du mail dans la zone de texte située au-dessous de la ligne Objet.

Les mots mal orthographiés sont automatiquement soulignés. Cliquez avec le bouton droit sur le mot souligné pour sélectionner la bonne orthographe.

  1. Pour ajouter une pièce jointe, cliquez sur et recherchez le fichier ou glissez-déposez le fichier de votre ordinateur vers l'en-tête du mail.

  2. Ajouter votre signature. Si vous avez plusieurs configurations de compte, dans le menu déroulant Comptes, sélectionnez l'adresse devant être utilisée pour ce mail. Si vous disposez d'un  style de signatures défini pour le compte, cliquez sur Signature dans la barre d'outils Écrire pour ajouter la signature.

  3. Cliquez sur Envoyer pour envoyer le mail.

Si vous ne souhaitez pas envoyer le message tout de suite, cliquez sur la flèche à côté d'Envoyer, puis sélectionnez Envoyer plus tard. Dans la boîte de dialogue Spécifier l'heure d'envoi du message entrez le jour et l'heure d'envoi du message. Le message est enregistré dans le dossier Brouillons et sera automatiquement envoyé à la date sélectionnée.
