Configurer des alertes et des rappels dans Zimbra Desktop

Vous pouvez créer des alertes pour les notifications de nouveaux mails et des rappels de rendez-vous. Pour recevoir des alertes, votre boîte aux lettres doit être ouverte.

  • Alertes Mails. Vous pouvez définir vos préférences dans l'onglet >Préférences Mail. Voici vos options d'alertes :

    • Émettre un son. Un signal sonore ("bip") vous avertit lorsque vous recevez un mail dans votre Boîte de réception.  Si vous avez mis le volume de votre ordinateur en sourdine, vous n'entendrez aucun son. Le logiciel QuickTime ou Windows Media doit être installé pour pouvoir entendre le signal sonore.

    • Activer l'onglet Mail. Si vous n'êtes pas en train d'utiliser l'onglet Mail, ce dernier est contrasté lorsque vous recevez un mail dans votre Boîte de réception.

    • Afficher un message de notification.   Lorsqu'un nouveau mail arrive, un message s'affiche. Vous devez disposer de Yahoo! BrowserPlus doit être installé.

  • Alertes de Rappel de rendez-vous.  Vous pouvez définir vos préférences dans le dossier Préférences>Calendrier.  Avant de pouvoir configurer une alerte, définissez l'heure du rappel de votre réunion. Vous pouvez la définir jusqu'à 60 minutes avant le début de la réunion. Une boîte de dialogue de rappel s'affiche à l'heure indiquée avant la réunion. Si vous définissez les rappels sur Jamais, vous ne recevrez pas de rappel au sujet de la réunion, même si l'option de rappel est activée.

Le rappel ne s'affichera pas en haut de la page si vous n'êtes pas en train d'utiliser le Calendrier. Vous pouvez activer les options suivantes pour bénéficier du message de notification :

    • Émettre un son.Lorsqu'un message de rappel de rendez-vous s'affiche, vous êtes averti par un signal sonore (un "bip"). Si vous avez mis le volume de votre ordinateur en sourdine, vous n'entendrez aucun son.

    • Afficher brièvement le nom du navigateur. Lorsqu'un message de rappel de rendez-vous s'affiche, le navigateur clignote jusqu'à ce que vous fermiez le message de notification.

    En plus des alertes envoyées à votre ordinateur, vous pouvez configurer une notification par mail ou SMS depuis le rendez-vous.  Vous pourrez configurer l'adresse mail et le numéro de téléphone depuis la page Préférences>Notification.
