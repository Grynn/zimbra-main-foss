Créer un document

Vous pouvez créer un document dans Porte-documents, concevoir la présentation de contenu en utilisant les outils en format RTF. Vous pouvez procéder à une vérification orthographique et utiliser des mots-clés pour créer des liens vers d’autres pages.

  1. Cliquez sur le dossier de porte-documents dans lequel vous souhaitez travailler. Le panneau de Contenu affiche les fichiers déjà contenus dans le porte-documents.

  2. Dans la liste Nouveau sur la barre d'outils, sélectionnez Document. Une nouvelle page s'affiche. Dans l'en-tête, entrez le nom du document.

  3. Rédiger le contenu de la page. Vous pouvez utiliser l’éditeur de format RTF pour ajouter des styles, des couleurs et des tableaux à votre page.

  1. Cliquez sur Enregistrer pour sauvegarder le fichier. Pour fermer le document, cliquez sur X dans le coin supérieur droit.

Remarque : Si vous avez créé le document dans le mauvais porte-documents, vous pouvez déplacer le fichier dans le bon porte-documents en sélectionnant le document et en le faisant glisser vers le bon dossier.
