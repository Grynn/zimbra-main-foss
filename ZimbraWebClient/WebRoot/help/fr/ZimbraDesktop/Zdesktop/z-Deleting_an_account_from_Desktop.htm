Suppression d’un compte sur Desktop

  1. }}}Cliquez sur Configuration sur Zimbra Desktop.{{{ }}}Ce lien apparaît dans l’angle droit.{{{

}}}La page Mes comptes apparaît et affiche vos comptes.{{{

  1. }}}Cliquez sur Supprimer pour supprimer le compte concerné.{{{ }}}Une boîte de dialogue affiche Les paramètres du compte ainsi que les données téléchargées seront supprimés{{{ }}}(les données stockées sur le serveur ne seront pas affectées).{{{ Voulez-vous vraiment effectuer cette opération ?

}}}Toutes les données sont supprimées de l'ordinateur, y compris les mails, les listes de contacts, les pages de carnets, les fichiers enregistrés dans le Porte-documents.{{{ }}}La suppression élimine les fichiers de l’ordinateur, mais pas du serveur.{{{

}}}Remarque :{{{  }}}Les fichiers que vous avez sauvegardés dans le Dossier local sont seulement enregistrés dans Zimbra Desktop.{{{ }}}Si vous souhaitez sauvegarder ces fichiers avant de supprimer votre compte, vous pouvez les exporter vers le Dossier local.{{{ }}}Cette opération peut être effectuée à partir de l’onglet Préférences>Importer/Exporter.{{{
