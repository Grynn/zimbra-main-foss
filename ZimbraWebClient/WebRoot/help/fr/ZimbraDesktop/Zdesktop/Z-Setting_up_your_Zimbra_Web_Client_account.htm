Configurer un compte Zimbra Web Client (client Web)

Avant d’utiliser Zimbra Desktop, vous devez spécifier les paramètres du compte existant. La première fois que vous activez Zimbra Desktop, un message vous demande de créer un compte. (Vous pourrez créer d’autres comptes à tout moment.)

Avant de commencer, vérifiez que vous êtes correctement connecté à Internet.

  1. Lancez Zimbra Desktop. Dans l’angle supérieur droit, cliquez sur Configuration.  

  2. Cliquez sur AJOUTER UN NOUVEAU COMPTE. Dans la liste Type de compte, sélectionnez Zimbra. La page des options de configuration apparaît. Entrez les informations ci-dessous.

    • Nom du compte – Nom qui apparaîtra pour ce compte dans Zimbra Desktop. Chaque compte doit être affecté d’un nom unique. Comme vous êtes en train de créer un compte ZWC, vous pourriez l’appeler Zimbra.

    • Adresse mail – Entrez l’adresse mail de votre compte ZWC.

    • Mot de passe – Entrez le mot de passe que vous spécifiez pour vous connecter à ce compte ZWC.

    • Serveur de courrier entrant. Entrez l’adresse du serveur Zimbra (généralement sous la forme suivante : mail.exemple.com). Vous devez spécifier le serveur qui héberge votre compte.

    • Sécurité – Si vous devez toujours utiliser une connexion cryptée, cochez Utiliser le cryptage SSL lorsque vous accédez à ce serveur. (si nécessaire, contactez votre administrateur système).

    • Dans la section Paramètres de synchronisation, sélectionnez la fréquence de lecture/synchronisation des données qui devra être respectée par Zimbra Desktop pour ce compte. Le défaut est de consulter les messages et les synchroniser quand de nouveaux mails arrivent.

Si vous sélectionnez la valeur manuellement, vous devrez cliquer sur le bouton Envoyer/Recevoir de la barre d’outils Zimbra Desktoppour synchroniser votre compte avec le serveur Zimbra.

Si vous sélectionnez un intervalle trop court (par exemple une minute), les performances de votre ordinateur risquent d’en être affectées.

Vous pouvez changer cette valeur à tout moment : cliquez sur Config compte, sélectionnez le compte concerné et modifiez le paramètre selon vos besoins.

  1. Lorsque vous avez terminé, cliquez sur Valider et enregistrer.

Zimbra Desktop valide les informations de votre nouveau compte. Si cette validation de compte s’exécute sans problème, la boîte de dialogue Mes comptes apparaît. Si cette validation de compte ne peut pas être exécutée, la boîte de dialogue initiale est réaffichée pour vous permettre de revoir vos entrées et de corriger les erreurs.

  1. Pour ouvrir Desktop, cliquez sur Lancer Desktop. Pendant la synchronisation du compte, vous pouvez continuer à travailler dans Zimbra Desktop.
