Travailler hors ligne

Les mails que vous téléchargez de vos comptes restent toujours accessibles dans Zimbra Desktop même si vous n’êtes pas connecté à Internet (mode "hors ligne").Zimbra Desktop " "

Lorsque vous travaillez hors ligne, Zimbra Desktop ne peut pas accéder à vos comptes (pour télécharger les nouveaux mails), mais vous pouvez continuer à travailler dans Zimbra Desktop comme si vous étiez connecté à Internet : consulter des mails, écrire de nouveaux mails, gérer le contenu des dossiers, etc. (toutes les modifications effectuées hors ligne seront synchronisées avec les données de votre compte lors de votre prochain accès "en ligne" à celui-ci).

Lorsque vous travaillez hors ligne, le carré affiché à côté du nom de votre compte est rouge.

 

Remarque : il est possible que vous soyez hors ligne AVANT que l’icône (cercle) devienne rouge. En effet, l’état du réseau est mis à jour au moment où Zimbra Desktop essaie de se synchroniser avec le serveur. Si cette tentative échoue, l’icône (cercle) devient rouge.

 

Vous pouvez forcer tous les comptes pour passer hors ligne en cliquant sur la petite icône à côté de Comptes dans le Panneau d'aperçu.

Cliquez à nouveau sur l'icône pour revenir en ligne.  

Actions possibles en mode hors ligne

  • Lire des mails, les déplacer d’un dossier à un autre, marquer d’un drapeau ceux qui requièrent votre attention.

  • Écrire la réponse à un mail et l’envoyer, faire suivre (transférer) un mail, créer un nouveau mail. Ces différents types de mail (réponse/transfert/nouveau) sont enregistrés dans le dossier Courrier sortant : ils sont traités dès votre prochaine connexion à Internet.

  • Supprimer des mails Ils sont placés dans le dossier Corbeille. (ceux-ci sont transférés dans le dossier Corbeille de Zimbra, puis dans le dossier Corbeille de votre compte dès votre prochaine connexion à Internet). Si vous videz votre dossier Corbeille, les mails contenus dans celle-ci sont supprimés définitivement.

  • Exécuter les fonctions Rechercher et Recherche avancée pour retrouver des mails ou/et des pièces jointes présents dans votre compte Zimbra Desktop.

  • Définir et enregistrer des recherches.

  • Vous pouvez modifier vos Préférences Zimbra Desktop. Ces options affectent la manière dont vous travaillez dans Zimbra Desktop. Les modifications apportées aux options du client Web Zimbra Desktop sont automatiquement répercutées dans les "préférences" de votre compte ZWC. Les modifications apportées aux options des autres types de compte de messagerie affectent uniquement les paramètres Zimbra Desktop. (et non les paramètres de votre compte).
