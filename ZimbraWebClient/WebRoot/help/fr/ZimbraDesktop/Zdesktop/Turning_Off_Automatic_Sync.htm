Désactivation de la synchronisation automatique

Remarque :  La désactivation de la synchronisation n'est pas disponible pour les dossiers Zimbra.

Lorsque vous configurez vos comptes pour un téléchargement sur Zimbra Desktopvous définissez les paramètres pour synchroniser tous les dossiers de votre compte OU uniquement votre Boîte de réception. Si vos paramètres ont été définis pour synchroniser tous vos dossiers, vous pouvez ensuite, si vous le souhaitez, désactiver la synchronisation pour vos dossiers personnels.  

  1. Cliquez avec le bouton droit sur un dossier.

  2. Sur le menu qui s'affiche, sélectionnez Désactiver la synchronisation.

Lorsque vous désactivez la synchronisation puis déplacez les mails de votre Boîte de réception vers un dossier qui n’est pas synchronisé avec votre compte, les mails déplacés seront supprimés du serveur. Le déplacement de mails vers des dossiers non synchronisés n’a aucun impact sur les dossiers de votre compte.

Pour synchroniser à nouveau ces dossiers, activez la synchronisation. Lors de la prochaine synchronisation de Zimbra Desktop avec votre compte, le contenu du dossier est synchronisé.
