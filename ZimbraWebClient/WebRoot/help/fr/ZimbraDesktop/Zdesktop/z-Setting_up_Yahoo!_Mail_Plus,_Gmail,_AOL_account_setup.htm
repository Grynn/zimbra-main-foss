Configurer votre compte

Avant de commencer, vérifiez que vous êtes correctement connecté à Internet.

  1. }}}Ouvrez Zimbra Desktop et en haut à droite, cliquez sur Configuration.{{{  

  1. }}}Cliquez sur AJOUTER UN NOUVEAU COMPTE.{{{ }}}Dans la liste Type de compte, sélectionnez le type de compte.{{{ }}}La page des options de configuration apparaît.{{{ }}}Entrez les informations ci-dessous.{{{

}}}Nom du compte –{{{}}}Nom qui apparaîtra pour ce compte dans Zimbra Desktop.{{{ }}}Si vous configurez plusieurs comptes, efforcez-vous de choisir des noms qui faciliteront leur identification.{{{ }}}Par exemple, pour un compte Yahoo!, vous pourriez utilisez Yahoo.{{{

Votre nom complet – Entrez votre nom et votre prénom (facultatif).

Adresse mail – Entrez l’adresse mail que vous utiliserez avec ce compte.

}}}Mot de passe –{{{ Entrez le mot de passe que vous spécifiez pour vous connecter à votre compte.

  1. }}}Dans Paramètres de synchronisation, sélectionnez la fréquence à laquelle Zimbra Desktop doit vérifier les messages de votre compte.{{{ (valeur par défaut : 15 minutes).

}}}-Si vous sélectionnez la valeur manuellement, vous devrez cliquer sur le bouton Envoyer/Recevoir de la barre d’outils Zimbra Desktop pour synchroniser votre compte avec le serveur Zimbra.{{{

}}}-Si vous sélectionnez un intervalle trop court (par exemple une minute), les performances de votre ordinateur risquent d’en être affectées.{{{

}}}-Vous pouvez changer cette valeur à tout moment : cliquez sur Config compte, sélectionnez le compte concerné et modifiez le paramètre selon vos besoins.{{{

}}}Dans votre boîte aux lettres, sélectionnez le ou les dossiers qui doivent être synchronisés.{{{

}}}* Synchroniser tous les calendriers{{{

}}}* Synchroniser tous les contacts et les groupes{{{

  1. }}}Lorsque vous avez terminé, cliquez sur Valider & Enregistrer.{{{

}}}Zimbra Desktop valide les informations de votre compte.{{{ }}}Si cette validation de compte s’exécute sans problème, la boîte de dialogue Mes comptes apparaît.{{{ }}}Si cette validation de compte ne peut pas être exécutée, la boîte de dialogue initiale est réaffichée pour vous permettre de revoir vos entrées et de corriger les erreurs.{{{

}}}La synchronisation du compte avec Zimbra Desktop démarre automatiquement.{{{

  1. }}}Cliquez sur Lancer Desktop pour activer votre Desktop.{{{ }}}Pendant la synchronisation du compte, vous pouvez continuer à travailler dans Zimbra Desktop.{{{
