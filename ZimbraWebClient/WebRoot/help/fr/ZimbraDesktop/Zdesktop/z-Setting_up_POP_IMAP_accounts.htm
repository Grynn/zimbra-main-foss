Configurer les comptes POP/IMAP, y compris Microsoft Exchange IMAP

}}}Pour configurer un compte, vous devez disposer de certains paramètres, dont les détails du serveur des mails entrants et du serveur des mails sortants{{{ }}}(si nécessaire, contactez votre administrateur système ou votre FAI).{{{

}}}Vous pouvez demander que la synchronisation porte sur tous les mails présents dans la boîte aux lettres (tous les dossiers), ou qu’elle soit limitée aux mails présents dans votre Boîte de réception.{{{

Configurer votre compte

Avant de commencer, vérifiez que vous êtes correctement connecté à Internet.

  1. }}}Ouvrez Zimbra Desktop et en haut à droite, cliquez sur Configuration.{{{  

  2. }}}Cliquez sur AJOUTER UN NOUVEAU COMPTE.{{{ }}}Dans la liste Type de compte, sélectionnez le type de compte.{{{ }}}La page des options de configuration apparaît.{{{ }}}Entrez les informations ci-dessous.{{{

}}}Nom du compte –{{{}}}Nom qui apparaîtra pour ce compte dans Zimbra Desktop.{{{ }}}Si vous configurez plusieurs comptes, efforcez-vous de choisir des noms qui faciliteront leur identification.{{{ }}}Par exemple, pour un compte Yahoo!, vous pourriez utilisez Yahoo.{{{

Votre nom complet – Entrez votre nom et votre prénom (facultatif).

Adresse mail – Entrez l’adresse mail que vous utiliserez avec ce compte.

  1. }}}Dans la section Réception des mails, entrez les informations ci-dessous.{{{

}}}Nom d'utilisateur.{{{ Entrez le nom d’utilisateur que vous spécifiez pour réceptionner les nouveaux mails }}}(généralement sous la forme suivante : nom d’utilisateur@exemple.com).{{{

}}}Mot de passe –{{{ Entrez le mot de passe que vous spécifiez pour vous connecter à ce compte.

}}}Serveur de courrier entrant.{{{ Entrez l’adresse du serveur de courrier entrant (SMTP). }}}(généralement sous la forme suivante : mail.exemple.com).{{{

Sécurité – }}}Précisez si les accès à ce serveur doivent être protégés en cochant (ou non) la case Utiliser le cryptage SSL pour accéder à ce serveur.{{{ }}}Si votre serveur de mails entrants utilise un port différent, cliquez sur Modifier pour changer le numéro de port{{{ }}}(si nécessaire, contactez votre administrateur système).{{{

  1. }}}Dans la section Envoi des mails, entrez les informations suivantes :{{{

}}}Serveur SMTP.{{{ Entrez l’adresse du serveur de courrier sortant }}}(généralement sous la forme suivante : smtp.exemple.com).{{{

Sécurité – }}}Précisez si les accès à ce serveur doivent être protégés en cochant (ou non) la case Utiliser le cryptage SSL pour accéder à ce serveur.{{{ }}}Si votre serveur de mails entrants utilise un port différent, cliquez sur Modifier pour changer le numéro de port{{{ }}}(si nécessaire, contactez votre administrateur système).{{{

}}}Si le nom d’utilisateur et le mot de passe sont obligatoires pour les envois de mail, cochez la case Authentification.{{{ }}}Si cette option est sélectionnée, vous devez spécifier des valeurs correctes de Nom d’utilisateur et de Mot de passe.{{{

}}}Répondre à.{{{ }}}Si vous souhaitez utiliser que les réponses de vos correspondants utilisent des valeurs de nom et d’adresse mail différentes de celles de vos mails sortants, entrez ce Nom et cette Adresse mail.{{{

  1. }}}Dans la section Paramètres de synchronisation, configurez le calendrier pour qu'il se synchronise avec votre serveur.{{{ Par défaut, la consultation des mails s’effectue toutes les 15 minutes. }}}Si vous sélectionnez la valeur manuellement, vous devrez cliquer sur le bouton Envoyer/Recevoir de la barre d’outils Zimbra Desktop pour synchroniser votre compte avec le serveur Zimbra.{{{ }}}Si vous sélectionnez un intervalle trop court (par exemple une minute), les performances de votre ordinateur risquent d’en être affectées.{{{

}}}Vous pouvez changer cette valeur à tout moment : cliquez sur Configuration, sélectionnez le compte concerné et modifiez le paramètre selon vos besoins.{{{

  1. }}}Lorsque vous avez terminé, cliquez sur Valider et enregistrer.{{{

}}}Zimbra Desktop valide les informations de votre compte.{{{ }}}Si cette validation de compte s’exécute sans problème, la boîte de dialogue Mes comptes apparaît.{{{ }}}Si cette validation de compte ne peut pas être exécutée, la boîte de dialogue initiale est réaffichée pour vous permettre de revoir vos entrées et de corriger les erreurs.{{{

  1. }}}Cliquez sur Lancer Desktop pour ouvrir Zimbra Desktop. Vous pouvez travailler dans Zimbra Desktop pendant que votre compte se synchronise.{{{
