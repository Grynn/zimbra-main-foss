Organisation de votre affichage de bureau

Quand vous configurez le bureau, vos comptes sont affichés séparément dans le Panneau d'aperçu. Si vous préférez avoir tous vos messages dans un compte, activez la préférence générale Afficher la zone "« Toutes les boîtes aux lettres » "avec mes autres comptes. Tous les messages à partir de chacun de vos comptes pour les dossiers Boîte de réception, Envoyés et Brouillons sont combinés sous cette section. Le nombre d'objets non lus pour tous les comptes est affiché. Vos comptes individuels sont indiqués en dessous de Toutes les boîtes aux lettres. Vous pouvez cliquer sur un compte pour voir uniquement les objets dans ce compte.

 

 

Quand vous affichez des messages dans l'affichage Toutes les boîtes aux lettres, une icône dans la liste de messages identifie de quel compte les messages viennent.

 

Compte par défaut

Quand vous ouvrez Zimbra Desktop, vos comptes sont affichés dans le Panneau d'aperçu dans l'ordre dans lequel vous les configurez. Le premier compte est le compte de bureau par défaut. C'est le compte qui est ouvert quand vous lancez Zimbra Desktop. Vous pouvez modifier quel compte est le compte par défaut.

  1. Cliquez sur Configuration sur Zimbra Desktop. La page Mes comptes affiche une liste de vos comptes.

  2. Pour modifier le compte par défaut, cliquez sur à côté du nom du compte. Ce compte est déplacé vers le haut.  

 

En plus que le compte est ouvert quand Zimbra Desktop est affiché, le compte par défaut utilisé si Zimbra Desktop est configuré comme l'application mail par défaut. Vous pouvez le configurer à partir de la page Préférences>Général dans la section Valeurs par défaut.
