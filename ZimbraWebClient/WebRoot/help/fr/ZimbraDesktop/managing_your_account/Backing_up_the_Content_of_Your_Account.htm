Déplacer des données d'un compte

Vous pouvez exporter toutes les données de votre compte (mails, pièces jointes, contacts, calendriers, tâches, dossiers du Porte-documents), ou limiter l'export à certains objets.  En enregistrant vos données dans votre ordinateur ou sur un support extérieur, vous disposez d'une copie de sauvegarde (backup). Les données exportées sont enregistrées sous la forme d'un fichier-archive (tar+gzip, tgz) que vous pouvez réimporter dans votre compte à tout moment.

Vous pouvez exporter tout ou partie des données d'un compte (mails, pièces jointes, contacts, calendriers, tâches, carnets ou/et dossiers du Porte-documents) à l'aide de la fonction Import/Export de Zimbra Desktop, puis importer ces données dans un autre compte Zimbra Desktop.  Les données exportées sont enregistrées sous la forme d'un fichier-archive via tar+gzip (tgz) que vous pouvez importer dans un autre compte Zimbra Desktop à tout moment.  Vous pouvez également utiliser la fonction Export/Import de Zimbra Desktop pour enregistrer tout ou partie des données de votre compte dans votre ordinateur ou sur un support extérieur, et disposer ainsi d'une copie de sauvegarde (backup). Les données sont enregistrées sous la forme d'un fichier-archive via tar+gzip (tgz) que vous pouvez réimporter à tout moment pour restaurer votre compte.

 Remarque : Les dossiers de contact exportés sont enregistrés sous forme de fichiers .csv. Les dossiers du calendrier exportés sont enregistrés sous forme de fichiers .ics.

Les données sont copiées sans être supprimées de votre compte.

Le fichier est exporté à partir de Préférences> Importer/Exporter.

Pour exporter des données

  1. Dans l'onglet Préférences du compte que vous souhaitez exporter, cliquez sur le dossier Importer/Exporter.

  2. Vous pouvez exporter toutes les données du compte ou sélectionner le type de dossier à exporter.

    • Pour exporter toutes les données du compte, sélectionnez Compte. Remarque : Les dossiers Corbeille et Spam ne peuvent pas être exportés.

    • Pour exporter une liste de contacts ou un calendrier, sélectionnez l'option correspondante (dans ce cas, vous devez également sélectionner le format du fichier d'export). Voir Exportation de carnets d'adresses et Exportation de votre calendrier iCal.

    • Pour sélectionner les différents types d'objet à exporter, activez l'option Paramètres évolués. Cochez la case du ou des types d'objet à exporter (tous les objets de ce type, dans le dossier actuel et ses éventuels sous-dossiers, seront inclus). Pour exporter des données spécifiques, vous pouvez utiliser le champ de filtrage de recherche. Voir également la description du langage de requête et les exemples de recherche.  

  1. Cliquez sur Exporter et sélectionnez le dossier de destination du fichier.

  2. Cliquez sur OK.

Remarque : Le fichier exporté peut être consulté à l'aide d'une application d'archivage telle que WinRAR.

Importer les données de compte enregistrées

Pour restaurer un compte, il vous suffit d'utiliser le fichier que vous avez exporté précédemment.  

  1. Dans l'onglet Préférences, cliquez sur le dossier Import/Export.

  2. Dans la section Importer, recherchez le fichier que vous souhaitez importer.  Si vous importez un calendrier ou des contacts, vous devez spécifier le format des données importées Voir Importer des contacts et Importer votre calendrier iCal. Si vous sélectionnez la valeur Compte, le seul fichier qui peut être importé dans votre compte est un fichier tgz exporté au préalable.  

Remarque : Le type d'objet Compte correspond au compte que vous utilisez actuellement dans (Zimbra, Yahoo!, Gmail, etc.).

  1. Pour retrouver le fichier à importer, cliquez sur Parcourir.

  2. Sélectionnez la destination du fichier.

  3. Si vous importez les données d'un compte, cochez la case appropriée dans les champs Éliminer les doublons, soit :

    • Ignorer. Les fichiers en double (doublons) ne sont pas analysés. Tous les fichiers sont réimportés, et la version existante des objets est conservée.

    • Remplacer. Les objets qui ont changé depuis l'exportation des données du compte reviennent à la version du fichier exporté. Les anciens objets sont réimportés. Les nouveaux objets qui ont été ajoutés dans votre compte depuis la création du fichier exporté sont conservés.

    • Réinitialiser. Tous les objets du type sélectionné qui sont stockés sous le dossier sélectionné sont supprimés avant l'importation des nouveaux objets.

  1. Si vous préférez importer des types de données spécifiques à partir du fichier tgz, cochez la case Paramètres évolués : dans la vue agrandie, sélectionnez le ou les types d'objet requis.

Dans la section Types de données, sélectionnez les types d'objet du fichier tgz enregistré que vous souhaitez importer. lorsque vous cliquerez sur "Importer", seuls ces types d'objet seront importés.

  1. Cliquez sur Importer.
