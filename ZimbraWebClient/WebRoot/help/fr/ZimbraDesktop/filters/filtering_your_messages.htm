Filtrer vos mails

Le filtre applique un ensemble de règles de correspondance au courrier entrant, puis exécute une action spécifiée.

Vous pouvez filtrer vos mails entrants pour les trier dans des dossiers, les marquer automatiquement, les faire suivre, ou les supprimer. Exemples de règle :

  • Tous les mails envoyés par votre directeur sont rangés dans le dossier "Directives commerciales".

  • Tous les mails contenus dans la liste de diffusion "événements de la société" sont marqués avec le tag "Événements".

Pour créer ou modifier une règle de filtre :

  1. Activez l’onglet Préférences .

  2. Activez le dossier Filtres, puis cliquez sur Nouveau filtre.

  3. La boîte de dialogue Ajouter un filtre s’affiche. Elle vous permet de définir les valeurs de règle et de destination qui devront être appliquées par le nouveau filtre.

  4. Dans le champ Nom du filtre, tapez le nom de votre choix pour la règle de filtre que vous êtes sur le point de définir (ce nom doit être unique).

  5. Dans la zone Si l'une des conditions/toutes les conditions suivantes sont satisfaites, choisissez une préférence de groupement.  

    • L'une des conditions : dès qu'une des conditions spécifiées dans le filtre est satisfaite, l'action définie est déclenchée.

    • Toutes les conditions : l'action définie est déclenchée seulement si TOUTES les conditions spécifiées dans le filtre sont satisfaites.  

Vous pouvez répéter les étapes suivantes afin de configurer plusieurs conditions et actions dans un seul filtre. Pour configurer plusieurs conditions, cliquez sur le lien +.

    • Choisissez dans la première liste déroulante une condition à utiliser.  

    • Choisissez un procédé de comparaison . Les valeurs disponibles dépendent de la sélection effectuée. Par exemple, "est" ou "n’est pas" peuvent s’afficher.

    • Saisissez un mot ou une phrase à comparer dans la zone du texte.

Cliquez sur pour ajouter des conditions. Après avoir défini la ou les conditions requises, vous allez sélectionner la ou les actions à associer à celles-ci.

  1. Dans le champ Exécuter les actions suivantes, choisissez une action dans le menu déroulant.

  • Spécifiez un dossier ou le nom d'un tag, si nécessaire. Ce que vous spécifiez dépend de l’action que vous choisissez.

Cliquez sur pour ajouter des actions. Après avoir défini la ou les actions requises, cliquez sur OK.

  1. Dans la partie supérieure de la boîte de dialogue Ajouter un filtre, notez la case. cochez cette case pour activer la règle de filtre ; décochez-la pour désactiver cette règle.

  2. Cochez Ne pas appliquer d'autres filtres, s'il s'agit du seul filtre à exécuter sur les mails qui remplissent les conditions du filtre. Remarque : dans chaque règle de filtre, Ne pas appliquer d'autres filtres doit toujours être la dernière action. (elle empêche l’application d'autres règles de filtre aux mails entrants qui satisfont la règle actuelle).

     Remarque : Si cette action n'est pas spécifiée, les règles des filtres ultérieurs continuent à être appliquées aux mails entrants – même si la ou les conditions du filtre actuel sont satisfaites –, ce qui risque de créer des conflits entre des actions contradictoires.

Les modifications que vous apportez aux règles des filtres sont enregistrées immédiatement. Le filtre sera appliqué de manière automatique dès la réception de nouveaux mails.  

Vous pouvez également appliquer des règles de filtre aux mails déjà stockés dans votre compte. Dans Préférences de filtres, cliquez sur Appliquer le filtre et sélectionnez les dossiers principaux auxquels le filtre sera appliqué.

Remarque : Vous pouvez également créer un nouveau filtre à partir d’un mail. Cliquez avec le bouton droit sur le mail, puis choisissez Nouveau filtre. La boîte de dialogue Modifier filtre s’affiche avec les champs De, À, et Objet configurés avec les données du mail.  Si les conditions du filtre sont correctes, donnez un nom de votre choix à celui-ci et cliquez sur OK.
