Impression d'une conversation

Pour imprimer l'ensemble des messages d'une conversation :

  1. Dans l'affichage Conversation, cliquez avec le bouton droit sur la conversation à imprimer.

  2. Dans la liste, cliquez sur Imprimer.

Une fenêtre distincte apparaît affichant tous les mails de la conversation dans une seule fenêtre de défilement.

  1. Dans la boîte de dialogue Impression, sélectionnez une imprimante, puis cliquez sur OKpour envoyer le message vers l'imprimante sélectionnée.

  2. Fermez la fenêtre du message.

 

Remarque : La taille des caractères utilisée pour imprimer les messages peut être configurée dans Préférences de l'onglet >Général. La police par défaut est 12 points.
