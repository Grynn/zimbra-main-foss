Joindre des fichiers à votre mail

Les mails peuvent inclure des pièces jointes. Vous pouvez joindre des documents, des feuilles de calcul, des photos, des diaporamas et d’autres types de fichiers.

Pour joindre un fichier à un mail :

  1. Rédigez le mail comme décrit dans la section Création d’un mail.

  2. Cliquez sur  . Le champ Joindre s’affiche sous la ligne de l’objet.

  3. Cliquez sur Parcourir... pour localiser le fichier.

  4. Sélectionnez le fichier puis cliquez sur Ouvrir.

  5. Pour joindre un autre fichier, cliquez à nouveau sur . Un nouveau champ Joindre s'affiche. .

  6. Lorsque le mail est prêt à être envoyé, cliquez sur Envoyer pour envoyer le mail ainsi que les pièces jointes.

 

  • Remarque : Les destinataires de votre mail doivent disposer du logiciel adéquat pour ouvrir et lire le fichier. Pour les fichiers créés avec des logiciels d'utilisation courante comme Microsoft Office, les utilisateurs d'un système bureautique classique pourront les ouvrir. Des formats de fichiers courants tels que les fichiers texte, HTML, et image comme .GIF ou .JPG peuvent être ouverts à partir de différents programmes. Néanmoins, cela dépend du type de système de l'utilisateur.
