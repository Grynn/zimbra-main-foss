Transfert de mail

Vous pouvez utiliser la fonction Faire suivre pour faire suivre un mail à un autre utilisateur. Le transfert de mail inclut toutes les pièces jointes, tandis que l’utilisation de la fonction Répondre conserve la partie texte mais n’inclut pas les pièces jointes.

Pour faire suivre un mail :

    1. Ouvrez le mail à faire suivre.

    2. Dans la barre d’outils, cliquez sur .  Une fenêtre de création de mail s’affiche. Le texte du mail transféré s’affiche dans le corps du message à moins que vous n’ayez choisi de faire suivre vos mails en pièces jointes dans l’onglet Préférences de messagerie.    

    3. Saisissez les adresses dans les champs À :, CC :, ou Cci : .

    4. Facultatif – Vous pouvez ajouter quelques lignes de texte en haut du message afin que le destinataire sache pourquoi vous avez décidé de le faire suivre.

    5. Cliquez sur Envoyer.
