Types de fichiers de pièces jointes pris en charge

Vous pouvez joindre n'importe quel fichier de votre système de fichiers reconnu en utilisant le bouton de navigation.

Vous pouvez ouvrir directement une pièce jointe depuis le client Web de Zimbra, à condition que votre poste de travail dispose de l’application adéquate et que l’extension ne soit pas bloquée par votre administrateur système.  Par exemple, pour ouvrir un document créé avec Microsoft Word, il vous faudra lancer l’application Microsoft Word sur votre ordinateur.

Sur la plupart des ordinateurs, le type de fichier est indiqué par son extension, généralement représentée par un point suivi d’un code de trois lettres. Par exemple, des fichiers tels que expenses.xls ou New-Items.doc. Si le type du fichier est supporté par le logiciel de votre ordinateur, vous pouvez généralement cliquer deux fois sur le fichier : votre ordinateur lancera automatiquement l’application adaptée à la lecture de ce fichier.

De nombreux types de fichiers peuvent être ouverts directement à partir de votre navigateur avec divers plug-ins. En général, si vous cliquez sur le lien d’un fichier sur une page Web, votre navigateur détecte automatiquement si vous disposez du plug-in adéquat, et vous invite à le télécharger si vous ne l’avez pas. Remarque : il est possible que les politiques d’administration de votre système n’autorisent pas les utilisateurs à télécharger et installer des logiciels.

  • Remarques :
    Pour les ordinateurs équipés de Windows, l’ordinateur considère que le type de fichier associé au nom du fichier correspond à son format réel. Si le fichier est correctement formaté de manière interne mais ne contient pas l'extension exacte du type de fichier dans le nom du fichier, il est possible que votre ordinateur ne puisse pas la reconnaître et ne puisse pas ouvrir l’application adéquate.

    Un nombre grandissant de virus informatiques ont été conçus pour se propager via les pièces jointes. Par conséquent, dans certains environnements, les administrateurs systèmes ont décidé de bloquer les mails entrants contenant certains types de pièces jointes, notamment les fichiers ayant pour extension .EXE ou dont le nom de fichier inclut .ZIP . Si vous envoyez un mail à un utilisateur dont le système de messagerie a été configuré pour bloquer certains types de pièces jointes, il est possible que vous receviez une notification indiquant que le mail a été bloqué.

Pour votre information, voici une liste des extensions de fichiers les plus couramment utilisées.

  • .TXT - indique un fichier texte, c’est-à-dire un fichier contenant uniquement des informations textuelles. Les fichiers textes peuvent être ouverts et lus avec de nombreux éditeurs de texte, notamment Bloc-notes, WordPad, Word, des éditeurs de ligne de commande, des navigateurs, et même des programmes de feuilles de calcul.

  • .HTM - Indique un fichier HTML, un type de fichier texte spécifique généralement ouvert avec un navigateur Web.

  • .ZIP – Indique un fichier qui a été compacté à l’aide d’une compression ZIP. Vous devrez disposer du programme WinZip ou d’un autre programme de compression pour pouvoir lire et extraire le fichier ZIP.

  • .EXE - Indique un fichier exécutable ou un fichier de programme. En double cliquant sur ce genre fichier, cela permet généralement de lancer l’application.

  • .DOC – Indique un fichier de document Microsoft Word.

  • .JPG – Indique une image graphique au format JPEG . Vous pouvez ouvrir des fichiers JPEG dans un navigateur ou en utilisant divers programmes graphiques tels que Paint ou Adobe Photoshop.

  • .GIF – Indique une image graphique au format GIF. Vous pouvez ouvrir des fichiers GIF dans un navigateur ou à l’aide d'un programme graphique.

  • .MPG – Indique un clip vidéo ou une animation au format MPEG . Ce type de fichier peut souvent être lu avec un plug-in de navigateur ou un lecteur comme par exemple Windows media Player.

  • .WMV - Indique un clip vidéo au format WMV, qui est un format exclusif à Windows.

  • .PDF - Indique un document au format d'Acrobat Reader. Vous pouvez généralement l’ouvrir dans votre navigateur à l’aide d’un plug-in.
