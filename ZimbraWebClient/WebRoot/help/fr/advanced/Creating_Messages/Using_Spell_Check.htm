Utilisation du vérificateur d’orthographe

Vous pouvez vérifier l'orthographe de votre mail avant de l'envoyer. Cliquez sur Vérification orthographique dans la barre d'outils de création

Si vous souhaitez que tous les messages soient vérifiés automatiquement avant d'être envoyés, activez la fonction Vérification orthographique obligatoire avant tout envoi de mail dans l'onglet Préférences de >création.

 Les mots inconnus sont contrastés par le vérificateur orthographique.

  1. Cliquez sur Vérification orthographique. Les mots qui ne sont pas correctement orthographiés sont contrastés.

  2. Cliquez sur un mot contrasté. Un menu contextuel affiche les suggestions de correction.

  3. Choisissez le mot correct. Le mot est contrasté d’une autre couleur.

  4. Pour accepter les modifications et fermer le vérificateur orthographique, cliquez sur Reprendre la modification.

 

Remarque : Avant de fermer le vérificateur orthographique, vous pouvez modifier un mot corrigé et récupérer son orthographe d'origine.  Cliquez sur le mot contrasté, puis sélectionnez l’orthographe initiale en haut du menu.

Ajout de mots à votre dictionnaire orthographique

Vous pouvez ajouter des mots à votre dictionnaire orthographique

  1. Cliquez sur Vérification orthographique. Les mots qui ne sont pas correctement orthographiés sont contrastés.

  2. Cliquez sur un mot contrasté. Un menu contextuel affiche les suggestions de correction.

  3. Si un mot correctement orthographié est contrasté, cliquez sur le mot, puis sur Ajouter. Ce mot est ajouté dans votre dictionnaire.

Vous ne pouvez pas supprimer des mots de votre dictionnaire une fois qu'ils ont été ajoutés.
