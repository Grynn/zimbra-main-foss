Utilisation du vérificateur orthographique

  1. Pour vérifier l’orthographe de votre mail, cliquez sur Vérifier l’orthographe, dans la barre d’outils.

Les mots inconnus sont contrastés par le vérificateur orthographique.

  1. Cliquez sur un mot contrasté. Un menu contextuel affiche les suggestions de correction.

  2. Choisissez le mot correct. Le mot est contrasté d’une autre couleur.

  3. Pour accepter les modifications et fermer le vérificateur orthographique, cliquez sur Reprendre la modification.

 

  • Avant de fermer le vérificateur orthographique, vous pouvez modifier un mot corrigé et récupérer son orthographe d'origine.  Cliquez sur le mot contrasté puis sélectionnez l’orthographe initiale en haut du menu.
