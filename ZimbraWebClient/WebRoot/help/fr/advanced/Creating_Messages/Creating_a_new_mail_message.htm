Création d’un mail

La première étape pour rédiger un mail est de cliquer dans la barre d’outils pour ouvrir une nouvelle page. Vous pouvez également cliquer avec le bouton droit dans la section De d’un mail et sélectionner Nouveau mail. Selon vos Préférences de messagerie, la page de création de mail s’ouvre dans le panneau Zimbra, ou dans une fenêtre séparée de création de mail.

Dans la fenêtre de création de mail Zimbra, saisissez l’adresse de la personne ou des personnes à qui vous voulez envoyer le mail dans le champ À : . Vous pouvez rechercher l’adresse mail d’une personne en cliquant sur À pour effectuer la recherche dans vos contacts ou dans la liste d'adresses globale. Pour ajouter des adresses Cci, cliquez sur Options dans la barre d’outils du nouveau mail.

  • Si vous avez configuré plus d’une identité mail , vous devrez sélectionner l’identité à utiliser avant de commencer à rédiger votre message.

Vous pouvez rédiger votre message au format HTML ou au format texte simple. Le format par défaut est configuré dans vos Préférences, dans l'onglet  Général. Pour modifier rapidement le format de ce message, cliquez surOptions dans la barre d’outils de nouveau mail puis choisissez le format HTML ou texte simple.

  • Le Format HTML vous permet de formater votre message avec différents styles de police, de créer des tableaux, et d’ajouter des couleurs.

  • Le Format texte simple génère un texte sans style ni formatage. Ce type de message est lisible sur n'importe quel ordinateur.

Pour écrire un mail :

  1. Cliquez sur dans la barre d’outils. La page de création de mail s’affiche.

  2. Si vous n’utilisez pas l’identité par défaut, choisissez l’identité à utiliser (appelée aussi « avatar ») dans la liste déroulante De.

  3. Remplissez le champ d’adresse, la ligne d’objet, et la zone de texte si nécessaire.

Si votre prénom et nom de famille ne forment pas une adresse mail correcte (nom@domaine.fr), une boîte de dialogue de confirmation s’affichera pour vous avertir que cette adresse ne paraît pas correcte. Vous pouvez quand même choisir d’envoyer le mail, même s'il est probable que sa livraison ne s'effectuera pas.

Saisissez le corps du mail dans la zone de texte située au-dessous de la ligne Objet.

Pour vérifier l’orthographe du message, cliquez sur Vérifier l'orthographe.

Pour ajouter une pièce jointe, cliquez sur .

    • Si vous avez défini une signature dans vos Préférences, mais qu’elle n’est pas activée, cliquez sur Ajouter Signature dans la barre d’outils de création de mail afin de l’ajouter.

  1. Cliquez sur Envoyer pour envoyer le mail.

Si vous ne souhaitez pas envoyer immédiatement le mail, cliquez sur Enregistrer le brouillon. Le mail est enregistré dans le dossier Brouillons.
