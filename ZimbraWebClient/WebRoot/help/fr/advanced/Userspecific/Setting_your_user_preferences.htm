}}}Configuration de vos préférences d'utilisateur{{{

Vos préférences d'utilisateur par défaut sont configurées lors de la création de votre compte. }}}Ces préférences définissent comme vos fonctions @ClientNameLong@ fonctionnent.{{{

}}}Vous pouvez modifier ces paramètres dans l’onglet Préférences.{{{ }}}Les pages de préférence s'affichent dans le Panneau d'aperçu.{{{ Si l’option décrite ici n’est pas visible dans votre affichage, cela signifie que cette préférence n’est pas disponible pour votre compte.

}}}Vous pouvez définir les préférences dans les dossiers suivants.{{{ }}}Cliquez sur le nom du dossier pour avoir une description du contenu.{{{

  • Général

    }}}Dans la page Général, vous pouvez{{{

    • }}}Configurer la version par défaut du ZWC à utiliser, avancé ou standard, sélectionner le thème, la langue et le fuseau horaire par défaut de votre boîte aux lettres.{{{

    • }}}Définir les Paramètres de recherche pour inclure les dossiers Spam et Corbeille dans vos recherches.{{{

    • }}}Affichage des cases à cocher dans le Panneau de contenu{{{

    }}}Voir Configurer vos préférences générales{{{

  • Mail

    }}}Dans la page Mail, vous pouvez{{{

    • }}}Configurer la fréquence à laquelle tester la présence de nouveau mail.{{{ }}}Par défaut, la consultation des mails s’effectue toutes les 5 minutes.{{{

    • }}}Sélectionner le comportement du bouton Actualiser pour exécuter votre recherche par défaut ou pour mettre à jour votre vue actuelle.{{{

    • }}}Sélectionner comment les messages sont affichés, comme HTML ou texte simple.{{{

    • }}}Configurer comment votre boîte aux lettres gère les messages qui sont reçus, y compris les options d'aperçu des messages, le panneau de lecture et les options de déplacement.{{{

    • }}}Configurer les alertes des messages.{{{

    • }}}Créer une réponse automatique par mail qui peut être utilisée quand vous êtes absent.{{{

    • }}}Activer la réponse Accusé de lecture.{{{

    • }}}Si votre configuration vous permet d’utiliser un client POP3, vous pouvez configurer le comportement d’accès POP lorsque des mails sont téléchargés sur @ClientNameShort@.{{{

    • }}}Définir les Options relatives au spam pour bloquer les messages afin de bloquer et autoriser les messages provenant des adresses spécifiées.{{{

    }}}Voir Configurer vos préférences mails{{{

  • Création de mail

    }}}Dans la page Écrire, vous pouvez{{{

    • }}}Définissez l’éditeur que vous voulez utiliser par défaut, HTML ou texte simple.{{{

    • Choisissez d’inclure le texte original dans le corps de votre message de réponse.

    • }}}Activez la vérification orthographique obligatoire.{{{

    • }}}Indiquez comment marquer les messages comme transférés ou répondus.{{{

    • }}}Par défaut, l’option Garder une copie des mails dans le dossier Envoyés est activée.{{{ Vous pouvez le modifier.

    }}}Voir Configurer vos préférences pour la fenêtre Écrire un mail{{{

  • Signatures

    Vous pouvez créer des signatures pour vos mails. Si vous créez plusieurs identités, vous pouvez créer plusieurs signatures et les attribuer à plusieurs adresses.

  • }}}Carnet d’adresses{{{

    }}}Dans la page Carnet d'adresses, vous pouvez{{{

    • }}}Activer la fonction pour ajouter automatiquement des adresses vers le dossier Contacts mail lorsque vous envoyez un mail.{{{

    • }}}Activer pour utiliser la Liste globale d'adresses en affichant les contacts dans le Sélecteur de contact.{{{

    • }}}Inclure la Liste globale d'adresses lors de la saisie automatique des adresses.{{{

    • }}}Inclure les adresses dans des carnets d'adresses partagés lors de la saisie automatique des adresses.{{{

    }}}Voir Configurer vos préférences de carnet d’adresses{{{

  • Comptes

    }}}La page Comptes sert à créer et à gérer vos avatars.{{{ Un avatar est une identité mail utilisée dans le champ De des mails que vous envoyez. }}}Vous pouvez créer votre et vos comptes POP.{{{

  • Filtres de mails

    }}}Utilisez le dossier Filtres de mails pour définir les règles de filtre de vos mails.{{{ Pour apprendre à configurer les règles de filtre des mails, consultez la rubrique Fonctionnement du filtre.

  • Calendrier

    }}}Dans la page Calendrier, vous pouvez{{{

    • }}}Définir votre affichage par défaut des calendriers{{{

    • }}}Définir les paramètres à utiliser lors de la création d'un rendez-vous.{{{

    • }}}Sélectionner pour afficher le mini-calendrier.{{{

    • }}}Saisir une adresse pour faire suivre vos invitations à partir d'un Calendrier partagé.{{{

    • Configurer la notification de rappel pour les réunions.

    • }}}Définir les autorisations pour les affichages libres/occupés et qui peut vous inviter aux réunions{{{

    }}}Voir Configurer vos préférences de calendrier{{{

  • }}}Partage{{{

    }}}La page Partage affiche les objets qui sont partagés avec vous et les objets qui sont partagés par vous.{{{ }}}Voir Gérer les objets partagés{{{

  • }}}Rappels Dans Rappels, vous ajoutez l'adresse mail qui doit être utilisée quand vous définissez un rappel dans Tâches ou les événements du calendrier.{{{

  • }}}MobilesLes mobiles que vous avez configurés pour être synchronisés avec ZWC sont indiqués ici.{{{ }}}Vous pouvez également supprimer votre mobile de cette page.{{{ }}}Voir Gérer des mobiles{{{

  • Importer/Exporter

    }}}La page Import/Export facilite l'importation des différents objets de votre compte : dossiers, calendriers, listes de contacts.{{{ }}}Il est utile si vous souhaitez sauvegarder votre compte ou un dossier spécifique localement.{{{

  • Raccourcis

    }}}Dans la page Raccourcis, vous pouvez afficher la liste des raccourcis clavier et imprimer la liste.{{{

  • Zimlets

    }}}Vous pouvez gérer quels Zimlets vous souhaitez utiliser à partir du dossier Zimlets.{{{ }}}Les Zimlets sont des applications complémentaires qui améliorent les fonctionnalités de votre client.{{{ }}}L'administrateur système active les Zimlets.{{{ }}}Les Zimlets que vous pouvez utiliser sont affichés dans le Panneau d'aperçu.{{{
