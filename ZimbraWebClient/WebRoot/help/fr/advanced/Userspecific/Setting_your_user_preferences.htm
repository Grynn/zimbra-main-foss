Configuration de vos préférences d'utilisateur

Vos préférences d'utilisateur par défaut sont configurées lors de la création de votre compte. Ces options définissent le fonctionnement de votre messagerie, de vos carnets d’adresses et de vos calendriers. Vous pouvez également configurer votre identités mail personnelles et vos comptes POP. Vous pouvez aussi choisir un thème pour votre messagerie.

Vous pouvez modifier les paramètres dans l’onglet Préférences. Si l’option décrite ici n’est pas visible dans votre affichage, cela signifie que cette préférence n’est pas disponible pour votre compte.

Vous pouvez définir les préférences dans les onglets suivants : Cliquez sur le mot pour obtenir une description détaillée de l’onglet.

  • Général

    • Vous êtes connecté au client Web standard. Pour configurer l’affichage de ce type de client par défaut lorsque vous vous connectez, sélectionnez Avancé.

    • Avancé – Propose la gamme complète des fonctions de collaboration Web. Ce client Web exige un navigateur récent et une connexion Internet rapide.

    • Standard – Recommandé pour les connexions Internet lentes, pour des navigateurs moins récents et pour les utilisateurs à accessibilité réduite.

    • Dans Thème choisissez la couleur de fond pour l’interface de votre messagerie.

    • Si le fuseau horaire affiché ici est incorrect, choisissez le fuseau horairepar défaut pour votre compte. Il détermine la date et l’heure des mails que vous envoyez ou recevez, et l’heure à afficher pour les rendez-vous de votre calendrier.  Celui-ci peut être différent de la configuration de fuseau horaire de votre ordinateur.

    • Si Changer le mot de passe s’affiche, vous pouvez modifier votre mot de passe. Si vous avez des difficultés à modifier votre mot de passe, contactez votre administrateur pour connaître les règles en matière de mot de passe.

      • Contactez votre administrateur système si cette option n’est pas disponible et si vous voulez changer votre mot de passe.

    • Dans Paramètres de recherche, vous pouvez choisir d’inclure automatiquement dans les dossiers Spam et Corbeille toutes les recherches que vous effectuez. Par défaut, ces dossiers ne sont pas pris en compte lors des recherches.

    • Dans Langue de recherche, cochez Toujours montrer la chaîne de recherche pour afficher les détails de cette chaîne dans la zone de texte de recherche.

    • Activez Toujours afficher les cases à cocher  pour afficher une case à cocher se rapportant à chaque élément contenu dans le panneau de contenu. Utilisez les cases à cocher pour sélectionner un ou plusieurs éléments sur lesquels exécuter la même action. Par exemple : supprimer, déplacer, marquer comme lu/non lu.

  • Mail

    • Sélectionnez le nombre d’éléments (mails ou conversations) à afficher par page. Par défaut, l’affichage est de 25 éléments par page, mais vous pouvez choisir 10, 50, ou 100 éléments par page.

    • Sélectionnez la manière dont vous voulez grouper votre messagerie, par conversation ou par mail. Si vous utilisez la fonction de conversation, le groupement par conversation est l'option par défaut.

    • L’intervalle de consultation est la fréquence selon laquelle votre ordinateur vérifie les nouveaux mails.  Par défaut, la consultation des mails s’effectue toutes les 5 minutes. Le laps de temps minimum est défini par votre administrateur Zimbra.

    • Spécifiez si voulez afficher les mails au format HTML pour les messages formatés en HTML. En désactivant cette option, les mails apparaissent désormais en format texte simple, affichant les balises HTML au lieu de les appliquer en tant que code.

    • Options d’Aperçu du mail

      • Vous pouvez choisir d’afficher le panneau de lecture lorsque vous consultez un mail. Par défaut, le panneau de lecture s’affiche toujours.

      • Si activé, le client affiche un extrait des mails de la liste des mails. Lorsque le curseur est placé sur l’objet d’un mail, les premiers mots du mail apparaissent.

      • Si activé, double-cliquez pour ouvrir les mails dans une nouvelle fenêtre

    • Activez Images pour télécharger automatiquement les images de votre mail en HTML. Lorsque cette fonction est activée, il n’est pas nécessaire de cliquer sur l’image du mail pour voir l’image.

    • La Recherche Mail par défaut définit la recherche à exécuter lorsque vous vous connectez. La recherche par défaut est dans votre Boîte de réception.  Lorsque vous vous connectez, les résultats de votre Boîte de réception apparaissent. Vous pouvez modifier ce champ pour qu'un autre dossier, tag, ou recherche enregistrée apparaissent lorsque vous ouvrez le client Web pour la première fois. Consultez la section Description du langage de demande pour connaître la syntaxe. Activez et spécifiez une adresse pour envoyer des notifications indiquant que vous avez reçu de nouveaux mails.

    • Spécifiez une adresse vers laquelle faire suivre votre mail. Vous pouvez supprimer le mail original de votre messagerie Zimbra.

    • Spécifiez si vous voulez envoyer une réponse automatique par mail pour les messages que vous recevez.  Cette fonction est fréquemment utilisée pour les messages d’absence.

    • Configurez les mails que j'envoie pour traiter les mails que vous envoyez et qui contiennent votre adresse parmi les destinataires du mail, ou une adresse telle qu'une liste de diffusion à laquelle vous appartenez.

     

  • Création de mail

    • Définissez l’éditeur que vous voulez utiliser par défaut, HTML ou texte simple.

    • Définissez les paramètres de police par défaut à utiliser pour rédiger en HTML dans Mail, Documents et Calendrier.  Vous pouvez définir le style, la taille et la couleur des polices à utiliser par défaut. La police par défaut est : Arial, 12 pt, noir.

    • Toujours ouvrir une nouvelle fenêtre pour créer un mail lance une nouvelle fenêtre de mail, séparée de votre messagerie Zimbra, lorsque vous rédigez un mail. Lorsque la fenêtre de création de mail apparaît, vous pouvez consulter vos mails et naviguer dans votre messagerie.

    • Les mails que vous rédigez sont automatiquement enregistrés en tant que brouillons après quelques minutes. Si vous préférez que les mails ne soient pas automatiquement enregistrés, désactivez cette fonction.

    • Choisissez d’inclure le texte original dans le corps de votre message de réponse. Vous pouvez choisir d’inclure le texte original, de ne pas l’inclure, de l’inclure sous forme de pièce jointe, de l’inclure dans le corps avec un préfixe de votre choix, ou d’inclure uniquement le mail le plus récent.  (autrement dit, il reprendra le texte écrit par la personne qui a envoyé le mail, mais pas le texte précédent.)

    • Si vous choisissez d’inclure le texte original, précisez si vous voulez ajouter un préfixe à chaque ligne avec > of | pour les mails précédents que vous avez fait suivre ou auxquels vous avez répondu.

    • Par défaut, l’option Garder une copie des mails dans le dossier Envoyés est activée. Une copie des mails que vous envoyez est enregistrée dans votre dossier Envoyés.

    • Spécifiez comment inclure le texte original d’un mail lorsque vous le faites suivre : dans le corps du mail, dans le corps avec un préfixe, ou en tant que pièce jointe.

  • Signatures

    Vous pouvez créer des signatures pour vos mails. Votre signature peut inclure votre nom et du texte supplémentaire comportant le nombre maximum de caractères autorisé par votre compte. Si vous créez plusieurs identités, vous pouvez créer plusieurs signatures et les attribuer à plusieurs adresses.

    • Insérez la signature qui devra apparaître dans vos mails.

    • Spécifiez l’emplacement de votre signature dans vos mails : Ajouter la signature au-dessous de tous les messages de votre mail, ou l'ajouter sous le texte que vous avez rédigé, ou auquel vous avez répondu, ou bien encore que vous avez fait suivre.

  • Carnet d’adresses

    • Activez la fonction pour ajouter automatiquement des adresses vers votre carnet d’adresses lorsque vous envoyez un mail. Si cette fonction est activée, les adresses sont enregistrées dans le carnet d'adresses Contacts mail.

    • Activez la fonction pour utiliser la liste d’adresses globale pour compléter automatiquement les adresses mail.

    • Choisissez si vous préférez voir les contacts sous forme de liste ou de cartes professionnelles.

    • Sélectionnez le nombre de contacts à afficher par page. Le nombre par défaut est 25.

    • Importez des contacts dans votre liste de contacts. Le fichier à importer doit être séparé d’une virgule.

    • Exportez votre liste de contacts. Le fichier s’exporte au format suivant : séparation avec une virgule.

     

  • Comptes

    L’onglet Comptes sert à créer et à gérer vos avatars. Un avatar est une identité mail utilisée dans le champ De des mails que vous envoyez.

    Le style par défaut est défini dans l’identité principale. Lorsque vous créez de nouvelles identités, vous pouvez utiliser les mêmes paramètres que l’identité par défaut, ou vous pouvez configurer des préférences différentes.

    Le bouton Ajouter un compte externe vous permet d’identifier vos comptes POP/IMAP externes afin que ce client Web puisse récupérer les mails à partir des autres comptes.  Vous pouvez également configurer votre compte pour que les réponses à ces mails utilisent l’adresse POP/IMAP correspondante.

  • Filtres de mails

    Utilisez l’onglet Filtres de mails pour définir les règles de filtre de vos mails. Pour apprendre à configurer les règles de filtre des mails, consultez la rubrique Fonctionnement du filtre.

  • Calendrier

    • Choisissez le mode d’affichage de votre calendrier Zimbra. Par défaut, l’affichage est défini par semaine ouvrée.

    • Spécifiez avec quel jour de la semaine vous souhaitez débuter la semaine de votre calendrier.

    • Si vous changez de fuseau horaire au cours de vos voyages, il est possible d’afficher la liste des fuseaux horaires dans la vue des rendez-vous.

    • Si vous créez plusieurs rendez-vous sans participants, cochez Utilisez la boîte de dialogue QuickAdd/Ajout rapide.

    • Un mini- calendrier s’affiche dans Calendrier.  Vous pouvez activer le mini-calendrier pour qu’il soit visible depuis n’importe quelle vue.

    • Configurez le nombre de minutes précédant le rappel d'un rendez-vous (l'option par défaut est cinq minutes.)

    • Importez et exportez d'autres programmes de calendrier.

  • Messagerie instantanée

    Vous pouvez définir la manière dont la messagerie instantanée (MI) est intégrée à votre compte.

    • Connexion. Si vous voulez vous connecter automatiquement aux services de messagerie instantanée, cette option doit être activée.

    • Notification. Voici comment la notification de MI est reçue et comment vous souhaitez apparaître aux autres utilisateurs.

    • Discussion.  Votre historique des discussions est automatiquement enregistré dans un dossier Discussions. Les messages de ce dossier sont similaires à des mails : vous pouvez les imprimer, les faire suivre sous forme de mail, et les supprimer du dossier. Vous pouvez également définir le délai d’expiration (si inactif).

  • Raccourcis

    Les raccourcis clavier vous permettent d’utiliser l’application sans vous servir de la souris. Il existe un grand nombre de raccourcis. Vous pouvez également créer des raccourcis personnalisés en affectant des alias numériques aux dossiers, tags, et recherches enregistrées.

    • La Liste de raccourcis indique tous les raccourcis clavier existants.

    • L’onglet Raccourcis des dossiers de mails sert à créer un raccourci pour afficher rapidement vos dossiers de messagerie et déplacer un mail vers le dossier désiré. Vous choisissez le dossier et lui attribuez un nombre. Par exemple, si vous attribuer 3 au dossier Boîte de réception : En saisissant v3, vous serez dirigé vers la Boîte de réception, quel que soit votre emplacement dans la messagerie. En sélectionnant un élément et en saisissant .3, l’élément sera déplacé vers la Boîte de réception.

    • L’onglet Raccourcis des recherches enregistrées sert à créer des raccourcis pour accéder rapidement aux recherches enregistrées. Vous choisissez une recherche enregistrée et lui attribuez un nombre. Par exemple, si vous attribuez 3 à votre recherche préférée, vous pouvez alors saisir s3 depuis n’importe quel emplacement de votre messagerie pour lancer rapidement cette recherche.

    • L’onglet Raccourcis de tag sert à créer des raccourcis pour accéder rapidement à tous les éléments marqués d’un tag spécifique ou pour appliquer un tag à un mail ou un contact. Vous choisissez un tag et lui attribuez un nombre. Par exemple, vous attribuez le chiffre 3 à un tag nommé Suivi. Depuis n’importe quel emplacement de votre messagerie, vous pouvez saisir y3 pour consulter tous les éléments marquésSuivi. Sélectionnez un élément puis saisissez t3 pour marquer cet élément comme étant Suivi.

    Vous pouvez attribuer n’importe quel nombre, mais il ne peut pas être réutilisé dans un onglet de raccourci.
