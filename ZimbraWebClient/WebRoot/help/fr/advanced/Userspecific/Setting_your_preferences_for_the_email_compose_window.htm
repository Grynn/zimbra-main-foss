Configurer vos préférences pour la fenêtre Écrire un mail

Utilisez les options de l’onglet Préférences>Création de mail de la manière suivante :

  • Créer. Choisissez votre éditeur de texte par défaut, qu’il s’agisse de HTML ou texte simple.

Définissez les paramètres de police par défaut à utiliser pour rédiger en HTML dans Mail, Documents et Calendrier. Vous pouvez définir le style, la taille et la couleur des polices à utiliser par défaut.

  • Les mails que vous rédigez sont automatiquement enregistrés en tant que brouillons après quelques minutes. Si vous préférez que les mails ne soient pas automatiquement enregistrés, désactivez cette fonction. Vous pouvez cliquer sur Enregistrer le brouillon pour enregistrer manuellement un message comme brouillon.

  • Répondre/Transférer en utilisant le format du mail original. Même si votre éditeur de texte par défaut est différent du mail reçu, cochez cette case pour répondre au mail au même format que celui reçu. Cette option est utile pour s'assurer que le destinataire reçoive le mail à un format adapté qu'il puisse lire.

  • Toujours ouvrir une nouvelle fenêtre pour créer un mail lance une nouvelle fenêtre de mail, séparée de votre messagerie Zimbra. Par défaut, un onglet Écrire s'ouvre dans la barre d'outils de l’application afin de rédiger vos mails. Dans les deux cas, vous pouvez consulter vos mails et naviguer dans votre messagerie lorsque la fenêtre Écrire un mail est ouverte.

  • Vérification orthographique obligatoire avant tout envoi de mail. Cochez pour que la vérification orthographique soit automatiquement appliquée avant l’envoi de votre mail. Si des erreurs sont détectées, une boîte de dialogue affiche le nombre de fautes d'orthographe trouvées et vous demande si vous voulez les corriger avant l'envoi du mail.  

  • Mails envoyés. Par défaut, Enregistrer une copie dans le dossier Envoyés est activé. Une copie des mails que vous envoyez est enregistrée dans votre dossier Envoyés. Vous pouvez cocher la case Ne pas enregistrer de copie des mails envoyés si vous n'acceptez pas cet enregistrement.

  • Répondre/Répondre à tousChoisissez d’inclure le texte original dans le corps de votre message de réponse. Vous pouvez choisir d’inclure le texte original, de ne pas l’inclure, de l’inclure sous forme de pièce jointe, de l’inclure dans le corps avec un préfixe de votre choix, ou d’inclure uniquement le mail le plus récent. (autrement dit, il reprendra le texte écrit par la personne qui a envoyé le mail, mais pas le texte précédent.) Quand Inclure les en-têtes est activé, les en-têtes À/De et Objet s'affichent dans votre message.

  • Faire suivre. Choisissez d’inclure le texte original dans le corps du mail que vous faites suivre. Vous pouvez choisir d’inclure le mail original et les titres avec le préfixe suivant ou inclure le mail original en pièce jointe. Quand Inclure les en-têtes est activé, les en-têtes À/De et Objet s'affichent dans votre message.

  • Préfixe. Si vous choisissez d’inclure le texte original, précisez si vous voulez ajouter un préfixe à chaque ligne avec > pour les mails précédents que vous avez fait suivre ou auxquels vous avez répondu.
