Quand dois-je enregistrer mon travail ?

Le client Web de Zimbra est une interface d’utilisateur graphique (GUI) qui inclut des fenêtres, des boutons, une zone de texte, et une souris.

Avec une GUI qui utilise la technologie du navigateur Internet pour afficher et exploiter les fonctions d’utilisateur, il n’est pas toujours possible pour l'application de détecter que vous avez effectué une tâche que vous voudriez enregistrer.

Dans Zimbra, les règles suivantes s’appliquent :

  • Lorsque vous êtes en train d'écrire un mail, et que vous cliquez sur une autre section de l'écran de Zimbra, l'application vous invite à enregistrer votre travail. Vous trouverez le mail ultérieurement dans votre dossier Brouillons.

  • En actualisant le client Web de Zimbra sur la même fenêtre du navigateur, votre session de messagerie continuera, mais il est possible que vous perdiez certaines modifications non enregistrées.
