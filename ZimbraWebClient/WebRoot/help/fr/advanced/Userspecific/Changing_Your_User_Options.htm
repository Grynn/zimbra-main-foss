Modifier vos préférences d’utilisateur

Vous pouvez modifier vos préférences d’affichage, de messagerie, de carnet d’adresses, de calendrier, et vous pouvez créer des règles de filtre à partir de l’application Préférences. Voici les indications générales pour modifier vos préférences :

  1. Cliquez sur Préférences.

  2. Cliquez sur l’onglet contenant les préférences que vous voulez modifier. Pour plus de détails, consultez la rubrique Aide pour chacun de ces onglets

  3. Modifier les paramètres.

  4. Cliquez sur Enregistrer.
