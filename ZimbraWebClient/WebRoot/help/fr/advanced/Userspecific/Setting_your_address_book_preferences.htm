Configurer vos préférences de carnet d’adresses

}}}Utilisez les options de l’onglet Préférences>Carnet d’adresses de la manière suivante :{{{

  • }}}Paramètres{{{

    • }}}Choisissez Ajouter les nouveaux contacts dans "Personnes contactées par mail" afin d’ajouter automatiquement des adresses à votre carnet d’adresses lorsque vous envoyez un mail.{{{ Si cette fonction est activée, les adresses sont enregistrées dans le carnet d'adresses Contacts mail.

    • }}}Première étape : explorer la liste GAL à l'aide du sélecteur de contact peut être activé si vous devez fréquemment trouver des adresses qui sont dans le carnet d'adresses de votre société mais pas dans votre carnet d'adresses.{{{

  • }}}Saisie automatique{{{

  • }}}Activez Inclure des adresses de la liste d'adresses globale (GAL) si vous souhaitez voir les adresses du carnet d'adresses de la société (liste d'adresses globale) quand vous tapez une adresse mail.{{{

  • }}}Activez Inclure les adresses dans les carnets d'adresses partagés si vous souhaitez voir des adresses dans les carnets d'adresses qui sont partagés avec vous quand vous tapez une adresse mail.{{{

}}}Notez que si vous activez l'une des deux fonctions ci-dessus, quand vous tapez une adresse, vous pouvez obtenir de nombreuses fausses adresses indiquées si ces listes d'adresses sont très grandes.{{{

  • }}}Quand Exécuter la saisie automatique à chaque frappe de virgule est activé, la saisie automatique reconnaît la virgule comme une "commande" pour compléter rapidement le nom et automatiquement l'ajouter au champ du destinataire.{{{

  • }}}Quand Ne pas afficher le groupe de contacts si l’un de ses membres correspond est activé, quand la saisie automatique sert à saisir des adresses d'utilisateur, le groupe de contacts indique que les utilisateurs présents ne sont pas affichés dans la liste de saisie automatique.{{{
