}}}Importer des objets dans votre compte{{{

}}}Le dossier des préférences d'import/export facilite l'importation des différents objets de votre compte : dossiers, calendriers, listes de contacts.{{{

}}}Importer des contacts{{{

}}}Vous pouvez importer les listes de contacts enregistrées au format .csv (dans le fichier généré, les différents contacts sont séparés par une virgule).{{{ }}}La fonction d'importation est disponible dans le dossier Préférences>Import/Export.{{{

  1. }}}Dans l'onglet Préférences, cliquez sur le dossier Import/Export.{{{

  2. }}}Dans la section Import, sélectionnez le type Contacts.{{{

  3. }}}Dans la liste Contacts, sélectionnez le type (format) du fichier à importer, parmi :{{{

    • Contacts Zimbra

    • }}}Contacts Yahoo!{{{

    • }}}Contacts Gmail{{{

    • Contacts Outlook 2000

    • Contacts Outlook 2003

    • Contacts Outlook

    • }}}Contacts Thunderbird{{{

}}}Les listes de contacts provenant de différents comptes risquent de ne pas afficher les informations dans le même ordre.{{{ }}}Par conséquent, il est important de sélectionner le type (format) du ou des fichiers de contacts à importer ; en effet, c'est cette valeur qui conditionnera l’ordre des champs et l'affichage approprié des informations dans votre carnet d’adresses.{{{

  1. }}}Recherchez le fichier de contacts .csv à importer.{{{

  2. }}}Cliquez sur Importer.{{{ }}}Dans la boîte de dialogue Sélectionnez un carnet d’adresses, spécifiez la cible du fichier de contacts importé : sélectionnez l'un des carnets d’adresses existants ou créez un carnet d’adresses.{{{

  3. }}}Cliquez sur OK.{{{ Le fichier de contacts est importé.

}}}Importation de votre calendrier iCal{{{

}}}Vous pouvez importer les calendriers iCalendars (iCal) enregistrés au format .ics. iCalendars est le standard pour les échanges des données de calendrier via Internet, et .ics est le format standard des informations iCalendar.{{{ }}}La fonction d'importation est disponible dans votre onglet Préférences >Import/Export.{{{

}}}Vous pouvez importer votre calendrier iCal dans un calendrier existant, ou créer un calendrier et importer le fichier .ics dans ce nouveau calendrier.{{{

Pour importer un calendrier iCal :

  1. }}}Dans l'onglet Préférences, activez le sous-onglet Import/Export.{{{

  2. }}}Dans le champ Type de la section Import, sélectionnez Calendrier.{{{

  3. }}}Dans le champ Fichier, retrouvez le fichier .ics à importer.{{{

  4. }}}Cliquez sur Importer.{{{ }}}Dans la boîte de dialogue Sélectionnez dossier, sélectionnez le dossier de destination du calendrier importé (l'un des calendriers existants, ou un calendrier que vous créez à cet effet).{{{

  5. }}}Cliquez sur OK.{{{ Le calendrier est importé dans votre compte.

}}}Les calendriers iCal sont mis à jour à intervalles réguliers (définis par l’administrateur système).{{{
