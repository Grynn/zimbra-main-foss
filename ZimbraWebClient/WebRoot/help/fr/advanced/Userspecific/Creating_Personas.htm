Créer des avatars

Un avatar est une identité mail utilisée dans le champ Envoyé par des mails que vous envoyez. La création de plusieurs identités mail vous permet d’utiliser plusieurs adresses mail dans votre messagerie. Vous pourriez vouloir faire cela pour gérer différents rôles dans votre vie professionnelle et personnelle à partir d'un seul et même compte de messagerie. Le nom de compte par défaut est défini dans l’identité principale.

Le bouton Ajouter un compte externe vous permet d’identifier vos comptes POP/IMAP externes afin que ce client Web puisse récupérer les mails à partir des autres comptes. Vous pouvez également configurer votre compte pour que les réponses à ces mails utilisent l’adresse POP/IMAP correspondante.

Pour créer un avatar, configurez ce qui suit :

  1. Dans la page Préférences Comptes, cliquez sur Ajouter un avatar.

  2. Dans la section Paramètres de l’avatar saisissez un nom pour cet avatar. Ce nom sert à identifier l’avatar dans la liste De lorsque vous rédigez un mail.

  3. Dans la zone de texte sous De : Saisissez le nom qui doit s’afficher dans le champ De de vos mails sortants (par exemple, John). Il s’agit du nom convivial qui s’affiche devant votre adresse mail. Dans la deuxième case, choisissez l'adresse mail à partir de laquelle vous voulez envoyer vos mails. En fonction de la configuration de votre compte, vous pouvez créer une autre adresse. Faites ce qui suit.

  4. Si les réponses doivent être envoyées à un nom et une adresse différents de ceux que vous avez configurés dans le champ De, cochez Répondre à. Saisissez le nom et l’adresse qui doivent recevoir les réponses pour cette identité. Par exemple, les réponses aux mails de formation de John doivent être envoyées à l'Équipe de formation à formation@exemple.com.

  5. Vous pouvez choisir une signature à utiliser pour cet avatar. Si vous n’avez pas créé de signature, vous pouvez revenir en arrière et modifier ultérieurement cet avatar. Si vous ne voulez pas que la signature soit appliquée de manière automatique, sélectionnez Ne pas inclure de signature.

  6. Si vous voulez que les mails contenus dans des dossiers spécifiques utilisent automatiquement cet avatar, cochez Répondre aux mails ou faire suivre les mails du ou des dossier(s) suivants. Cliquez sur l’icône du dossier pour sélectionner un ou plusieurs dossiers dans la liste des dossiers existants ou pour créer un nouveau dossier. Insérez une virgule ou un point-virgule pour séparer les dossiers saisis dans le champ.

  7. Cliquez sur Enregistrer pour enregistrer cet avatar.  

Si vous devez créer une signature, affichez l’onglet Signature après avoir cliqué sur Enregistrer. Voir la section Utiliser une signature.
