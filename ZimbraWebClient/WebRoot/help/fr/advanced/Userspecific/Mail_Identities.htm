Gestion de vos identités de compte

Lorsque vous envoyez un nouveau mail, répondez ou transférez un mail, une identité est toujours associée à ce mail.  Il s’agit de l’identité par défaut créée en tant que nom de compte. Elle est utilisée à moins que vous ne configuriez d’autres identités.  La création de plusieurs identités mail, appelées aussi avatars, vous permet d’utiliser plusieurs adresses mail dans votre messagerie. Il est possible que vous souhaitiez effectuer cette opération pour gérer plusieurs fonctions dans votre travail ou plusieurs rôles dans votre vie privée à partir de ce seul compte de messagerie.

Vous pouvez également ajouter des adresses de comptes externes pour récupérer vos mails depuis vos autres comptes POP ou IMAP. Vous pouvez aussi configurer les paramètres du compte externe pour que l’ adresse correspondante soit insérée dans le champ De lorsque vous répondez à ces mails.  

Si vous avez configuré plusieurs avatars ou ajouté des comptes externes, vos identités apparaissent dans le champ De de l’en-tête lorsque vous ouvrez une nouvelle fenêtre de mail.

Les avatars, comptes externes et signatures sont configurés dans l’onglet Préférences, Comptes et dans l’onglet Signatures.
