Répondre à des mails

Vous pouvez répondre à l’expéditeur d’un mail, ou vous pouvez répondre à l’expéditeur ainsi qu’à tous les destinataires du mail.

  • Répondre permet de répondre uniquement à l'expéditeur du mail original. Cette option n’inclut aucune autre adresse présente dans les champs À, ou Cc . En cliquant sur Répondre une nouvelle fenêtre pour rédiger un mail apparaît, et remplit le champ À : avec le nom de la personne qui vous a envoyé le mail.

  • Répondre à tous inclut tous les destinataires du mail précédent. Vous utilisez cette option si vous voulez que toutes les personnes ayant reçu le mail précédent puissent également recevoir votre réponse.

L'utilisation de l’option Répondre permet uniquement de conserver la zone de texte. Les pièces jointes du mail original ne sont pas comprises dans la réponse.

Lorsque vous cliquez sur Répondre  ou sur Répondre à tous la page de création de mail s’affiche, contenant le texte du mail auquel vous répondez. Suivant la configuration de vos préférences d'utilisateur, le texte des mails précédents peut contenir un caractère indiquant le nombre d’échanges précédents effectués depuis que ce texte a été rédigé. Par exemple, si la réponse fait partie d’une conversation en cours, et que vous avez activé l’option Inclure le mail original avec le préfixe... afin d’utiliser le caractère >, le texte de chaque mail de la conversation est affiché avec des crochets obliques :

> Mail précédent

> > Mail avant le mail précédent

> > > Et ainsi de suite

Remarque : Si le message fait partie d'une conversation, la modification de la ligne Objet: fera apparaître votre réponse dans une nouvelle conversation.

Pour répondre à un mail :

    1. Ouvrez le mail, puis cliquez sur Répondre ou surCliquez sur la flèche pour choisir Répondre à tous. Une fenêtre de nouveau mail s’affiche. Elle contient les adresses dans l'en-tête et le mail original dans le corps du texte.

    2. Rédigez votre réponse.

    3. Si vous ajoutez automatiquement une signature à vos mails,  elle est insérée dans le mail selon les préférences que vous avez définies, au bas de tous les mails dans la fenêtre de nouveau mail ou au bas du mail que vous rédigez, sous le texte cité.

    1. Si vous voulez modifier la partie du mail original à inclure dans votre réponse, cliquez sur Options dans la barre d’outils du nouveau mail. Votre option par défaut est cochée mais vous pouvez uniquement la modifier pour ce mail.

    2. Cliquez sur Envoyer.
