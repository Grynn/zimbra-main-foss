}}}Impression de mails individuels{{{

}}}Vous pouvez sélectionner un ou plusieurs messages et vous pouvez sélectionner une conversation complète à imprimer.{{{

Pour envoyer un mail vers l’imprimante :

  1. }}}Sélectionnez le message, puis cliquez sur Imprimer.{{{

Le mail choisi apparaît dans une fenêtre séparée, accompagné d’une boîte de dialogue d’impression standard.

}}}Quand vous sélectionnez plusieurs messages, ils sont imprimés comme un seul message imprimé.{{{

  1. Dans la fenêtre de dialogue, sélectionnez une imprimante puis cliquez sur Imprimer, pour envoyer le message vers l’imprimante sélectionnée.

}}}La taille de police par défaut pour la sortie est 12 points.{{{ }}}Vous pouvez modifier la taille de police qui est utilisée dans votre section Préférences>Général>Autre.{{{

}}}Remarque :{{{ }}}n'utilisez pas la fonction Fichier>Imprimer du navigateur pour imprimer votre message.{{{ La page ne sera pas formatée de manière correcte, et la lecture de votre message pourrait s'avérer difficile.
