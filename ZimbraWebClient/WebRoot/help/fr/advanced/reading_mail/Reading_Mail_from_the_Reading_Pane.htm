Affichage des messages à partir du Panneau de lecture

Vous pouvez utiliser le Panneau de lecture pour avoir un aperçu des messages dans votre Boîte de réception et d'autres dossiers.

Par défaut, l’affichage des mails dans le panneau de lecture est activé. Vous pouvez le désactiver dans le lien Affichage de la barre d’outils. Vous pouvez personnaliser l'emplacement du Panneau de lecture.

Sélection de l'affichage du Panneau de lecture

Le Panneau de lecture peut s'afficher en bas ou à droite du Panneau de contenu.

  1. Cliquez sur Afficher dans la barre d'outils Mail.

  2. Sélectionnez pour afficher le Panneau de lecture en bas ou à droite du Panneau de contenu. Vous pouvez également désactiver le Panneau de lecture.

Affichage des messages dans le Panneau de lecture

Le Panneau de lecture est vide jusqu'à ce que vous cliquiez sur un message.

Ajustez la taille du Panneau de lecture en plaçant votre pointeur sur le haut de la ligne entre le Panneau de contenu et le Panneau de lecture. Le pointeur devient une flèche bidirectionnelle. Cliquez sur le pointeur et faites glisser pour modifier la taille de la zone du Panneau de lecture.

Quand vous cliquez sur un message dans la liste, le texte du message s'affiche dans le Panneau de lecture.

Marquage des messages comme lus à partir du Panneau de lecture

Dans Préférences mail, vous pouvez configurer si vous souhaitez marquer ou non les messages que vous voyez dans le Panneau de lecture comme lus.

  • Le marquer comme lu immédiatement. Dès que vous cliquez sur un message dans le Panneau de lecture, il est marqué comme lu.

  • Le marquer comme lu après "x " secondes. Le délai par défaut est de 5 secondes. Si vous cliquez sur un message et l'afficher pendant 5 secondes, le message est marqué comme lu.

  • Ne pas le marquer comme lu. Les messages que vous lisez dans le Panneau de lecture ne sont jamais marqués comme lus.
