}}}Récupération d'objets supprimés{{{

}}}Le dossier Corbeille est disponible pour chaque application @CompanyName@ que vous utilisez.{{{ }}}Seuls les objets spécifiques à l'application sélectionnée sont affichés quand vous ouvrez le dossier Corbeille à partir de l'application.{{{ }}}Mais quand vous sélectionnez Vider la corbeille à partir de toute application, les objets de toutes les applications sont supprimés.{{{

}}}Avant de vider le dossier Corbeille, vous pouvez récupérer les fichiers de la Corbeille en les faisant glisser et en les déposant dans un autre dossier.{{{

}}}Si Restaurer objets supprimés s'affiche quand vous cliquez avec le bouton droit sur le dossier Corbeille, vous pouvez récupérer les mails que vous aviez supprimés dans la Corbeille.{{{

  1. }}}Cliquez avec le bouton droit sur le dossier Corbeille de messagerie, puis sélectionnez Restaurer objets supprimés.{{{

  2. }}}Sélectionnez un ou plusieurs messages à récupérer.{{{

  3. }}}Cliquez sur Restaurer à, puis sélectionnez le dossier de destination ou créez un nouveau dossier.{{{

  4. }}}Cliquez sur OK.{{{

  5. }}}Fermez.{{{

}}}Il est possible de récupérer les objets supprimés de la Corbeille dans un délai de 30 jours après la suppression de la Corbeille.{{{ }}}Passés ces 30 jours, les objets sont supprimés de manière permanente.{{{

}}}Les messages qui restent dans la Corbeille peuvent être automatiquement supprimés, auquel cas vous ne pouvez pas les récupérer.{{{
