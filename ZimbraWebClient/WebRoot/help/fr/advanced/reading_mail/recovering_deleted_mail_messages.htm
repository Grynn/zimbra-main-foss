Récupération de mails supprimés

Lorsque vous supprimez des mails et des contacts, le fichier est déplacé vers le dossier Corbeille. Vous pouvez récupérer ces fichiers dans le dossier Corbeille en les sélectionnant et les faisant glisser vers un autre dossier..

Attention : il est possible que la Corbeille soit vidée de manière automatique, ce qui implique que les mails qu'elle contient seront définitivement supprimés.
