Lecture d’un mail

Par défaut, tous les nouveaux mails sont enregistrés dans votre Boîte de réception.  Si vous avez configuré des règles de filtre de mails, les nouveaux messages peuvent apparaître dans d’autres dossiers que votre Boîte de réception. Le nombre entre parenthèses indiqué à côté de chaque dossier correspond au nombre de mails non lus contenus dans ce dossier.

Si vous utilisez l’affichage de conversation, les mails sont regroupés en conversations.  Si vous déplacez le curseur de votre souris sur la conversation, une info-bulle affiche la première ligne du corps du mail le plus récent.

Pour lire un mail dans la fenêtre de conversation

  1. Double-cliquez sur la conversation contenant le mail.

  2. Le panneau de contenu change, et affiche à présent les mails individuels de la conversation. La colonne Fragment affiche la première ligne du corps du message. Pour ouvrir le mail, cliquez sur la ligne qui contient le mail que vous souhaitez lire.

Si le panneau de lecture est ouvert, le corps du message apparaît dans la partie inférieure du panneau. Dans le cas contraire, vous devez double cliquer sur le message dans la liste des mails. Cela permettra de remplir le panneau de contenu.

Pour lire un mail dans la vue de mails

  • Double cliquez sur le mail.

Le corps du message apparaît dans la partie inférieure du panneau. S’il ne s’affiche pas, double cliquez sur le message dans la liste des mails.
