Exportation de carnets d’adresses

Vos  carnets d’adresses sont exportés et enregistrés au format de fichier texte (.csv) avec une virgule pour les séparer.

  1. Affichez Préférences, Carnet d’adresses.

  2. Cliquez sur Exporter. La liste de vos carnets d’adresses s’affiche.

  3. Choisissez le carnet d’adresses à exporter.

  4. Cliquez sur OK.

  5. Cochez Enregistrer sur le disque puis cliquez sur OK.  Sélectionnez l’emplacement où enregistrer le fichier et choisissez le nom du fichier. Le nom par défaut est Contacts.csv.

  6. Cliquez sur Enregistrer.
