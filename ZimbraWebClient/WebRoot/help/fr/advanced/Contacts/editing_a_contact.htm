Modification des informations d’un contact

Pour rechercher et ouvrir une fenêtre de contact :

  1. Dans le champ Rechercher sur la barre de Recherche, saisissez les critères de recherche comme par exemple le prénom, nom de famille ou un nom de groupe.

Pour effectuer une recherche par adresse mail, vous devez saisir l’adresse complète. Le prénom ou le nom de famille doivent correspondre à un mot complet. Par exemple, vous pouvez saisir Dupont ou dupont@acme.com.

  1. Sélectionnez le carnet d’adresses ou la liste d’adresses globale (GAL) dans la liste déroulante se trouvant à la droite du champ de recherche.

  2. Dans la fenêtre de Modification de contact, double-cliquez sur le contact à modifier et effectuez les changements.

  3. Cliquez sur Enregistrer pour valider vos modifications.

Pour ouvrir un contact à modifier :

  1. Dans le champ Contacts, cliquez avec le bouton droit sur le contact à modifier puis choisissez Modifier contact.

  2. Dans la fenêtre de Modification de Contact, effectuez les modifications puis cliquez sur Enregistrer pour valider vos changements.

Pour déplacer un contact vers un autre carnet d’adresses, procédez comme suit :

  1. Choisissez le contact puis ouvrez la fenêtre de modification de contact.

  2. Dans le menu Carnet d'adresses qui apparaît, sélectionnez le carnet d'adresses vers lequel vous souhaitez déplacer ce contact.

  3. Cliquez sur Enregistrer.
