Partage de votre carnet d’adresses

 Vous pouvez partager vos carnets d’adresses avec les utilisateurs suivants :

  • Utilisateurs ou groupes internes. Vous sélectionnez le type de privilèges à accorder aux utilisateurs internes, c’est-à-dire :

    • Visionneur – Le bénéficiaire peut consulter mais ne peut pas modifier le contenu du carnet d’adresses

    • Gestionnaire – Le bénéficiaire dispose de l’autorisation complète pour consulter, modifier, ajouter des contacts au carnet d’adresses et les supprimer.

    • Admin – Le bénéficiaire peut appliquer les fonctions suivantes aux contacts de votre carnet d’adresses : consulter, modifier, ajouter, supprimer, partager les listes de contacts avec d’autres utilisateurs.

Aucune est une option pour désactiver temporairement l’accès au carnet d’adresses partagé d’un cédant sans retirer les privilèges de partage. Le bénéficiaire dispose toujours du carnet d'adresses mais n'est pas autorisé à consulter ou gérer son contenu.

  • Invités externes. Vous créez un mot de passe pour accéder au carnet d’adresses, et les invités doivent le saisir pour pouvoir consulter les contacts. Le carnet d’adresses s’affiche sous forme de fichier .csv. Ces invités ne sont pas autorisés à effectuer des modifications sur le carnet d’adresses de votre compte.

  • Public. Tout utilisateur ayant connaissance de l’URL menant au carnet d’adresses peut consulter la liste des contacts. Le carnet d’adresses s’affiche sous forme de fichier .csv. Ces invités ne sont pas autorisés à effectuer des modifications sur le carnet d’adresses de votre compte.

Pour partager votre carnet d’adresses

  1. Cliquez avec le bouton droit sur le carnet d’adresses à partager, puis choisissez Partager Carnet d'adresses.

  2. Sélectionnez la personne avec laquelle partager le calendrier dans la boîte de dialogue Propriétés du partage.

Utilisateurs ou groupes internes

  1. Si vous choisissez l’option Interne, vous avez la possibilité de sélectionner des adresses dans la liste s’affichant lorsque vous commencez à les saisir – saisie automatique – ou vous pouvez saisir des adresses qui ne figurent pas dans la liste. L’utilisateur ou le groupe doit disposer d’une adresse valide dans le serveur mail de votre société.

  2. Dans la section Rôle, sélectionnez les autorisations d’accès.

  1. Dans la zone Mail , choisissez le type de mail à envoyer. Les options sont les suivantes :

  • Envoyer un mail standard – Le mail standard de notification de partage est envoyé à l’adresse mail.  

  • Ajouter une note au mail standard – Pour ajouter des informations complémentaires au mail standard (facultatif).

  1. Cliquez sur OK.

 

Invités externes

  1. Si vous choisissez des invités Externes, saisissez leurs adresses mail. Vous avez la possibilité de saisir plusieurs adresses mail dans le champ . Insérez un point-virgule (;) entre les adresses.

  2. Saisissez le mot de passe que les utilisateurs devront utiliser pour consulter votre carnet d’adresses.

  1. Dans la zone Mail , choisissez le type de mail à envoyer.  L’URL menant à votre carnet d’adresses ainsi que le mot de passe sont inclus dans le contenu du mail.

    • Envoyer un mail standard – Le mail standard de notification de partage est envoyé à l’adresse mail.

    • Ajouter une note au mail standard – Pour ajouter des informations complémentaires au mail standard (facultatif).

  2. Cliquez sur OK.

Public

Si vous choisissez l’option Public, toute personne disposant de l’URL peut avoir accès à votre carnet d’adresses. L’utilisation d’un mot de passe est facultative.

Modification ou retrait de l’autorisation

Vous pouvez retirer les autorisations à tout moment.

  1. Cliquez avec le bouton droit sur le carnet d’adresses à partager.

  2. Cliquez sur Propriétés d'édition. Les utilisateurs partageant le dossier sont répertoriés.

  3. Cliquez sur Modifier pour modifier les autorisations, ou
    sur Retirer pour supprimer les autorisations.

  4. Choisissez si un mail décrivant les modifications doit être envoyé.

  5. Cliquez sur OK.
