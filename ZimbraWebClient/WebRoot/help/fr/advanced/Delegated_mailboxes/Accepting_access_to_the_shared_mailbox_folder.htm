Accepter l’accès à un dossier de boîte de réception partagée

Si vous recevez une notification par mail indiquant que vous êtes autorisé à partager le dossier de messagerie d'une autre personne, vous pouvez accepter ou refuser le partage.

Si vous acceptez le partage, le dossier de messagerie partagé  s’affiche dans la liste de vos Dossiers  dans le panneau Aperçu.

Pour accepter

  1. Cliquez sur Accepter le partage dans le mail. La boîte de dialogue Accepter le partage apparaît et détaille le rôle dont vous bénéficiez.

  2. Avant d’accepter, vous pouvez modifier le nom du dossier de messagerie puis choisir une couleur pour contraster le dossier.

  3. Facultatif : choisissez d'envoyer une confirmation au cédant.

  4. Cliquez sur Oui. Le dossier de messagerie est ajouté à votre liste de dossiers. Ce message est déplacé vers le dossier Corbeille.
