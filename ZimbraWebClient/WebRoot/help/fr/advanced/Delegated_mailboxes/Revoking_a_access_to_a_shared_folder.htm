Modification d’accès à un dossier partagé

Vous pouvez modifier les autorisations ou retirer le partage à tout moment.

  1. Cliquez avec le bouton droit sur un dossier de messagerie que vous partagiez et cliquez sur Éditer propriétés. Les utilisateurs partageant le dossier sont répertoriés dans la section Partages pour ce dossier de la fenêtre.

  2. Cliquez sur Modifier pour modifier les autorisations, ou
    Cliquez sur Retirer pour supprimer les autorisations.

  3. Choisissez si un mail décrivant les modifications doit être envoyé.

  4. Cliquez sur OK.
