Personnalisation de votre page de messagerie initiale

Par défaut, votre boîte aux lettres s’ouvre avec l’affichage Boîte de réception. Tous les mails de votre Boîte de réception s'affichent dans le panneau de contenu. Vous pouvez modifier l'affichage initial de tous les dossiers, recherches, ou tags dans le panneau d'aperçu. Effectuez ces modifications à partir de Préférences mails, Recherche Mail par défaut.  

Voici des exemples pour afficher seulement les messages spécifiques quand vous ouvrez votre boîte aux lettres :

  • Pour afficher seulement vos messages non lus quand vous ouvrez votre boîte aux lettres, tapez est : non lu dans le champ Recherche Mail par défaut.

  • Pour afficher les mails qui sont tagués avec des tags spécifiques, tapez le tag :< nom de tag> dans le champ Recherche Mail par défaut. Par exemple, pour afficher seulement les messages marqués avec le tag ToDo, tapez tag : ToDo.

  • Pour afficher les messages que vous avez envoyés vers un dossier spécifique via un filtre mail, tapez dans :< nom de dossier> dans le champ Recherche Mail par défaut. Par exemple, pour afficher les messages dans le dossier Travail, tapez dans : travail.

Remarque :  Utilisez des guillemets si le nom du dossier ou du tag comporte plusieurs mots. Par exemple, dans :"Mail non lu."

Configuration du comportement de la boîte aux lettres pour actualiser

Dans la section Préférences mail Affichage des mails, activez l'une des options dans les ensembles Lorsque je clique sur "Actualiser" que l'affichage de votre boîte aux lettres change ou non quand vous cliquez sur Actualiser.

  • Si Exécuter ma recherche par défaut est sélectionné, quand vous cliquez sur Actualiser, la recherche mail décrite dans le champ Recherche Mail par défaut fonctionne. Si votre recherche par défaut n'est pas le dossier que vous affichez, votre affichage est automatiquement redirigé vers l'affichage de dossier par défaut. Dans la plupart des cas, c'est votre Boîte de réception.  

  • Si Actualiser la vue actuelle est sélectionné, quand vous cliquez sur Actualiser à partir d'un dossier, Actualiser demande des recherches de mises à jour pour l'affichage de votre dossier actuel et pour un autre nouveau mail. L'affichage de votre dossier actuel ne change pas.

Remarque :  Si un nouveau mail est reçu vers d'autres dossiers, ces dossiers passent en gras et le nombre de messages non lus est mis à jour.
