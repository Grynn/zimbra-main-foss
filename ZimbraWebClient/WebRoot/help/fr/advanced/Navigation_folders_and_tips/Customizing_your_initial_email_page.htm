Personnalisation de votre page de messagerie initiale

Par défaut, votre boîte aux lettres s’ouvre avec l’affichage Boîte de réception. Tous les mails de votre Boîte de réception s'affichent dans le panneau de contenu. Vous pouvez modifier l'affichage initial de tous les dossiers, recherches, ou tags dans le panneau d'aperçu.  Par exemple, si vous souhaitez uniquement afficher vos mails non lus en ouvrant votre boîte aux lettres, affichez Préférences>Mail>Recherche Mail par défaut et, dans le champ qui apparaît, saisissez la recherche avec la mention is :unread.

Ou, pour afficher les mails marqués par le tag "ToDo"que vous avez créés au moment de votre connexion, saisissez la recherche avec le tag :ToDo. Pour afficher un mail dans un dossier autre que votre Boîte de réception, saisissez une recherche de dossier, par exemple in:Work.

Utilisez des guillemets si le nom du dossier ou du tag comporte plusieurs mots. Par exemple dans :"Mail non lu."
