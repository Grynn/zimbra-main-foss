Affichage dans le panneau d’aperçu

À gauche de l'interface du client Web de Zimbra, vous trouvez vos dossiers, recherches, tags et Zimlets (pour le client Web avancé).

 

Dossiers

  • Boîte de réception : emplacement par défaut pour la réception de vos mails.

  • Envoyés : contient les messages que vous avez envoyés.

  • Brouillons : contient les mails que vous avez écrits mais pas encore été envoyés.

  • Spams : contient les messages qui ont été marqués comme spam, par l'utilisateur (vous-même) ou par le système.

  • Corbeille : contient les éléments que vous avez supprimés.

  • Les Dossiers que vous avez créés sont affichés au-dessous. Si votre compte mail a été migré d’un autre système de messagerie tel que Microsoft® Outlook, il est possible que certains dossiers présents sur l’interface Zimbra représentent des dossiers du système de messagerie précédent.

Nombre de mails non lus. Le nombre entre parenthèses indiqué à côté de chaque dossier correspond au nombre de mails non lus contenus dans ce dossier. Les dossiers Tags indiquent également le nombre de mails non lus. Tout dossier comportant des mails non lus apparaît en gras.

Pour connaître le nombre de mails et la taille totale du dossier, déplacez le curseur sur un dossier.

.

Recherches

Ce champ contient les demandes de recherche que vous avez créées et enregistrées pour les utiliser ultérieurement.

Tags

Cette zone indique les tags que vous avez créés. Cliquez sur un tag pour afficher rapidement tous les mails marqués avec ce tag. Pour plus d’informations, consultez la section Utiliser des tags pour classer les mails.

Zimlets

Les Zimlets servent de mécanisme pour intégrer des informations et contenus tiers avec les fonctions du client Web de Zimbra. Les Zimlets vous permettent d’interagir avec différents types de contenus dans vos mails. Selon la configuration de votre interface, il est possible que vous n'ayez pas accès aux Zimlets depuis le panneau d'aperçu.
