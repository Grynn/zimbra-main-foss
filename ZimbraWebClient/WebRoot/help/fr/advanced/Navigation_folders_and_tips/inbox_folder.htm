Dossiers du système de messagerie

}}}Les dossiers système sont Boîte de réception, Envoyés, Brouillons, Spams et Corbeille.{{{ Vous ne pouvez pas déplacer, renommer ni supprimer ces dossiers. Les dossiers système s’affichent toujours dans la partie supérieure de votre liste de dossiers, suivis par les dossiers que vous avez créés.

Si vous déplacez le curseur sur un dossier, le nombre de mails présents dans le dossier et la taille de ce dossier s'afficheront.

Boîte de réception

Les nouveaux mails sont réceptionnés dans la Boîte de réception. Par défaut, votre Boîte de réception s’affiche immédiatement après votre connexion.

Envoyés

Une copie de tous les mails que vous avez envoyés est enregistrée dans le dossier Envoyés.

Brouillons

}}}Vous pouvez enregistrer les mails que vous avez écrits mais pas encore envoyés dans le dossier Brouillons.{{{ Lorsque vous ouvrez un mail dans le dossier Brouillons, il s’affiche dans la fenêtre Écrire.

}}}Spam{{{

Le filtrage de la plupart des mails automatiques non sollicités (appelés "spam" ou "courrier indésirable") est effectué par un filtre anti-spam avant que ces mails ne soient délivrés sur votre Boîte de réception. }}}Des mails pouvant être considérés comme du courrier indésirable (mais avec le bénéfice du doute) peuvent être placés automatiquement dans le dossier Spams.{{{ Vous pouvez consulter ces mails, les déplacer vers un autre dossier ou les supprimer. }}}Si vous ne les supprimez pas, le dossier Spam sera vidé après un laps de temps défini par votre administrateur Zimbra.{{{

}}}Remarque :{{{  }}}si vous trouvez que vous recevez un grand nombre de mails indésirables, contactez votre administrateur de messagerie.{{{ En rapportant les mails marqués comme du spam, vous permettez à votre administrateur de messagerie d'affiner le filtrage des spams en vigueur sur votre site.

}}}Si vous trouvez un message qui n'est clairement pas pertinent, vous pouvez le contraster, puis cliquer sur Reporter un Spam dans la barre d'outils.{{{ }}}Ce message a été déplacé vers le dossier Spam.{{{ }}}À l'inverse, il peut arriver qu'un mail ait été placé dans le dossier Spam alors qu'il est légitime et qu'il pourrait vous être utile ;{{{ }}}Glissez-déposez le message dans un autre dossier.{{{  

}}}Vous pouvez vider le dossier Spams en cliquant avec le bouton droit sur le dossier (dans le panneau situé à gauche) et en sélectionnant ViderSpams.{{{ }}}Notez en outre que le dossier Spam est vidé automatiquement au bout d'un certain délai (défini par l'administrateur).{{{

Corbeille

Les éléments supprimés sont placés dans le dossier Corbeille et sont conservés jusqu’à ce que vous vidiez manuellement la corbeille ou que le dossier soit automatiquement vidé. }}} Mail est vidé de ce dossier après une période spécifiée, tel que spécifié par votre administrateur.{{{

Vous pouvez vider la corbeille en cliquant avec le bouton droit sur le dossier (dans le panneau situé à gauche), puis en sélectionnant Vider la corbeille.
