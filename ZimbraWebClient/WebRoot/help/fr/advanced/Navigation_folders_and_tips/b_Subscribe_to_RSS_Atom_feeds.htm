S’abonner à un fil RSS/Atom

Vous pouvez vous inscrire aux sites Web qui vous fournissent vos fils RSS / podcast favoris et envoyer des informations mises à jour directement vers votre boîte aux lettres.

Vous créez un dossier pour chaque fil RSS.

Avant de procéder à la création d'un dossier, vous devez connaître l'URL du fil RSS.

  1. Cliquez sur Nouveau dossier au dessus  du titre Dossiers dans le volet de navigation.

  2. Dans le champ Nom, saisissez le nom du dossier de fil RSS

  3. Cochez S’inscrire à un fil RSS/Atom.

  4. Dans le menu Dossiers, choisissez l'emplacement vers lequel ajouter le dossier de fil RSS. En choisissant l’onglet racine, cela vous permet de créer un dossier qui s'affiche dans la partie supérieure de votre affichage de dossiers – au même niveau que votre dossier Boîte de réception.

  5. Dans le champ URL, saisissez l’URL du fil.

  6. Cliquez sur OK.

L’icône de fil RSS,, s’affiche sous forme de dossier dans votre volet de navigation. Ouvrez le dossier RSS pour accéder au lien. Pour mettre à jour le contenu, ouvrez le dossier RSS puis cliquez sur Charger fil RSS.
