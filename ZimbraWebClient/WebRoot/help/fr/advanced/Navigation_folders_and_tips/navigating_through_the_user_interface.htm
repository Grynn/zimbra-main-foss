Navigation sur le client Web de Zimbra

Lorsque vous vous connectez au client Web de Zimbra, l’interface affichera un écran complet similaire à celui ci-dessous.

La page du client Web de Zimbra inclut les zones suivantes :

  1. Commandes du navigateur. Affiche les boutons standard du navigateur, comme par exemple Accueil, l'adresse actuelle vers laquelle vous pointez, et d’autres fonctions qui dépendent de votre logiciel et des paramètres de votre navigateur. Les boutons de la barre du navigateur ne font pas partie du client Web de Zimbra, et apparaissent sur chaque page Web que vous visitez.

  2. Barre de recherche, Nom d’utilisateur et Affichage de quota. La Recherche, Recherche avancée, le nom d’utilisateur avec lequel vous êtes connecté et le quota de votre boîte aux lettres s’affichent dans cette zone.

  3. Onglet d’applications. Les applications accessibles sont répertoriées dans les onglets situés dans la partie supérieure. Vous trouvez également les boutons Aide et Déconnexion.

  1. Barre d’outils. Affiche les actions disponibles pour l’application que vous utilisez actuellement. Dans cet exemple, la barre d’outils de l’applicationMail s’affiche.

  2. Panneau d’aperçu. Affiche vos Dossiers, y compris les dossiers système Boîte de réception, Envoyés, Brouillons, Spams, et Corbeille, ainsi que tous les dossiers personnalisés que vous avez créés, les Recherches que vous avez sauvegardées, et les Tags que vous avez créés, les liens Zimlet qui ont été créés pour intégrer les applications tierces à votre boîte aux lettres. Lorsque vous utilisez l’application Calendrier, la fenêtre d’aperçu affiche votre liste de calendriers et les Zimlets.

  3. Un mini-calendrier peut s’afficher au-dessous de la fenêtre d’aperçu. L’affichage du mini-calendrier est facultatif et peut être activé dans Préférences, dans l’onglet Calendrier.

  4. Panneau de contenu. Le contenu de cette zone change selon l’application utilisée. Dans l’affichage Boîte de réception, le panneau affiche tous les mails ou conversations dans votre Boîte de réception.
