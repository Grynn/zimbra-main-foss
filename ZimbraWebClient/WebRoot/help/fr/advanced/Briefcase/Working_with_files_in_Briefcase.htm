Travailler avec des fichiers dans un Porte-documents

Ouvrir les fichiers dans le porte-documents

Vous pouvez ouvrir des fichiers de vos Porte-documents depuis n'importe quel ordinateur. Vous devez disposer de l'application logicielle adéquate pour pouvoir ouvrir, lire ou mettre à jour le fichier. Des formats de fichiers courants comme les fichiers texte et HTML et les fichiers images comme les . gif ou . jpg peuvent être ouverts à partir de différents programmes.

Pour ouvrir un fichier, cliquez avec le bouton droit sur le fichier, puis sélectionnez Ouvrir.

Remarque : Vous pouvez également double-cliquer sur le fichier pour l'ouvrir.

  • Si le fichier est de type inconnu, une boîte de dialogue s'affiche, vous demandant ce que vous voulez faire : l'enregistrer sur l'ordinateur ou l'ouvrir avec un programme spécifique. Si vous avez partagé le Porte-documents avec d'autres utilisateurs, vous pouvez extraire un fichier, ceci afin d'empêcher que d'autres utilisateurs ne le modifient dans le Porte-documents.

    • Cliquez avec le bouton droit sur le fichier, puis sélectionnez Extraire un fichier. Une icône d'un verrou s'affiche à côté du fichier.

    • Si vous voulez extraire le fichier et y apporter des modifications, dans la boîte de dialogue des fichiers téléchargés, sélectionnez Extraire le fichier pour les modifications. Cliquez sur Télécharger.

    • Uploadez le fichier après l'avoir modifié. La boîte de dialogue Conflit d'upload apparaît, vous demandant si vous voulez envoyer la nouvelle version ou conserver celle qui se trouve dans le Porte-documents. Sélectionnez Nouveau, puis cliquez sur OK.

  • Si le fichier est au format Zimbra, double-cliquez pour ouvrir le fichier dans une page du navigateur. Vous pouvez modifier un document Zimbra depuis cette page.  

Modifier un fichier Zimbra

  • Cliquez avec le bouton droit sur le document Zimbra, puis sélectionnez Modifier. Le fichier s'ouvre dans une nouvelle page du navigateur.

  • Modifiez la page, puis cliquez sur Enregistrer et fermer pour fermer la page. Si vous ne souhaitez pas enregistrer votre travail, fermez l'onglet du navigateur.

Remarque : En cliquant sur Enregistrer ou Enregistrer et fermer, une nouvelle version du fichier est créée.

Déplacer des fichiers

Vous pouvez déplacer des fichiers d'un porte-documents à un autre.

  1. Sélectionner le(s)fichier(s) à déplacer

  2. Dans la barre d'outils, cliquez sur .

  3. Choisissez le dossier de destination ou créez un nouveau dossier de porte-documents.

  4. Cliquez sur OK.

Marquer des fichiers

Vous pouvez marquer des fichiers.

  1. Cliquez sur le fichier, puis dans la barre d'outils, et enfin sur .

  2. Choisissez le tag. Vous pouvez uniquement afficher les tags dans la Vue Détails.

  3. Cliquez sur OK.
