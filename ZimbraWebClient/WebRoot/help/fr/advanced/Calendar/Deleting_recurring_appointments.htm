Modification de rendez-vous récurrents

Si vous avez créé des réunions récurrentes, vous avez la possibilité de modifier et de supprimer une occurrence ou la série de réunions. Un mail est envoyé aux participants.

Pour modifier une réunion

  1. Double-cliquez sur la réunion à modifier, ou cliquez avec le bouton droit puis sélectionnez Supprimer.

  2. Choisissez d’ouvrir uniquement la date sélectionnée ou d’ouvrir la série. La page du Rendez-vous s’affiche.

  3. Effectuez vos modifications. Vous pouvez modifier le programme, les participants, ajouter une explication dans le champ de Texte puis ajouter une pièce jointe.

  4. Cliquez sur OK. Un mail est envoyé aux participants.

Pour supprimer une réunion récurrente

  1. Cliquez sur le rendez-vous.

  2. Cliquez avec le bouton droit puis sélectionnez Supprimer, ou faites glisser l’objet vers.

  3. Choisissez de supprimer l’instance ou de supprimer la série.

  4. Un mail est envoyé aux participants et le rendez-vous est supprimé de leurs calendriers.
