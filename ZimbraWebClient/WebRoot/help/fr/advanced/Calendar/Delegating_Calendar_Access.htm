Partager votre calendrier avec d’autres personnes

Vous pouvez partager votre calendrier avec :

  • Les utilisateurs ou groupes internes.  Vous choisissez le type de privilèges à accorder, ou :

    • Visionneur. Le bénéficiaire peut consulter les activités inscrites sur votre calendrier ainsi que l’état des demandes de réunion, mais il ne peut pas apporter de modifications à votre calendrier.

    • Gestionnaire. Le bénéficiaire a toute liberté de créer des réunions, accepter ou refuser des invitations, ou modifier et supprimer des activités de votre calendrier.

    • Aucune est une option pour désactiver temporairement l’accès au calendrier partagé du cédant sans retirer les privilèges de partage. Le bénéficiaire dispose toujours du calendrier sur sa liste du Calendrier mais ne peut plus afficher ou gérer les activités sur le calendrier partagé du cédant.

  • Invités externes. Vous créez un mot de passe pour accéder au calendrier partagé. Les invités doivent saisir ce mot de passe pour pouvoir consulter le calendrier. Ils ne sont pas autorisés à apporter des modifications au calendrier.

  • Public. Tout utilisateur ayant connaissance de l’URL menant au calendrier partagé peut le consulter. Ils ne sont pas autorisés à apporter des modifications au calendrier.

Pour partager votre calendrier

Cliquez avec le bouton droit sur le calendrier à partager puis sélectionner Partager le calendrier.

Sélectionnez la personne avec qui partager le calendrier dans la boîte de dialogue Propriétés du partage.

Utilisateurs ou groupes internes

  1. Si vous choisissez l’option Interne, vous avez la possibilité de sélectionner des adresses dans la liste qui s’affiche lorsque vous commencez à les saisir – saisie automatique – ou vous pouvez saisir des adresses qui ne figurent pas dans la liste. L’utilisateur ou le groupe doit disposer d’une adresse valide dans le serveur mail de votre entreprise.

  2. Dans la section Rôle, sélectionnez les autorisations d’accès.

  1. Dans la zone Message, sélectionnez le type de message à envoyer.  Les options sont les suivantes :

  • Envoyer un mail standard. Le mail standard de notification de partage est envoyé à l’adresse mail.  

  • Ajouter une note au mail standard. Vous pouvez ajouter des informations supplémentaires au mail standard.

  1. Cliquez sur OK.

 

Invités externes

  1. Si vous choisissez des invités Externes, saisissez leurs adresses mail.   Vous avez la possibilité de saisir plusieurs adresses mail dans le champ de messagerie. Insérez un point-virgule (;) entre les adresses.

  2. Saisissez le mot de passe que les utilisateurs devront saisir pour consulter votre calendrier.

  1. Dans la zone Mail , choisissez le type de mail à envoyer.  L’URL menant à votre calendrier ainsi que le mot de passe sont inclus dans le contenu du mail.

    • Envoyer un mail standard. Le mail standard de notification de partage est envoyé à l’adresse mail.

    • Ajouter une note au mail standard. Vous pouvez ajouter des informations supplémentaires au mail standard.

  2. Cliquez sur OK.

Public

Si vous choisissez l’option Public, toute personne disposant de l’URL peut consulter votre calendrier.  Les mots de passe sont facultatifs.

Modification ou retrait de l’autorisation

Vous pouvez modifier ou retirer les autorisations à tout moment.

  1. Cliquez avec le bouton droit sur un calendrier partagé.

  2. Cliquez sur Éditer propriétés. Les utilisateurs partageant le dossier sont répertoriés.

  3. Cliquez sur Modifier pour modifier les autorisations, ou
    Cliquez sur Retirer pour supprimer les autorisations.

  4. Choisissez si un mail décrivant les modifications doit être envoyé.

  5. Cliquez sur OK.
