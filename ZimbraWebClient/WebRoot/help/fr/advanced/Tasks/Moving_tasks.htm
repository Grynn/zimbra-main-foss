Déplacer des tâches

Vous pouvez déplacer une tâche vers une autre liste de l’une des manières suivantes :

  • Faites un clic droit sur la tâche puis, dans la boîte de dialogue Déplacer une tâche, choisissez la liste de tâches vers laquelle la déplacer, ou bien créez une nouvelle liste. Cliquez sur OK.

  • Pour sélectionner la tâche à déplacer, maintenez enfoncé le bouton gauche de la souris puis déplacez la tâche vers les listes du panneau d’aperçu.

  • Ouvrez la page de tâches puis dans le champ Listes de tâches, sélectionnez la liste des tâches.
