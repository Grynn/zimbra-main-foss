Ajouter une tâche

Créez une nouvelle tâche en saisissant simplement son nom dans votre liste de tâches. Vous pouvez également ouvrir la page des tâches et y ajouter des informations détaillées, notamment des notes et des pièces jointes.

  • Vous pouvez utiliser l’Assistant Zimbra pour ajouter rapidement une tâche tout en poursuivant celle que vous avez commencée (qu'il s'agisse d'un mail, d'une réunion dans un calendrier, ou de l'affichage d'un contact).  Lorsque la tâche est créée, toutes les informations disponibles sont ajoutées dans la zone de texte de la tâche.

Pour ajouter rapidement une tâche

  1. Choisissez la liste de tâches dans le panneau d’aperçu

  2. Dans le panneau de contenu, cliquez sur la ligne supérieure qui affiche Cliquez ici pour ajouter une nouvelle tâche.    

  3. Un champ de texte s’affiche. Saisissez le nom de la tâche.

  4. Appuyez sur Entrée. La tâche est ajoutée à votre liste de tâches.

  5. Si vous avez créé une tâche dans la mauvaise liste, ouvrez la tâche puis sélectionnez la liste correcte dans le menu déroulant des Listes de tâches. Vous pouvez aussi sélectionner et faites glisser la tâche du panneau de contenu vers la liste qui convient.

Pour ajouter une tâche comportant des informations détaillées

Vous pouvez utiliser vos listes de tâches pour créer et suivre l'avancement d’une tâche. De plus, vous pouvez écrire des notes dans votre tâche et joindre des fichiers pour faciliter l’accès à la tâche. Lorsque vous partagez votre liste de tâches avec d’autres utilisateurs, il est très utile de disposer de toutes les informations appropriées pour cette tâche.

 Vous pouvez évaluer la durée du projet en saisissant la date de début et d’échéance, et établir sa priorité. Lorsque vous démarrez la tâche, vous pouvez choisir un statut similaire à Non commencée ou En cours, puis sélectionner l'état d'avancement.

  1. Choisissez la liste de tâches dans le volet de navigation.

  2. Dans la barre d’outils de tâches, cliquez sur Nouvelle.  Une page de tâche sans titre s’affiche.

  3. Dans le champ Objet, donnez un nom à la tâche.

  4. (Facultatif) Dans la section Date, sélectionnez la Priorité, Date de début, Date d’échéance, et état.

  5. Si vous avez cliqué sur Nouvelle avant d’avoir ouvert la liste de tâches que vous souhaitiez, vous pouvez sélectionner celle qui convient dans le menu déroulant Liste de tâches.  

  6. Ajoutez des notes dans le champ de texte. Pour joindre des pièces, cliquez sur Ajouter pièces jointes dans la barre d’outils.

  7. Cliquez sur Enregistrerpour sauvegarder la tâche. La nouvelle tâche est ajoutée à votre liste de tâches.
