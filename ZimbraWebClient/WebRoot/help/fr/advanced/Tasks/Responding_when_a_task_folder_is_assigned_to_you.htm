Accepter l’accès à une autre liste de tâches

Si vous recevez une notification par mail indiquant que vous êtes autorisé à partager la liste des tâches d'une autre personne, vous pouvez alors accepter ou refuser le partage.

Si vous acceptez le partage, la liste de tâches partagée s’affiche dans votre liste de tâches du panneau d’aperçu.

Pour accepter

  1. Cliquez sur Accepter le partage dans le mail. La boîte de dialogue Accepter le partage s’affiche et décrit le rôle qui vous a été attribué.

  2. Avant d’accepter, vous pouvez personnaliser le nom de la liste de tâches, puis sélectionner une couleur à utiliser pour afficher la liste dans votre volet de navigation.

  3. Facultatif : choisissez d'envoyer une confirmation au cédant.

  4. Cliquez sur Oui. La liste de tâches partagée est ajoutée à votre liste de tâches dans le volet de navigation. Ce message est déplacé vers le dossier Corbeille.
