Créer une page

Vous pouvez créer une page, élaborer une présentation du contenu à l’aide des outils au format RTF ou saisir des tags HTML. Vous pouvez procéder à une vérification orthographique et utiliser des mots-clés pour créer des liens vers d’autres pages.

  1. Cliquez sur le carnet sur lequel vous souhaitez travailler. La page de table des matières s’affiche dans le panneau Contenu.

  2. Cliquez sur Nouveau dans la barre d’outils. Une page vierge s’affiche. Dans le champ Page de l’en-tête, saisissez le nom de la nouvelle page. Son nom s’affiche dans la table des matières.

  3. Rédiger le contenu de la page. Vous pouvez utiliser l’éditeur de format RTF pour ajouter des styles, des couleurs, des tableaux, et des feuilles de calculs à votre page.

Si vous êtes familier avec HTML et préférez utiliser les tags HTML, cliquez sur Format dans la barre d’outils, puis sélectionnez Source HTML. La page se transforme en champ vide. Vous pouvez alors utiliser la syntaxe standard des tags HTML pour créer la page.

  1. Cliquez sur Enregistrer pour sauvegarder le fichier. Vous pouvez alors procéder à l’ajout de détails sur la page, ou vous pouvez cliquer sur Fermer pour revenir à la page de la table des matières.
