Créer un nouveau message d’état

}}}La messagerie instantanée de @CompanyName@ vous permet de créer plusieurs messages d’état personnalisés.{{{

Pour créer un message d’état :

  1. }}}Dans la barre de messagerie instantanée, cliquez sur l’icône pour afficher la liste des états.{{{

  2. Choisissez Nouveau message d’état.

  3. }}}Dans la boîte de dialogue, qui apparaît, tapez le texte décrivant votre nouvel état,{{{ }}}Cliquez sur OK.{{{ }}}Exemple de message d’état : "Je reviens de suite".{{{ }}}Le message que vous venez de créer apparaît dans la liste des états.{{{

}}}Pour afficher l'un des états par défaut, déroulez la liste "En ligne" et sélectionnez la valeur souhaitée.{{{
