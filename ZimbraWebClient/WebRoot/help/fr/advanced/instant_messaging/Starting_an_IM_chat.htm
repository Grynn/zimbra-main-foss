}}}Démarrer une discussion de messagerie instantanée{{{

}}}Après avoir ajouté des amis, vous pouvez démarrer une discussion de messagerie instantanée avec eux.{{{

}}}Pour démarrer une discussion :{{{

  1. }}}Double-cliquez sur le nom du contact avec lequel vous souhaitez discuter.{{{ }}}Une fenêtre de discussion de messagerie instantanée s'affiche avec le nom de l'ami dans l'en-tête.{{{

  2. }}}En bas de la fenêtre de discussion, tapez le texte que vous souhaitez envoyer, puis appuyez sur la touche Entrée ou cliquez sur Envoyer.{{{ }}}Le message s'affiche dans la fenêtre de discussion sous votre nom.{{{

  3. }}}Quand votre ami vous envoie une réponse, vous pouvez la voir dans la fenêtre de discussion de messagerie instantanée sous son nom.{{{

  4. }}}Pour terminer une discussion, appuyez sur la touche Échap. ou cliquez sur la X rouge pour fermer la fenêtre.{{{

}}}Votre transcription de discussion est enregistrée par défaut dans le dossier Discussions auquel vous pouvez accéder dans le Panneau d'aperçu sous l'onglet Mail.{{{ }}}Pour modifier les configurations d'enregistrement, consultez Configuration des préférences de messagerie instantanée.{{{

}}}Utilisation des Émoticônes dans votre messagerie instantanée{{{

}}}Vous pouvez utiliser les émoticônes Yahoo! dans votre discussion de messagerie instantanée.{{{ }}}Cliquez sur la flèche située sur le bouton smiley jaune pour ouvrir le menu déroulant des émoticônes.{{{

}}}Cliquez sur l'émoticône souhaité pour l'insérer dans le champ de texte.{{{ }}}Tapez le texte qui accompagne l'émoticône ou appuyez simplement sur Entrée ou cliquez sur Envoyer pour simplement envoyer l'émoticône.{{{ }}}Vous pouvez le voir s'afficher dans la fenêtre de discussion.{{{

}}}D'autres manières de démarrer une discussion de messagerie instantanée{{{

  • }}}Cliquez avec le bouton droit sur le nom d'un ami, puis sélectionnez Nouvelle discussion.{{{

  • }}}Double-cliquez sur un contact dans votre Liste d'amis pour ouvrir une fenêtre de discussion.{{{
