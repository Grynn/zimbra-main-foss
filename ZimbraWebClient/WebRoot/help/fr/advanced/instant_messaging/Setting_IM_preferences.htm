Configuration des préférences de messagerie instantanée

Utilisez les options de l’onglet Préférences>Messagerie instantanée de la manière suivante : N’oubliez pas de cliquer sur Enregistrer pour enregistrer les modifications que vous avez apportées à vos préférences.

  • Connexion. Cochez la case si vous souhaitez vous connecter automatiquement à la messagerie instantanée quand vous vous connectez à @ClientNameLong@ . Si la case n'est pas cochée, vous devez vous connecter manuellement en modifiant votre état de Hors ligne à En ligne dans la liste des états.

  • Notifications. Sélection des options pour voir les notifications de messagerie instantanée

    • Activer les notifications instantanées affiche de brefs messages contextuels pour informer les autres de votre état.

    • Afficher les notifications de présence affiche un bref message contextuel pour vous informer si un ami est En ligne ou Hors ligne.

    • Afficher une notification chaque fois qu'un de mes amis modifie son état affiche un bref message contextuel pour vous informer si un ami modifie son état dans la messagerie instantanée @CompanyName@.

  • Lorsqu'un message arrive. Sélectionnez les paramètres à afficher dans la messagerie instantanée.

    • Émettre un son émet un son lorsqu'un message arrive. Un module d'extension Quicktime ou Windows Media Player doit être installé pour entendre les sons.

    • Afficher brièvement le nom du navigateur affiche brièvement le nom du navigateur si vous travaillez dans un onglet ou une fenêtre de navigateur différent(e) et si vous recevez un message de messagerie instantanée.

    • Afficher un message de notification. Yahoo! BrowserPlus doit être installé.

  • Discussion. Personnaliser votre discussion.

    • Enregistrer l'historique des discussions enregistre vos transcriptions de discussion. Vous pouvez accéder aux discussions enregistrées dans le dossier Discussions sous l'onglet Mail

    • Signaler l'état d'inactivité vous signale comme inactif à vos amis si vous êtes absent de @ClientNameShort@ pendant la durée spécifiée dans Délai d'inactivité.

  • Délai d'inactivité vous permet de sélectionner la durée à passer avant que votre état ne change. Votre état change en fonction de vos spécifications sous l'option Définir la présence.

  • Définir la présence vous permet de sélectionner l'état à afficher lorsque vous n'accédez pas à la messagerie instantanée. L'état change en fonction de la durée spécifiée sous Délai d'inactivité.
