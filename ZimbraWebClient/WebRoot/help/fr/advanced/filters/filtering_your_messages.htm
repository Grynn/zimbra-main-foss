Filtrer vos messages

Le filtre applique un ensemble de règles de correspondance au mail entrant, puis exécute une action précise.

Vous pouvez filtrer vos mails entrants pour les trier dans des dossiers, les marquer automatiquement, les faire suivre, ou les supprimer. Par exemple, vous pouvez créer une règle de la manière suivante :

  • Tous les mails envoyés par votre directeur sont rangés dans le dossier "Directives commerciales".

  • Tous les mails contenus dans la liste de diffusion "événements de la société" sont marqués avec le tag "Événements".

Pour créer ou modifier une règle de filtre :

  1. Activez l’onglet Préférences  .

  2. Activez l’onglet Filtres de mails puis cliquez sur Nouveau filtre.

  3. La boîte de dialogue Ajouter un filtre s’affiche.

  4. Dans Nom du filtre, saisissez le nom de la règle.

  5. Dans la zone Si les conditions ci-dessous sont satisfaites, sélectionnez une préférence de groupement.  

  • Condition indifférente signifie que si n'importe quelle condition du filtre est satisfaite, l'action devient effective.

  • Toutes les conditions signifie que toutes les conditions du filtre doivent être satisfaites afin d’exécuter l’action du filtre.  

Vous pouvez répéter les étapes suivantes afin de configurer plusieurs conditions et actions dans un seul filtre.

  • Choisissez dans la première liste déroulante une condition à utiliser.  

  • Choisissez un procédé de comparaison . Les options affichées dépendent de votre choix dans le menu précédent. Par exemple, "est" ou "n’est pas" peuvent s’afficher.

  • Saisissez un mot ou une phrase à comparer dans la zone du texte.

Cliquez sur Ajouter pour ajouter des conditions. Vous pouvez continuer à ajouter des conditions ou passer à l'étape suivante, qui consiste à ajouter une ou plusieurs actions.

  1. Dans le champ Exécuter les actions suivantes, choisissez une action dans le menu déroulant.

  • Spécifiez un dossier ou le nom d'un tag, si nécessaire. Ce que vous spécifiez dépend de l’action que vous choisissez.

Cliquez sur , pour ajouter des actions. Vous pouvez continuer à ajouter des actions ou cliquer sur OK pour terminer.

  1. Cochez la case Actif pour activer la règle de filtre.

Le filtre sera appliqué de manière automatique dès la réception de nouveaux mails.

  • Vous pouvez également créer un nouveau filtre à partir d’un mail. Cliquez avec le bouton droit sur le mail, puis choisissez Nouveau filtre. La boîte de dialogue Modifier filtre s’affiche avec les champs De, À, et Objet configurés avec les données du mail.  Si les conditions du filtre sont correctes, donnez un nom au filtre puis cliquez sur OK.
