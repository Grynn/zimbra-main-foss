Conditions et actions de filtre supportées

Les règles de filtre ne sont pas sensibles aux majuscules, ce qui signifie que les règles ignorent les lettres en majuscules.

Conditions

Les champs de comparaison incluent les éléments suivants :

  • De. À utiliser pour spécifier un nom dans l'en-tête De d'un mail.

  • À. Identique au champ De, mais la recherche des noms spécifiés s'effectue dans l’en-tête À .

  • Cc. Identique au champ De, mais s’affiche dans le champ Cc .

  • Objet. S’affiche dans l’en-tête Objet du mail.

  • En-tête nommé. Lorsque vous sélectionnez cette option, un champ d’entrée de texte supplémentaire s’affiche devant l’opérateur de comparaison (la section "contient"). Cette option vous permet d'indiquer l’en-tête mail de votre choix. Vous pouvez non seulement spécifier les champs standard À, ou De, mais également les champs de Date, Répondre à, et autre champs personnalisés qui peuvent être inclus dans l’en-tête du message. Vous pouvez utiliser cette option pour filtrer les mails qui ont des en-têtes "manquants", c’est-à-dire des en-têtes auxquelles il manque certaines informations qui sont normalement considérées comme classiques pour un mail. Parfois le spam (généré automatiquement), omet des informations telles que les champs Expéditeur et Répondre à. Utilisez la deuxième zone de texte (le champ situé immédiatement à droite de l'opérateur de comparaison) pour spécifier l'en-tête à tester.

  • Taille. Utilisez cette option pour sélectionner les mails qui ont une taille plus importante ou plus faible que celle spécifiée. La taille d'un mail comprend celle des pièces jointes. Vous pouvez l’utiliser pour supprimer les mails trop volumineux.

  • Date. Utilisez cette option pour spécifier les mails envoyés avant ou après une date précise.

  • Contenu. Les options de contenu sont Contient/Ne contient pas, et vous permettent de spécifier des mots correspondants dans le corps du mail. Vous ne pouvez pas effectuer d'opération de filtrage par mots dans les pièces jointes.

  • Pièce jointe. Vous pouvez filtrer vos mails selon qu’ils contiennent ou pas des pièces jointes.

  • Adresse dans. Vérifie la présence d’une adresse dans vos contacts. Le champ suivant vous sert à indiquer le carnet d'adresses à rechercher.

Les opérateurs de comparaison incluent ce qui suit :

  • Correspond exactement/Ne correspond pas exactement. Recherche une correspondance exacte. Par exemple, en recherchant Objet correspond exactement à  Banane, la correspondance exacte est "Banane" et non pas "Bananes" ou "Un camion rempli de feuilles de bananiers".

  • Contient/ne contient pas. Signifie que le champ Objet doit contenir l’élément spécifié.  Par exemple, la recherche de Objet contient Banane correspondrait à "Je vais manger des bananes".

  • Correspond au modèle/Ne correspond pas au modèle. Le caractère de substitution * est utilisé dans le champ de comparaison du filtre de mail pour représenter un ou plusieurs caractères du filtre. Par exemple, en spécifiant les correspondances d’objet "banane*" les correspondances seraient "bananes" ou "casserole de feuilles de bananes" mais pas "ma casserole de feuilles de bananes."

  • Existe/n’existe pas. Établit une correspondance si le champ spécifié est présent dans le mail. Utilisé avec les champs d’en-tête nommé et de pièces jointes.

  • Inférieur/pas inférieur à, supérieur/pas supérieur à. Utilisé pour établir une comparaison avec le champ Taille.

  • Avant/pas avant, après/pas après. Utilisé pour établir une comparaison avec le champ Date.

  • Dans/pas dans. Utilisé pour tester la présence d’une adresse dans vos contacts.

Actions

  • Conserver dans la Boîte de réception. Enregistre les mails dans la boîte de réception. Si aucune règle de filtre ne correspond à un mail, cette action s'exécute par défaut.

  • Ranger dans un dossier. Déplace un mail vers un dossier spécifié.

  • Supprimer. Supprime le mail sans le distribuer. Le mail ne fait pas partie de vos dossiers, même pas du dossier Corbeille.

  • Faire suivre. Transfère un mail vers une adresse spécifiée.

  • Arrêter l’évaluation. Cette action empêche les règles de filtre ultérieures de s’appliquer à un mail correspondant à celle qui est actuellement en application.

  • Marquer. Sélectionnez comme lu ou avec drapeau.

  • Libeller avec. Vous pouvez marquer les mails correspondants avec un tag sélectionné.

Actions multiples

Une règle de filtre permet de définir plusieurs actions. Les combinaisons d’actions sont interprétées de la manière suivante :

  • Supprimer. Si elle est combinée avec d’autres actions, l’action de suppression est ignorée et les autres actions deviennent effectives.

  • Arrêter l’évaluation. Les actions spécifiées après celle-ci sont ignorées.

  • Ranger dans un dossier. Plusieurs actions "Ranger dans" entraînent la création de plusieurs copies du mail stocké dans différents dossiers. Si aucun dossier n’est spécifié, le mail sera conservé dans la Boîte de réception.

  • Tag/Drapeau Ces actions s'exécutent pour le mail renvoyé à l'expéditeur à partir de l'action précédente la plus récente. Dans le cas de plusieurs actions "Ranger dans", il peut arriver que plusieurs copies du mail soient sauvergardées sans tag, et d'autres avec.

  • Conserver dans la Boîte de réception. Plusieurs actions "conserver" peuvent être définies, mais une seule copie du mail est conservée dans la Boîte de réception.
