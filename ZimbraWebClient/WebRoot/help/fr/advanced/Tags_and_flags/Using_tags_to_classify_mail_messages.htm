Utiliser les tags pour classer les mails et les contacts

Les tags servent de système de classement personnel pour vos mails, contacts et rendez-vous. Les tags facilitent la recherche de ces éléments. Par exemple, vous pourriez avoir un tag pour Rotation immédiate, et un autre pour Priorité moyenne. Les mails qui sont marqués d'un tag peuvent se trouver dans différents dossiers. Vous pouvez rechercher un tag précis, de sorte que tous les mails marqués avec ce tag s'affichent. Peu importe le dossier dans lequel ils sont rangés.

  • Vous pouvez créer autant de tags que vous le souhaitez.

  • Vous pouvez appliquer plusieurs tags au même mail ou au même contact.

Créer des définitions de tags

Vous pouvez accéder à la fonction Nouveau tag de différentes manières :

  • Dans la barre d'outils, cliquez sur la flèche Bas à côté de Nouveau, puis sélectionnez Tag.

  • Sélectionnez Tag dans la barre d'outils.

  • Faites un clic droit sur un élément affiché puis choisissez Marquer la conversation.

  • Faites un clic droit sur Tags ou sur un tag dans le panneau d'aperçu, puis choisissez Nouveau tag.

Lorsque la boîte de dialogue Nouveau tag s'affiche : Saisissez un nom puis cliquez sur OK. Les noms de tags peuvent contenir toutes sortes de caractères, hormis les suivants :

: (deux points)

/ (slash, barre oblique)

" (guillemets)

Le nouveau tag est répertorié sous Tags dans le panneau d'aperçu et peut être appliqué à vos mails ou à vos contacts.
