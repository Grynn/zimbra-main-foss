Gestion de l’affichage de vos mails

Vous pouvez afficher vos mails sous la forme de simples messages ou en tant que conversations. Si vous utilisez la fonction Conversations , l’affichage par défaut est celui d’une conversation.

Pour modifier la manière dont les mails sont regroupés lors de cette session, cliquez sur la flèche à côté d' Affichage dans le champ situé au-dessus du panneau Aperçu. Pour modifier l’affichage par défaut, affichez l’onglet Préférences, Boîte de réception.  

Affichage des mails

L’affichage traditionnel présente une liste de mails en commençant par le mail le plus récent. Les mails non lus sont affichés en gras. Les informations affichées sur une ligne contiennent les éléments suivants : colonnes de priorité (drapeaux), libellés (tags), De, présence de pièces jointes, objet, emplacement du dossier, taille du mail, y compris pièces jointes et heure à laquelle le mail a été reçu.

 

Affichage de conversation

La vue de conversation affiche vos mails regroupés par objet. Le nombre de messages dans la conversation s’affiche entre parenthèses après l’objet. Chaque conversation contient le message d'origine et l'ensemble des réponses. Les conversations contenant des messages non lus sont affichées en gras.

 

Double-cliquez sur la conversation pour afficher les messages dans la conversation. Cliquez sur un message pour afficher son contenu.

Tous les messages associés à la conversation sont affichés, même s'ils sont conservés dans d’autres dossiers. La colonne Dossier affiche le dossier dans lequel le message est conservé.
