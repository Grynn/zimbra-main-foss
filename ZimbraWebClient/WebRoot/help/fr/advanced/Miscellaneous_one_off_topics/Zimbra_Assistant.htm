Assistant Zimbra

L’assistant Zimbra est une fonction d’utilisateur avancée qui permet d'interagir rapidement avec le client Web à l'aide du clavier pour effectuer les tâches suivantes :

  • Créer des rendez-vous

  • Aller à une date précise de votre calendrier

  • Créer des contacts

  • Envoyer un mail sans modifier l’application que vous utilisez actuellement

Vous pouvez ouvrir l’assistant Zimbra à partir de n'importe quelle vue du client Web de Zimbra – à l’exception de l’onglet Préférences. Le raccourci de clavier pour ouvrir l’assistant Zimbra est ` (guillemet simple fermant) ou maj~ (tilde). Cette touche se trouve au-dessous de la touche Echap (Echap).

  • Maj ~ (tilde)  ne fonctionne pas avec l’utilisation de Safari ou Firefox dans Mac OS X. Le guillemet simple fermant (`) ne fonctionne pas.

La boîte dialogue de l’assistant Zimbra fournit une aide détaillée. Vous trouverez ci-dessous les étapes générales pour utiliser l’assistant Zimbra.

  1. Utilisez la touche de raccourci (le guillemet simple ou le tilde Maj, au choix) dans le client Web de Zimbra. La boîte de dialogue de l’assistant Zimbra s’affiche.

  2. Saisissez l’une des commandes suivantes :

    • Pour créer un rendez-vous, saisissez rendez-vous. Vous pouvez alors saisir un objet entre guillemets, l'heure, une date et un jour de la semaine, et tout autre commentaire pour le rendez-vous. Cliquez sur OK.  Remarque : Si vous voulez utiliser des fonctions de rendez-vous plus avancées, cliquez sur Plus de détails sur la page de Rendez-vous.

    • Pour aller à une date spécifique de votre calendrier, saisissez le calendrier puis la date, ou le jour de la semaine. Cliquez sur OK.  Remarque : Après la date, vous pouvez saisir l’affichage du calendrier que vous préférez – par jour, semaine de travail, semaine ou par mois. Le calendrier s’ouvre en suivant cet affichage.

    • Pour créer un contact, allez dans Contacts puis saisissez le nom du contact, son adresse mail, etc. Cliquez sur OK.  Remarque : Si vous voulez ouvrir la fiche détaillée des contacts, cliquez sur Plus de détails. Vous pouvez spécifier sur cette fiche le carnet d’adresses dans lequel vous voulez ajouter le nom du contact.

    • Pour envoyer un mail sans ouvrir la fenêtre Écrire un mail, saisissez mail. Saisissez l’objet entre guillemets; Entrez À :name@adress puis saisissez le corps du message.  Cliquez sur OK.
