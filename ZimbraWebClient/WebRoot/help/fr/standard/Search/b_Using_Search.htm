Utiliser la recherche

La fonction de recherche de Zimbra est un outil puissant. Bien que vous puissiez apprendre à créer des requêtes de texte complexes, comme indiqué dans Description de la langue de recherche, il existe un raccourci vous permettant de saisir simplement un nom ou un mot, sans aucun mot clé. Par exemple, pour rechercher tous les mails contenant le mot "tim" n’importe où dans une ligne d’objet, dans les champs À, De, Cc ou Bcc , dans le corps du mail, ou dans les pièces jointes, vous pouvez simplement saisir tim dans la zone Rechercher puis cliquez sur Recherche.

Vous choisissez l’emplacement où effectuer la recherche. Vous pouvez effectuer une recherche dans les mails, les contacts personnels, les calendriers personnels et les listes de tâches personnelles.

Vous pouvez saisir n'importe quelle valeur, comme par exemple un mot, un prénom, un nom de famille, un numéro de téléphone, ou le nom d'un domaine.   Pour plus d’options, vous pouvez utiliser  l’astérisque * en tant que caractère de substitution.  

Utiliser le caractère * de substitution dans une recherche

L’astérisque * peut servir de caractère de substitution  pour rechercher dans une zone de texte des mots ayant une orthographe similaire.  

Utiliser l’astérisque * en tant que caractère de substitution après un préfixe. Une recherche du type ca* donne comme résultat les éléments avec le mot cacher, carotte, etc.

Pour rechercher des contacts dans vos carnets d’adresses, vous devez saisir un mot complet pour que la demande soit correcte. Par exemple, pour rechercher des contacts par rapport à leur numéro de téléphone, vous devrez saisir le numéro de téléphone complet tel qu'il apparaît dans l'entrée du contact. Vous ne pouvez pas effectuer une recherche avec l'indicatif régional uniquement.

Pour rechercher des contacts dans votre carnet d’adresses :

  1. Dans la zone Rechercher, choisissez Contacts personnels dans le menu déroulant. Cela vous permet d'effectuer une recherche dans l'ensemble de vos carnets d'adresses.

  2. Saisissez un nom ou toutes autres données du contact dans la zone Rechercher.

  3. Cliquez sur Recherche, pour lancer la recherche.

Pour effectuer une recherche rapide dans les mails :

  1. Dans la zone Rechercher, choisissez Mails dans le menu déroulant.

  2. Saisissez un nom ou toutes autres données relatives au mail dans la zone Rechercher.

  3. Cliquez sur Recherche, pour lancer la recherche.

Créer une entrée de recherche complexe

Si vous lancez fréquemment la même requête de recherche ou si vous voulez rechercher des informations détaillées spécifiques, il est dans votre intérêt de créer une entrée de recherche et de l’enregistrer dans votre dossier Recherche. Lorsque vous voulez relancer la recherche, vous pouvez simplement cliquer sur la recherche.

Pour créer et enregistrer une recherche

  1. Dans le panneau d’aperçu de recherche, cliquez sur modifier. Un panneau Nouvelle recherche s’affiche.

  2. Sélectionnez un emplacement pour créer le dossier de recherche.  Si vous l’enregistrez dans l'onglet racine, la recherche s’affichera dans la zone Recherche. Si vous choisissez un autre dossier, la recherche sera enregistrée dans ce dossier.

  3. Dans le champ Chercher nom, donnez un nom à la recherche.

  4. Dans le champ Demande de recherche, saisissez votre requête en utilisant le langage de demande décrit dans Description du langage de demande. Vous pouvez également utiliser Et ou Ou dans votre demande.

    Utilisation de ET ou OU pour établir votre demande de recherche

    Pour effectuer des recherches à l’aide de plusieurs critères, vous pouvez rechercher des éléments correspondant à l’un des critères spécifiés ou à l’ensemble des critères. :

    • La recherche de mails correspondant à l’un des critères spécifiés s’appelle une recherche OU. En effet, si l’un des mails contient le terme X ou Y, il est alors considéré comme une correspondance.

    • La recherche de mails contenant à la fois X et Y s’appelle une recherche ET. En effet, le mail doit correspondre à l’ensemble des critères spécifiés afin d'être considéré comme une correspondance.

  5. Cliquez sur Nouvelle recherche pour créer et rechercher, et l’enregistrer dans le dossier spécifié.

Pour lancer l'une de vos recherches, cliquez sur le nom de la recherche dans le panneau d’aperçu. Les résultats de la recherche s’affichent dans le panneau de contenu.
