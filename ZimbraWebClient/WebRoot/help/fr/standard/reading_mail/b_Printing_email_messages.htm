Imprimer des mails

Pour envoyer un mail vers l’imprimante :

  1. Ouvrez la conversation ou le mail à imprimer, puis, sur la barre d’outils, cliquez sur Imprimer.

  2. Cliquez sur OK.  

Pour modifier la taille de police avant d'imprimer, puis envoyer la conversation vers l'imprimante :

  1. Ouvrez la conversation à imprimer, puis, sur la barre d’outils des messages, cliquez sur Imprimer. Le message s'affiche dans la boîte de dialogue Imprimer.

  2. Cliquez sur Annuler dans la boîte de dialogue Imprimer.

  3. Sur la page d'impression de message, cliquez sur les liens Taille de police + ou - à gauche.

  4. Sur la barre d'outils du navigateur, cliquez sur l'icône Imprimer.

  5. Dans la boîte de dialogue Imprimer, sélectionnez une imprimante, puis cliquez sur OK.

  6. Fermez la fenêtre du message.
