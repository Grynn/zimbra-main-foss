Ouvrir des pièces jointes

Les mails peuvent comporter des pièces jointes, comme par exemple des documents de traitement de texte, des feuilles de calculs, des fichiers textes, des fichiers ZIP, des images, des applications exécutables, ou tout autre type de fichier informatique évoluant dans le réseau informatique de l'expéditeur. Un trombone s’affiche à côté de l’objet pour les mails contenant des pièces jointes.

}}}Certains types de fichiers, comme les images JPEG ou les fichiers textes, peuvent être ouverts dans la plupart des systèmes informatiques.{{{ }}}D’autres fichiers, comme les documents Word de Microsoft, doivent être ouverts avec le même programme que celui avec lequel ils ont été créés.{{{

Lorsque vous recevez un mail contenant des pièces jointes, vous pouvez cliquer dans l’en-tête sur le lien de pièce jointe du trombone afin de vous diriger au bas du mail. Vous pouvez alors choisir d’ouvrir la pièce jointe, d’ouvrir la pièce jointe au format HTML ou de télécharger la pièce jointe.

  • Si vous choisissez d’ouvrir la pièce jointe, votre ordinateur doit être équipé du programme adéquat pour pouvoir lire ce type de fichier.

  • Si vous décidez de télécharger la pièce jointe, une boîte de dialogue de navigation apparaît et affiche votre système de fichiers actuel. Choisissez un emplacement approprié puis enregistrez le fichier. Vous pouvez ouvrir le fichier ultérieurement en dehors de votre client de messagerie.

Visualiser les pièces jointes au format HTML

Il est possible que vous disposiez de la fonction de visualisation des pièces jointes au format HTML. Cette fonction vous permet d’ouvrir la pièce jointe même si l’application adéquate n’est pas installée. }}}Cliquez sur afficher au format html pour visualiser le contenu du fichier.{{{ Cette caractéristique est configurée par votre administrateur système.

Antivirus

Les virus informatiques peuvent se propager via des pièces jointes de mails. Il est courant que les administrateurs de messageries appliquent un filtre antivirus pour les mails entrants, et effectuent une analyse antivirus des ordinateurs. Néanmoins, il est toujours possible qu’un virus passe au travers.

Dans le cas des virus propagés par mails, la majorité d’entre eux sont activés lors de l'ouverture ou de la visualisation de la pièce jointe contenant le virus. Un grand nombre de ces virus est connu pour se propager à l’aide des entrées dans les carnets d’adresses des utilisateurs. Par conséquent, nous vous recommandons de rester vigilant lorsque vous ouvrez des pièces jointes, même si elles proviennent d’expéditeurs que vous connaissez.
