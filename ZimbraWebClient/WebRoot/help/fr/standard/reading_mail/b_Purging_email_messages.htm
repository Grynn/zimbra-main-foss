Purger des mails

Pour supprimer des mails contenus dans la Corbeille, ouvrez le dossier Corbeille, puis, dans le panneau de contenu, cliquez sur Supprimer. Tous les mails sont définitivement supprimés.

De plus, Zimbra Collaboration Suite peut être configuré pour purger automatiquement les mails les plus anciens d’un dossier après un laps de temps spécifié. L’administrateur Zimbra établit les règles concernant la durée pendant laquelle les mails sont conservés dans vos dossiers.  

Une fois purgés, les mails sont définitivement supprimés.
