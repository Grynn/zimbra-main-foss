Affecter des tags à des mails et à des contacts

Les tags servent de système de classement personnel pour vos mails, contacts et rendez-vous. Les tags vous permettent de rechercher ces éléments. Par exemple, vous pourriez avoir un tag pour rotation immédiate, et un autre pour Priorité moyenne. Les mails qui sont marqués d’un tag peuvent se trouver dans différents dossiers. Ils fournissent un moyen d’afficher tous les mails portant un tag spécifique, où qu'ils se trouvent.

Vous pouvez créer et appliquer plusieurs tags à un seul élément afin de le classer simultanément dans plusieurs catégories. Après la création d’un tag, vous pouvez l’appliquer à des conversations, des mails ou des contacts.

Pour créer un tag :

  1. Dans l’en-tête du panneau d’aperçu du tag, cliquez sur modifier.

  2. Dans la fenêtre de contenu, cliquez sur Nouveau tag.

  3. Saisissez le nom du tag puis choisissez une couleur pour ce dernier.

  4. Cliquez sur Créer un tag. Le nouveau tag s’affiche dans la section du panneau d’aperçu de tags.

Pour appliquer/retirer un tag à plusieurs éléments à la fois :

  1. Cochez les cases pour les éléments à marquer d’un tag.

  2. Dans le menu déroulantAutres actions, choisissez le tag à appliquer.

  3. Cliquez sur Accepter.

Pour supprimer un tag de plusieurs éléments :

  1. Cochez les cases pour les éléments marqués du tag à retirer.

  2. Dans le menu déroulant Autres actions, et la section Retirer tag, choisissez le tag à supprimer.

  3. Cliquez sur Accepter.
