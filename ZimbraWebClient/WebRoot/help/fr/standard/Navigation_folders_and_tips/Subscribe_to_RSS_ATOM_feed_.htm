S’abonner au fil RSS/ATOM

Vous pouvez vous inscrire aux sites Web qui vous fournissent vos fils RSS / podcast favoris et envoyer des informations mises à jour directement vers votre boîte aux lettres.

Vous créez un dossier pour chaque fil RSS.

Avant de procéder à la création d'un dossier, vous devez connaître l'URL du fil RSS.

  1. Dans la fenêtre d’aperçu de la section Dossiers, cliquez sur modifier.

  2. Dans la barre d’outils, cliquez sur Nouveau fil RSS/Atom.

  3. Saisissez le nom du dossier, choisissez l’emplacement du dossier puis tapez  l’URL RSS du site Web dans le champ URL.

  4. Cliquez sur OK.

Le dossier est créé dans votre panneau d’aperçu. Ouvrez le dossier pour accéder au lien. Pour mettre à jour le contenu, faites un clic droit puis sélectionnez Recharger dans le dossier.
