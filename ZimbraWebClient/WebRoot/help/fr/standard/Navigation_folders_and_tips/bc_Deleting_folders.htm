Supprimer des dossiers

Vous pouvez supprimer des dossiers personnalisés, mais pas de dossiers système.

Lorsque vous supprimez un dossier, son contenu est déplacé vers le dossier Corbeille. Vous pouvez récupérer ce contenu dans le dossier Corbeille en le sélectionnant et le faisant glisser vers un autre dossier.

  • Le dossier Corbeille est régulièrement vidé, par conséquent les objets qu'il contient risquent d'être définitivement supprimés.

Pour supprimer un dossier :

  1. Choisissez le dossier à supprimer puis, dans le titre du dossier, cliquez sur modifier.

  2. Cochez Supprimer ce dossier.

  3. Cliquez sur Supprimer dossier.

Si vous souhaitez sauvegarder certains mails du dossier, déplacez-les vers un autre dossier avant de le supprimer.

 

Supprimer le contenu d’un dossier

Si vous voulez supprimer tout le contenu de l’un de vos dossiers, mais conserver le dossier :

  1. Choisissez le dossier puis, dans le titre Dossier, cliquez sur modifier.

  2. Cochez Supprimer définitivement tous les objets de ce dossier.

  3. Cliquez sur Supprimer tous les objets. Le contenu du dossier est immédiatement supprimé.   Cette action ne peut pas être défaite.
