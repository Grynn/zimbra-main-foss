Options d’affichage du Calendrier

L’Affichage Semaine Ouvrée s’affiche lorsque le Calendrier est ouvert pour la première fois.  Vous pouvez modifier l’affichage à tout moment depuis la barre d’outils du Calendrier. La modification de l’affichage du Calendrier effectuée depuis la barre d'outils fonctionne uniquement pour cette session.  Vous pouvez modifier le type d’affichage du calendrier dans Préférences, sous l’onglet Calendrier.

Les vues sont les suivantes :

  • L’affichage Jour correspond aux activités d’une journée

  • L’affichage Semaine ouvrée affiche les activités du lundi au vendredi

  • L’affichage Semaine indique les activités portant sur sept jours  . Dans l’onglet Calendrier, sous Préférences, vous pouvez sélectionner le premier jour de la semaine.

  • L’affichage Mois indique les activités d’un mois

  • La vue de Programme est la mieux adaptée si vous avez plusieurs calendriers. Chaque calendrier s’affiche dans une colonne séparée.
