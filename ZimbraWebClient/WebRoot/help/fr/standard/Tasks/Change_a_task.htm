Modifier une tâche

Pour actualiser votre tâche

  1. }}}Ouvrez la liste de tâches qui contient votre tâche.{{{

  2. }}}Dans le panneau de contenu, cliquez sur la tâche que vous souhaitez modifier .{{{ La page de la tâche s’affiche.

  3. Modifiez les informations de la tâche.

  4. }}}Cliquez sur Enregistrer, lorsque vos modifications sont terminées.{{{

Pour marquer une tâche accomplie

}}}Deux manières de marquer les tâches comme terminées :{{{

  • }}}Cochez une ou plusieurs tâches à marquer comme terminée(s).{{{ }}}Cliquez avec le bouton droit sur une ligne de tâche, puis sélectionnez Marquer comme terminé.{{{ Le % d'avancement passe à 100%.

  • }}}Cochez une ou plusieurs tâches à marquer comme terminée(s).{{{ }}}Cliquez sur Marquer comme terminé dans la barre d'outils.{{{

}}}Les informations de la tâche restent dans votre dossier de tâches.{{{
