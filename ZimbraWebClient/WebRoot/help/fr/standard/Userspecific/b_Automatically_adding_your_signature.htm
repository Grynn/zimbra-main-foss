Ajouter automatiquement votre signature

Vous pouvez créer des signatures pour vos mails. Votre signature peut inclure votre nom et du texte supplémentaire comportant le nombre maximum de caractères autorisé par votre compte.

Par exemple, votre signature peut indiquer :

John Smith
Vice Président du département technique Division
Widgets
Acme Corporation, Inc.
303-555-1212 x111
john.smith@exemple.com

Si vous créez plusieurs identités, vous pouvez créer plusieurs signatures et les attribuer à plusieurs adresses. Voir Gestion de vos identités de compte.

Pour créer une signature :

  1. Ouvrez Préférences puis sélectionnez l’onglet Signatures .

  2. Cliquez sur Modifier la signature.

  3. Dans le champ Nom associé à la signature insérez votre signature et un nom identifiable. Ce n'est pas votre signature mais un titre identifiable. Par exemple, vous pouvez appeler la signature "Formelle" pour décrire qu'elle inclut votre nom, votre titre, le nom de la société, etc.

  4. Dans la zone de texte Signature, saisissez les informations de la signature telles que vous voulez qu’elles s’affichent dans vos mails.

  5. Cliquez sur Enregistrer.

  6. Allez dans l'onglet Comptes pour sélectionner quelle signature afficher dans vos messages par défaut.

  7. Cliquez sur Enregistrer.
