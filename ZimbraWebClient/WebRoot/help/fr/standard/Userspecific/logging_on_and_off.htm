Se connecter et se déconnecter

}}}Vous devez vous connecter et être authentifié afin d'utiliser @ClientNameLong@.{{{ }}}Deux versions du @ClientNameLong@ sont disponibles : avancée et standard.{{{

  • Le client Web avancé offre un ensemble complet de fonctions de collaboration Web sous Ajax. Le client Web avancé fonctionne avec les navigateurs les plus récents et les connexions Internet les plus rapides.

  • Le client Web standard est l'option idéale lorsque les connexions Internet sont lentes ou que les utilisateurs préfèrent utiliser une messagerie au format HTML pour naviguer dans leur boîte aux lettres.

}}}Lorsque la boîte de dialogue de connexion apparaît, vous pouvez choisir la version à laquelle vous voulez vous connecter.{{{

}}}Remarques :{{{ }}}Dans la plupart des cas, le client Web avancé est le client par défaut et s’ouvre automatiquement lorsque vous vous connectez. Si vous voulez que le client Web standard soit votre client par défaut, affichez vos Préférences, dans l'onglet Général, après vous être connecté.{{{

Pour se connecter

  1. }}}Ouvrez une fenêtre du navigateur puis saisissez l'URL fournie par votre administrateur Zimbra.
    {{{
    Par exemple :
    }}}
    http:// mail.domain.com/ zimbra/mail    {{{

  2. }}}Lorsque l’écran de connexion apparaît, saisissez votre nom d’utilisateur.{{{ On vous demandera peut-être de saisir votre adresse mail complète pour le nom d'utilisateur et votre mot de passe.  

}}}Exemple d'adresse mail complète, tapez :{{{
}}}
myra@exemple.com
myra123{{{

Votre mot de passe apparaît à l’écran sous la forme ******, afin de protéger votre confidentialité.

  1. Afin que l’ordinateur garde en mémoire votre session jusqu'à son expiration ou votre déconnexion, cochez Se rappeler de moi sur cet ordinateur.

Si vous cochez cette option, vous n'aurez pas à vous reconnecter chaque fois que vous relancez le navigateur pendant la journée. }}}Lorsque vous sélectionnez cette option et que vous ne vous déconnectez pas, votre session reste active jusqu’à l’expiration de la session.{{{ L’administrateur système détermine la durée d’activité d’une session.

}}}Remarque :{{{ Si vous laissez votre client Web ouvert dans votre navigateur mais que vous ne consultez pas votre messagerie pendant un certain laps de temps, il est possible que votre session expire. Vous devrez alors vous reconnecter.

  1. Si vous ne voulez pas utiliser le type de client configuré par défaut, modifiez la version affichée.

  2. Cliquez sur Connexion.

 

Pour se déconnecter :

Afin d’empêcher que d’autres utilisateurs ne se connectent à votre compte de messagerie, vous devez toujours utiliser le lien Déconnexion pour fermer votre session Zimbra.
