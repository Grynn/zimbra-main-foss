Joindre des fichiers à votre mail

Les mails peuvent inclure des pièces jointes. Vous pouvez joindre des documents, des feuilles de calcul, des photos, des diaporamas et d’autres types de fichiers.

Pour joindre un fichier à un mail :

  1. Rédigez le mail comme décrit dans la section Création d’un mail.

  2. Cliquez sur Ajouter pièces jointes. Une nouvelle fenêtre apparaît, vous permettant de télécharger jusqu’à 10 pièces jointes.

  3. Pour uploader un fichier, cliquez sur Parcourir... pour localiser le fichier.

  4. Sélectionnez le fichier puis cliquez sur Ouvrir.

  5. Uploadez des pièces jointes supplémentaires en appliquant le même procédé.

  6. Pour afficher les images que vous avez téléchargées dans le corps du mail, cochez la case Afficher les images dans le corps des mails.  

  7. Une fois toutes les pièces jointes ajoutées, cliquez sur Joindre. Les pièces jointes sont répertoriées sous la ligne d’objet.

  8. Lorsque le mail est prêt à être envoyé, cliquez sur Envoyer pour l'envoyer avec ses pièces jointes.

  • Remarque : Les destinataires de votre mail doivent disposer du logiciel adéquat pour pouvoir ouvrir et lire le fichier. Pour les fichiers créés avec des logiciels d'utilisation courante comme Microsoft Office, les utilisateurs d'un système bureautique classique pourront les ouvrir. Des formats de fichiers courants tels que les fichiers texte, HTML, et image comme .GIF ou .JPG peuvent être ouverts à partir de différents programmes. Néanmoins, cela dépend du type de système de l'utilisateur.

Pour supprimer une pièce jointe, décochez la case à côté de la pièce jointe avant d’envoyer le mail.
