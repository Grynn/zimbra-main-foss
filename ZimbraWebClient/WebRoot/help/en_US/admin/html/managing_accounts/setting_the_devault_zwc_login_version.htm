Setting the Default ZWC Login Version

ZCS offers a standard and an advanced Zimbra Web Client version. Users can select the client to use before they log on. The default ZWC for login is the advanced Zimbra Web Client. When users log in, they view the advanced Zimbra Web Client, unless they use the menu on the login screen to change to the standard version.

You can change the default in the COS Preferences tab to make the standard Zimbra Web Client the default. The following are the two versions.

  • Advanced Web Client includes Ajax capability and offers a full set of Web collaboration features. This Web client works best with newer browsers and fast internet connections.

  • Standard Web Client is a good option when Internet connections are slow or users prefer HTML-based messaging for navigating with their mailbox. If a user's screen resolution is 800 x 600 is detected upon log in, the standard Zimbra Web Client is opened automatically as the default. You can disable this redirect action in the COS and users can set their log in preference to log in to the advanced ZWC in all cases.
