User Preferences

How mailboxes behave when a message is composed or received are preferences that are enabled in the COS. The preferences in the COS become the default settings for accounts.

The COS default preferences can be changed for individual accounts, and users can modify their preferences from the Zimbra Web Client Options tab.  The account's preferences are updated when users change their options in the Web Client.

The following preferences can be configured.

Preference

General

  • Show IMAP search folders

  • Show search string

  • Enable keyboard shortcuts. Allows users to use keyboard shortcut keys. If Keyboard shortcuts aliases is enabled on in Features tab, users can also create their own shortcut keys.

  • Initial mail search, which folder in the mailbox to search first. The default is the Inbox.

  • Display a warning when users try to navigate away from ZCS.  

  • Show selection checkbox for selecting email and contact in a list view for batch operations. This makes it easy for users to select multiple items in the Content pane and then perform an action such as move, delete, forward, tag, or mark as read/unread.

  • Automatically log in instant messages services (beta). If the instant messaging feature is enabled, the  instant messaging feature will be available when users log in.

Mail

  • View mail as HTML (when possible).

  • Display external images in HTML mail. If this enabled, images send in an email are displayed.

  • Group mail by Conversation (default) or Message.

  • Number of items to display per page, 10, 25, 50, or 100. The default is 25.

  • Polling interval. This is the number of minutes to wait before checking for new mail. The default is 5 minutes. Minimum mail polling interval limits the user's setting their polling frequency.

  • Out of office cache lifetime. When out of office reply is enabled, a reply message is sent to received messages only once during the cache lifetime.  The default is one time every 7 days.

  • Save to sent. A copy of messages that are sent from the account are saved to the Sent folder.

  • Allow sending email from any address. Used with Mail Identities to allow users to create different email addresses. If this is not checked users can only select from email addresses or aliases that you configured.

  • Always compose in new window.

  • Reply/Forward using format of the original message.

  • Always compose mail using Text (default) or HTML.

  • Signature style, either Internet standard, with the signature at the end of the message thread or basic, with the signature at the end of the new message.

  • You can configure the signature length. The maximum length of a mail signature is 1024 characters.

Address Book

  • Use GAL when autocompleting addresses.

  • Enable automatic adding of contacts.

  • Number of contacts per page, 10, 25, 50, or 100.  The default is 25.

Calendar

  • Timezone is used to set the default time zone for  new accounts in this COS. The time zone you set during the ZCS install sets the default COS time zone.

  • Number of minutes before an appointment to show reminder. The default is 5 minutes.

  • Always show the mini-calendar. A mini-calendar will always display in the lower left corner of the Web client.

  • Use the QuickAdd dialog when creating new appointments. If the Calendar feature is enabled, users can click on a date to open the QuickAdd dialog to create an appointment.

  • Show timezone list in appointment view. The timezone list will be displayed when creating appointments.
