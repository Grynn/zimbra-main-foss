Account Limits

You can set the following account limits by domain:

  • Maximum number of accounts for a domain

  • Maximum number of  accounts that can be assigned to a specific COS

  • Maximum number of accounts that can be assigned a specific feature

 

How to set up account limits

By default, no limits are set. Note: entering 0 in these fields also means that no limits are set.

  1. Select a domain to edit and open the Accounts Limits tab

  2. Each of the following can be set:

    • Enter the number of accounts that can be assigned to this domain.

    • To set the maximum accounts per COS, click Add and in the Class of Service field enter a COS name. This will start auto-complete that shows all COS.  Select a COS and in the Limit field, enter the maximum number of accounts allowed. Click OK.

    • To set account limits for specific features, in the Maximum accounts per feature section, click Add. Select a feature from the drop-down and set the limit. Click OK.  The Feature limit is counted when a feature is set in an account, overriding the COS setting. For instance, the COS for an account does not enable Instant Messaging. The account is edited to enable Instant Messaging.

  3. Click Save.
