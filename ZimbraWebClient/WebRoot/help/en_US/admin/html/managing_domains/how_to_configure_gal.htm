How to Configure GAL

The Global Address List (GAL) is configured on a per-domain basis.  The GAL Configuration Wizard directs you through the steps to configure GAL

GAL is configured when a new domain is created and can be edited from the Domain toolbar.

To edit the GAL configuration for a domain, select the domain, and click Configure GAL.

 

To configure the GAL setting:

  1. Select the GAL mode, Internal, External, or  Both. See Configure the Global Address List

Enter the value for the maximum results that should be returned from a GAL search. The default is 100.

Note: The maximum results settings can also be set in Global Settings. The value entered in Global Settings becomes the default for domains that do not have this value explicit set.

If Internal is selected, the configuration is complete. Click Finish to save the configuration.

If External or Both is selected, go to step 2.

  1. For External or Both, enter the LDAP type; select either LDAP or Active Directory.

  2. In the LDAP URL: field type the external LDAP URL. The external server port is 389.

    If you check Use SSL, the server port is 636.

  3. To add additional LDAP servers, click Add. A new LDAP URL field displays.

  4. In the LDAP filter: field, enter the search expression that is used.  

  5. The LDAP search base is configured by default. Click Next.

  6. To use DN to bind to the external server, check Use DN/Password to bind to external server.

    Enter the Bind DN and Bind Password. Click Next. The GAL Sync Settings dialog displays. You can set GAL search settings and GAL sync settings separately.

  7. If you want to set your sync settings different from your GAL search settings, uncheck the box Use GAL search settings for GAL sync, and continue to complete the GAL sync settings in this and the next dialog.

    If the GAL Search and GAL sync settings are the same, on the Gal Sync Settings dialog, click Next.

  8. The GAL Settings Summary page displays the information you entered. If the information is correct, to test the GAL settings, enter a name from the directory in the Please provide a search term field. Click Test.

  9. The GAL Test Results page displays the test results. If the results failed, click Previous, to resolve the issues described in the Details text box. Click Finish, if the test past.
