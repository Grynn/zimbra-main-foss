Changing Access Permission for Documents

You can change the access permissions to the domain Documents Notebook from the administration console.

  • You cannot delete the domain Documents account. If the Documents Notebook should not be accessed, delete the access permissions.

To change access to domain Documents account

  1. From Configuration>Domains, select the domain to be edited.

  2. Click the Documents tab. The page displays the list of users and permissions that are set.

  3. Select the user type to edit and click Edit. The Edit Permissions dialog displays.

  4. Make the changes and click OK. The changes are immediately reflected on the page.

  5. Click Save on the toolbar.

To revoke (delete) user access to domain Documents account

  1. From Configuration>Domains, select the domain to be edited.

  2. Click the Documents tab. The page displays the types of users and permissions.

  3. Select the user type to delete and click Delete. The user is immediately deleted.

  4. You cannot delete types defined as Public or as All users in Domain. To revoke their permissions, select the type and click Edit. Remove all checks from the permissions.

  5. Click Save on the toolbar.
