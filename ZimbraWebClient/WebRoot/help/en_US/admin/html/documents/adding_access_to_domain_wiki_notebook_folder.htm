Adding Access to Domain Notebook Folder

You can add users who can access the domain Documents Notebook folder from the administration console.

To add users who can access the domain Documents account

  1. From Configuration>Domains, select the domain to be edited.

  2. Click the Documents tab. The page displays the list of users and permissions that are set.

  3. Click Add. The Add Access dialog displays.

  4. From the drop-down list select type of user you are adding, Groups, Users, or All users in domain.

  5. In the text field, select a user, group, or domain from the list, or type a user's valid email address.

  6. Check the boxes for the type of access allowed.

  7. Click Save on the toolbar.
