About Zimbra Documents

The Zimbra Documents application is a document sharing and collaboration application. The Zimbra Documents feature enables users to create, organize, and share web documents. Images, spreadsheets, and other rich web content objects can be embedded into Documents. External documents can also be uploaded.

The Documents feature consists of a global Documents account, one optional domain Documents account per domain, and Documents notebook folders for individual accounts.

  • The global Documents account is automatically created when ZCS is installed. This account includes the Notebook folder and the Template Notebook folder.  The Template Notebook contains the Documents templates that defines how pages within all the Documents folders are displayed. These cannot be changed.

  • One domain Documents account can be created per domain. This account is created from the administration console. This account includes the Notebook folder which can be used to collect, organize, and share information with users on the domain. Different user groups can be given permissions to view, edit, add, and delete information within the domain Notebook.

  • The Documents application can be enabled from the Class of Service and for individual accounts. When Documents is enabled, one Notebook folder is created for the account.  Users can create additional notebook folders and share any of their folders with others.
