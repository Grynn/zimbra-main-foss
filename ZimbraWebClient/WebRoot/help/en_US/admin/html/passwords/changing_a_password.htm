Changing Passwords from the Admin Console

If you use internal authentication, you can change an account's password quickly from the Accounts toolbar.  Click Change Password and enter the new password. The user must be told the new password to use to log on.

Note: Passwords cannot contain accented characters, such as these examples:  ñ, é, ó, etc.

Must Change password can be enabled for the account.  The user must change the password the next time he logs on.
