Importing your iCal calendar

You can import iCalendars (iCal) calendars that are saved in the .ics format. iCalendars are the standard for calendar data exchange over the Internet, and .ics is the standard format for iCalendar information. Import is available from your Preferences > Import/Export tab.

You can import your iCal calendar to an existing calendar or create a new calendar and import the .ics file to it.

To import an iCal calendar:

  1. Select Preferences and click the Import / Export Folder.

  2. In the Import section's Type field,  select Calendar.

  3. In the File field, browse to locate the .ics file to import.

  4. Click Import. In the Choose folder dialog, select the folder to import the calendar, either select one of your existing calendars or create a new calendar.

  5. Click OK. The calendar is imported into our account.

iCal calendars are updated periodically according to the remote calendar polling intervals set by the system administrator.
