Exporting Address Book Contact Lists

Your  contact lists are exported and saved in a comma-separated (.csv) text file format.

To export your address book:

  1. From the  Options tab, click the Import/Export tab.

  2. In the Export section, select the type as Contacts.

  3. In the File field, browse to select the address book folder to export.

  4. In the Type drop-down, select the contact format you want to export the contact file as:

    • Account contacts. This exports the file as it displays in your account.

    • Yahoo! contacts

    • GMail contacts

    • Outlook 2000 contacts

    • Outlook 2003 contacts

    • Outlook contacts

    • Thunderbird contacts

This formats the contact information in the proper order for that type of contact list.

  1. Click Export.

  2. Select where to save the exported file and click OK. To import them into another contact manager program, see documentation for that program.
