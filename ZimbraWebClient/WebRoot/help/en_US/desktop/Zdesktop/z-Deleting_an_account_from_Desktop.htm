Deleting an account from Desktop

  1. Click Account Setup on the Yahoo! Zimbra Desktop. You can find this link in the right corner.

The My Accounts page opens displaying your accounts.

  1. Click Delete for the account to be deleted. An information dialog displays Account settings and downloaded data will be deleted. Data on the server will not be affected. OK to proceed?

All data is deleted, including email messages, contact lists, calendars, Documents Notebook pages, files saved to Briefcase and data archived to the Local Folder.  Delete removes the files from the computer, not from the server.

  • Files you saved to the Local Folder are only saved in Yahoo! Zimbra Desktop. If you want to save these files, before you delete your account, export the files in the Local Folder. This can be done from the Options>Import/Export tab.
