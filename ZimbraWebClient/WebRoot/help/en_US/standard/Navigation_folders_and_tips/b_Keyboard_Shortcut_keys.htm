Keyboard Shortcuts

Keyboard shortcuts are keys or key combinations that let you quickly select an action when the access key sequence is entered.  

The following is a list of favorite shortcuts.  Go to your Options>Shortcut tab to see the complete list of shortcut keys.

nm  or  c

New message (compose)

gm

Go to your mailbox

go

Go to Options

gc

Go to Calendar

ga

Go to Address Book

Shortcuts used in Mail

o

Open the current email message

d

Delete selected item (moves it to trash)

r

Reply to sender

a

Reply to all recipients of the message

f

forward message

Shortcuts used in Address Book

nc

New contact

ng

New contact group

Shortcuts in Calendar

na

New appointment
