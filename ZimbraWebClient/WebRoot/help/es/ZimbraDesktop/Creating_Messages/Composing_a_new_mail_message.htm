Redactar un nuevo mensaje de correo

Para redactar un nuevo mensaje:

  1. Haz clic en en la barra de herramientas. Se abrirá la página de redacción.

  2. Rellena los campos con la dirección del destinatario, el asunto y el texto del mensaje.

Nota:  Si introduces un nombre y apellido que no están en un formato de dirección válida (nombre@dominio.es), se abrirá un cuadro de diálogo de confirmación para avisarte de que no parece ser una dirección válida. Puedes optar por enviar el mensaje de todas formas aunque no se pueda entregar.

  1. Si quieres recibir una notificación de lectura, haz clic en Opciones y selecciona Solicitar notificación de lectura.

  2. Introduce el texto del mensaje en el campo de redacción debajo del asunto.

Las palabras mal escritas se subrayan de forma automática. Haz clic con el botón derecho del ratón en la palabra subrayada para seleccionar la ortografía correcta.

  1. Para añadir un archivo adjunto, haz clic en y localiza el archivo, o arrástralo y suéltalo desde tu ordenador hasta el encabezado del mensaje.

  2. Añadir tu firma. Si tienes una configuración diferente para cada cuenta, en el menú desplegable de Cuentas, selecciona las direcciones de correo electrónico para este mensaje. Si has definido un estilo de firma para la cuenta, haz clic en Firma en la barra de herramientas de redacción para añadir esa firma.

  3. Pulsa en Enviar para enviar tu mensaje.

Si no quieres guardar el mensaje en ese momento, pulsa en Guardar borrador. Se guardará el mensaje en la carpeta Borradores.
