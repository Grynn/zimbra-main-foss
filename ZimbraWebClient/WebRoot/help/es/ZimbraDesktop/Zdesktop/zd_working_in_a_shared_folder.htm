Trabajar con una carpeta compartida desde Zimbra Desktop

Nota: Disponible únicamente en las cuentas de Zimbra.

Al aceptar compartir una carpeta de cesionario, verás esta carpeta en el panel de resumen de Zimbra Desktop.  Si estás conectado a Internet  en Zimbra Desktop,  podrás ver el contenido de esa carpeta. En realidad, no se guarda ese contenido ni en tu buzón ni en tu ordenador.

Cuando el otorgante comparte una de sus carpetas, puede seleccionar uno de los siguientes niveles de acceso:

  • Espectador. El cesionario podrá leer el contenido de la carpeta, pero no podrá realizar ningún cambio.

  • Controlador. El cesionario tiene permiso para ver, modificar el contenido de una carpeta, crear nuevas subcarpetas, enviar mensajes de tu parte y eliminar mensajes.

  • Administrador. El cesionario tiene permiso total para ver, modificar el contenido de una carpeta compartida, crear nuevas subcarpetas, enviar mensajes de tu parte, eliminar mensajes y compartir tus carpetas con otros.

Rol de Controlador o Administrador de las carpetas de correo

  • Controlador. El cesionario tiene permiso para ver, modificar el contenido de la carpeta, crear subcarpetas, enviar mensajes de tu parte y eliminar mensajes.

  • Administrador. El cesionario tiene permiso para ver, modificar el contenido de la carpeta, crear subcarpetas, enviar mensajes de tu parte, eliminar mensajes y compartir tus carpetas con otros.

Se guardan copias de los mensajes que se envían en el buzón de correo del dueño de la carpeta y si eliminas un mensaje de su buzón o lo señalas como spam, se moverá a la Papelera o a la carpeta Spam del dueño.

Si tienes acceso a más de una carpeta del dueño del buzón, podrás mover el contenido de una carpeta a otra dentro de ese buzón.  Puedes copiar un mensaje de correo de las carpetas del dueño a una de tus carpetas de correo; se dejará una copia del mensaje en la carpeta del dueño del buzón.

Puedes crear subcarpetas en la carpeta de correo compartida y esas nuevas subcarpetas se verán en el buzón del dueño.

Rol de Controlador o Administrador de contactos

  • Controlador. Tendrá pleno permiso para ver, modificar los contactos, añadir nuevos contactos y eliminarlos.

  • Administrador. El cesionario tendrá pleno permiso para ver, modificar los contactos, añadir nuevos contactos, eliminarlos y compartir la lista de contactos con otros usuarios.

Rol de Controlador o Administrador de las agendas

  • Controlador. El cesionario tiene permiso para crear reuniones, aceptar o rechazar invitaciones y modificar y eliminar las actividades que ya figuran en tu agenda.

  • Administrador. El cesionario tiene pleno permiso para crear reuniones, aceptar o rechazar invitaciones y modificar y eliminar las actividades que ya figuran en tu agenda, compartir tu agenda con otros y cambiar los permisos de acceso de otros a tu agenda.

Rol de Controlador o Administrador para las listas de tareas

  • Controlador. El cesionario tendrá permiso para crear tareas y editar o eliminarlas de tu lista de tareas.

  • Administrador. El cesionario tiene permiso para crear tareas, modificar y eliminarlas de tu lista de tareas y compartir esta lista con otros.

Rol de Controlador o Administrador para el Bloc de notas

  • Controlador. El cesionario tiene permiso para ver y modificar el contenido de la carpeta de blocs de notas, añadir nuevas páginas o eliminar otras.

  • Administrador. El cesionario tiene permiso para ver y modificar el contenido de la carpeta de blocs de notas, añadir nueva información, eliminar tus blocs y compartirlos con otros.

Rol de Controlador o Administrador de las carpetas del maletín

  • Controlador. El cesionario tiene permiso pleno para ver, modificar y eliminar los archivos de tu maletín.

  • Administrador. El cesionario tiene permiso para ver, modificar y añadir nueva información a una carpeta del maletín, eliminar una carpeta y compartirla con otros.
