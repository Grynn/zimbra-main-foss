Configurar las alertas y los recordatorios de Desktop

Puedes crear alertas para notificarte mensajes nuevos, recordatorios de tus citas y notificaciones de nuevos mensajes instantáneos. Para recibir estas alertas, tu buzón de correo debe estar abierto.

  • Alertas de mensajes. Configura estas alertas en la carpeta Preferencias>Correo. Estas son las opciones para las alertas:

    • Reproducir un sonido. Escucharás un pitido cuando recibas un mensaje nuevo en tu bandeja de entrada.  Si has silenciado el volumen, no escucharás esta alerta. Se requiere tener QuickTime o Windows Media instalado en el ordenador.

    • Resaltar la pestaña Correo. Si no estás trabajando en la pestaña Correo, se resaltará esa pestaña al recibir un nuevo mensaje en tu bandeja de entrada.

    • Mostrar una notificación emergente.  Al recibir un mensaje nuevo, se abrirá una ventana emergente. Debes disponer de Yahoo! BrowserPlus instalado.

  • Recordatorios de tus citas.  Configura estas alertas en la carpeta Preferencias>Agenda.  Para poder configurar una alerta, debes indicar cuándo notificarte la reunión. Puede ser hasta 60 minutos antes de la reunión. Verás un recordatorio en el momento indicado antes del comienzo de la reunión. Si configuras esta opción como Nunca, no recibirás ningún recordatorio incluso si has habilitado la opción de recordatorios.

El recordatorio de ZWC no se abrirá en la parte superior de la página si no estás trabajando con la agenda de ZWC. Por lo contrario, activar una de las siguientes opciones ayuda a notificarte el recordatorio emergente:

    • Reproducir un sonido. Se te notificará mediante un pitido al abrirse un recordatorio de una cita. Si has silenciado el volumen, no escucharás esta alerta.

    • Título del navegador intermitente. Al abrirse un recordatorio, parpadeará el navegador hasta que cierres la ventana emergente.
