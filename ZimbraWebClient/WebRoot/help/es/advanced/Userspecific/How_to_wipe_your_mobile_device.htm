Cómo borrar los datos de tu dispositivo móvil

Nota: La función para móviles se encuentra disponible únicamente con la @CollaborationProductFullName@ Network Edition

Si pierdes o te roban el dispositivo móvil, puedes borrar todo el contenido del dispositivo mediante el enlace Limpiar dispositivo en la página de Preferencias para Dispositivos móviles. Este comando borrará todos los datos del dispositivo para restablecer la configuración original del fabricante.

  1. Haz clic en Preferencias y selecciona la carpeta Dispositivos móviles

  2. Selecciona el dispositivo en cuestión y haz clic en Limpiar dispositivo.  

El estado cambiará a Borrado pendiente.  El servidor está esperando a que el dispositivo se conecte al servidor  para iniciar la operación.  Cuando el dispositivo se conecte al servidor, se borrarán todos los datos.  El estado cambiará a Borrado finalizado.

Nota:  Puedes cancelar tu solicitud de borrar los datos del dispositivo mientras el estado sea Borrado pendiente.
