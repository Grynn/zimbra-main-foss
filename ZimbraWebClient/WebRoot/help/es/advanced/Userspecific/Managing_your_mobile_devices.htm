Gestionar tus dispositivos móviles

Nota: La función para móviles se encuentra disponible únicamente con la @CollaborationProductFullName@ Network Edition

Si has habilitado la función ZCS para móviles, verás la carpeta Dispositivos móviles en la pestaña Preferencias. En esta carpeta se muestran los dispositivos móviles configurados para sincronizarse con tu cuenta ZWC.

En esta carpeta, puedes

  • Iniciar una sincronización desde ZWC a tu dispositivo

  • Suspender una sincronización

  • Eliminar dispositivos móviles que ya no utilizas.

Además, si pierdes o te roban el dispositivo, puedes borrar todos los datos del dispositivo.

Funciones de seguridad para móviles

Es posible que tu administrador haya configurado una política de seguridad para móviles que requiera crear un número de identificación personal (PIN) en tu dispositivo móvil. Si olvidas tu PIN, ponte en contacto con el administrador del sistema.

Si dispones de un PIN, es posible que la política esté configurada con una función para bloquear a los usuarios si no introducen el PIN correcto.  El administrador del sistema configura el ajuste de bloqueo del PIN para que si alguien intenta iniciar sesión en un dispositivo con una contraseña incorrecta un número de veces superior al configurado, se llevará a cabo un borrado iniciado localmente del dispositivo. Esta operación borrará todos los datos del dispositivo. Sólo el administrador puede determinar la configuración de la política que no puede ser modificada por el usuario.  Solicítale al administrador del sistema más información.
