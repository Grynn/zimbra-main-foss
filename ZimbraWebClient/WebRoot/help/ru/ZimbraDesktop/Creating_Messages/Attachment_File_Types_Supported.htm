xxxxПоддерживаемые типы файлов вложенийxxxx

Любой вложенный файл можно открыть прямо из Zimbra Web Client, если на компьютере установлено соответствующее приложение и открытие файлов этого типа не заблокировано системным администратором.  Например, чтобы открыть документ, созданный с помощью Microsoft Word, вы должны иметь возможность запустить Microsoft Word на своем компьютере.

xxxxНа большинстве компьютеров тип файла обозначается расширением в виде точки с последующим трехбуквенным кодом.xxxx В качестве примера можно взять файлы расходы.xls илиНовые_позиции.doc. xxxxЕсли тип файла поддерживается установленным на компьютере программным обеспечением, можно просто дважды щелкнуть этот файл и компьютер автоматически запустит нужное приложение для его чтения.xxxx

xxxxФайлы многих типов можно открывать непосредственно в браузере с помощью различных подключаемых модулей.xxxx xxxxОбычно, если щелкнуть на странице ссылку на файл, браузер автоматически определит, установлен ли необходимый подключаемый модуль, и предложит загрузить его.xxxx xxxxИмейте в виду, что политика администрирования системы может запрещать конечным пользователям загружать и устанавливать какое-либо программное обеспечение.xxxx

Примечание.
На компьютерах с ОС Windows предполагается, что тип файла, указанный в расширении файла, совпадает с действительным форматом файла. xxxxДаже если у файла правильный внутренний формат, но нет правильного расширения в имени файла, компьютер может не распознать его и не откроет нужное приложение.xxxx

xxxx Постоянно растущее количество компьютерных вирусов создается для распространения с помощью вложенных файлов.xxxx Поэтому в некоторых средах системные администраторы вынуждены блокировать входящую почту, содержащую вложения определенных типов, в особенности файлы с расширениями .EXE или .ZIP в имени файла. xxxxПри отправке сообщения пользователю, система электронной почты которого настроена на блокировку вложений определенных типов, отправитель может получить или не получить извещение о блокировке.xxxx

xxxxДля справки ниже приведены некоторые широко используемые расширения файлов.xxxx

  • .TXT — текстовый файл, т. е. файл, содержащий только текстовую информацию. Текстовые файлы можно открывать и читать с помощью большого количества текстовых редакторов, включая Блокнот, WordPad, Word, редакторы командной строки, браузеры и даже программы обработки электронных таблиц.

  • .HTM — HTML-файл, который является особым типом текстового файла, обычно открываемого с помощью браузера.

  • xxxx.ZIP xxxxxxxx—xxxxxxxx файл, который был сжат с помощью ZIP-сжатия.xxxx Необходима программа WinZip или другая программа сжатия, которая может читать и распаковывать ZIP-файлы.

  • .EXE — исполняемый или программный файл. xxxxДважды щелкнув мышью такой файл обычно можно запустить приложение.xxxx

  • xxxx.DOC xxxxxxxx—xxxxxxxx файл документа Microsoft Word.xxxx

  • .JPG — графическое изображение в формате JPEG. Графические изображения в формате JPEG можно открывать в браузере или с помощью одной из множества программ для работы с изображениями, например Paint или Adobe Photoshop.

  • xxxx.GIF xxxxxxxx—xxxxxxxx графическое изображение в формате GIF.xxxx Графические изображения в формате GIF можно открывать в браузере или с помощью программ для работы с изображениями.

  • .MPG — видеоклип или анимация в формате MPEG. xxxxЗачастую воспроизводится с помощью подключаемого модуля браузера или программами воспроизведения, такими как Windows Media Player.xxxx

  • .WMV — видеоклип в формате WMV, который поддерживается только в ОС Windows.

  • .PDF — документ в формате Acrobat Reader. xxxxОбычно он открывается в браузере с помощью подключаемого модуля.xxxx
