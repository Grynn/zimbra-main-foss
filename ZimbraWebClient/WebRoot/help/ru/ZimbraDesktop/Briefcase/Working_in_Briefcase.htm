xxxxРабота в портфелеxxxx

Портфель позволяет сохранять файлы в учетной записи, поэтому доступ к этим файлам можно получить с любого компьютера, войдя в свой почтовый ящик.

В портфель можно загружать документы, электронные таблицы, презентации, изображения, pdf-файлы с компьютера или из сети. Кроме того, в нем можно сохранять вложения почтовых сообщений.  В портфеле можно создавать разные папки для систематизации файлов,.

xxxxВ портфеле хранятся копии исходных файлов.xxxx  Изменения, вносимые в файлы в портфеле, не сохраняются в исходных файлах.

Загрузка файлов

В процессе загрузки файлы передаются с компьютера в почтовый ящик. xxxxЭто позволяет получить доступ к файлам в любое время, для этого нужно только войти в почтовый ящик.xxxx Количество файлов, которое можно загрузить в портфель, определяется лимитом учетной записи.

xxxxЗагрузка файловxxxx

  1. Откройте портфель и на панели инструментов щелкните Загрузить файл.

  2. В диалоговом окне Загрузить новый файл в портфель нажмите кнопку Обзор, чтобы найти файлы для загрузки. Можно выбрать несколько файлов для одновременной загрузки, если эти файлы расположены в одной папке. Для добавления других файлов нажмите кнопку Добавить и найдите необходимые файлы.

Сортировка файлов

Если выбран подробный вид отображения портфеля, то содержащиеся в нем файлы можно сортировать по имени, размеру и дате. При изменении вида порядок сортировки сохраняется.

Общий доступ к портфелю

Доступ к папкам портфеля может быть предоставлен другим пользователям внутри и за пределами организации. Другим пользователям можно предоставить разрешение на создание и редактирование содержимого в портфеле или только на просмотр страниц. См. раздел «Общий доступ к папкам Zimbra».
