Добавление задачи

Новую задачу можно создать, введя имя задачи в списке задач, или открыв страницу задач и добавив дополнительные данные о задаче, включая заметки и вложения.  После открытия страницы задачи отобразится вкладка «Задача» на панели инструментов «Приложение», а также в области содержимого откроется форма «Задача». На панели инструментов «Приложение» могут быть открыты несколько вкладок «Задача», между которыми можно переключаться.

Быстрое добавление задачи

  1. Выберите список задач в области просмотра

  2. В области содержимого щелкните верхнюю строчку со словами Щелкните здесь, чтобы добавить новую задачу.    

  3. Появится текстовое поле. Введите имя задачи.

  4. Нажмите клавишу Enter. Задача будет добавлена в ваш список задач.

Примечание. Если вы по ошибке создали задачу не в том списке, откройте задачу и выберите нужный список в раскрывающемся списке задач, или перетащите задачу в нужный список в области содержимого.

Добавление задачи с подробностями

Списки задач можно использовать для создания задачи и отслеживания ее выполнения. Кроме того, к задаче можно добавить заметки и вложить файлы для простого доступа в пределах задачи.

Оценить длительность проекта можно с помощью даты начала и окончания проекта и установки приоритета. Начиная выполнение задачи, можно выбрать для нее статус, Не началась или Выполняется, и указать процент выполнения.

  1. Выберите папку списка задач в области обзора.

  2. На панели задач «Задачи» нажмите кнопку Создать.  Откроется страница задачи без названия, а на панели инструментов отобразится вкладка «Задача».

  3. В разделе Детали

    • Введите имя задачи в поле «Тема». Если требуется, введите расположение в поле «Место».

    • Выберите Приоритет - обычный, низкий или высокий.

    • Если нажать кнопку «Создать» до выбора нужного списка задач, можно выбрать соответствующий список задач в раскрывающемся списке Список задач.

  4. В разделе Выполнение укажите значения в полях «Дата начала» и «Срок». Если выполнение задачи уже началось, можно заменить состояние Не начато одним из пунктов списка и обновить процент выполнения задачи.

  5. Установите дату и время, а также, в случае настройки адреса электронной почты на странице Настройки>Напоминания, установите флажок Отправить по почте, чтобы получать напоминание по электронной почте автоматически в указанный день и время.

  1. Добавьте заметки в текстовом поле, чтобы добавить вложения, нажмите на панели задач кнопку Добавить вложения.

  2. Нажмите кнопку Сохранить, чтобы сохранить задачу. Новая задача будет добавлена в список задач.
