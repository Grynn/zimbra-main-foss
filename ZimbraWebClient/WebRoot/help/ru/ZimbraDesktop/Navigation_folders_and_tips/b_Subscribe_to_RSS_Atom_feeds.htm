xxxxПодписка на RSS/Atom-лентыxxxx

xxxxМожно подписаться на веб-сайт, предоставляющий новости RSS-ленты (Really Simple Syndication) и подкасты, и обновления будут поступать прямо в ваш почтовый ящик.xxxx

xxxxСоздайте для каждой RSS-ленты папку.xxxx

xxxxПрежде чем продолжить, узнайте URL-адрес RSS-ленты.xxxx

  1. В области обзора над заголовком Папки щелкните ссылку Новая папка.

  2. xxxxВ поле xxxxxxxxИмяxxxxxxxx введите имя папки для RSS-ленты.xxxx

  3. xxxxУстановите флажок xxxxxxxxПодписаться на RSS-ленту/Atom-лентуxxxxxxxx.xxxx

  4. xxxxВ меню папок выберите папку для добавления RSS-ленты.xxxx xxxxЕсли выбрать корневую папку, будет создана папка, отображаемая на самом высоком уровне на виде папок, т.е. на одном уровне с папкой «Входящие».xxxx

  5. xxxxВ поле xxxxxxxxURL-адресxxxxxxxx введите URL-адрес ленты новостей.xxxx

  6. xxxxНажмите кнопку xxxxxxxxОКxxxxxxxx.xxxx

Значок RSS-ленты отображается в виде папки в области обзора. xxxxОткройте папку RSS-ленты, чтобы открыть ссылку.xxxx

RSS-лента автоматически обновляется каждые 12 часов. Чтобы обновить содержимое, откройте папку RSS-ленты и щелкните ссылку Загрузить ленту.
