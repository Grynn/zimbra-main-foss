xxxxНазначение тегов сообщениям и другим элементамxxxx

xxxxНазначение тегов подобно установке флагов, за исключением того, что можно создать и применить несколько тегов для одного элемента для его систематизации одновременно по нескольким категориям.xxxx xxxxПосле xxxxxxxxсоздания тегаxxxxxxxx, его можно применить к разговорам, сообщениям или контактам.xxxx

Примечание. Теги создаются в пределах учетной записи, с которой вы работаете. К ним не имеют доступа другие настроенные вами учетные записи.

xxxxПометка тегом элемента с помощью перетаскиванияxxxx

xxxxПеретащите тег из области обзора слева на элемент в области содержимого.xxxx

xxxxПрименение тега к элементу с помощью менюxxxx

  1. xxxxЩелкните правой кнопкой мыши элемент в области обзора.xxxx

  2. xxxxВыберите пункт xxxxxxxxПометить сообщениеxxxxxxxx (или xxxxxxxxПометить разговорxxxxxxxx или xxxxxxxxПометить контактxxxxxxxx).xxxx

  3. xxxxВыберите имя тега из открывшегося списка.xxxx xxxxВ списке отображаются только те теги, которые не применены к выбранному элементу.xxxx

xxxxСоздание нового тега и применение его к элементуxxxx

  1. xxxxЩелкните правой кнопкой мыши элемент из списка и выберите вариант меню «Тег».xxxx

  2. xxxxВыберите пункт xxxxxxxxНовый тегxxxxxxxx.xxxx

  3. xxxxВведите имя тега и нажмите кнопку xxxxxxxxOKxxxxxxxx.xxxx

xxxxУдаление тега из элементаxxxx

  1. xxxxЩелкните правой кнопкой мыши элемент из списка и выберите вариант меню «Тег».xxxx

  2. xxxxВыберите пункт xxxxxxxxУдалить тегxxxxxxxx.xxxx xxxxЕсли элемент содержит несколько тегов, можно удалить один или все теги.xxxx

xxxxОдновременное применение и удаление тега для нескольких элементовxxxx

  1. xxxxУдерживая нажатой клавишу Ctrl или Shift, выберите все элементы, которые необходимо пометить тегом.xxxx

  2. xxxxЩелкните правой кнопкой мыши любой из выбранных элементов и выберите вариант меню «Тег».xxxx

  3. xxxxВыберите тег для удаления или добавления.xxxx

xxxxБыстрый поиск помеченных тегами сообщенийxxxx

xxxxЩелкните тег в области обзора.xxxx

xxxxИли можно выполнить поиск тега.xxxx В поле Поиск введите запрос в виде тега: <"имя тега"> и нажмите кнопку Поиск. xxxxРезультатом запроса xxxxxxxxtag:Фиолетовыйxxxxxxxx будут все элементы, к которым был применен тег с именем «Фиолетовый».xxxx
