Изменение повторяющихся встреч

Если вы создали повторяющиеся встречи, можно изменить и удалить одну из них или весь ряд собраний. Участникам будет отправлено сообщение электронной почты.

Изменение собрания

  1. Щелкните правой кнопкой мыши собрание, которое следует изменить.

  2. Выберите, что необходимо открыть, — только выбранную дату, экземпляр или весь ряд. Откроется страница Детали встречи.

  3. Внесите необходимые изменения. Можно изменять расписание, участников, добавлять пояснения в поле Текст и добавлять вложения.

Если о внесённых изменениях не следует уведомлять пользователей, снимите флажок Отправить почтовые уведомления. Параметр «Отправить почтовые уведомления» позволяет вносить изменения в собрание без уведомления всех участников.

  1. Нажмите кнопку ОК. Участникам будет отправлено сообщение электронной почты.

Удаление одного собрания из ряда

  1. Щелкните правой кнопкой мыши встречу, которую следует удалить, и выберите пункты Копия > Удалить копию, чтобы удалить только это собрание.

  2. Нажмите кнопку ОК.

Удаление ряда повторяющихся собраний

  1. Щелкните правой кнопкой мыши встречу, которую следует удалить, и выберите пункты Ряд > Удалить ряд, чтобы удалить повторяющееся собрание.  

  2. Выберите из двух параметров:

    • отменить весь ряд, чтобы удалить все ссылки на собрание и для прошедших, и для будущих дат;

    • отменить только выбранную копию и все последующие за ней, чтобы удалить все ссылки на собрание, начиная с этой даты и до даты окончания повторяющегося собрания.  Предыдущие собрания в этом ряду остаются в ежедневнике.

  3. Нажмите кнопку ОК.

  4. Участникам будет отправлено сообщение электронной почты, а повторяющаяся встреча будет удалена из их ежедневников.
