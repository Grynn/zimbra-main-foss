xxxxПоддерживаемые условия и действия фильтровxxxx

xxxxРегистр букв правил фильтра не имеет значения, т.е. в правилах не учитываются заглавные буквы.xxxx

xxxxУсловияxxxx

xxxxПоля сравнения включают следующие данные.xxxx

  • xxxxОт.xxxx Используется для указания имени в заголовке сообщения «От».

  • xxxxКому.xxxx Используется аналогично параметру «От», но выполняется поиск указанных имен в заголовке «Кому».

  • Копия. Используется аналогично параметру «От», но поиск выполняется в заголовке «Копия».

  • Кому или Копия. Используется аналогично параметру «От», но поиск выполняется в заголовках «Кому» и «Копия».

  • xxxxТема.xxxxxxxx Указываются данные из заголовка «Тема».xxxx

  • xxxxИменованный заголовок.xxxxxxxx При выборе этого параметра появляется дополнительное поле ввода текста перед оператором сравнения (часть «содержит»).xxxx xxxxЭтот параметр позволяет указывать любой заголовок сообщения.xxxx Можно указать не только стандартные поля «Кому» или «От», но также поля «Дата», «Обратный адрес» и другие специальные поля, которые могут содержаться в заголовке сообщения. Этот параметр можно использовать для фильтрации сообщений с «неправильными» заголовками, т. е. заголовками, которые не содержат стандартную для электронных сообщений информацию. xxxxИногда в спаме, который генерируется автоматически, отсутствует информация в таких полях, как «Отправитель» или «Получатель».xxxx xxxxДля указания проверяемого заголовка используется второе текстовое поле (поле, которое находится справа от оператора сравнения).xxxx

  • xxxxРазмер.xxxxxxxx Этот параметр используется для указания сообщения, размер которого больше или меньше указанного.xxxx xxxxРазмер сообщения включает любые файловые вложения.xxxx xxxxЭтот параметр можно использовать для удаления сообщений слишком большого размера.xxxx

  • xxxxДата.xxxxxxxx Этот параметр используется для указания сообщения, отправленного до или после определенной даты.xxxx

  • xxxxТело письма.xxxxxxxx Параметры для тела письма «содержит» или «не содержит» позволяют указать соответствующие слова в теле письма.xxxx xxxxУстанавливать фильтры для слов в файловых вложениях нельзя.xxxx

  • Подтверждение прочтения.Возможна фильтрация сообщений с подтверждением прочтения.

  • xxxxВложение.xxxxxxxx Можно установить фильтр для сообщений, которые содержат или не содержат файловые вложения.xxxx

  • xxxxАдрес в.xxxxxxxx Проверка наличия адреса в контактах.xxxx xxxxВ следующем поле можно указать, какой адрес.xxxx

  • Приглашение из ежедневника. Возможна фильтрация сообщений, имеющих отношение к ежедневнику.

xxxxОператоры сравнения включают следующие данные.xxxx

  • xxxxТочно соответствует/не точно соответствуетxxxxxxxx.xxxx xxxxУказывает точное соответствие.xxxx Например, если в поле Тема точно соответствует указать Яблоко, то соответствием будет «Яблоко», но не «Яблоки» или «Грузовик с яблоками».

  • xxxxСодержит/не содержитxxxxxxxx.xxxx Этот параметр указывает, что строка темы должна содержать указанную подстроку. Например, если указать Тема содержит «яблоко», то соответствием может быть «Я хочу яблоко».

  • Соответствует/не соответствует условию поиска по шаблону. xxxxПодстановочный знак звездочка (*) используется в поле сравнения почтового фильтра для подстановки одного или более знаков фильтра.xxxx Например, если указать, что тема должна соответствовать слову «яблоко*», то соответствиями будут «яблоки», «яблоки-пироги», но не «мои яблоки».

xxxxДействияxxxx

  • xxxxХранить в папке «Входящие».xxxxxxxx Сохраняет сообщения в папке «Входящие».xxxx xxxxЕсли почтовое сообщение не соответствует ни одному из правил фильтра, это действие выполняется по умолчанию.xxxx

  • xxxxЗаписать в папку.xxxxxxxx Перемещает сообщения в указанную папку.xxxx

  • xxxxОтклонить.xxxxxxxx Удаляет сообщение, не доставляя его.xxxx xxxxСообщения нет ни в одной папке, даже в Корзине.xxxx

  • xxxxПереслатьxxxxxxxx.xxxx Сообщения перенаправляются на указанный адрес.

  • xxxxПометить.xxxxxxxx Выберите вариант xxxxxxxxПрочитанноеxxxxxxxx или xxxxxxxxС флагомxxxxxxxx.xxxx

  • xxxxПометить тегом.xxxxxxxx Соответствующие сообщения можно пометить указанным тегом.xxxx

xxxxНесколько действийxxxx

Можно создать правило фильтра, состоящее из нескольких действий. Сочетания действий в правиле трактуются следующим образом.

  • xxxxОтклонить.xxxxxxxx В сочетании с другими действиями этот параметр игнорируется, выполняются другие действия.xxxx

  • xxxxЗаписать в папку.xxxxxxxx В результате нескольких действий «файл в» несколько копий сообщения сохраняются в разных папках.xxxx xxxxЕсли указанная папка не существует, сообщение сохраняется в папке «Входящие».xxxx

  • xxxxТег/пометить.xxxxxxxx Эти действия применяются к сообщению, к которому применялось последнее действие.xxxx xxxxВ случае нескольких действий «файл в» какие-то копии сообщений сохраняются с тегами, какие-то xxxxxxxx—xxxxxxxx без.xxxx

  • xxxxХранить в папке «Входящие».xxxxxxxx Можно определить несколько действий «сохранить», но при этом только одна копия сообщения будет сохраняться в папке «Входящие».xxxx

  • Переслать по адресу. xxxxСообщения будут пересылаться по указанному адресу.xxxx
