  2. Чтобы сохранить профиль поиска, щелкните значок на панели Поиск. xxxxОткроется xxxxxxxxдиалоговое окно xxxxxxxx«Сохранить профиль поиска».xxxx

  3. xxxxВведите имя для сохраненного профиля поиска.xxxx xxxxПрофили поиска сохраняются по умолчанию в виде папки поиска в разделе xxxxxxxxПрофили поискаxxxxxxxx в области просмотра, но можно выбрать другую папку.xxxx

  4. xxxxНажмите кнопку xxxxxxxxОКxxxxxxxx.xxxx xxxxПрофиль поиска сохранен.xxxx
