Поиск типа AND и OR

Для поиска с использованием нескольких критериев, можно найти элементы соответствующие одному или всем указанным критериям. Можно выполнить оба типа поиска с использованием функции расширенного поиска.

Для всех областей поиска, кроме обычного поиска, правила следующие.

  • Поиск сообщений, соответствующих любым указанным критериям, называется поиском с условием OR (ИЛИ), потому что, если сообщение содержит X или Y, оно считается соответствующим условиям поиска. Для панелей расширенного поиска с несколькими установленными флажками выбор нескольких вариантов на одной панели создает поиск для этих элементов с условием OR.

  • Поиск сообщений, которые содержат X и Y, называется поиском с условием AND (И), потому что сообщение должно соответствовать всем указанным критериям, чтобы считаться соответствием. Для панелей расширенного сервера с установленными флажками открытие нескольких копий одной и той же панели и установка разных флажков на каждой из них делает необходимым определение критериев поиска как поиска с условием AND.

Например, предположим, вы ищете сообщения, содержащие файлы формата Adobe PDF или изображения формата JPEG. На одной и той же панели «Вложения» нужно установить оба эти флажка.

Примечание. В запросе появится только «OR». Если задан параметр показывать запрос поиска на панели поиска по мере выбора в поле «Расширенный поиск», текстовое окно поиска будет обновляться и отображать полученный запрос. В профиле поиска типа AND слово AND не отображается.
