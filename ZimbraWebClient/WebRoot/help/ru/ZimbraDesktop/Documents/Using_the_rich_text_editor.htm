xxxxИспользование форматированного текстаxxxx

Можно создавать сообщения электронной почты и документы в портфеле, используя редактор форматированного текста для оформления содержимого на странице.

При редактировании с помощью редактора форматированного текста панель инструментов позволяет легко форматировать разметку страницы. xxxxВыберите размер, гарнитуру и цвет шрифта, стиль текста и цвет фона.xxxx xxxxЭти инструменты форматирования работают также, как и многие другие инструменты обработки текста.xxxx Кроме того, на страницу можно добавлять таблицы, вложения и изображения.

xxxxДобавление стилей абзацаxxxx

Значок  на панели инструментов отображает доступные стили абзаца. Эти стили позволяют легко форматировать текст и сохранять единообразие стиля в документе. Стили включают: Обычный стиль для большинства абзацев, Заголовки шести уровней для определения заголовков разделов, Адрес для выделения текста курсивом и Форматированный для отображения текста шрифтом Courier. Эти стили нельзя изменить, но можно использовать возможности редактирования для изменения шрифта, размера, цвета и т. п.

xxxxДобавление таблиц на страницуxxxx

xxxxДобавление таблиц на страницу выполняется в три этапа.xxxx xxxxДобавление таблицы, задание свойств таблицы и добавление содержимого.xxxx Чтобы добавить таблицу, щелкните значок таблицы на панели инструментов, выберите пункт «Вставить таблицу», а затем укажите, сколько строк и столбцов должно быть в таблице.

xxxxДобавление вложений и изображений на страницуxxxx

Можно загрузить файлы, на которые есть ссылка в файле документа. Щелкните значок скрепки или значок изображений, чтобы найти файл для загрузки. Появится уведомление, если загружаемый файл уже есть в портфеле.

Создание ссылки на веб-страницу

Можно создать ссылку на внешний URL-адрес и дать ей описательный заголовок, который будет отображаться вместо URL-адреса. Щелкните значок и введите адрес страницы или URL-адрес, на который следует создать ссылку.
