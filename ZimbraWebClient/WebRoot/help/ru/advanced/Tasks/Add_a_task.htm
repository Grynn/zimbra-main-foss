Добавление задачи

Новую задачу можно создать, введя имя задачи в списке задач, или открыв страницу задач и добавив дополнительные данные о задаче, включая заметки и вложения.

  • Для быстрого добавления задачи во время работы с почтовым сообщением, собранием в ежедневнике или во время просмотра контакта можно использовать Помощник Zimbra.  После создания задачи вся доступная информация добавляется в область текста задачи.

Быстрое добавление задачи

  1. Выберите список задач в области обзора

  2. В области содержимого щелкните верхнюю строчку со словами Щелкните здесь, чтобы добавить новую задачу.    

  3. Появится текстовое поле. Введите имя задачи.

  4. Нажмите клавишу Enter. Задача будет добавлена в ваш список задач.

  5. Если вы по ошибке создали задачу не в том списке, откройте задачу и выберите нужный список в раскрывающемся списке задач, или перетащите задачу в нужный список в области содержимого.

Добавление задачи с подробностями

Списки задач можно использовать для создания задачи и отслеживания ее выполнения. Кроме того, к задаче можно написать заметки и вложить файлы для простого доступа в пределах задачи. Наличие всей необходимой информации в задаче полезно при предоставлении общего доступа к списку задач.

 Оценить длительность проекта можно с помощью даты начала и окончания проекта и установки приоритета. Начиная выполнение задачи, можно выбрать для нее статус, Не началась или Выполняется, и указать процент выполнения.

  1. Выберите список задач на панели навигации.

  2. На панели задач «Задачи» нажмите кнопку Создать.  Откроется страница задачи без заглавия.

  3. В поле Тема введите имя задачи.

  4. (Необязательно). В разделе Дата выберите Приоритет, Дату начала, Дату окончания и статус.

  5. Если нажать кнопку «Создать» до открытия нужного списка задач, можно выбрать соответствующий список задач в раскрывающемся списке Список задач.  

  6. Добавьте заметки в текстовом поле, чтобы добавить вложения, нажмите на панели задач кнопку Добавить вложения.

  7. Нажмите кнопку Сохранить, чтобы сохранить задачу. Новая задача будет добавлена в ваш список задач.
