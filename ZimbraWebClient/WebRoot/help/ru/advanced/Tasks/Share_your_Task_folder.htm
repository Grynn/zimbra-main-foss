Общедоступный список задач

Списки задач можно сделать доступными для других пользователей и определить их права доступа к общедоступному списку следующими способами.

  • Внутренние пользователи или группы. Можно выбрать один из перечисленных типов предоставляемых полномочий.

    • Наблюдатель. Обладатель этого статуса может читать содержимое списка задач, но не может вносить изменения в его содержимое.

    • Менеджер. Обладатель этого статуса имеет все полномочия создавать задачи, изменять и удалять задачи из списка задач.

    • Нет. Этот параметр временно отключает доступ к списку задач распорядителя без отмены прав общего доступа. Обладатель этого статуса сохраняет за собой список задач, но не может просматривать задачи или управлять ими.

  • Внешние гости. Для доступа к общедоступному списку задач необходимо создать пароль. Для просмотра списка задач гости должны вводить пароль. Они не могут вносить изменения в задачи.

  • Открытый. Все, кто знает URL-адрес общедоступного списка задач, могут его просматривать. Они не могут вносить какие-либо изменения.

Общедоступный список задач

Щелкните правой кнопкой мыши список задач, который необходимо сделать общедоступным и выберите пункт «Общедоступный список задач».

В диалоговом окне «Свойства профиля общего доступа» введите пользователей, которым должен быть предоставлен доступ, и укажите необходимые сведения.

 

Общий доступ для внутренних пользователей и групп

1. Выберите пункт Внутренние пользователи или группы для общего доступа

2. В поле Электронный адрес введите электронные адреса лиц, которым вы предоставляете доступ. По мере ввода адресов отображается список имен, соответствующих вводимым буквам.  Можно выбрать имя из списка.

3. Выберите роль наблюдателя или менеджера.

4. В разделе Сообщение выберите тип сообщения, отправляемого для уведомления получателя статуса.

Нажмите кнопку ОК.

Общий доступ для внешних гостей

1. В разделе Общий доступ с выберите пункт Внешние гости. После этого откроется поле пароля.

2. Введите электронные адреса гостей.

3. Введите пароль, который должны вводить гости для доступа к этому списку задач.

4. Выберите тип уведомления, которое будет им отправлено. Сообщение будет содержать сведения, необходимые для доступа к списку задач.

5. Нажмите кнопку ОК.

 

Общий доступ в режиме открытый

1. В разделе Общий доступ с выберите пункт «Открытый». После этого откроется диалоговое окно свойств общей папки с URL-адресом списка задач.

2. Запомните этот URL-адрес и сообщите его пользователям, обладающих правом доступа в режиме «Открытый». Когда они перейдут по этому URL-адресу, они смогут прочесть содержимое списка задач, но не смогут вносить в него изменения.

При выборе параметра «Открытый» все, у кого есть URL-адрес, могут просматривать список задач.  Пароль не требуется.

Изменение или отмена разрешения

Полномочия можно отменить в любое время.

  1. Щелкните правой кнопкой мыши общедоступный список задач.

  2. Выберите пункт Изменить свойства. Появится список пользователей, имеющих доступ к данной папке.

  3. Щелкните ссылку Изменить для изменения разрешений или
    щелкните ссылку Отменить для отмены разрешений.

  4. Выберите, необходимо ли отправить сообщение с описанием изменений.

  5. Нажмите кнопку ОК.
