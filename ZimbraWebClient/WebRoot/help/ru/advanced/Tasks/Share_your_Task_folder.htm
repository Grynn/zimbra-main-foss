xxxxОбщедоступный список задачxxxx

xxxxСписки задач можно сделать доступными для других пользователей и определить их права доступа к общедоступному списку следующими способами.xxxx

  • xxxxВнутренние пользователи или группы.xxxx xxxxМожно выбрать один из перечисленных типов предоставляемых полномочий.xxxx

    • xxxxНаблюдательxxxxxxxx.xxxx xxxxОбладатель этого статуса может читать содержимое списка задач, но не может вносить изменения в его содержимое.xxxx

    • xxxxМенеджерxxxxxxxx.xxxx Обладатель этого статуса имеет право создавать задачи, изменять и удалять задачи из списка задач.

    • xxxxАдминистраторxxxxxxxx.xxxx Обладатель этого статуса имеет полномочия создавать задачи, изменять и удалять задачи из списка задач и предоставлять доступ к вашему списку задач.

    • xxxxНетxxxxxxxx. Этот параметр временно отключает доступ к списку задач распорядителя без отмены прав общего доступа.xxxx xxxxОбладатель этого статуса сохраняет за собой список задач, но не может просматривать задачи или управлять ими.xxxx

  • xxxxВнешние гости.xxxx xxxxДля доступа к общедоступному списку задач необходимо создать пароль.xxxx xxxxДля просмотра списка задач гости должны вводить пароль.xxxx xxxxОни не могут вносить изменения в задачи.xxxx

  • xxxxОткрытый.xxxx xxxxВсе, кто знает URL-адрес общедоступного списка задач, могут его просматривать.xxxx xxxxОни не могут вносить какие-либо изменения.xxxx

xxxxОбщедоступный список задачxxxx

xxxxЩелкните правой кнопкой мыши список задач, который необходимо сделать общедоступным и выберите пункт «Общедоступный список задач».xxxx

xxxxВ диалоговом окне «Свойства профиля общего доступа» введите пользователей, которым должен быть предоставлен доступ, и укажите необходимые сведения.xxxx

xxxx 3. Выберите роль наблюдателя или менеджера. xxxx
xxxx xxxx
xxxx 4. В разделе xxxxxxxxСообщениеxxxxxxxx выберите тип сообщения, отправляемого для уведомления получателя статуса.xxxx

5. Нажмите кнопку OK.

xxxxОбщий доступ для внешних гостейxxxx

1. xxxxВ разделе xxxxxxxxОбщий доступ сxxxxxxxx выберите пункт xxxxxxxxВнешние гостиxxxxxxxx.xxxx xxxxПосле этого откроется поле пароля.xxxx

2. xxxxВведите электронные адреса гостей.xxxx

3. xxxxВведите пароль, который должны вводить гости для доступа к этому списку задач.xxxx

4. xxxxВыберите тип уведомления, которое будет им отправлено.xxxx xxxxСообщение будет содержать сведения, необходимые для доступа к списку задач.xxxx

5. xxxxНажмите кнопку xxxxxxxxОКxxxxxxxx.xxxx

 

xxxxОбщий доступ в режиме открытыйxxxx

1. xxxxВ разделе xxxxxxxxОбщий доступ сxxxxxxxx выберите пункт «Открытый».xxxx xxxxПосле этого откроется диалоговое окно свойств общей папки с URL-адресом списка задач.xxxx

2. xxxxЗапомните этот URL-адрес и сообщите его пользователям, обладающих правом доступа в режиме «Открытый».xxxx xxxxКогда они перейдут по этому URL-адресу, они смогут прочесть содержимое списка задач, но не смогут вносить в него изменения.xxxx
