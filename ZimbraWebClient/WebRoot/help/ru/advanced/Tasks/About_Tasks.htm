О задачах

Функция «Задачи» ZCS позволяет создавать список задач и управлять их выполнением. Задачи можно добавлять в список задач ZCS по умолчанию, можно создавать другие списки задач для систематизации списков задач по более конкретным видам деятельности, например, по работе или личным проектам.

Для простой задачи нужно лишь создать задачу в области содержимого. Если задачей нужно управлять, можно добавить дополнительные данные, определить дату начала и окончания, установить приоритет задачи — высокий, обычный, низкий — и проследить выполнение задачи.

На иллюстрациях ниже показано создание трех списков задач. Для задач, приведенных в области содержимого, отображается статус, процент выполнения и дата, когда задача должна быть выполнена.

 

 

Для быстрого добавления задачи во время работы с почтовым сообщением, собранием в ежедневнике или во время просмотра контакта можно использовать Помощник Zimbra. После создания задачи вся доступная информация добавляется в область текста задачи.

Общедоступные списки задач

Списки задач можно сделать доступными для других сотрудников в офисе, для внешних гостей, или сделать их открытыми. Делая списки задач общедоступными для внутренних пользователей, вы можете выбрать для них тип доступа: статус управляющего с полным доступом для просмотра и изменения задачи, или статус наблюдателя только с правом чтения. Списки со статусом открытых и для внешних гостей предоставляют только право чтения.
