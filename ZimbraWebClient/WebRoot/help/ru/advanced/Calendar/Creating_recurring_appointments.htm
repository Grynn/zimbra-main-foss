Создание повторяющихся встреч

Повторяющиеся встречи — это встречи, которые повторяются в соответствии с определенным расписанием. Можно запланировать ежедневное, еженедельное, ежемесячное или ежегодное повторение повторяющихся собраний. Кроме того, можно изменять расписания повторяющихся встреч.

  1. На любом виде, кроме вида «Настройки», щелкните стрелку на значке и выберите пункт Встреча.

  2. Введите тему. Тема становится описанием в ежедневнике и ее ввод является обязательным.

  3. Введите место проведения. Можно ввести любое место. Места, созданные системным администратором как учетные записи ресурсов, могут быть зарезервированы. Если известно название места, введите его в поле «Место». Если не известно, перейдите на вкладку Найти место для его поиска. После выбора места оно появится в данном поле.

  4. Выберите дату и время. Введите дату или щелкните стрелку вниз для отображения календаря и выбора даты. Выберите дату окончания.

  1. Примечание. Если показан часовой пояс, то он отображает текущий часовой пояс пользователя. Обычно нет необходимости изменять данный параметр. При планировании собраний с участниками из разных часовых поясов отправляемое приглашение отражает время встречи в часовом поясе участников. Например, при создании собрания с участниками из Минска и Москвы, в приглашении будет показано минское время для участников из Минска и московское время (на один час больше) для участников из Москвы.

  1. Щелкните Повтор для просмотра шаблонов повтора: ежедневный, еженедельный, ежемесячный, ежегодный или специальный. Собрание будет запланировано по повторяющемуся шаблону с указанных даты и времени начала. Специальный повтор дает возможность создать расписание более индивидуальных шаблонов.

  2. Если у повторяющегося собрания есть дата окончания, щелкните ссылку рядом с полем «Повтор».  

  3. Выберите шаблон повтора, и если необходима дальнейшая настройка шаблона, щелкните ссылку, которая находится рядом с ним.

    • Каждый день. Можно настроить ежедневные повторения, выбрав повторение каждый день или в какой-либо другой последовательности, например каждые два или четыре дня.

    • Каждую неделю. Можно настроить еженедельные повторения, выбрав повторение встречи каждую неделю или по какому-либо другому недельному шаблону, например каждые две недели.  Можно выбирать день недели, в который должно состояться собрание.

    • Каждый месяц. Можно настроить ежемесячные повторения, выбрав повторение встречи каждый месяц или по какому-либо другому месячному шаблону, например каждые два месяца.  Отображается календарь и можно выбрать, на какой день или дни необходимо запланировать собрание.

    • Каждый год. Можно настроить ежегодные повторения, выбрав месяц и дату.

  4. Добавьте ресурсы. Если известно название ресурса, введите его в поле «Ресурс». Если неизвестно, перейдите на вкладку Найти ресурсы для поиска доступных ресурсов.

  5. Введите имена участников. Имена участников можно ввести одним из следующих способов.

    • Перейдите на вкладку Найти участников. Введите имя и выберите, какой список использовать — «Контакты» или «Глобальный список адресов». Выберите имена и нажмите кнопку Добавить.  После завершения нажмите OK.

    • В текстовом поле Участники введите адреса электронной почты, разделяя адреса точкой с запятой (;). Во время ввода в списке контактов появляются имена, отвечающие вводимому тексту.

    • Чтобы просмотреть расписание занятости участников, перейдите на вкладку Расписание участников. Во время ввода имен участников и адресов электронной почты, если расписания участников известны, на горизонтальных панелях рядом с именами будет отображаться доступность участника. После завершения введения списка участников вернитесь на вкладку Встреча.  

  6. В поле Текст введите дополнительные сведения, которые необходимо включить в сообщение электронной почты. Чтобы вложить файлы, щелкните Вложить файлы на панели инструментов.

  7. Нажмите кнопку Сохранить. Приглашение по электронной почте будет отправлено всем участникам, а встреча будет отображаться в их ежедневниках.
