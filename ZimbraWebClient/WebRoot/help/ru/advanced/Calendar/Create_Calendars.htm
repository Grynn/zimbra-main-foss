Создание ежедневников

Для пользователя создается один ежедневник, также можно создавать дополнительные ежедневники для ведения учета функций различных типов.  Например, можно создать ежедневник для рабочих, личных мероприятий и мероприятий компании.  Можно использовать вид Расписание для одновременного просмотра ежедневников.

Cоздание ежедневника

  1. Перейдите на вкладку «Ежедневник».

  2. Щелкните правой кнопкой мыши Ежедневники в области обзора или нажмите кнопку  на панели инструментов и выберите Ежедневник.

  3. В диалоговом окне Создать новый ежедневник введите название ежедневника и выберите цвет для отображения мероприятий данного ежедневника.

  4. Нажмите кнопку ОК. Новый ежедневник появится в списке ежедневников.

Создание ежедневника и синхронизации встреч из удаленного ежедневника

  1. Перейдите на вкладку «Ежедневник».

  2. Щелкните правой кнопкой мыши Ежедневники в области обзора или нажмите кнопку  на панели инструментов и выберите Ежедневник.

  3. В диалоговом окне Создать новый ежедневник введите название ежедневника и выберите цвет для отображения мероприятий данного ежедневника.

  4. Установите флажок Синхронизировать встречи из удаленного ежедневника. Введите допустимый URL-адрес файла iCal удаленного ежедневника.

  5. Нажмите кнопку ОК.
