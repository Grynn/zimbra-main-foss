Планирование событий на весь день

Можно создать событие на весь день или на несколько дней, например конференцию. Событие отображается в верхней части ежедневника в день события.

Планирование встречи на весь день

  1. В любом виде, кроме вида «Настройки», щелкните стрелку на значке и выберите пункт Встреча.

  2. Поле От отображается, если создано несколько образов для учетной записи. Выберите образ, который следует использовать для создания этой встречи.

  3. Введите тему собрания. Поле Тема необходимо заполнить обязательно. Тема становится описанием в ежедневнике.

  4. Введите имена участников. Имена участников можно ввести одним из следующих способов.

    • В поле Участники введите адреса электронной почты, разделяя их точкой с запятой (;). Во время ввода появляются имена из списков контактов, отвечающие вводимому тексту. В Планировщике, который откроется, можно просмотреть существующую информацию о доступности.

    • Щелкните Участники, чтобы найти адреса для добавления.

    • Щелкните Планировщик, чтобы просмотреть информацию о доступности при вводе имен участников. Если расписания участников известны, информация о доступности отображается на горизонтальных панелях рядом с именами.

В разделе «Планировщик» щелкните стрелку раскрывающегося списка в первом столбце, чтобы обозначить участника как необязательного.

  1. Щелкните Место, чтобы найти помещение. Или можно ввести текст в поле, чтобы выбрать место, не обозначенное как ресурс, например адрес улицы или название здания.

Если следует зарезервировать ресурс, щелкните Показать ресурс и найдите ресурс, например аудио- или видеооборудование.

  1. Щелкните Весь день и выберите дату начала и окончания.

Примечание. Если показан часовой пояс, то он отображает текущий часовой пояс пользователя. Обычно нет необходимости изменять данный параметр. При планировании собраний с участниками из разных часовых поясов отправляемое приглашение отражает время встречи в часовом поясе участников. Например, при создании собрания с участниками из Минска и Москвы, в приглашении будет показано минское время для участников из Минска и московское время (на один час больше) для участников из Москвы.

  1. Щелкните Повтор, чтобы выбрать шаблон повторения: ежедневно, еженедельно, ежемесячно, ежегодно или специальный. Собрание будет запланировано по повторяющемуся шаблону с указанных даты и времени начала. Специальный повтор дает возможность создать расписание более индивидуальных шаблонов.

    • Каждый день. Можно настроить ежедневные повторения, выбрав повторение каждый день или в какой-либо другой последовательности, например каждые два или четыре дня.

    • Каждую неделю. Можно настроить еженедельные повторения, выбрав повторение встречи каждую неделю или по какому-либо другому недельному шаблону, например каждые две недели.  Можно выбирать день недели, в который должно состояться собрание.

    • Каждый месяц. Можно настроить ежемесячные повторения, выбрав повторение встречи каждый месяц или по какому-либо другому месячному шаблону, например каждые два месяца.  Отображается календарь, и можно выбрать, на какой день или дни необходимо запланировать собрание.

    • Каждый год. Можно настроить ежегодные повторения, выбрав месяц и дату.

Щелкните Настроить, чтобы выбрать шаблон повторения.

  1. В разделе Напоминание можно выбрать время отправления напоминания участникам.

  2. Если следует отправить уведомление электронной почтой или SMS-сообщением, установите соответствующий флажок.  Если эти функции не настроены на странице настроек уведомления, щелкните «Настройка».

  3. В поле Текст введите дополнительные сведения, которые необходимо включить в сообщение электронной почты. Чтобы вложить файлы, щелкните Вложить файлы на панели инструментов. На панели инструментов в разделе Параметры можно отключить запросы на ответы, отправку уведомления и пометить встречу как личную.

  4. Нажмите кнопку Отправить. Всем участникам будет отправлено приглашение по электронной почте.

Если приглашение следует отправить позже, нажмите кнопку Сохранить. Сохраняется локальная копия встречи, а приглашения участникам не отправляются.
