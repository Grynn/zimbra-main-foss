Образ — это личные данные почты, используемые в поле От отправляемых сообщений. Создание различных личных данных почты позволяет использовать несколько адресов электронной почты в одном почтовом ящике. xxxxЭто может понадобится для управления различными ролями на работе и в личной жизни с помощью одной учетной записи.xxxx Имя учетной записи по умолчанию определено в основных личных данных.

xxxxКнопка xxxxxxxxДобавить внешнюю учетную записьxxxxxxxx позволяет определить внешние учетные записи POP/IMAP для получения почты в веб-клиенте с других учетных записей.xxxx Также можно установить, чтобы в ответах на эти сообщения использовался соответствующий адрес POP/IMAP.

Чтобы создать образ, выполните следующие действия.

  1. На странице Настройки, Учетные записи нажмите кнопку Добавить образ.

  2. xxxxВ разделе xxxxxxxxНастройки образаxxxxxxxx введите имя образа.xxxx xxxxЭто имя используется для идентификации образа в списке «От:» при создании сообщения.xxxx

  3. В текстовом поле От введите имя, которое должно быть указано в поле «От» исходящих сообщений, например Иван. xxxxЭто описательное имя, котрое будет отображаться перед вашим электронным адресом.xxxx xxxxВо втором окне выберите электронный адрес, с которого будут отправляться сообщения.xxxx В зависимости от настроек может быть доступна возможность создания другого адреса. Сделайте это далее.

  4. xxxxЕсли ответы должны отправляться на имя и адрес, отличные от указанного в поле «От:», установите флажок xxxxxxxxОбратный адресxxxxxxxx.xxxx xxxxВведите имя и адрес для получения ответов для этих личных данныхxxxx Например, ответы на сообщения о тренировках, отправляемые Иваном, должны отправляться Тренировочной команде по адресу: training@xample.com.

  5. xxxxМожно выбрать подпись для этого образа.xxxx xxxxЕсли вы не создали подпись, вы можете вернуться позднее и отредактировать этот образ.xxxx xxxxЕсли вы не хотите использовать автоматическую подпись, выберите пункт xxxxxxxxНе включать подпись.xxxx

  6. Если нужно, чтобы этот образ автоматически использовался для сообщений в определенных папках, установите флажок Ответ или пересылка сообщений в папки. xxxxЩелкните значок папки, чтобы выбрать одну или несколько папок из списка существующих папок или создайте новую папку.xxxx xxxxВведите запятую или точку с запятой для разделения папок в поле.xxxx

  7. xxxxНажмите кнопку xxxxxxxxСохранитьxxxxxxxx, чтобы сохранить образ.xxxx
