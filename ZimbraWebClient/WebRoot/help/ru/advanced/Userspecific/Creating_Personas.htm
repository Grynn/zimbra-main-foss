Создание образов

Создание различных личных данных для почты, называемых образами, позволяет использовать несколько электронных адресов в одном почтовом ящике. Это может понадобится для управления различными ролями на работе и в личной жизни с помощью одной учетной записи.

Чтобы создать образ, откройте вкладку Настройки и выполните следующие действия.

  1. На вкладке Учетные записи щелкните Добавить образ.

  2. В разделе Настройки образа введите имя образа. Это имя используется для идентификации образа в списке «От:» при создании сообщения.  Это имя не появляется в сообщении.

  3. В текстовом поле, расположенном ниже поля От: введите имя, которое должно быть указано в поле «От:» исходящих сообщений, например Иван. Это описательное имя, котрое будет отображаться перед вашим электронным адресом. Во втором окне выберите электронный адрес, с которого будут отправляться сообщения. Настройки вашей учетной записи позволяют создать адрес.

  4. Если ответы должны отправляться на имя и адрес, отличные от указанного в поле «От:», установите флажок Обратный адрес. Введите имя и адрес для получения ответов для этих личных данных Например, ответы на сообщения о тренировках, отправляемые Иваном, должны отправляться Тренировочной команде по адресу: training@komanda.ru.

  5. Можно выбрать подпись для этого образа. Если вы не создали подпись, вы можете вернуться позднее и отредактировать этот образ. Если вы не хотите использовать автоматическую подпись, выберите пункт Не включать подпись.

  6. Если нужно, чтобы этот образ автоматически использовался для сообщений в определенных папках, установите флажок Ответ или пересылка сообщений в папку (папки). Щелкните значок папки, чтобы выбрать одну или несколько папок из списка существующих папок или создайте новую папку. Введите запятую или точку с запятой для разделения папок в поле.

  7. Нажмите кнопку Сохранить, чтобы сохранить образ.  

Если нужно создать подпись, откройте вкладку «Подпись» после нажатия кнопки «Сохранить». См. раздел Использование подписи.
