Изменение пользовательских настроек

Можно изменить настройки вида, почтовых сообщений, адресной книги, ежедневника, можно также создать правила фильтра из приложения «Настройки». Общие указания по изменению настроек следующие.

  1. Нажмите кнопку Настройки.

  2. Откройте вкладку с настройками, которые нужно изменить. Подробнее см. раздел Справки для каждой вкладки.

  3. Измените настройки.

  4. Нажмите кнопку Сохранить.
