Управление личными данными учетной записи

При отправке нового сообщения, ответе или пересылке сообщения, с электронным адресом всегда связывается одна запись личных данных.  Это личные данные по умолчанию, созданные как имя учетной записи и используемые до тех пор, пока не будут установлены другие личные данные.  Создание различных личных данных для почты, называемых образами, позволяет использовать несколько электронных адресов в одном почтовом ящике. Это может понадобится для управления различными ролями на работе и в личной жизни с помощью одной учетной записи.

Можно также добавить адреса внешних учетных записей для получения почты с других почтовых ящиков POP или IMAP, кроме того можно настроить внешние учетные записи так, чтобы при ответе на сообщения в поле «От:» появлялся соответствующий адрес.  

Если установлено несколько образов или добавлены внешние учетные записи, при создании нового сообщения личные данные появляются в поле «От:» в заголовке.

Образы, внешние учетные записи и подписи можно настроить на вкладке Настройки->Учетные записи и на вкладке Подписи.
