xxxxДобавление внешних учетных записей к учетной записи веб-клиентаxxxx

  1. Откройте вкладку Настройки, Учетные записи и нажмите кнопку Добавить внешнюю учетную запись. xxxxИмя новой внешней записи добавлено к вашим учетным записям.xxxx xxxxТеперь надо настроить эту учетную запись.xxxx

  2. В разделе Настройки внешней учетной записи в поле Имена учетных записей введите имя этой учетной записи. xxxxИмя учетной записи можно выбрать из списка адресов «От:», который появляется во время создания нового сообщения.xxxx

  3. xxxxВ поле xxxxxxxxЭлектронный адресxxxxxxxx введите точный электронный адрес внешней учетной записи.xxxx

  4. xxxxНастройте учетную запись.xxxx xxxxДля этого вам, возможно, понадобится обратиться к провайдеру.xxxx

    • xxxxУстановите флажок xxxxxxxxТип учетной записиxxxxxxxx, POP или IMAPxxxx

    • Имя пользователя учетной записи. xxxxВведите имя почтового ящика.xxxx

    • xxxxПочтовый серверxxxxxxxx.xxxx xxxxВведите имя хост-сервера электронной почты.xxxx

    • xxxxПарольxxxxxxxx.xxxx xxxxВведите пароль для входа в эту внешнюю учетную запись.xxxx xxxxЧтобы проверить правильность введенного пароля, установите флажок xxxxxxxxПоказать парольxxxxxxxx.xxxx

    • Дополнительные параметры. Если ваш провайдер использует не принятый по умолчанию порт, смените порт. В случае использования безопасного подключения включите параметр «Во время доступа к серверу используется зашифрованное подключение (SSL)».

  5. xxxxВ разделе xxxxxxxxЗагрузить сообщения в:xxxxxxxx укажите, где должны храниться сообщения.xxxx xxxxПо умолчанию создается новая папка, куда отправляются сообщения.xxxx  Если необходимо, укажите папку «Входящие».

  6. Включите параметр Удалять сообщения с сервера после загрузки, если нужно, чтобы сообщение удалялось из внешнего почтового ящика после отправки в почтовый ящик ZWC.

  7. В текстовом поле Отвведите имя, которое должно быть указано в поле «От» исходящих сообщений, например Иван. xxxxЭто описательное имя, котрое будет отображаться перед вашим электронным адресом.xxxx

  8. xxxxЕсли ответы должны отправляться на имя и адрес, отличные от указанного в поле «От:», установите флажок xxxxxxxxОбратный адресxxxxxxxx.xxxx xxxxВведите имя и адрес для получения ответов для этих личных данныхxxxx Например, ответы на сообщения о тренировках, отправляемые Иваном, должны отправляться Тренировочной команде по адресу: training@example.com.

  9. xxxxМожно выбрать подпись для этого образа.xxxx xxxxЕсли соответствующая подпись не создана, можно вернуться и изменить данные этой учетной записи позже.xxxx xxxxЕсли вы не хотите использовать автоматическую подпись, выберите пункт xxxxxxxxНе включать подпись.xxxx

  10. xxxxНажмите кнопку xxxxxxxxСохранитьxxxxxxxx, чтобы сохранить эту внешнюю учетную запись.xxxx

xxxxЕсли нужно создать подпись, откройте вкладку «Подпись» после нажатия кнопки «Сохранить».xxxx xxxxСм. раздел xxxxxxxxИспользование подписиxxxxxxxx.xxxx
