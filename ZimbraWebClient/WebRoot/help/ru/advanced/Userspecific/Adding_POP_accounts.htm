Добавление внешних учетных записей

Вместо входа в каждую внешнюю учетную запись для проверки почты, ее можно проверять непосредственно из этой учетной записи. См. раздел Доступ к внешним учетным записям.

Добавление внешних учетных записей к учетной записи веб-клиента

  1. Откройте вкладку Настройки, Учетные записи и нажмите кнопку Добавить внешнюю учетную запись. Имя новой внешней записи добавлено к вашим учетным записям. Теперь надо настроить эту учетную запись.

  2. В разделе Настройки внешней учетной записи в поле Имена учетных записей введите имя этой учетной записи. Имя учетной записи можно выбрать из списка адресов «От:», который появляется во время создания нового сообщения.

  3. В поле Электронный адрес введите точный электронный адрес внешней учетной записи.

  4. Настройте учетную запись. Для этого вам, возможно, понадобится обратиться к провайдеру.

    • Установите флажок Тип учетной записи, POP или IMAP

    • Имя пользователя учетной записи. Введите имя почтового ящика.

    • Почтовый сервер. Введите имя хост-сервера электронной почты.

    • Пароль. Введите пароль для входа в эту внешнюю учетную запись. Чтобы проверить правильность введенного пароля, установите флажок Показать пароль.

    • Дополнительные параметры. Если ваш провайдер использует не принятый по умолчанию порт, смените порт. В случае использования безопасного подключения включите параметр «Во время доступа к серверу используется зашифрованное подключение (SSL)».

  5. В разделе Загрузить сообщения в: укажите, где должны храниться сообщения. По умолчанию создается новая папка, куда отправляются сообщения.  Если необходимо, укажите папку «Входящие».

  6. Включите параметр Удалять сообщения с сервера после загрузки, если нужно, чтобы сообщение удалялось из внешнего почтового ящика после отправки в почтовый ящик ZWC.

  7. В текстовом поле От введите имя, которое должно быть указано в поле «От» исходящих сообщений, например Иван. Это описательное имя, котрое будет отображаться перед вашим электронным адресом.

  8. Если ответы должны отправляться на имя и адрес, отличные от указанного в поле «От:», установите флажок Обратный адрес. Введите имя и адрес для получения ответов для этих личных данных Например, ответы на сообщения о тренировках, отправляемые Иваном, должны отправляться Тренировочной команде по адресу: training@example.com.

  9. Можно выбрать подпись для этого образа. Если соответствующая подпись не создана, можно вернуться и изменить данные этой учетной записи позже. Если вы не хотите использовать автоматическую подпись, выберите пункт Не включать подпись.

  10. Нажмите кнопку Сохранить, чтобы сохранить эту внешнюю учетную запись.

Если нужно создать подпись, откройте страницу «Подпись» после нажатия кнопки «Сохранить». См. раздел Использование подписи.
