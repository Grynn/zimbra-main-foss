Использование тегов для систематизации сообщений и контактов

Теги — это ваша личная система систематизации почтовых сообщений, контактов, задач и встреч.  Теги помогают найти эти элементы. Например, вы назначили один тег для элемента Немедленный оборот и другой тег для элемента Средний приоритет. Помеченные тегом почтовые сообщения могут находиться в разных папках. Если выполнить поиск тега, будут отображены все сообщения, помеченные этим тегом, независимо от того, в какой папке они находятся.

  • Можно создать столько тегов, сколько необходимо.

  • К одному и тому же сообщению и контактам можно применить несколько тегов.

Создание определений нового тега

Доступ к функции Новый тег можно получить одним из следующих способов.

  • На панели задач щелкните стрелку вниз, расположенную рядом с пунктом Создать и выберите пункт Тег.

  • Выберите на панели инструментов Тег.

  • Щелкните правой кнопкой мыши элемент в списке и выберите пункт Пометить разговор.

  • Щелкните правой кнопкой мыши Теги или тег в области обзора и выберите пункт Новый тег.

Откроется диалоговое окно Новый тег. Введите имя и нажмите кнопку OK. Имена тегов не должны содержать следующих знаков

: (двоеточие)

/ (косая черта)

" (двойные кавычки)

Новый тег появится в списке Теги в области обзора и может быть применен к любому сообщению или контакту.
