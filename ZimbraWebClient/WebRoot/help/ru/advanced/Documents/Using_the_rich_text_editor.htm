Использование форматированного текста

Для создания и форматирования страниц можно использовать форматированный текст или теги  кода HTML. Между этими двумя форматами можно переключаться, чтобы воспользоваться преимуществами форматированного текста и вставить таблицы или электронные таблицы.

При редактировании с помощью форматированного текста двухрядная панель инструментов позволяет легко форматировать разметку страницы. Выберите размер, гарнитуру и цвет шрифта, стиль текста и цвет фона. Эти инструменты форматирования работают также, как и многие другие инструменты обработки текста.

Кроме того, на страницу можно добавлять таблицы, электронные таблицы, вложения и изображения.

Добавление стилей абзаца

Значок    на панели инструментов показывает имеющиеся стили абзаца. Эти стили позволяют легко  форматировать текст и сохранять единообразие стиля на страницах блокнота. Стили включают: Обычный стиль для большинства абзацев, Заголовки шести уровней для определения заголовков разделов, Адрес для выделения текста курсивом и Форматированный для отображения текста шрифтом Courier. Эти стили нельзя изменить, но можно использовать возможности редактирования для иного форматирования стиля, например изменение шрифта, размера, цвета и т.п.

Добавление таблиц на страницу

Добавление таблиц на страницу выполняется в три этапа. Добавление таблицы, задание свойств таблицы и добавление содержимого. Чтобы добавить таблицу, щелкните значок таблицы на панели инструментов, выберите пункт «Вставить таблицу», затем укажите, сколько строк и столбцов должно быть в таблице.

Добавление электронной таблицы на страницу 

Чтобы добавить электронную таблицу, щелкните значок таблицы на панели инструментов. На страницу будет добавлена электронная таблица из шести столбцов. Электронные таблицы можно использовать для создания  баз данных, для ведения списков деятельности, расписаний, задач и т.п.

Добавление вложений и изображений на страницу

Файлы, на которые есть ссылка на странице «Документы», можно загрузить. Щелкните значок скрепки или значок изображений ,  чтобы найти файл для загрузки. Если загружаемый файл уже есть в документах или портфеле, вам поступит уведомление.

Подключение к блокноту или к веб-страницам

Можно подключиться к другим документам (блокнотам) или внешним веб-страницам и  дать ссылке описательный заголовок без отображения URL-адреса.
