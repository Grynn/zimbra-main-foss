Создание новой страницы

Создать страницу или оформить содержимое можно с помощью инструментов форматирования или тегов HTML. На странице можно использовать проверку орфографии и ключевые слова для простого создания ссылок на другие страницы.

  1. Откройте блокнот, в котором вы хотите работать. На панели Содержание появится содержание страницы.

  2. Щелкните ссылку Создать на панели задач. Откроется чистая страница. В заголовке поля Страница введите имя новой страницы. Это имя появится в содержании.

  3. Создайте содержимое страницы. По умолчанию можно использовать форматированный текст, чтобы добавить на страницу стили, цвет, таблицы и электронные таблицы.

Если вы знакомы с кодом HTML и предпочитаете использовать теги HTML, откройте на панели инструментов меню Формат и выберите пункт Код HTML. Откроется чистое поле, в котором можно будет создать страницу с помощью языка разметки HTML.

  1. Нажмите кнопку Сохранить, чтобы сохранить файл. Затем можно добавить содержимое на страницу или нажать кнопку Закрыть, чтобы вернуться к содержанию страницы
