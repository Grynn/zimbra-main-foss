Создание нового документа

Можно создать документ в портфеле и оформить содержимое с помощью инструментов форматирования. На странице можно использовать проверку орфографии и ключевые слова для простого создания ссылок на другие страницы.

Каждая страница содержит нижний колонтитул, в котором отображается расположение страницы, версия, открытая в данный момент, автор и дата последнего изменения.

  1. Щелкните папку портфеля, в которой необходимо работать. На панели Содержание отображаются файлы, уже содержащиеся в портфеле.

  2. В списке Создать на панели инструментов выберите Документ. Отобразится пустая страница. В заголовке введите имя документа.

  3. Создайте содержимое страницы. По умолчанию можно использовать редактор форматированного текста, чтобы добавить на страницу стили, цвет и таблицы.

  1. Нажмите кнопку Сохранить, чтобы сохранить файл. Теперь можно продолжить редактирование страницы или щелкнуть Сохранить и закрыть, чтобы вернуться на страницу «Содержание».

Примечание. Если документ был создан не в том портфеле, можно выделить документ и перетащить его в правильную папку.
