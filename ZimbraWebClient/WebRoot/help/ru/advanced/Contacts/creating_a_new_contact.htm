Добавление новых контактов

Чтобы добавить новый контакт, щелкните стрелку на кнопке или щелкните правой кнопкой мыши имя в заголовке сообщения и выберите пункт Контакты. При добавлении нового контакта вы вводите имя и сведения о компании в разные поля. Можно выбрать, каким образом вносить контакт в список. Можно ввести до трех адресов электронной почты и трех почтовых адресов.

Добавление нового контакта

  1. На панели инструментов щелкните стрелку рядом с кнопкой Создать и выберите пункт Новый Контакт. Откроется форма Новый контакт.

  2. Введите контактные данные, включая имя, фамилию, адрес электронной почты, должность, сведения о компании, подробные сведения об электронной почте и системе обмена мгновенными сообщениями, а также примечания. Для контакта можно также загрузить фотографию.

  3. В поле Записать как выберите, как внести имя. По умолчанию контакт записывается в порядке фамилия, имя.

  4. В Адресной книге выберите одну из личных адресных книг, чтобы сохранить имя.

  5. Нажмите кнопку Сохранить.

Добавление нового контакта из существующего сообщения

  1. Откройте сообщение электронной почты, вверху в области информации сообщения в сером цвете отображается заголовок. Можно добавлять адреса электронной почты из полей  «От:»,  «Кому:», «Копия:» и «Скрытая копия:».

  2. Щелкните правой кнопкой мыши имя, которое необходимо добавить в список контактов, и выберите пункт Добавить в контакты.

  3. Откроется форма Новый контакт с внесенными в нее данными, доступными из заголовка сообщения. Внесите дополнительные данные, а также проверьте правильность предварительно заполненных полей.

  4. В поле Записать как выберите, как внести имя. По умолчанию контакт записывается в порядке фамилия, имя.

  5. В Адресной книге выберите одну из личных адресных книг, чтобы сохранить имя.

  6. Нажмите кнопку Сохранить.

 

Также можно создавать списки групповых контактов.
