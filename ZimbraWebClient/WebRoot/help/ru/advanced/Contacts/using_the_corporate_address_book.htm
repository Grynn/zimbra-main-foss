Использование адресной книги организации

В пределах организации не всегда известен правильный адрес электронной почты, который необходимо использовать. Например, у Ивана Петрова из организации «Контора» могут быть адреса ivan@kontora.ru или ipetrov@kontora.ru, или даже ip123@prodam.kontora.ru.

При создании сообщения обычно доступны для использования два типа адресных книг: адресная книга компании (называемая иногда глобальным списком адресов) и личные адресные книги. Адресная книга организации доступна всем, тогда как личные адресные книги доступны пользователям, которым они принадлежат и которым предоставлен доступ.

Добавление имен из адресной книги в новое сообщение электронной почты

  1. Откройте новое сообщение, щелкнув в меню значок .

  2. Щелкните поле Кому:. Появится диалоговое окно выбора адреса.

  3. В поле Показывать имена из списка выберите для поиска одну из адресных книг: Контакты или Глобальный список адресов.

  4. Введите имя или часть имени в поле «Поиск» слева и нажмите кнопку Поиск. Слева появится список имен и адресов электронной почты.

  5. Дважды щелкните имя, чтобы добавить его в список получателей справа. С помощью кнопок Кому, Копия и Скрытая копия укажите, в какие поля вставлять имена при отправке. Нажмите необходимую кнопку перед перемещением имени.

  • С помощью кнопки Удалить можно удалить имя из списка получателей справа.

  1. При необходимости повторяйте поиск, пока не добавите все имена.

  2. По завершении нажмите кнопку OK.
