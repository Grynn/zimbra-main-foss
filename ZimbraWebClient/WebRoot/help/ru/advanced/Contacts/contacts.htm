Адресная книга

Адресная книга — это место для хранения контактных данных в личных адресных книгах. По умолчанию в «Адресной книге» создаются адресные книги Контакты и Отправленные по электронной почте контакты. Можно создавать дополнительные адресные книги и предоставлять доступ к ним другим пользователям.

Можно добавлять контакты в любую из адресных книг. Для создания контакта требуется только имя, также можно создать подробную карточку контакта, включающую полное имя, несколько адресов электронной почты, рабочий, домашний и другие адреса, номера телефонов и примечания к контакту. Также можно создавать списки групповых контактов.

Адресная книга Отправленные по электронной почте контакты заполняется при отправлении электронной почты на новый адрес, который не содержится в других ваших адресных книгах. Эту функцию можно отключить на вкладке «Адресная книга» в меню «Настройки». Снимите флажок Включить автоматическое добавление контактов.

Администратор электронной почты может установить ограничение на максимальное количество записей, которые могут содержаться во всех ваших адресных книгах. При достижении лимита исчезнет возможность добавлять контакты. Можно удалить неиспользуемые контакты, чтобы освободить место.

При составлении сообщения электронной почты функция автозаполнения показывает список имен из адресной книги, соответствующих вводимому тексту.

Просмотр данных контактов

Можно просматривать имена контактов в списке или в виде подробных карточек. Вид Список контактов является вертикальным списком всех контактов. Вид Карточки отображает все сведения, введенные в форме контакта.

Можно изменить вид на один сеанс при помощи раскрывающегося списка Адресная книга. Вид адресных книг по умолчанию можно изменить, открыв вкладки «Настройки->Адресная книга», параметр Вид контактов по умолчанию.

Полные контактные данные отображаются в виде всплывающей подсказки при наведении курсора мыши на имя в сообщении или при просмотре содержимого электронной почты, если это имя содержится в одной из адресных книг.

Предоставление доступа к спискам контактов

Можно предоставлять доступ к любой из адресных книг. Эта функция полезна, если адресные книги используются для определенных категорий контактов и другим пользователям необходимо иметь доступ к именам. В зависимости от роли, которая назначена для получившего доступ пользователя, общий список контактов может иметь права только на просмотр или права полного доступа для изменения, добавления и удаления имен.
