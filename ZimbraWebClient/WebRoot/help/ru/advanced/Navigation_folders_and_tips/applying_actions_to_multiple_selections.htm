Применение действий к нескольким выбранным элементам

Если щелкнуть имя папки слева от нее для просмотра, отобразится содержимое папки с одним выбранным (выделенным) элементом. Можно выбрать один или несколько элементов для выполнения следующих действий.

  • Щелкните элемент, чтобы выбрать его. Если щелкнуть другой элемент, выбор предыдущего элемента будет отменен.

  • Дважды щелкните элемент, чтобы открыть его.

  • Щелкните правой кнопкой мыши элемент, чтобы отобразить меню действий, которые можно применить к нему. Если выбрать несколько элементов и щелкнуть правой кнопкой мыши, то же самое действие будет применено ко всем элементам сразу.

  • Одновременное нажатие клавиши Ctrl и кнопки мыши выделяет несколько элементов. Эту комбинацию можно также использовать для отмены выбора текущего элемента.

  • Одновременное нажатие клавиши Shift и кнопки мыши выделяет несколько элементов в последовательности.  Если выделить один элемент, а затем — другой элемент в списке, удерживая нажатой клавишу Shift, будут выделены все элементы между этими двумя элементами.

Можно выбрать несколько элементов, а затем с помощью правой кнопки мыши выбрать пункт «Пометить как прочитанное», «Тег» или «Переместить» для всех выбранных элементов.
