Быстрые клавиши ZWC

Быстрые клавиши используются для перемещения в веб-клиенте Zimbra (ZWC), кроме того, можно создать собственные комбинации быстрых клавиш. Перейдите в меню Настройки>Быстрые клавиши, чтобы посмотреть список быстрых клавиш и создать собственные комбинации клавиш. На четырех вкладках можно увидеть следующие пункты

  • Список быстрых клавиш — список быстрых клавиш для всех приложений

  • Быстрые клавиши папок почты — позволяет создавать и назначать клавиши для доступа к папкам почты

  • Быстрые клавиши для сохраненных профилей поиска — позволяет создавать и назначать клавиши для сохраненных профилей поиска

  • Быстрые клавиши для тегов — позволяет создавать и назначать клавиши для созданных тегов
