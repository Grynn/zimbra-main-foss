Создание специальных папок

В дополнение к предопределенным системным папкам «Входящие», «Отправленные» и Корзина можно создать специальные папки. Специальные папки можно создавать в других папках, в том числе в системных.

Нельзя создавать две папки с одинаковыми именами в одной родительской папке. Например, в папке «Входящие» не может быть двух папок с именем «Покупки», но одна папка с именем «Покупки» может быть в папке «Входящие», а другая папка с таким именем может быть в папке «Отправленные».

Создание новой папки

  1. На панели задач щелкните стрелку на кнопке «Создать» ( ) и выберите пункт Папка, или щелкните правой кнопкой мыши область обзора в списке папок и выберите пункт Новая папка.  Откроется диалоговое окно Создать новую папку.

  2. Введите имя папки.

  3. Выберите каталог для новой папки. Укажите каталог Папки, если нужно, чтобы папка была на верхнем уровне, или выберите подкаталог из списка уже существующих папок. Новая папка будет помещена в указанную папку.

  4. Нажмите кнопку ОК.

Чтобы создать папку, связанную с лентой новостей RSS или ATOM, установите флажок Подписаться на RSS-ленту/Atom-ленту, затем введите URL-адрес. Новости будут автоматически подаваться в папку в почтовом ящике Zimbra.
