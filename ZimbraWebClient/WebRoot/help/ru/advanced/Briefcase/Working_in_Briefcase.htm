Работа в портфеле

Портфель позволяет сохранять файлы в учетной записи, поэтому доступ к этим файлам можно получить с любого компьютера, войдя в свой почтовый ящик.

В портфель можно загружать документы, электронные таблицы, презентации, изображения, pdf-файлы с компьютера или из сети. Кроме того, в нем можно сохранять вложения почтовых сообщений.  В портфеле можно создавать разные папки для систематизации файлов, а также делать их общедоступными.

В портфеле хранятся копии исходных файлов.  Изменения, вносимые в файлы в портфеле, не сохраняются в исходных файлах.

Создание папок портфеля

  1. Откройте вкладку Портфель.

  2. Выберите пункт Портфель из списка Создать на панели инструментов.


  1. Откроется диалоговое окно Создать новый портфель. Введите имя новой папки портфеля.

Это имя должно быть уникальным в иерархии папок вашего почтового ящика. Папки портфеля не могут иметь такие же имена, как у папок верхнего уровня в почте, ежедневнике или адресной книге @CompanyName@. Например, если у вас есть ежедневник с именем «Праздники», то дать имя «Праздники» портфелю верхнего уровня нельзя. Имя «Праздники» можно дать портфелю в другом портфеле.

  1. Выберите иерархию папки. Можно создать новый портфель верхнего уровня или поместить его в уже существующий портфель.

  2. Нажмите кнопку ОК. Новый портфель появится в области обзора.

Загрузка файлов

В процессе загрузки файлы передаются с компьютера в почтовый ящик. Это позволяет получить доступ к файлам в любое время, для этого нужно только войти в почтовый ящик. Количество файлов, которое можно загрузить в портфель, определяется лимитом учетной записи.

Загрузка файлов

  1. Откройте портфель и на панели инструментов щелкните Загрузить файл.

  2. В диалоговом окне Загрузить новый файл в портфель нажмите кнопку Обзор, чтобы найти файлы для загрузки. Можно выбрать несколько файлов для одновременной загрузки, если эти файлы расположены в одной папке. Для добавления других файлов нажмите кнопку Добавить и найдите необходимые файлы.

Загружать файлы с одинаковыми именами в один и тот же портфель нельзя. В диалоговом окне «Загрузить» можно указать, как сохранять файлы с одинаковыми именами при загрузке в портфель. Внесенные в этот параметр изменения сохраняются в диалоговом окне.

    • Чтобы получать уведомления о загрузке файлов с одинаковыми именами, выберите пункт Спрашивать меня. Если файл с именем, аналогичным имени загружаемого файла уже существует, откроется диалоговое окно Конфликт загрузки. Можно сохранить уже существующий файл или заменить его загружаемым. Можно также просмотреть новый файл. Это настройка по умолчанию.

    • Если нужно, чтобы загружаемый файл всегда заменял уже существующий файл с таким же именем, выберите пункт Заменить моим файлом. Загруженный файл будет сохранен вместо существующего. Уведомление об этом не поступает.

    • Если файлы в портфеле должны быть сохранены, выберите пункт Оставить существующий файл. Файл не загружается, уведомление об этом не поступает.

  1. Если нужно загрузить все файлы, нажмите кнопку ОК. Файл отображается в портфеле.  

  2. Выберите Вид, чтобы указать, как файлы должны отображаться в портфеле.

    • Вид обозревателя. Файлы отображаются по типу (PDF, Word, PowerPoint и т. п.)

    • Подробный вид. Файлы отображаются в виде списка с указанием имени, типа, размера, даты и владельца файла.  Если выбран подробный вид, щелкните заголовок для сортировки файлов по имени, размеру или дате.

    • Вид браузера «Столбцы». Имена файлов отображаются в столбцах.

Примечание. В почтовом ящике можно установить предельный размер для загружаемых файлов. Если файл слишком большой, появится предупреждение.
Загружаемые файлы влияют на лимит почтового ящика.  

Добавление почтовых вложений

В портфель можно быстро добавлять почтовые вложения.

  1. Чтобы добавить вложение, щелкните ссылку Добавить в портфель, которая находится в заголовке сообщения рядом со ссылкой на вложение.

  2. В диалоговом окне Добавить в портфель выберите папку назначения или создайте новую папку.

  3. Нажмите кнопку ОК.

Просмотр файлов

Можно выбрать режим просмотра файлов в папках портфеля. В меню Вид на панели задач портфеля выберите один из трех вариантов.

  • Вид обозревателя. Файлы отображаются в виде эскизов.

  • Подробный вид. Файлы отображаются горизонтальной строкой с указанием размера файла, даты загрузки, владельца (если файл общий, то владельцем может быть другое лицо) и папки, в которой он хранится.

  • Вид браузера «Столбцы». Файлы отображаются по вертикали. В этом виде можно щелкнуть файл и посмотреть информацию о нем, включая размер, дату создания, создателя, дату изменения и кто внес изменения.

Присвоение тегов файлам

Файлы можно помечать.

  1. Выделите файл и на панели инструментов щелкните значок .

  2. Выберите тег. Теги видны только на виде «Подробный».

  3. Нажмите кнопку ОК.

Сортировка файлов

Если выбран подробный вид отображения портфеля, то содержащиеся в нем файлы можно сортировать по имени, размеру и дате. При изменении вида порядок сортировки сохраняется.

Общий доступ к портфелю

Доступ к папкам портфеля может быть предоставлен другим пользователям внутри и за пределами организации. Другим пользователям можно предоставить разрешение на создание и редактирование содержимого в портфеле или только на просмотр страниц.

Как сделать портфель общедоступным см. в разделе Общий доступ к портфелю
