В сообщения можно вкладывать файлы с компьютера пользователя, из портфеля @ClientNameShort@, списков контактов и сообщений электронной почты.

xxxxПримечание.xxxx У получателей сообщения должно быть установлено соответствующее программное обеспечение для открытия и чтения файла. xxxxПользователи, использующие обычный настольный компьютер, смогут открыть файлы с помощью таких распространенных программ, как Microsoft Office.xxxx xxxxФайлы распространенных форматов, например текстовые файлы, HTML-файлы и изображения (.GIF или .JPG), можно открывать с помощью различных программ.xxxx xxxxОднако, все зависит от типа используемой системы.xxxx

Добавление вложений с помощью ссылки «Вложить файл»

  1. xxxxСоздайте сообщение, как указано в разделе xxxxxxxxСоздание нового сообщенияxxxxxxxx.xxxx

  2. Нажмите кнопку . В диалоговом окне «Вложить файл(ы)» отображаются вкладки «Мой компьютер», «Портфель», «Вложить сообщение» и «Контакты». Выберите вкладку согласно типу вложения.

    • Мой компьютер. Перейдите в любую папку на компьютере и выберите файл. Выберите файл и нажмите кнопку Открыть. Также можно перетащить файлы из папки в заголовок создаваемого сообщения.

    • Портфель. Выберите папку портфеля. Справа отобразятся файлы из этого портфеля. Отметьте файлы, которые необходимо вложить.

    • Вложить сообщение. Если сообщение находится не в папке «Входящие», выберите папку. В выбранной папке можно выполнить поиск сообщения. Сообщения отображаются справа. Выберите сообщение, которое необходимо вложить.

    • Контакты. Выберите список контактов. Справа отобразится список имен. В списке контактов можно выполнить поиск имени. Отметьте имена, которые необходимо вложить. Они будут вложены как файлы .vcf.

Примечание. Чтобы добавить несколько вложений за один раз, щелкайте их мышью, удерживая нажатой клавишу Ctrl или Shift.

  2. xxxxПосле добавления всех вложений нажмите кнопку xxxxxxxxВложитьxxxxxxxx.xxxx xxxxСписок вложений находится под полем «Тема».xxxx

  3. xxxxКогда сообщение готово к отправке, нажмите кнопку xxxxxxxxОтправитьxxxxxxxx, чтобы отправить сообщение и вложения.xxxx

Добавление вложений с компьютера с помощью перетаскивания

  1. Выберите один или несколько файлов, наведите на них курсор, а затем нажмите и удерживайте левую кнопку мыши.

  2. Перетащите файлы в область заголовка сообщения и отпустите кнопку. В заголовке отобразятся имена файлов.

 

 

 

Удаление вложения

Для удаления вложения снимите флажок рядом с вложением перед отправкой сообщения электронной почты.
