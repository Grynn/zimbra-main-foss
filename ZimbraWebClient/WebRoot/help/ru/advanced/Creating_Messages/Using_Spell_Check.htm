xxxxИспользование проверки правописанияxxxx

Перед отправкой сообщения можно проверить правописание. На панели инструментов создания сообщения нажмите кнопку Проверить правописание.

Если необходима автоматическая проверка правописания в сообщениях перед их отправкой, установите флажок Обязательно проверять правописание перед отправкой сообщения на странице «Настройки» > «Создание сообщения».

 xxxxБудут выделены слова, неизвестные программе проверки правописания.xxxx

  1. Нажмите кнопку Проверить правописание. Будут выделены слова с ошибками.

  2. xxxxЩелкните выделенное слово.xxxx xxxxВо всплывающем окне отобразятся варианты исправления.xxxx

  3. xxxxВыберите правильное слово.xxxx xxxxСлово будет выделено другим цветом.xxxx

  4. xxxxЧтобы принять изменения и закрыть программу проверки правописания, нажмите кнопку xxxxxxxxПродолжить редактированиеxxxxxxxx.xxxx

 

Примечание. xxxxПеред закрытием программы проверки правописания можно вернуть исходное написание исправленного слова.xxxx  Щелкните выделенное слово и выберите его исходное написание вверху всплывающего окна.

Добавление слов в словарь функции проверки правописания

В словарь функции проверки правописания можно добавлять слова.

  1. Нажмите кнопку Проверить правописание. Будут выделены слова с ошибками.

  2. xxxxЩелкните выделенное слово.xxxx xxxxВо всплывающем окне отобразятся варианты исправления.xxxx

  3. Если выделено правильно написанное слово, щелкните его и выберите пункт Добавить. Слово будет добавлено в словарь.

Возможность удалять добавленные в словарь слова отсутствует.
