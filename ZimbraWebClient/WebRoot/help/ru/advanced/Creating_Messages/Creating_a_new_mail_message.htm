Создание нового сообщения

Первое, что нужно сделать для создания нового сообщения, это нажать кнопку на панели инструментов, чтобы открыть пустую страницу сообщения. Можно также щелкнуть правой кнопкой мыши в поле сообщения От: и выбрать пункт Написать письмо, чтобы открыть пустую страницу для создания сообщения. В зависимости от настроек почты пустая страница откроется в области содержимого Zimbra или как отдельное окно создания сообщения.

В окне Написать письмо Zimbra, в поле Кому: введите адрес электронной почты пользователя или пользователей, которым должно быть отправлено сообщение. Для поиска адреса электронной почты пользователя среди своих контактов или в глобальном списке адресов щелкните поле Кому:. Чтобы ввести адреса в поле «Скрытая копия», нажмите кнопку Параметры на панели инструментов создания сообщения.

  • Если есть несколько вариантов личных данных почты, перед началом создания сообщения необходимо выбрать, какие личные данные использовать.

Сообщение можно создавать в формате HTML или в формате обычного текста. Формат по умолчанию настраивается на вкладке «Настройки->Общие». Для быстрого изменения формата только для текущего сообщения нажмите кнопку Параметры на панели инструментов создания сообщения и выберите «HTML» или «Обычный текст».

  • Формат HTML — позволяет использовать в сообщении различные виды шрифтов, создавать таблицы и добавлять цвет.

  • Формат обычного текста — используется текст без стилей и форматирования. Этот тип сообщений может быть прочитан на любом компьютере.

Создание нового сообщения

  1. На панели инструментов нажмите кнопку . Появится страница создания сообщения.

  2. Если личные данные по умолчанию не используются, в раскрывающемся списке От: выберите, какие личные данные (второе название — образ) использовать.

  3. Введите адрес, тему и текст сообщения.

Если ввести имя и фамилию в виде недопустимого адреса в Интернете (имя@домен.com), появится диалоговое окно подтверждения с сообщением, что адрес недопустимый. Можно выбрать отправку сообщения в любом случае, даже если оно может быть не доставлено.

Введите текст сообщения в текстовом поле, расположенном ниже поля Тема.

Для проверки правописания в сообщении нажмите кнопку Проверить правописание.

Чтобы добавить вложение, нажмите кнопку .

    • Если подпись на вкладке «Настройки->Подписи» есть, но не включена, нажмите кнопку Добавить подпись на панели инструментов «Написать письмо».

  1. Чтобы отправить сообщение, нажмите кнопку Отправить.

Если отправлять сообщение в данный момент не нужно, нажмите кнопку Сохранить черновик. Сообщение будет сохранено в папке Черновики.
