Поддерживаемые типы файлов вложений

В сообщение можно вложить любой файл, который можно найти с помощью кнопки «Обзор» в файловой системе.

Любой вложенный файл можно открыть прямо в веб-клиенте Zimbra, если на компьютере установлено соответствующее приложение и открытие файлов этого типа не заблокировано системным администратором.  Например, чтобы открыть документ, созданный с помощью Microsoft Word, вы должны иметь возможность запустить Microsoft Word на своем компьютере.

На большинстве компьютеров тип файла обозначается расширением в виде точки с последующим трехбуквенным кодом. В качестве примера можно взять файлы expenses.xls или New-Items.doc. Если тип файла поддерживается установленным на компьютере программным обеспечением, можно просто дважды щелкнуть этот файл и компьютер автоматически запустит нужное приложение для его чтения.

Файлы многих типов можно открывать непосредственно в браузере с помощью различных подключаемых модулей. Обычно, если щелкнуть на странице ссылку на файл, браузер автоматически определит, установлен ли необходимый подключаемый модуль, и предложит загрузить его. Имейте в виду, что политика администрирования системы может запрещать конечным пользователям загружать и устанавливать какое-либо программное обеспечение.

  • Примечание.
    В компьютерах с ОС Windows предполагается, что тип файла, указанный в расширении файла, совпадает с действительным форматом файла. Даже если у файла правильный внутренний формат, но нет правильного расширения в имени файла, компьютер может не распознать его и не откроет нужное приложение.

    Постоянно растущее количество компьютерных вирусов создается для распространения с помощью вложенных файлов. Поэтому в некоторых средах системные администраторы вынуждены блокировать входящую почту, содержащую вложения определенных типов, в особенности файлы с расширениями .EXE или .ZIP в имени файла. При отправке сообщения пользователю, система электронной почты которого настроена на блокировку вложений определенных типов, отправитель может получить или не получить извещение о блокировке.

Для справки ниже приведены некоторые широко используемые расширения файлов.

  • .TXT — текстовый файл, т.е. файл, содержащий только текстовую информацию. Текстовые файлы можно открывать и читать с помощью большого количества текстовых редакторов, включая Блокнот, WordPad, Word, редакторы командной строки, браузеры и даже программы обработки электронных таблиц.

  • .HTM — HTML-файл, который является особым типом текстового файла, обычно открываемого с помощью браузера.

  • .ZIP — файл, который был сжат с помощью ZIP-сжатия. Необходима программа WinZip или другая программа сжатия, которая может читать и распаковывать ZIP-файлы.

  • .EXE — исполняемый или программный файл. Дважды щелкнув мышью такой файл обычно можно запустить приложение.

  • .DOC — файл документа Microsoft Word.

  • .JPG — графическое изображение в формате JPEG. Графические изображения в формате JPEG можно открывать в браузере или с помощью одной из множества программ для работы с изображениями, например Paint или Adobe Photoshop.

  • .GIF — графическое изображение в формате GIF. Графические изображения в формате GIF можно открывать в браузере или с помощью программ для работы с изображениями.

  • .MPG — видеоклип или анимацию в формате MPEG. Зачастую воспроизводится с помощью подключаемого модуля браузера или программами воспроизведения, такими как Windows Media Player.

  • .WMV — видеоклип в формате WMV, который поддерживается только в ОС Windows.

  • .PDF — документ в формате Acrobat Reader. Обычно он открывается в браузере с помощью подключаемого модуля.
