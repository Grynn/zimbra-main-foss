Автоматическое добавление подписи

Можно создавать подписи для сообщений. Подпись может включать имя и дополнительный текст длиной не более максимального количества знаков, допустимого для вашей учетной записи.

Например, подпись может быть такой:

Иван Сидоров
Заместитель директора по технической части
Отдел штуковин
Контора фиговин Inc.
303-555-1212 x111
ivan.sidorov@example.com

Если вы создаете несколько личных данных, можно создать несколько подписей и назначить их разным адресам. См. раздел Личные данные почты.

Cоздание подписи

  1. Откройте Настройки и выберите вкладку Подписи.

  2. В поле Имя подписи введите подпись и соответствующее имя.  Это не ваша подпись.

  3. В текстовом окне Подпись введите данные подписи точно так, как вы хотите их видеть в сообщениях.

  4. Нажмите кнопку Сохранить.

  5. Перейдите на вкладку «Учетные записи» и выберите подпись для отображения в сообщениях. В Настройках основной учетной записи, в поле Подпись выберите имя подписи, созданной при выполнении шага 2.

  6. Нажмите кнопку Сохранить.
