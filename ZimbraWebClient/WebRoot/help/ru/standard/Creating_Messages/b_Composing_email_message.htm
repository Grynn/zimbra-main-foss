Создание нового сообщения

 

  1. Для создания нового сообщения электронной почты перейдите на вкладку Написать письмо. Появится пустая область создания сообщения.  

  2. Введите адрес электронной почты пользователя или пользователей, которым должно быть отправлено сообщение, в поле Кому. Также можно ввести адреса в поле Копия или Скрытая копия. Для поиска адреса электронной почты пользователя на панели инструментов создания сообщения нажмите кнопку . В списках контактов можно осуществлять поиск конкретных имен. Выберите, в какое поле необходимо ввести имя – «Кому», «Копия» или «Скрытая копия».

  3. В поле Тема введите тему сообщения.

  4. Введите текст сообщения в текстовое поле, расположенное ниже поля «Тема».

При наличии настроенной подписи она будет автоматически добавлена в сообщение.

  1. Чтобы добавить вложение, щелкните Вложить файлы.

    Добавление вложений

    Можно вложить до 10 файлов.

    1. Нажмите кнопку «Обзор», чтобы найти файл.

    2. Выберите файл  и нажмите кнопку «Открыть». Имя файла и папки появятся в списке.

    3. Вложив файлы, нажмите кнопку Готово. Имена вложенных файлов отображаются под полем «Тема».

    Примечание. В случае решения не отправлять файл снимите флажок.

  2. Чтобы отправить сообщение, нажмите кнопку Отправить. Если включена функция Сохранять копии сообщений в папке Отправленные, сообщение будет сохранено в папке «Отправленные».

Если отправлять сообщение в данный момент не нужно, нажмите кнопку Сохранить черновик. Сообщение будет сохранено в папке «Черновики».
