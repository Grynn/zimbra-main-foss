Создание новых ежедневников

Для пользователя создается один ежедневник, также можно создавать дополнительные ежедневники для ведения учета функций различных типов.  Например, можно создать ежедневник для рабочих, личных мероприятий и мероприятий организации, можно подписаться на доступный в Интернете ежедневник, а также можно подключиться к общему ежедневнику.

Можно использовать вид «Расписание» для одновременного просмотра нескольких ежедневников.

Cоздание ежедневника

  1. Перейдите на вкладку «Ежедневник».

  2. В области «Обзор» ежедневника щелкните пункт Изменить.

  3. На панели инструментов нажмите кнопку Новый ежедневник.

  4. Введите название ежедневника и выберите цвет для отображения мероприятий данного ежедневника. Если просмотр другими лицами мероприятий ежедневника нежелателен,  установите флажок Исключить этот ежедневник при составлении отчета о занятости.

  5. По умолчанию параметр Помечено в UI включен. Мероприятия данного ежедневника будут отображаться при открытии вида «Ежедневник».  Если данный ежедневник не нужно отображать вместе с другим ежедневником, снимите флажок в этом поле. В таком случае для просмотра мероприятий ежедневника необходимо выбрать ежедневник и открыть его.

  6. Щелкните пункт Создать ежедневник.

Подписка на другой ежедневник

  1. Перейдите на вкладку Ежедневник.

  2. В области «Обзор» ежедневника щелкните пункт Изменить. Отобразятся сведения ежедневника по умолчанию.  

  3. На панели инструментов нажмите кнопку Подписаться на ежедневник.  

  4. Введите название ежедневника, введите допустимый URL-адрес файла iCal удаленного ежедневника и выберите цвет для отображения мероприятий данного ежедневника.

  5. Щелкните пункт Создать ежедневник.
