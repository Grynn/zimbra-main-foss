Открытие вложений

Электронные сообщения могут содержать вложенные файлы: документы обработки текста, электронные таблицы, текстовые файлы, ZIP-файлы, изображения, исполняемые файлы и любые другие компьютерные файлы, которые могут находиться в компьютерной сети отправителя. В сообщениях, содержащих вложения, рядом с темой отображается скрепка.

Некоторые типы файлов, например, изображения формата JPEG или текстовые файлы, можно открыть практически на любом компьютере. Другие типы файлов, например документы Microsoft Word, должны открываться в том же приложении, в котором они были созданы.

Получив сообщение с вложениями, щелкните скрепку, указывающую на вложение в заголовке, чтобы перейти вниз сообщения. Внизу сообщения можно выбрать варианты: просмотр вложения, просмотр вложения в виде HTML или загрузка вложения.

  • Для просмотра файла на компьютере должно быть установлено приложение, с помощью которого можно прочитать этот тип файла.

  • При загрузке вложения откроется диалоговое окно с отображением текущей файловой системы. Выберите соответствующую папку и сохраните файл. Файл можно открыть позже не в почтовом клиенте.

Просмотр вложений в виде HTML

Может быть доступна функция просмотра вложений в виде HTML. Она позволяет открывать вложения, даже если соответствующее приложение не установлено.  Щелкните ссылку просмотреть в виде HTML, чтобы просмотреть содержимое файла. Эта функция устанавливается системным администратором.

Вирусная безопасность

Компьютерные вирусы могут распространяться через почтовые вложения. Стандартной практикой у почтовых администраторов является использование вирусных фильтров для входящей почты, а также сканирование вирусов на персональных компьютерах. Однако всегда существует опасность проникновения вируса.

Что касается почтовых вирусов, то большинство из них активизируется при открытии или просмотре файлового вложения, содержащего вирус. Известно, что многие из этих вирусов распространяются через записи в адресных книгах пользователя. Поэтому мы рекомендуем проявлять осторожность при открытии файловых вложений даже от известных отправителей.
