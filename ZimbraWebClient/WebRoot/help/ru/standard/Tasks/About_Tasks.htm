О задачах

Функция «Задачи» ZCS позволяет создавать список задач и управлять их выполнением. Задачи можно добавлять в список задач ZCS по умолчанию, можно создавать другие списки задач для систематизации списков задач по более конкретным видам деятельности, например, по работе или личным проектам.

Для простой задачи нужно лишь создать задачу в области содержимого. Если задачей нужно управлять, можно добавить дополнительные данные, определить дату начала и окончания, установить приоритет задачи — высокий, обычный, низкий — и проследить выполнение задачи.
