xxxxНазначение меток сообщениям и контактамxxxx

xxxxТеги xxxxxxxx—xxxxxxxx это ваша личная система систематизации почтовых сообщений, контактов и встреч.xxxx xxxxТеги помогают найти эти элементы.xxxx xxxxНапример, вы назначили один тег для элемента «Немедленный оборот» и другой тег для элемента «Средний приоритет».xxxx xxxxПомеченные тегом почтовые сообщения могут находиться в разных папках.xxxx xxxxТеги позволяют показывать все сообщения с определенными тегами где бы они не находились.xxxx

xxxxМожно создать и применить несколько тегов к одному элементу, чтобы систематизировать его сразу по нескольким категориям.xxxx xxxxПосле создания тега его можно применить к разговорам, сообщениям или контактам.xxxx

xxxxСоздание нового тегаxxxx

  1. xxxxВ заголовке «Тег» области просмотра нажмите кнопку xxxxxxxxИзменитьxxxxxxxx.xxxx

  2. xxxxВ области содержимого щелкните xxxxxxxxНовый тегxxxxxxxx.xxxx

  3. xxxxВведите имя тега и выберите для него цвет.xxxx

  4. xxxxВыберите пункт xxxxxxxxСоздать новый тегxxxxxxxx.xxxx xxxxНовый тег появится в области обзора тегов.xxxx

xxxxОдновременное применение и удаление тега для нескольких элементовxxxx

  1. xxxxУстановите флажки для элементов, которые нужно пометить тегами.xxxx

  2. xxxxВ раскрывающемся меню xxxxxxxxДополнительные действияxxxxxxxx выберите тег.xxxx

  3. xxxxНажмите кнопку xxxxxxxxПерейтиxxxxxxxx.xxxx

xxxxУдаление тега из элементаxxxx

  1. xxxxУстановите флажки для элементов, теги которых нужно удалить.xxxx

  2. xxxxВ раскрывающемся меню xxxxxxxxДополнительные действияxxxxxxxx в разделе «Удалить тег» выберите тег для удаления.xxxx

  3. xxxxНажмите кнопку xxxxxxxxПерейтиxxxxxxxx.xxxx
