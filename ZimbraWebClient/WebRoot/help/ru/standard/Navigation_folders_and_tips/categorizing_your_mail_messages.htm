Систематизация почтовых сообщений

Веб-клиент предоставляет несколько способов систематизации почтовых сообщений. Ими можно пользоваться вместе с функцией поиска для быстрого нахождения и выборки сообщений. Это следующие методы.

  • Фильтры. Открыв вкладки «Настройки», «Фильтры почты», можно создать правила систематизации входящих сообщений по заголовку, отправителю, получателю или другим параметрам.

  • Папки. Можно создать папки для систематизации сообщений, можно также перемещать сообщения из одной папки в другую, например, можно переместить сообщения из папки «Задачи» в папку «Выполнено».

  • Теги. Теги используются как метки почтовых сообщений, которые потом можно найти с помощью функции поиска и отобразить все сообщения с определенным тегом.

  • Разговоры. Сообщения можно группировать по темам разговоров. Эта встроенная функция автоматически группирует все отправленные и полученные сообщения с одной и той же темой в один разговор.
