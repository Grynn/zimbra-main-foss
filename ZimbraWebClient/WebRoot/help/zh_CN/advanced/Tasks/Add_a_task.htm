添加任务

您只需在任务列表中输入任务名称即可创建一项新任务，也可以打开任务页面并添加有关该任务的详细信息，其中包括注释和附件。

  • 在处理邮件，参加会议或查看联系人时，可用 Zimbra 助手添加任务。 创建任务后，所有可用的信息均被添加到“任务”区。

若需快速添加任务

  1. 从窗格中选择任务列表

  2. 在“内容”窗格中，单击第一行，其上显示的是单击此处添加新任务。    

  3. 屏幕上随即显示一个字段。键入任务的名称。

  4. 按 Enter 键。 此任务即被添加到您的任务列表中。

  5. 若误将任务放入了另一列表，可打开任务并在“任务列表”下拉选单中选择正确的列表，或将“内容”窗格中的如任务拖放到正确的列表中。

若需添加有详细资料的任务

您可使用任务列表创建任务并跟踪其进程。 另外，您可在任务和附加文件中书写注释，以方便任务的处理。当您与他人共享任务列表时，列出与任务相关的所有信息是必要的。

 通过输入开始日期、结束日期和设置优先级，您可估算出项目的时间长度。启动任务时，您可选择其状态，如未开始和进行中，并可选择完成的百分比。

  1. 从“浏览”窗格中选择任务列表。

  2. 在“任务”工具条中，单击新建。  一个无标题任务页面随即打开。

  3. 在主题字段，输入任务名称。

  4. （可选）在日期部分，选择优先级、开始日期、结束日期以及状态。

  5. 如果在打开所需任务列表之前单击了“新建”，则可从任务列表选单中选择所需的任务列表。  

  6. 您可在文字区中添加注释，若需添加附件，可单击工具条上的添加附件。

  7. 单击保存便可保存该任务。该项新任务随即被添加到您的任务列表中。
