用户界面功能和快捷键

Zimbra Web Client 界面包括许多方便的功能。

  • 下箭头选单。有些指令按钮的右面有一个小小的下箭头图标。单击该按钮便可执行一项功能，单击下箭头选单即可显示更多的选项。

  • 拖放。 单击并在文件夹之间拖动邮件或会话。

  • 右键选单。右键单击文件夹、会话、邮件、邮件或会话中的“发件人”列等项目或邮件标题中的项目时，一个选单随即显示，列出可施加于该项目的各种功能。此例为用右键单击发件人时的情况。

  • 可移动边框 对于分隔窗格，如“会话”视图，您可用边框增大或缩小上下窗格的相对大小。单击并拖动分隔窗格的横条。

 

  • 可点击 项。Zimbra 遵循标准浏览器的惯例，当鼠标经过某个活跃链接上时，即变为“手”状。

  • 提示工具。 对于某些项，如指令按钮、会话以及邮件标题中的邮件地址，当鼠标经过上面的时候，就会显示一个小框，用以显示相关的说明。

 

  • 翻页。 如果列表太长而无法在一个窗口中显示，可用列表窗口的左右箭头前后翻页。请注意，翻页按钮在不同的画面上可能有不同的功效。数字表示邮件在页面中的顺序。您可用提示工具的显示，了解翻页功能在当前画面是什么功能。
