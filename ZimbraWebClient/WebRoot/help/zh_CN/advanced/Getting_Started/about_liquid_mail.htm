Zimbra Collaboration Suite 开源伙伴

 

欢迎使用 Zimbra™ Web Cient, Advanced Edition

Zimbra Collaboration Suite 是一全功能的通信及协作办公应用程序，提供可靠和高效能的邮件、地址簿、效率手册、任务列表以及网络文档制作功能。  

  • 所用帐户有可能不包括上述全部功能。若需了解您的帐户都启用了哪些功能，请与您的系统管理员联系。

从 Zimbra 网络客户端 (ZWC)，您可使用下列功能。

电子邮件

  • 写信和发送邮件

  • 阅读和回复邮件

  • 透过会话功能跟踪来往邮件，也可转换到传统邮件视图

  • 在邮件中加入附件

  • 将邮件转发给一个或多个收件人

  • 通过特征或指定的文字搜索邮件和附件

  • 创建您自己的文件夹用以整理邮件

  • 创建标记用以整理邮件

  • 创建过滤器以便将来信汇入指定的文件夹

  • 设置不同的帐户身份和地址，用以管理您在工作和个人生活中的不同角色。

  • 设定您的帐户，以便从 ZWC 邮箱直接收取 POP3 或 IMAP 帐户的邮件。

  • 授权他人查看或管理您的邮件文件夹

地址簿

  • 创建和管理多个地址簿

  • 导入和导出联系人列表

  • 共享个人地址簿

效率手册

  • 创建和管理多个效率手册

  • 创建约会、会议和事件

  • 授权他人查看或管理您的效率手册

  • 查看参会者日程的空闲/忙碌状态

  • 导入/导出其他效率手册程序。

任务

  • 创建多个“任务列表”

  • 创建待办事项

  • 将附件添加到任务

  • 管理任务、设置优先级和跟踪进度

  • 授权他人查看或管理您的“任务列表”

文档

  • 创建多个笔记本，用以整理您的网络文档

  • 使用 HTML 编辑器设计页面，创建表格和电子表格，添加图片，创建网页链接

  • 授权给所在组织机构中的组或个人创建和编辑您的文档笔记本

  • 允许公众查看您的文档笔记本

公文包

  • 上传任何类型的文档到您的公文包，以便随时登录 ZWC 存取这些文件

  • 创建文件夹，用以整理上传的文件

  • 授权他人查看或管理您的“公文包”文件夹

即时通信 (IM) Beta

  • 即时通信

  • 创建聊天组或与好友聊天

  • 建立好友名单并查看他们的在线状态

  • 选择您的在线状态或创建自定义状态讯息

  • 通过电子邮件快速发送您的聊天内容

  • 聊天内容自动保存到“聊天”文件夹

首选项

  • 设定邮箱的各项功能，使之与邮件、地址簿和效率手册配合使用

  • 创建邮件过滤器

  • 设置快捷键，用于快速存取邮件文件夹、保存的搜索项和标记
