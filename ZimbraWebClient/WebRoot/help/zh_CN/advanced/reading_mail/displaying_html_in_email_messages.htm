显示 HTML 格式邮件

您可用两种格式查看电子邮件。

  • 纯文本。  纯文本邮件即不含 HTML 格式的邮件。以纯文本格式发送的邮件，在其他任何邮件客户端中都可以阅读。即便邮件中含 HTML 格式，您也可以用纯文本模式阅读。在纯文本视图中，HTML 标记会在邮件的正文部分显示。

  • HTML。HTML 邮件可具有文本格式、编号、项目符号、背景色以及链接，以便阅读。有些电子邮件客户端可能不接受 HTML 格式的邮件。

若需始终以 HTML 格式查看邮件，请勾选首选项，邮件标签上的以 HTML 格式查看邮件 (若可能)复选框。 如果是以 HTML 格式发送的，邮件将显示为格式化的版面。
