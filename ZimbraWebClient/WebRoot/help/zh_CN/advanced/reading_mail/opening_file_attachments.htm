打开附加文件

邮件中可含附加文件，例如文字处理文档、电子表格、文本文件、ZIP 文件、图像、可执行应用程序、以及保存在发件人计算机网络中的任何其他类型的计算机文件。含附件的邮件通常会在主题旁边后显示回形针图标。

有些类型的文件，如 JPEG 图像文件或文本文件，几乎可在任何计算机系统上打开。其他文件，如Microsoft Word 文档，则必须使用创建时使用的程序才能打开。

在收到含附件的邮件时，可双击邮件标题处的附件部分。一个对话框随即出现，询问是打开还是保存该文件。

  • 如果选择查看附件，工作站上则必须装有可阅读该类型文件的应用程序。

  • 如果选择下载附件，则会显示一个浏览对话框，并显示当前的文件系统。选择一个合适的位置并保存文件。此后，您便可在邮件客户端打开该文件。

以 HTML 查看附件

您可能具备以 HTML 格式查看附件的功能。这样，即便您没有安装相应的应用程序也可打开附件。  单击以 html格式查看便可查看文件的内容。此功能由系统管理员设定。

防范病毒

计算机病毒可通过邮件附件传播。邮件系统管理员可按标准作法对邮件进行病毒过滤，并对个人计算机进行病毒扫描。但是，总还是有一些病毒漏网。

这种邮件病毒，大多数是在用户打开或查看附件（含病毒者）时发作的。众所周知，这类病毒许多是利用用户地址簿上的条目进行传播的。故建议用户在打开附件时要格外小心，即便发件人是您认识的人。
