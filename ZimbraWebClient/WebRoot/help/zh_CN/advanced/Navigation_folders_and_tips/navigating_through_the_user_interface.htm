浏览 Zimbra Web Client

如下图所示，登录到 Zimbra Web Client 后，您会看到一个全屏视图。

Zimbra Web Client 页面由以下区域组成：

  1. 浏览器控件。可显示标准的浏览器按钮，如主页、当前地址及其他功能，具体内容取决于所用的浏览器软件和设置。浏览器工具条上的按钮不属于 Zimbra Web Client，您访问的每个网页都会显示这些按钮。

  2. 搜索工具条、用户名以及限额显示。搜索、高级搜索、您登录时使用的用户名以及您的邮箱限额显示在此区域。

  3. 应用程序标签。可使用的应用程序列在顶端的各个标签中。右边是帮助和退出。

  1. 工具条。 显示当前使用的应用程序的可用操作项。在此例中显示的是邮件应用程序工具条。

  2. 窗格。显示您的文件夹，包括系统文件夹收件箱、 发件箱, 草稿 垃圾邮件和已删除邮件，以及您所创建的任何自定义文件夹、已保存的搜索项和您创建的标记、从邮箱创建的用于与第三方应用程序集成的Zimlet 链接。如果您在“效率手册”中，窗格会显示您的效率手册列表和 Zimlet。

  3. 在窗格下可显示一个小日历。小日历是一选项，可在首选项的效率手册标签中启用。

  4. 内容窗格。此区域中的内容取决于所使用的应用程序。在“收件箱”视图中，内容窗格显示的是收件箱中所有的邮件和会话。
