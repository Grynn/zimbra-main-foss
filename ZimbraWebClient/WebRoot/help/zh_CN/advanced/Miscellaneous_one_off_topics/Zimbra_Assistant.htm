Zimbra 助手

Zimbra 助手是一种高级用户功能，可用键盘与网络客户端快速交互，以完成下列任务：

  • 创建新约会

  • 转到您效率收册中的某一日期

  • 创建新联系人

  • 可在不离开当前正在使用的应用程序的情况下发送新邮件

除了“首选项”标签之外，您可从 Zimbra Web Client 视图的任何地方打开 Zimbra 助手。打开 Zimbra 助手的快捷键是 ` （后引号）或 shift~（波浪号）。此键位于 Escape （Esc）键下面。

  • Shift ~ （波浪号） 在 Mac OS X 下使用 Safar 或 Firefox 时不起作用。后引号（`）起作用。

Zimbra 助手对话框提供详细的帮助信息。以下是使用 Zimbra 助手的一般步骤。

  1. 在 Zimbra Web Client 中，使用快捷键（后引号或 shift 波浪号）。Zimbra 助手对话框随即显示。

  2. 输入下列其中一个指令

    • 若需创建约会，键入约会。然后在引号内输入主题、时间、日期和星期以及对约会的任何注释。单击确定。 备注：如果需要更高级约会功能，可单击更多详细资料，以前往效率手册的约会页面。

    • 若需前往您效率手册中的某一具体日期，可输入效率手册和日期或星期。单击确定。 备注：在日期之后，您可在相关的“效率手册”视图中输入日、工作周、周或月。“效率手册”即以所选视图打开。

    • 若需创建新联系人，键入联系人并输入联系人姓名、邮件地址等。最后单击确定。 备注：如果想打开“联系人”的详细资料格式，可单击“更多详细资料”。在这个格式中，您可指定向哪个地址簿添加联系人。

    • 若需在不打开写信窗口的情况下发送邮件，可键入邮件。在引号内输入主题；键入 to:name@adress ，然后输入邮件正文。 单击确定。
