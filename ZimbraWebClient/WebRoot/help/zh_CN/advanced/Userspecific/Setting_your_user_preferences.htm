设置用户首选项

默认的用户首选项是在创建您的帐户时配置的。 这些选项界定了您的邮箱、地址簿和效率手册应用程序的使用方法。您还可以设置个人邮件 ID和 POP 帐户，并为您的邮箱选择一个基调主题。

您可在首选项标签中改变这些设置。如果此处描述的选项在您的视图中看不到，则说明该选项对于您的帐户不适用。

您可在下列标签中设置首选项。单击相关字词后，可获得相关标签的详尽说明。

  • 常规

    • 您登入的是高级网络客户端。要将此设置为登录时使用的默认客户端类型，请选择高级。

    • 高级具有全套的网上协同办公功能。此种网络客户端最适合于较新版的浏览器和更快的互联网连接。

    • 如果互联网连接速度较慢、所用浏览器为早期版本或为了访问更简易，可采用标准。

    • 在主题处选择邮箱的背景色。

    • 如果此处显示的时区不正确，请为您的帐户选择默认时区。时区功能可确定您发送和接收邮件的正确日期和时间，并正确显示您“效率手册”中的约会时间。  这可能与所用计算机上所配置的时区不同。

    • 如果显示变更密码，您可修改您的密码。如果在变更密码时遇到问题，请向您的管理员询问密码规则。

      • 如果没有此选项，但您需要变更密码，请与系统管理员联系。

    • 在搜索设置中，您可选择自动在每次执行搜索时包含“垃圾邮件”和“已删除邮件”文件夹。默认情况下，这两个文件夹是不搜索的。

    • 在搜索语言中，勾选总显示搜索字符串以便在搜索字框中显示搜索字符串的细节。

    • 启用总显示选项复选框  ，以便“内容”窗格中的每个条目都显示一个复选框。通过此复选框，您可以选择一项或多项同时执行形同的操作，例如删除、移动、标记为已读/未读。

  • 邮件

    • 选择每页显示多少条目（邮件或会话）。默认设定值为每页显示 25 个条目，但您可选择 10、50 或 100。

    • 选择邮件如何分组，按会话或按邮件。如果有“会话”功能，则“会话”是默认的。

    • 轮询间隔是指您的计算机检查新邮件的频率。 默认间隔时间为 5 分钟。最短的时间长度由 Zimbra 管理员设定。

    • 对于已经格式化为 HTML 的邮件，可指定是否以 HTML 方式查看。若关闭此项，邮件则以纯文本形式显示，即只显示 HTML 标记，但不显示标记语言版本。

    • 邮件预览选项。

      • 用于设置查看邮件时是否显示阅读窗格。默认设置为总显示阅读窗格。

      • 若启动此项，客户端会在邮件列表中显示邮件片段。当光标经过邮件主题时，邮件开头的部分文字会显示出来。

      • 若启用此项，双击后可在新窗口中打开邮件

    • 启用图像后，HTML 邮件便可自动下载图片。若启用，您无需单击“显示图像邮件”就可以看到图像。

    • 默认的“邮件搜索”为登录时执行搜索。默认的搜索目标是您的收件箱。  在您登录时，“收件箱”的搜索结果会即刻显示。 您可修改此字段，以使在刚打开 Zimbra 网络客户端时显示的是其他的文件夹、标记或保存的搜索。有关句法，请见查询语言说明。启用并指定一个地址，以便系统发送关于收到新邮件的通知。

    • 指定一个邮件转发地址。您可指定将原始邮件从 Zimbra 邮箱中删除。

    • 指定是否对收到的邮件发送一个自动回复讯息。 自动回复讯息经常被用作外出不在讯息，或休假讯息，以表明您不在办公室。

    • 设定“我发的邮件”部分，用以处理以下两种情况：您发送的邮件，但您自己的地址是邮件的收件人之一；或所含地址为包括您自己在内的邮件列表。

     

  • 写信

    • 设置默认的文本编辑器，HTML 或纯文本。

    • 设置以 HTML 格式书写邮件、文档和效率手册时所使用的默认字体。 您可确定字体的样式、大小和颜色，用以作为默认的样式。 默认字体为 Arial、12 point black。

    • 总在新窗口中书写，这一选项可使您在书写邮件时打开一个与 Zimbra 邮箱分开的新书写窗口。在书写窗口打开的情况下，您可查看和浏览您的邮箱。

    • 每隔几分钟，您所书写的邮件都会自动保存到草稿箱。如果不需要此自动保存功能，可关闭此功能。

    • 选择是否在您的回复邮件正文中包含原始邮件。您可选择包含原始邮件，不包含原始邮件，以附件形式包含原始邮件，在正文中包含但加上您选的前缀，或只包含最近的邮件。 （也就是说，只引用发件人写下的内容，而不是前面的文本。）

    • 如果选择引用原始文本，请指定是否需要为转发或回复的前一封邮件的每一行加上前缀 > 或 |。

    • 默认情况下，在“已发送邮件”文件夹保存副本为启用状态。发送邮件的副本在“已发送邮件”文件夹中保存。

    • 指定在转发邮件时如何包含邮件的原始文本：放入邮件正文、加上前缀放入邮件正文或作为附件一并转发。

  • 签名

    您可为您的邮件创建各种签名。签名中可包括姓名和其他结束语句，帐户设定中对语句的长短和字符数量上限均有限制。如果您有多个身份，则可创建不同的签名，并分别指派给具体的地址。

    • 输入您的签名。

    • 指定签名在邮件中的位置：将签名添加到邮件的最下面，或添加到您书写、回复或转发的文本的结束处。

  • 地址簿

    • 启用此功能后，您可在发送邮件时自动将地址添加到地址簿。如果启动此功能，地址将保存到您的“邮件联系人”文件夹中。

    • 启用后，可在自动完成地址时使用“全局地址列表”。

    • 选择以何种形式查看联系人，列表或名片。

    • 选择每页显示的联系人的人数。默认值为 25。

    • 导入联系人到您的“联系人”列表。导入的文件必须是逗号分隔格式。

    • 导出联系人列表。导出的文件采用逗号分隔格式。

     

  • 帐户

    帐户标签用于创建和管理您的身份。身份是一种邮件 ID，在您发送邮件的“发件人”字段中使用。

    默认的样式在主标识中定义。在创建新的标识时，可使用与“默认标识”相同的设置，也可设置不同的首选项。

    添加外部帐户按钮用于识别您的外部 POP/IMAP 帐户，从而使网络客户端能从您的其他帐户收取邮件。 您可通过设置，使回信也使用相应的 POP/IMAP 地址。

  • 邮件过滤器

    使用“邮件过滤器”标签定义邮件过滤规则。若需了解如何设置邮件过滤规则，请参见过滤工作原理。

  • 效率手册

    • 用于选择如何查看 Zimbra 效率手册。默认值为按照工作周。

    • 指定在效率手册中把哪一天作为一周的第一天。

    • 如果经常往返于不同的时区，应该选择在约会视图中显示时区列表。

    • 若需创建许多没有参会者的约会，请勾选使用“快速添加”对话框。

    • “效率手册”中会显示一个小日历。 您可在任何视图中启用小日历。

    • 设定在约会前多少分钟进行提醒。默认值为五分钟。

    • 导入或导出其他效率手册程序。

  • 即时通信

    您可设置如何将即时通信（IM）集成到您的帐户。

    • 登录。 如果您需要自动登录到即时通信服务，须启用此项。

    • 通知。 此选项可用来设定如何接收IM通知，以及您希望别人如何看到您的在线状态。

    • 聊天。 您的聊天记录会自动保存到“聊天”文件夹。此文件夹中的邮件与任何电子邮件一样，可以打印、作为邮件转发或从文件夹中删除。您还可以设置空闲超时。

  • 快捷方式

    快捷键可使您在使用应用程序时不用鼠标。有大量的快捷键可用。您也可以创建自定义的快捷键，方法是：将数字别名指派给文件夹、标记和保存的搜索。

    • 快捷键列表显示了全部现有的快捷键。

    • 邮件文件夹快捷键标签用于创建能快速转到您的邮件文件夹并将邮件移至该文件夹的快捷键。您须选择一个文件夹，然后便可指定一个数字作为快捷方式。例如，将 3 指派给“收件箱”文件夹：在邮箱的任何地方，键入v3可使您转移到“收件箱”。选择一个条目然后键入 .3，将把该条目移动到“收件箱”。

    • 已存搜索项快捷方式标签用于创建一个快捷键，用以快速使用已保存的搜索项。您须选择一个保存的搜索项，然后便可指定一个数字作为快捷方式。例如，如果把 3 指派给您常用的一个搜索项，则从您的邮箱的任何地方，可通过键入s3快速执行该搜索。

    • 用标记快捷键标签创建的快捷键，可用于快速存取具有指定标记的所有条目，或将一个标记应用于邮件或联系人。您须选择一个标记，然后便可指定一个数字作为快捷方式。例如，将 Follow up 标记赋予数字 3 后，您可从邮箱的任何地方键入 y3 即可看到所有标记为 Follow up的项目。选择一个条目并键入 t3 ，便可将该条目标记为 Follow up。

    您可指定任何数字，但不能在同一快捷键标签内重复。
