使用 rich text 编辑器

编写页面可用 rich-text 编辑器格式化页面内容，或用  HTML 源码通过 HTML 标记格式化页面。可在两种格式之间来回切换，以便利用 rich-text 的编辑功能并插入表格和电子表格。

在用 Rich-text 编辑器时，您可通过双行工具条格式化页面布局。选择字号、字体、文本样式、字体颜色和背景颜色。这些格式化工具的用法与许多文字处理工具相同。

另外，您可透过单击的方式将表格、电子表格、附件和图像添加到页面中。

添加段落样式

工具条上的    图标显示有可用的段落样式。利用这些段落样式，您可轻而易举地 在笔记本页面中格式化文本并保持一致的风格。这些样式包括：用于大多数段落的普通选项目，表示章节标题的六级标题，可使文本变为斜体的 字型，以及可将文本显示为 Courier 字体时使用的预定的格式。这些样式不能修改，但可使用各种编辑选项分别对某一样式进行格式化处理，如改变字体、字号、颜色等等。

在页面上添加表格

添加表格到页面时，您须分三步进行。即添加表格，配置表格属性和向表格添加内容。若需添加表格，单击工具条上的表格图标，选取“插入表格”，然后选择表格的行数和列数。

在页面上添加电子表格  

要添加电子表格，单击工具条上的电子表格图标 。一个六列电子表格即被添加到页面中。电子表格可用于创建 数据库，记录活动、日程安排、任务等事项的完成情况。电子表格包括基本的电子表格功能和一系列预定义的函数。

在页面上添加附件和图像

您可上传“文档”页面所链接的文件。单击别针图标 ，或图像图标  ，然后寻找要上传的文件。如果要上传的文件在您的“文档”中或“公文包”中已经存在，您会得到提醒。

链接到笔记本页面或网页

您可以链接到别的“文档”笔记本页面或一个外部的网络 URL 并 为该链接配一个描述性的标题，这样 URL 地址就不会显示出来。
