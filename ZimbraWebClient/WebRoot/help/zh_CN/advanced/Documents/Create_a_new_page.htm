创建新页面

您可创建新页面并通过 rich-text 格式工具或输入 HTML 标签来设计内容版面。您可检查拼写，并用关键字创建前往其他页面的链接。

  1. 单击要在其中工作的笔记本。目录页面随即在内容窗格中显示。

  2. 单击工具条中的新建。一个空白页面随即打开。在标题页面字段中，输入新页面的名称。名称随即在目录中显示。

  3. 书写页面内容。您可使用默认的 rich text 编辑器在页面中添加样式、颜色、表格和电子表格。

如果您熟悉 HTML 并想使用 HTML 标签，单击工具条上的格式，然后选择 HTML 源码。页面随即变为一空白字段，此时您就可以使用标准的 HTML 标记语法创建页面了。

  1. 单击保存保存该文件。您现在可向页面添加细节，也可单击关闭回到目录页面。
