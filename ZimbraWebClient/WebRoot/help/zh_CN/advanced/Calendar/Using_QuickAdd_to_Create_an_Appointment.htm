使用快速添加创建约会

“快速添加”对话框可用来快速创建约会。在效率手册上，先选择开始时间，双击或拖曳时间，“快速添加约会”对话框随即出现。

快速添加功能是一个选项。如果您习惯以双击或在效率手册上拖曳方式打开“约会”页面，请在“选项”页面关闭此功能。

若需快速创建一个约会

  1. 在“效率手册”上，单击约会的时间 或单击约会的开始时间并拖放到结束时间。 快速添加约会对话框随即出现。

  2. 要从“日”或“周”视图中创建全天约会，请在效率手册的顶端，单击并拖放划过一天或几天。

  3. 输入主题、地点，如果是重复约会，选择重复模式。

  4. 单击确定。 该约会即显示在效率手册上。

若需邀请参会者或改变重复选项，可单击更多详情，以打开“约会”页面。
