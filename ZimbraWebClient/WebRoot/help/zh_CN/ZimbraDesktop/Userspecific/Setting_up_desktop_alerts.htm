设置桌面提示和提醒

您可为新邮件通知、约会提醒和即时讯息通知创建桌面提示。要接收提示，您的邮箱必须打开。

  • 电子邮件提示。在“首选项”>“邮件”文件夹中设置您的首选项。有下列提示选项：

    • 播放声音。当有新邮件到达收件箱时就会发出“哔”声提醒您。如果把音量调到静音，则听不到声音。须安装 QuickTime 或 Windows Media。

    • 高亮显示“邮件”选项卡。如果没有在“邮件”选项卡操作，则在新邮件到达收件箱时，邮件选项卡会高亮显示。

    • 显示弹出的通知。当新邮件到达时，就会弹出提示消息。您必须安装 Yahoo!BrowserPlus 。

  • 约会提醒提示。在“首选项”>“效率手册”文件夹中设置您的首选项。在设置提示之前，必须先为您的会议设置提醒时间。提醒时间最早可设定在会议前 60 分钟。提醒对话框将在会议前的指定时间显示。如果您将提醒设置为从不，即便已启用提醒选项，您也不会得到会议提醒。

如果您不在 ZWC 效率手册，ZWC 提醒信息不会弹出在页面上。启用下列选项之一有助于提示您有提醒信息弹出：

    • 播放声音。当有约会提醒信息弹出时发出“哔”声提醒您。如果您把音量调到静音，您将听不到声音。

    • 闪烁显示浏览器标题。当有约会提醒信息弹出时，浏览器将会闪烁，直到您关闭弹出的窗口。
