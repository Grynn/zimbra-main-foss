Zimbra Collaboration Suite 开源伙伴

开始使用 ZCS 标准版网络客户端

标准版客户端可通过以下功能为您提供一个易于使用的电子邮件帐户。

  • 所用帐户有可能不包括上述全部功能。若需了解您的帐户都启用了哪些功能，请与您的系统管理员联系。

参见浏览您的信箱。

电子邮件

    • 写信和发送邮件

    • 阅读和回复邮件

    • 透过会话功能跟踪来往邮件，也可转换到传统邮件视图

    • 在邮件中加入附件

    • 将邮件转发给一个或多个收件人

    • 创建您自己的文件夹用以整理邮件

    • 创建过滤器以便将来信汇入指定的文件夹

地址簿

    • 创建和管理多个地址簿

    • 导入和导出联系人列表

    • 共享个人地址簿

效率手册

    • 创建和管理多个效率手册

    • 创建约会、会议和事件

    • 共享效率手册

    • 导入/导出其他效率手册程序。

任务

    • 创建多个“任务列表”

    • 创建待办事项

    • 将附件添加到任务

    • 管理任务、设置优先级和跟踪进度

    • 授权他人查看或管理您的“任务列表”

首选项

  • 设定邮箱，使之与邮件、地址簿和效率手册配合使用

  • 配置您的帐户设置，包括“发件人”字段中的邮件地址以及是否在您的邮件中自动添加签名。

  • 设定邮件签名

  • 创建邮件过滤器

  • 查看可用的快捷键组合，用以快速浏览邮箱

 

此外，您还可在任意视图内搜索邮件、个人联系人、效率手册和任务列表等。
