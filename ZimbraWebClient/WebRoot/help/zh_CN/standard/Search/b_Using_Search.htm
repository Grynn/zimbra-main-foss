使用搜索功能

Zimbra 的搜索功能是一功能强大的工具。尽管您可以学会使用查询语言描述所介绍的创建复杂文字查询条件，但也可以采用较快捷的方式，如此，您仅需要输入一个姓名或单词，而无需任何关键字。例如，查找所有含 tim 的邮件，不论是在主题行、收件人行、发件人行、抄送行、密送：行、邮件正文或附件中，都可在 查找字框输入tim ，并单击搜索即可。

您须指定搜索位置。您可在邮件、个人联系人、个人效率手册和个人任务列表中进行搜索。

您可输入任何值，如单词、名、姓、电话号码或域名。  若需更多选择，可使用 星号 * 作为通配符。  

在“搜索”中使用 * 作为通配符

星号 * 可以在 搜索中作为通配符使用，用以寻找词拼写相似的内容。  

请在前缀后使用星号 *。如要搜索 do*，返回的条目可以是 dog、door 等。

要在地址簿中搜索联系人，查询条件必须是一个完整的词。例如，如果按照电话号码搜索联系人，您必须输入完整的电话号码，不能仅按照区码进行搜索。

若需在地址簿中搜索联系人：

  1. 在“查找”框中，从下拉选单中选取个人联系人。这将搜索您所有的地址簿。

  2. 在查找框中输入姓名或其他联系人资料。

  3. 单击搜索即可执行。

若需快速搜索邮件：

  1. 在“查找”框中，从下拉选单中选择邮件。

  2. 在查找框中输入单词、姓名或其他相关的资料。

  3. 单击搜索即可执行。

创建复杂的搜索条件

如果您需频繁地执行同一搜索条件或要搜索特定的详细信息，您可能需要创建一个搜索条件并将其保存在“搜索”文件夹中。若想再次运行该项搜索，只需单击一下即可。

若需创建并保存某个搜索条件

  1. 在“搜索”面板中，单击编辑。创建新搜索窗格随即打开。

  2. 选择一个位置，以创建搜索文件夹。 如果保存在根文件夹中，此搜索将显示在“搜索”区。 如果选择其他文件夹，此搜索即被保存到那个文件夹。

  3. 在搜索名称字段中，输入此搜索的名称。

  4. 在搜索查询字段中，按照查询语言说明中的说明输入查询条件。您也可在查询条件中使用And 或 Or。

    建立搜索条件时使用 And 或 OR

    对于有多种条件的搜索项，您可以查找匹配其中任意条件的项目，也可查找符合全部条件的项目。 :

    • 匹配指定条件中任意条件的搜索称为 OR （或）搜索，意思是如果邮件中包含 X 或Y，则认为该邮件是匹配的。

    • 同时包含 X 与 Y 的搜索称为 AND（与）搜索，也就是说，邮件必须符合所有指定的条件才是匹配的。

  5. 单击创建搜索条件后，便可创建，同时执行搜索并将其保存到指定的文件夹中。

运行搜索时，单击窗格中的搜索名称。搜索结果将在“内容”窗格中显示。
