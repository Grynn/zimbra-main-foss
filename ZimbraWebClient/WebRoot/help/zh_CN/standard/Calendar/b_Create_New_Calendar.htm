创建新的效率手册

系统已为您创建了一个效率手册，但您可创建更多的效率手册，用以跟踪不同类型的活动。 比如，您可以为工作、个人、公司的各项活动创建分开的效率手册，也可订用互联网上的效率手册，还可以链接到共享的效率手册。

可使用日程安排视图对照查看两个或更多的效率手册。

若需创建效率手册

  1. 选择“效率手册”标签。

  2. 在“效率手册”窗格中，单击编辑。

  3. 在工具栏中，单击新建效率手册。

  4. 键入效率手册的名称，选择显示“效率手册”活动的颜色。如果不希望其他人看到效率手册中的活动，请 勾选报告空闲/忙碌时间时排除此效率手册。

  5. 默认情况下，已在 UI 中勾选为启用状态。此效率手册中的活动将在打开“效率手册”视图时显示。 如果不希望随同别的效率手册一起查看此效率手册，请取消勾选此复选框。在这种情况下，要查看此效率手册的活动，须选择该效率手册并将其打开。

  6. 单击创建效率手册。

若需订阅其他效率手册

  1. 选择效率手册标签。

  2. 在“效率手册”窗格中，单击编辑。默认的效率手册信息随即显示。  

  3. 在工具栏中，单击订阅效率手册。 

  4. 键入该效率手册的名称，输入该远程效率手册的有效的iCal 文件 URL，选择一种显示该效率手册的活动的颜色。

  5. 单击创建效率手册。
