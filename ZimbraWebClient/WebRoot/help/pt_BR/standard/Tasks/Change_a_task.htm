Alterar uma tarefa

Para atualizar sua tarefa

  1. Abra a lista que contém a sua tarefa

  2. No painel Conteúdo, clique na tarefa que você deseja alterar. A página da tarefa é aberta.

  3. Faça as alterações nas informações da tarefa.

  4. Clique em Salvar quando suas alterações estiverem concluídas.

Para marcar uma tarefa como concluída

Quando você terminar uma tarefa, no painel de conteúdo, marque a caixa próxima à tarefa para marcá-la rapidamente como concluída. O % concluído é alterado para 100%.
