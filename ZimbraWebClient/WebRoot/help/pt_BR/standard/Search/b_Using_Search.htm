Usando a busca

O recurso de busca do Zimbra é uma ferramenta poderosa. Apesar de você poder aprender a criar consultas complexas de texto, descritas em Descrição da linguagem de consultas, um atalho rápido é digitar um nome ou uma palavra, sem nenhuma palavra-chave. Por exemplo, para localizar todos os e-mails contendo a palavra "duda" em qualquer parte da linha de assunto, nas linhas para: de: cc: ou cco: , no corpo da mensagem, ou arquivo anexo, basta digitar duda na caixa de texto Localizar e clicar em Buscar.

Você seleciona onde buscar. É possível buscar em mensagens, contatos pessoais, agendas pessoais e na listas de tarefas pessoais.

Você pode inserir qualquer valor, como uma palavra, nome, sobrenome, número de telefone ou nome de domínio.   Para ter mais opções, use o  asterisco * como curinga.  

Usando * como curinga na busca

O asterisco * pode ser usado como curinga  em uma busca para localizar conteúdo que contenha palavras com ortografia similar.  

Use o asterisco * como curinga após um prefixo. Ou seja, a busca por co* retorna itens com a palavra colar, coração, comércio.

Para pesquisar contatos nas suas listas de endereços, a consulta deve ser uma palavra inteira. Por exemplo, para pesquisar contatos por número de telefone, você precisa inserir o número de telefone inteiro conforme ele aparece na entrada do contato. Não é possível pesquisar apenas pelo código de área.

Para buscar contatos em sua lista de endereços:

  1. Na seção de texto Localizar, selecione Contatos pessoais na lista suspensa. Isso fará uma busca em todas as suas listas de endereços.

  2. Digite o nome ou outro dado do contato na caixa de texto Localizar.

  3. Clique em Buscar para executar a busca.

Para fazer uma busca rápida por mensagens de e-mail:

  1. Na seção de texto Localizar, selecione Mensagens de e-mail na lista suspensa.

  2. Digite uma palavra, nome ou outro dado relacionado ao e-mail na caixa de texto Localizar.

  3. Clique em Buscar para executar a busca.

Criar uma consulta de busca complexa

Se você executar uma mesma consulta de busca com freqüência e desejar buscar informações detalhadas específicas, talvez deseje criar uma consulta de busca e salvá-la na pasta Busca. Quando desejar executá-la, basta clicar na busca.

Para criar e salvar um critério de busca

  1. No painel Visão geral da busca, clique em editar. O painel Criar nova busca é exibido.

  2. Selecione onde colocar a pasta.  Se salvá-la na pasta raiz, a busca será exibida na seção Buscar. Se você selecionar outra pasta, a busca será salva naquela pasta.

  3. No campo Nome da busca, dê um nome a ela.

  4. No campo Critério da busca, digite seu critério usando a linguagem de critérios descrita em Descrição da linguagem de critérios. Você também pode usar And ou Or.

    Usando AND ou OR ao construir sua consulta de Busca

    Para buscas usando vários critérios, você pode localizar itens que atendam a um critério especificado ou a todos eles.

    • A busca por mensagens que correspondam a qualquer um dos critérios especificados é chamada uma busca OR, porque se a mensagem contiver X ou Y, é considerada uma correspondência.

    • Buscar mensagens que contêm tanto X quanto Y é chamada uma busca AND, porque a mensagem deve atender a todos os critérios especificados para que seja considerada uma correspondência.

  5. Clique em Criar busca para criar e pesquisar, e salvar na pasta especificada.

Para executar sua busca, clique no nome desejado no painel de Visão geral. Os resultados da busca são exibidos no painel de Conteúdo.
