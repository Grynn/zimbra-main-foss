Atalhos do teclado

Atalhos do teclado são teclas ou combinações delas que possibilitam que você selecione rapidamente uma ação quando a seqüência da tecla de acesso é informada.  

A seguir apresentamos uma lista dos atalhos favoritos.  Vá para Opções>aba Atalho para ver a lista completa de teclas de atalho.

nm  ou  c

Nova mensagem (escrever)

gm

Mudar (“ir”) para E-mail

go

Mudar para opções

gc

Mudar para agenda

ga

Muda para Contatos (“lista de endereços”)

Atalhos usados no E-mail

o

Abrir a mensagem de e-mail atual

d

Apagar o item selecionado (movê-lo para a lixeira)

r

Responder ao remetente

a

Responder a todos os destinatários da mensagem

f

encaminhar mensagem

Atalhos usados na Lista de endereços

nc

Novo contato

ng

Novo grupo de contatos

Atalhos na Agenda

na

Novo compromisso
