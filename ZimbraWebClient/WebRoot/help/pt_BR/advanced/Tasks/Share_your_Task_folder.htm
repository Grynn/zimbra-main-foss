xxxxCompartilhar sua lista de tarefasxxxx

xxxxVocê pode compartilhar suas listas de tarefas com outras pessoas e pode definir permissões para a lista compartilhada conforme descrito a seguir:xxxx

  • xxxxUsuários ou grupos internos.xxxx Selecione o tipo de privilégio que deseja conceder:

    • xxxxVisualizadorxxxxxxxx.xxxx xxxxO Beneficiado pode ver o conteúdo da lista de tarefas, mas não pode fazer alterações no conteúdo da lista.xxxx

    • xxxxGerenciadorxxxxxxxx.xxxx O Beneficiado tem permissão para criar, editar e apagar tarefas na lista de tarefas.

    • xxxx xxxxxxxxAdminxxxxxxxx.xxxx O beneficiado tem permissão para criar, editar e apagar tarefas da lista de tarefas, bem como compartilhar sua lista de tarefas com outras pessoas.

    • xxxxNenhumxxxxxxxx é uma opção para desativar temporariamente o acesso a uma lista do concedente sem revogar os privilégios de compartilhamento.xxxx xxxxO Beneficiado ainda terá a lista de tarefas na sua visualização, mas não poderá visualizar ou gerenciar as tarefas.xxxx

  • xxxxConvidados externos.xxxx xxxxVocê pode criar uma senha para acessar a lista de tarefas compartilhada.xxxx xxxxOs convidados devem digitar essa senha para visualizar a lista.xxxx xxxxEles não poderão fazer alterações em nenhuma das tarefas.xxxx

  • xxxxPúblico.xxxx xxxxQualquer pessoa que saiba a URL para a lista compartilhada pode visualizá-la.xxxx xxxxEles não podem fazer nenhuma alteração.xxxx

xxxxPara compartilhar sua Lista de tarefasxxxx

xxxxClique com o botão direito do mouse na lista a ser compartilhada e selecione Compartilhar lista de tarefas.xxxx

xxxxNa caixa de diálogo Propriedades do compartilhamento, selecione com quem compartilhar e configure as informações adequadas.xxxx

 

Compartilhando com usuários ou grupos internos
1. Selecione os Usuários ou grupos internos com os quais compartilhar

2. No campo E-mail, digite os endereços de e-mail dos beneficiários. xxxxÀ medida que você digita um endereço, uma lista de nomes que corresponde às letras digitadas é exibida.xxxx Você pode selecionar o nome na lista.

3. Selecione o papel, Visualizador ou Gerenciador

4. Na seção Mensagem, selecione o tipo de mensagem de e-mail a ser enviada para notificar o beneficiário.

5. Clique em OK.

xxxxCompartilhando com convidados externosxxxx

1. xxxxNa seção xxxxxxxxCompartilhar comxxxxxxxx, selecione xxxxxxxxConvidados externosxxxxxxxx.xxxx xxxxQuando você seleciona essa opção, é exibido um campo Senha.xxxx

2. xxxxDigite os endereços de e-mail para os convidados.xxxx

3. xxxxDigite a senha que os convidados devem usar para acessar essa lista de tarefas.xxxx

4. xxxxSelecione que tipo de mensagem de notificação deve ser enviada.xxxx xxxxEssa mensagem fornece a eles as informações necessárias para acessar a lista.xxxx

5. xxxxClique em xxxxxxxxOKxxxxxxxx.xxxx

 

xxxxCompartilhando com o públicoxxxx

1. xxxxNa seção xxxxxxxxCompartilhar comxxxxxxxx, selecione Público.xxxx xxxxQuando você seleciona essa opção, a caixa de diálogo propriedades do compartilhamento exibe a URL de sua lista de tarefas.xxxx

2. Anote essa URL e notifique os usuários "públicos" sobre esse endereço. xxxxQuando eles clicarem na URL, poderão ler o conteúdo da lista de tarefas, mas não poderão alterar nada.xxxx
