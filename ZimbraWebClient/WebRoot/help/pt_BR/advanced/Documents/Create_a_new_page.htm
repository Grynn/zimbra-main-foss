Criar uma nova página

Você pode criar uma página, formular o layout do conteúdo usando as ferramentas de texto com formatação ou inserir tags HTML. É possível verificar a ortografia na página e usar palavras-chave para criar facilmente links para outras páginas.

  1. Clique no notebook em que você deseja trabalhar. A página tabela de conteúdo é exibida no painel Conteúdo.

  2. Clique em Novo na barra de ferramentas. Uma página em branco é aberta. No campo Página do cabeçalho, digite o nome da nova página. O nome é exibido no índice.

  3. Escreva o conteúdo da página. Você pode usar o editor de texto com formatação padrão para adicionar estilos, cores, tabelas e planilhas à sua página.

Se estiver familiarizado com HTML e preferir usar tags HTML, clique em Formatar na barra de ferramentas e selecione Fonte HTML. A página muda para um campo em branco e você pode usar a sintaxe padrão de tags HTML para criar a página.

  1. Clique em Salvar para salvar o arquivo. Agora você pode adicionar detalhes à página ou clicar em Fechar para retornar para a página da tabela de conteúdo
