Usando o editor de texto com formatação

Você pode criar suas páginas usando ou o editor de texto com formatação para formatar o conteúdo na página ou  Fonte HTML para formatar sua página usando tags HTML. Você pode alternar entre os dois formatos para tirar proveito dos recursos do editor de texto com formatação e para inserir tabelas e planilhas.

Quando se utiliza o editor de texto com formatação, a barra de ferramentas de duas linhas possibilita que você formate facilmente o layout da sua página. Selecione tamanhos de fonte, tipos de fonte, estilos de texto, cores de fonte e de plano de fundo. Essas ferramentas de formatação funcionam exatamente da mesma forma que muitas ferramentas de processamento de texto.

Além disso, você pode clicar para adicionar tabelas, planilhas, anexos e imagens à sua página.

Adicionando estilos de parágrafo

O ícone     na barra de ferramentas exibe os estilos de parágrafo disponíveis. Esses estilos de parágrafo facilitam a  formatação de texto e mantêm um estilo consistente nas páginas do notebook. Os estilos incluem Normal para a maioria dos parágrafos, seis níveis de Cabeçalhos para identificar títulos de seções,  Endereço para deixar o texto em itálico e Pré-formatado para usar quando se deseja exibir texto na fonte Courier. Esses estilos não podem ser modificados, mas é possível usar as opções de edição para formatar um estilo diferentemente – alterar a fonte, o tamanho, a cor, etc.

Adicionando tabelas à sua página

Adicionar tabelas à sua página é um processo de três etapas. Você adiciona uma tabela, configura as propriedades da tabela e depois adiciona conteúdo a ela. Para adicionar uma tabela, clique no ícone na barra de ferramentas, selecione Inserir tabela e depois selecione quantas linhas e colunas deve haver na tabela.

Adicionando uma planilha à sua página  

Para adicionar uma planilha, clique no ícone de planilha, na barra de ferramentas. Uma planilha de seis colunas é adicionada à página. As planilhas podem ser usadas para criar  bancos de dados para acompanhar listas de atividades, programações, tarefas a fazer, etc. A planilha contém a funcionalidade básica de planilhas e um conjunto de funções predefinidas.

Adicionando anexos e imagens à sua página

Você pode carregar arquivos vinculados à página Documentos. Clique no ícone de clipe, ou de imagens,  e procure o arquivo a ser carregado. Você será notificado se carregar um arquivo que já existe em seu Documentos ou Porta-arquivos.

Criando link para uma página de notebook ou para páginas da web

Você pode criar link para outras páginas de Notebook de documentos ou para uma URL externa da Web e  fornecer ao link um título descritivo para que o endereço URL não seja exibido.
