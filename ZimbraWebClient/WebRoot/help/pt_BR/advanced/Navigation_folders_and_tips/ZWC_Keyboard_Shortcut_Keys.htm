Teclas de atalho de teclado do CZW

As teclas de atalho podem ser usadas para navegar no Cliente Zimbra para Web (CZW) e você pode criar suas próprias combinações de tecla de atalho. Vá para Preferências>Atalhos> para ver a lista de teclas de atalho e para criar suas próprias teclas de atalho. As quatro abas exibem o seguinte

  • Lista de atalhos inclui uma lista de atalhos para todos os aplicativos

  • Atalhos da pasta E-mail possibilita que você crie e atribua atalhos para acessar suas pastas de e-mail

  • Atalhos de busca salva possibilita que você crie e atribua atalhos às suas buscas salvas

  • Atalhos para marcadores possibilita que você crie e atribua atalhos aos marcadores criados
