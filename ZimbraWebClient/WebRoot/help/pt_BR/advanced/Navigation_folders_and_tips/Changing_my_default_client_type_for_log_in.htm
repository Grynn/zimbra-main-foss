Alterando meu tipo de cliente padrão para login

Toda vez que você fizer login, poderá escolher que versão de cliente web deseja usar, seja a Avançada ou a Padrão.

  • Padrão é o tipo de cliente configurado em Preferências, aba Geral. Ele pode ser o Avançado ou o Padrão.

  • O Cliente Web Avançado oferece o conjunto completo de recursos de colaboração web baseado em Ajax. O cliente web avançado funciona com os navegadores mais novos e conexões de Internet mais rápidas.

  • O Cliente Web Padrão é uma boa opção quando as conexões de Internet são lentas ou os usuários preferem mensagens em HTML para navegar em sua caixa de e-mail.

Se seu tipo de cliente preferido não for o padrão, você pode alterá-lo depois de efetuar o login.

1. Clique na aba Preferências.

2. Clique na aba Geral e na seção Opções de login selecione Avançado ou Padrão.

3. Clique em Salvar.

Na próxima vez que fizer login, sua seleção será a padrão.
