Navegando no Cliente Zimbra para Web

Quando se faz login no Cliente Zimbra para Web, você vê uma visualização de tela inteira semelhante à que é mostrada abaixo.

A página do Cliente Zimbra para Web inclui as seguintes áreas:

  1. Controles do navegador. Mostra os botões padrão do navegador, tais como Página inicial, o endereço atual que você está acessando e outros recursos que podem depender do seu software navegador e suas configurações. Os botões na barra do navegador não são parte do Cliente Zimbra para Web e são exibidos para todas as páginas da Web que você visita.

  2. Barra de busca, Nome de usuário e Exibição de cota. A Busca, Busca avançada, o nome de usuário com o qual você fez login e sua cota de caixa de e-mails são exibidos nessa área.

  3. Abas de aplicativos. Os aplicativos que você acessar estão listados nas abas na parte superior. À direita temos os links Ajuda e Sair.

  1. Barra de ferramentas. Mostra as ações disponíveis para o aplicativo que você está usando no momento. Neste exemplo, a barra de ferramentas do aplicativo E-mail está sendo exibida.

  2. Painel de Visão geral. Exibe suas Pastas, incluindo as pastas de sistema Caixa de entrada, Enviadas, Rascunhos, Spam e Lixeira, bem como quaisquer pastas personalizadas que você tenha criado, Buscas que você tenha salvo e Marcadores criados, links para Zimlet que possam ter sido criados para integrar a aplicativos de terceiros em sua caixa de entrada. Quando você está no aplicativo Agenda, o painel de Visão geral exibe sua lista de agendas e Zimlets.

  3. Uma mini agenda pode ser exibida abaixo do painel de Visão geral. A exibição da mini agenda é opcional e pode ser ativada em Preferências, aba Agenda.

  4. Painel de Conteúdo. O conteúdo dessa área muda dependendo de que aplicativo está em uso. Na visualização da Caixa de entrada, são exibidas todas as mensagens ou tópicos de conversação de sua Caixa de entrada.
