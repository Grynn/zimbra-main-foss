xxxxSolicitando uma confirmação de leituraxxxx

Você pode sinalizar uma de suas mensagens para que o destinatário envie uma confirmação de leitura de volta assim que abri-la. Depois de escrever uma mensagem, selecione Retornar confirmação de leitura no menu Opções. xxxxO destinatário pode escolher enviar a confirmação ou não.xxxx

Os usuários configuram suas preferências de E-mail de confirmação de leitura para mensagens sinalizadas com uma solicitação de confirmação de leitura. xxxxAs seguintes opções podem ser definidas:xxxx

  • xxxxNunca enviar uma confirmação de leituraxxxxxxxx.xxxx xxxxEsse é o padrão.xxxx Quando uma mensagem é recebida com uma confirmação de leitura, a solicitação é ignorada. xxxxO destinatário nem fica sabendo da solicitação.xxxx

  • Sempre enviar uma confirmação de leitura. Uma mensagem de confirmação de leitura é enviada automaticamente quando é aberta uma mensagem sinalizada com uma confirmação de leitura.

  • xxxxPergunte-mexxxxxxxx.xxxx Os destinatários são perguntados se desejam enviar a confirmação.

Uma confirmação de leitura não será enviada se o destinatário selecionar mais de uma mensagem, clicar com o botão direito do mouse e selecionar Marcar como lida.
