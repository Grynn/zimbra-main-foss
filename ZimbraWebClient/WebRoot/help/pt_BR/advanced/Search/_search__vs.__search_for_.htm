Buscando mensagens

Você pode buscar mensagens, tópicos de conversação ou contatos

  • Use a caixa de texto de Busca para inserir sua consulta, conforme está descrito em Configurações de busca rápida.

  • Use a Busca avançada para especificar critérios de busca mais detalhados.

Para buscar mensagens usando a caixa de texto Busca:

  1. Insira o critério de busca na caixa Busca como has:anexo ou talvez o nome do remetente.

  2. Clique na seta de Busca, e selecione o tipo de busca na lista.

  3. Clique em Buscar.

Para buscar uma mensagem usando critérios detalhados:

  1. Clique em Avançado para abrir a área de Busca avançada.

  2. Clique nos ícones na barra de ferramentas para abrir o painel do tipo de critério que você deseja usar para sua busca.

  3. Insira as informações. Se você ativou as Preferências para Sempre exibir a string de busca, a caixa de texto Busca é atualizada enquanto você digita ou faz seleções.

  4. Assim que você insere um critério no painel de busca ou assim que você faz qualquer mudança de seleção em qualquer outro painel de busca, como alterar um botão de opção ou uma opção de busca em menu suspenso, a pesquisa é realizada.

  5. Os itens que correspondem ao critério de busca são listados na parte inferior direita. Se o recurso de Tópico de conversação estiver ativado, os tópicos de conversação contendo mensagens que corresponderem ao critério da busca são listados.

  6. Para atualizar a área de resultados de busca, clique em Buscar.
