Definindo suas preferências de usuário

Suas preferências padrão de usuário são configuradas quando sua conta é criada. Essas opções definem como funcionarão seus aplicativos de caixa de e-mails, listas de endereços e agenda. Também é possível configurar suas identidades de e-mail pessoal e contas POP e selecionar um tema para sua caixa de e-mails.

É possível alterar essas configurações na aba Preferências. Se uma opção descrita aqui não estiver visível na sua visualização, a preferência não está disponível para sua conta.

É possível definir preferências nas seguintes abas. Clique na palavra para obter uma descrição detalhada.

  • Geral

    • Você executou o login no cliente web padrão. Para definir essa opção como Tipo de cliente padrão quando fizer login, selecione Padrão.

    • Avançadooferece um conjunto completo de recursos de colaboração na Web. Esse Cliente Web funciona melhor com navegadores mais novos e conexões de Internet mais rápidas.

    • Padrãoé recomendado quando as conexões de Internet forem lentas, quando usar navegadores antigos ou para uma acessibilidade facilitada.

    • Em Tema selecione a cor de plano de fundo da interface de sua caixa de e-mails.

    • Se o fuso horário exibido aqui não estiver correto, selecione o fuso horário padrão para sua conta. Essa opção determina a data e a hora de mensagens que você enviou ou recebeu e a hora a ser exibida em compromissos da Agenda.  Ele pode ser diferente da configuração de fuso horário do seu computador.

    • Se a opção Alterar senha for exibida, você poderá alterar sua senha. Caso tenha dificuldades para alterar sua senha, entre em contato com o administrador para conhecer as regras de senha.

      • Entre em contato com o administrador caso não tenha essa opção e deseje alterar sua senha.

    • Em Configurações de busca, você pode escolher incluir automaticamente as pastas Spam e Lixeira em qualquer busca que realizar. Por padrão essas pastas não são pesquisadas.

    • Em Idioma da busca, marque Exibir o idioma da busca avançada na barra de ferramentas de busca para mostrar detalhes da string de busca na caixa de texto relacionada.

    • Ative a opção Exibir caixas de verificação para selecionar rapidamente itens em listas (precisa de atualização)  para exibir uma caixa de verificação para cada item no painel de Conteúdo. Use a caixa de verificação para selecionar um ou mais itens para realizar a mesma ação, como apagar, mover, marcar como lido/não lido.

  • E-mail

    • Selecione quantos itens (mensagens ou tópicos de conversação) deseja exibir por página. O padrão é exibir 25 itens por página, mas você pode selecionar 10, 50 ou 100 itens por página.

    • Selecione como deseja agrupar seus e-mails: por tópico de conversação ou mensagem. Caso tenha o recurso Tópico de conversação, o modo Tópico de conversação é o padrão.

    • O intervalo de verificação é a freqüência com a qual o computador verifica novos e-mails.  O padrão é a cada 5 minutos. O intervalo mínimo é definido pelo administrador Zimbra.

    • Especifique se deseja visualizar e-mails como HTML, para mensagens que foram formatadas em HTML. Desativar essa opção faz com que os e-mail sejam exibidos como texto sem formatação, mostrando os tags HTML em vez de aplicá-los como linguagem de marcação.

    • Visualização de mensagens: opções

      • Defina se irá mostrar o painel de leitura quando visualizar e-mail. O padrão é sempre mostrar o painel de leitura.

      • Se for ativado, o cliente exibe trechos das mensagens na lista de e-mails. Quando o cursor estiver sobre o assunto da mensagem, as primeiras palavras da mensagem são exibidas.

      • Se ativado, clique duas vezes para abrir as mensagens em uma nova janela

    • Ative Imagens para fazer download de fotos automaticamente nas mensagens de e-mail em HTML. Quando a opção está ativada, não é preciso clicar em exibir mensagem com imagem para visualizá-la.

    • A Busca de e-mail padrão define a busca a ser executada quando se faz login. A busca padrão é sua Caixa de entrada.  Quando você faz login, os resultados da sua Caixa de entrada são exibidos. É possível alterar esse campo para outra pasta, marcador ou busca salva quando se abre o Cliente Zimbra para Web pela primeira vez. Consulte Descrição da linguagem de critérios para conhecer a sintaxe. Ative e especifique um endereço para enviar notificações de que você recebeu novos e-mails.

    • Especifique um endereço para encaminhar seus e-mails. É possível optar por apagar o e-mail original de sua caixa de e-mails Zimbra.

    • Especifique se deseja que uma mensagem de resposta automática seja enviada para os e-mails recebidos.  Esse recurso é usado freqüentemente como mensagem de ausência ou mensagem de férias informando que você não está no escritório e o que a mensagem deve dizer.

    • Configure a seção Mensagens vindas de mim para lidar com mensagens que você envia e que incluam seu endereço como um dos destinatários da mensagem ou que vão para um endereço cuja lista de distribuição o inclua.

     

  • Escrevendo

    • Defina que editor de texto deseja usar como padrão: HTML ou texto sem formatação.

    • Defina as configurações de fonte padrão a serem usadas quando a mensagem for escrita em HTML em E-mails, Documentos e Agenda.  É possível definir o estilo, o tamanho e a cor da fonte a serem usados como estilo padrão. Arial, 12 pontos, preto é o padrão.

    • Sempre escrever e-mail em uma nova janela abre uma nova janela de criação, separada da caixa de e-mails Zimbra quando se escreve uma mensagem. Você pode visualizar e navegar em sua caixa de e-mails quando a janela de criação está aberta.

    • As mensagens que você escreve são salvas automaticamente como rascunhos a intervalos de alguns minutos. Se preferir não salvar automaticamente o que está sendo escrito, desative o recurso.

    • Selecione se deseja incluir o texto original no corpo de sua mensagem de resposta. É possível escolher incluir o texto original, não incluí-lo, incluí-lo como anexo ou no corpo da mensagem com um prefixo escolhido, ou mesmo incluir apenas a mensagem mais recente.  (Em outras palavras, será citado apenas o que foi escrito pela pessoa que enviou a mensagem e não o texto anterior.)

    • Especifique se o prefixo em cada linha será > ou | para as mensagens de e-mail anteriores que são encaminhadas ou respondidas, se escolher citar o texto original.

    • Por padrão, a opção Salvar cópias das mensagens na pasta Enviadas está ativada. Cópias de mensagens enviadas serão salvas na pasta Enviadas.

    • Especifique como incluir o texto original da mensagem quando você a encaminha: no corpo da mensagem, no corpo com um prefixo ou como anexo.

  • Assinaturas

    Você pode criar assinaturas para suas mensagens de e-mail. Sua assinatura pode incluir seu nome e um texto adicional de fechamento até o número máximo de caracteres que sua conta permitir. Caso crie várias identidades, poderá criar diferentes assinaturas e atribuí-las a endereços específicos.

    • Insira a assinatura que deve ser exibida nas mensagens de e-mail.

    • Especifique o posicionamento da assinatura nas suas mensagens de e-mail: adicionar a assinatura abaixo de todas as mensagens no e-mail ou no final do texto que você escreveu, respondeu ou encaminhou.

  • Lista de endereços

    • Ative o recurso de adicionar endereços automaticamente à sua lista de endereços quando se envia e-mails. Caso a opção seja ativada, os endereços são salvos na pasta de Contatos de e-mail.

    • Ative a opção para usar a Lista de endereços globais quando estiver completando automaticamente endereços.

    • Selecione se deseja visualizar contatos como uma lista ou em forma de cartões de visita.

    • Selecione o número de contatos a exibir por página. O valor padrão é 25.

    • Importe contatos para sua Lista de contatos. O arquivo a ser importado deve estar no formato de texto separado por vírgulas.

    • Exporte sua lista de contatos. O arquivo é exportado em um formato de texto separado por vírgulas.

     

  • Contas

    A aba de contas é usada para criar e gerenciar seus personagens. Personagem é a identidade de e-mail usada no campo De de suas mensagens de e-mail enviadas.

    O estilo padrão é definido na identidade principal. Quando você cria novas identidades, pode usar as mesmas configurações que a Identidade padrão ou pode definir preferências diferentes.

    O botão Adicionar conta externa possibilita que você identifique suas contas POP/IMAP externas de forma que este cliente web possa recuperar e-mails de suas outras contas.  Também é possível configurar para que as respostas a essas mensagens usem o endereço POP/IMAP correspondente.

  • Filtros de e-mail

    Use a aba Filtros de e-mail para definir as regras de filtragem de e-mail. Para saber como configurar regras de filtro de e-mail, consulte Como funciona a filtragem.

  • Agenda

    • Selecione como deseja visualizar sua agenda Zimbra. O padrão é por semana útil.

    • Especifique que dia da semana deve ser o primeiro dia em sua agenda.

    • Se você costuma viajar para lugares com fusos horários diferentes, talvez queria selecionar para exibir a lista de fusos horários na visualização de compromissos.

    • Se você criar muitos compromissos sem participantes, marque Usar a caixa de diálogo QuickAdd.

    • Uma mini agenda é exibida na Agenda.  Você pode ativar para que a mini agenda fique visível em qualquer visualização.

    • Defina o número de minutos de antecedência com que o compromisso será lembrado. O padrão é cinco minutos.

    • Importar e exportar outros programas de agenda.

  • Sistema de troca de mensagens instantâneas

    Você pode configurar a forma como a troca de mensagens instantâneas (MI) é integrada à sua conta.

    • Login. Caso deseje efetuar login automaticamente em serviços de troca de mensagens instantâneas, essa opção deverá ser ativada.

    • Notificação. É a forma como receberá notificações de MI e a forma como deseja que os outros o vejam

    • Bate-papo.  Seu histórico de bate-papos é salvo automaticamente na pasta de Bate-papo. As mensagens nessa pasta são como qualquer outra mensagem de e-mail: você poderá imprimir, encaminhar como e-mail ou apagá-las da pasta. Também é possível definir o tempo limite de espera.

  • Atalhos

    Os atalhos de teclado permitem que você use o aplicativo sem precisar do mouse. Um grande número de atalhos está disponível. Você também pode criar atalhos personalizados, atribuindo nomes alternativos numéricos a pastas, marcadores e pesquisas salvas.

    • A Lista de atalhos mostra todos os atalhos de teclado existentes.

    • A aba Atalhos de pasta de e-mail é usada para criar um atalho para ir rapidamente para suas pastas de e-mail e para mover uma mensagem para aquela pasta. Você seleciona a pasta e atribui um número a ela. Por exemplo, se você atribuir 3 para a pasta de Caixa de entrada: De qualquer lugar de sua caixa de e-mails, digitar v3 irá movê-lo para a Caixa de entrada. Selecionar um item e digitar .3 irá mover o item para a Caixa de entrada.

    • A aba Atalhos de busca salva é usada para criar atalhos para acesso rápido às Buscas salvas. Você seleciona uma busca salva e atribui um número a ela. Por exemplo, se você atribuir 3 à sua busca favorita, de qualquer lugar de sua caixa de e-mails, poderá digitar s3 para executar rapidamente aquela busca.

    • A aba Atalhos de marcador é usada para criar atalhos para acesso rápido a todos os itens com um determinado marcador ou aplicar um marcador a uma mensagem ou contato. Você seleciona um marcador e atribui um número a ele. Por exemplo, atribui-se 3 a um marcador chamado Acompanhamento. De qualquer lugar de sua caixa de e-mails, você pode digitar y3 para ver todos os itens marcados com Acompanhamento. Selecione um item e digite t3 para marcá-lo com Acompanhamento.

    Você pode atribuir qualquer número, mas ele não pode ser repetido na aba de atalhos.
