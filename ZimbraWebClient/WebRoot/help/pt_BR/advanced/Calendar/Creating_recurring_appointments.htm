Criando compromissos recorrentes

Compromissos recorrentes são compromissos que se repetem em algum tipo de programação. Você pode programar reuniões recorrentes para que elas se repitam diariamente, semanalmente, mensalmente ou anualmente. Além disso, você pode personalizar programações de compromisso recorrente.

  1. Em qualquer visualização, exceto na visualização Preferências, clique na seta em e selecione Compromisso.

  2. Insira o assunto. O assunto torna-se a descrição na agenda e é obrigatório.

  3. Insira um local. Você pode inserir qualquer local. Os locais que foram criados como contas de recursos pelo administrador do sistema podem ser reservados. Se você souber o nome do local, insira-o no campo Local. Se não souber, vá para a aba Encontrar local para fazer uma busca. Depois de selecioná-lo, o local será exibido nesse campo.

  4. Selecione a data e a hora. Insira a data ou clique na seta para baixo para exibir a agenda e escolher uma data. Selecione a data final.

  1. Observação: se o fuso horário for exibido, ele refletirá o fuso horário em que você está. Em geral não é necessário fazer alterações. Quando você programa reuniões com participantes em fusos horários diferentes, o convite é enviado refletindo a hora no fuso horário deles. Por exemplo, se você criar uma reunião com participantes na Califórnia e em Nova York, o convite exibirá a hora do Pacífico para os participantes na Califórnia e a hora da Costa leste (três horas depois) para os participantes em Nova York.

  1. Clique em Repetir para visualizar os padrões de recorrência: diário, semanal, mensal, anual ou personalizado. A reunião será programada no padrão recorrente de hora e data inicial especificadas. A repetição personalizada permite que você programe padrões mais específicos.

  2. Se a reunião recorrente tiver uma data final, clique no link ao lado do campo Repetir.  

  3. Selecione um padrão de recorrência e, se necessário, para personalizar mais o padrão, clique no link que aparece ao lado dele.

    • Diariamente. Personalize repetições diariamente para escolher se é para todos os dias ou alguma outra seqüência, como a cada dois dias ou quatro dias.

    • Toda semana. Personalize as repetições semanalmente para selecionar se é um compromisso recorrente semanal ou algum outro padrão semanal, como a cada duas semanas.  Você pode selecionar que dia da semana deseja a reunião.

    • Mensalmente. Personalize as repetições mensalmente para selecionar se é um compromisso recorrente mensal ou algum outro padrão mensal, como a cada dois meses.  Uma agenda é exibida e com isso é possível selecionar que dia(s) deseja programar a reunião.

    • Anualmente. Personalize as repetições anualmente para selecionar o mês e a data.

  4. Adicione recursos. Se você souber o nome do recurso, insira-o no campo Recurso. Se não souber, vá para a aba Localizar recursos para buscar os recursos disponíveis.

  5. Digite os nomes dos participantes. Você pode digitar nomes de participantes de qualquer uma das formas a seguir:

    • Vá para a aba Localizar participantes. Digite um nome e selecione que lista será usada, seja a lista Contatos ou a Lista de endereços globais. Selecione os nomes e pressione Adicionar.  Quando concluir, clique em OK.

    • No campo de texto Participantes, digite os endereços de e-mail, separando os endereços por ponto-e-vírgula (;). Conforme você digita, os nomes na lista de contatos que coincidem são exibidos.

    • Para ver as programações de livre/ocupado dos participantes, clique na aba Programar participantes. À medida que você insere os nomes e endereços de e-mail dos participantes, se as programações dos participantes forem conhecidas, a disponibilidade aparecerá nas barras horizontais ao lado dos nomes. Retorne para a aba Compromisso quando a lista de participantes estiver completa.  

  6. Use o campo Texto para adicionar outras informações que serão incluídas no e-mail. Para adicionar anexos, clique em Adicionar anexos na barra de ferramentas.

  7. Clique em Salvar. Um convite de e-mail é enviado a todos os participantes e o compromisso é exibido nas agendas deles.
