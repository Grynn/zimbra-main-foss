Criando agendas

Depois de criar uma agenda, você pode criar agendas adicionais para ter controle dos diferentes tipos de funções.  Por exemplo, você pode criar uma agenda para atividades profissionais, pessoais e empresariais.  Use a Visualização de programação para ver as agendas lado a lado.

Para criar uma agenda

  1. Selecione a aba Agenda.

  2. Clique com o botão direito do mouse em Agendas no painel Visão geral, ou na barra de ferramentas clique em   e selecione Agenda.

  3. Na caixa de diálogo Criar nova agenda, digite o nome e selecione uma cor para exibir as atividades da agenda.

  4. Clique em OK. A nova agenda é exibida na lista de agendas.

Para criar uma agenda e sincronizar os compromissos de uma agenda remota

  1. Selecione a aba Agenda.

  2. Clique com o botão direito do mouse em Agendas no painel Visão geral, ou na barra de ferramentas clique em   e selecione Agenda.

  3. Na caixa de diálogo Criar nova agenda, digite o nome e selecione uma cor para exibir as atividades da Agenda.

  4. Marque Sincronizar compromissos da agenda remota. Insira uma URL de arquivo iCal válida para a agenda remota.

  5. Clique em OK.
