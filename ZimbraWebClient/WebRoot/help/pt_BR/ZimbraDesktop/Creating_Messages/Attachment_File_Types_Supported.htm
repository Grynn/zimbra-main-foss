xxxxTipos de arquivo de anexo suportadosxxxx

É possível abrir qualquer anexo de e-mail diretamente do Zimbra Web Client, contanto que sua estação de trabalho tenha o aplicativo certo e que a extensão não esteja bloqueada pelo administrador do sistema. Por exemplo, para abrir um documento criado com o Microsoft Word, é preciso conseguir executar o Microsoft Word no seu computador.

xxxxNa maioria dos computadores, o tipo de arquivo é indicado pela extensão, que é normalmente um ponto seguido por um código de três letras.xxxx Entre os exemplos estão arquivos como despesas.xls ou Novos-itens.doc. xxxxSe o tipo de arquivo for suportado por um software instalado no seu computador, você em geral pode clicar duas vezes sobre o arquivo e seu computador iniciará automaticamente o aplicativo certo para ler aquele arquivo.xxxx

xxxxMuitos tipos de arquivos podem ser abertos diretamente do seu navegador usando vários plugins.xxxx xxxxEm geral, se você clica em um link de arquivo em uma página, o navegador detecta automaticamente se você tem ou não o plugin certo e pergunta se você deseja fazer seu download.xxxx xxxxObserve que as diretrizes do administrador do sistema talvez não permitam que os usuários finais façam download e instalem qualquer tipo de software.xxxx

Observações:
Em computadores com Windows, o computador entende que o tipo de arquivo no nome do arquivo corresponde ao formato real. Mesmo se o arquivo estiver formatado internamente de forma correta, caso ele não tenha a extensão do tipo certo no seu nome, o computador pode não conseguir reconhecê-lo e abrir o aplicativo certo para você.

Um número cada vez maior de vírus de computador tem sido criado para se espalhar via anexos de arquivo. Portanto, em alguns ambientes, os administradores de sistema optaram por bloquear mensagens recebidas contendo certos tipos de anexos, especialmente arquivos com a extensão .EXE ou .ZIP como parte do nome do arquivo. xxxxSe você enviar uma mensagem de e-mail para alguém cujo sistema de e-mail tenha sido configurado para bloquear certos tipos de anexo, você pode ou não receber alguma notificação de que foi bloqueado. xxxx

xxxxPara sua informação, veja a seguir algumas das extensões de arquivo mais comumente usadas.xxxx

  • .TXT – Indica um arquivo de texto, que é um arquivo contendo apenas informações de texto. Arquivos de texto podem ser abertos e lidos usando uma ampla gama de editores de texto, incluindo o Bloco de Notas, o WordPad, o Word, editores de linha de comando, navegadores e até mesmo programas para planilhas.

  • .HTM – Indica um arquivo HTML, que é um tipo especial de arquivo de texto normalmente aberto em um navegador da Web.

  • .ZIP – Indica um arquivo que foi compactado usando a compressão ZIP. É preciso ter o WinZip ou outro programa de compactação que possa ler e expandir o arquivo ZIP.

  • .EXE – Indica um arquivo de programa ou executável. xxxxClicar duas vezes nesse arquivo normalmente inicia o aplicativo.xxxx

  • .DOC – Indica um arquivo de documento do Microsoft Word.

  • .JPG – Indica uma imagem gráfica no formato JPEG. Você pode abrir JPEGs em um navegador ou usando um dos vários programas gráficos tais como Paint ou Adobe Photoshop.

  • .GIF – Indica uma imagem gráfica no formato GIF. É possível abrir GIFs em um navegador ou com um programa gráfico.

  • .MPG – Indica um clipe de filme ou animação no formato MPEG. xxxxÉ normalmente reproduzível com um plugin de navegador como o Windows Media Player.xxxx

  • .WMV – Indica um clipe de filme no formatoWMV, que é um formato exclusivo para Windows.

  • .PDF – Indica um documento no formato Acrobat Reader. xxxxVocê pode normalmente abri-lo a partir do navegador usando um plugin.xxxx
