Abwesenheitsmeldungen erstellen

Hinweis: Nur für Zimbra Web Client-Accounts verfügbar.

Abwesenheitsmeldungen sind automatische Antworten auf empfangene Mails. Sie können eine Abwesenheitsmeldung einrichten, die automatisch den Nutzern gesendet wird, von denen Sie Mails empfangen, während Sie für einen längeren Zeitraum nicht im Büro sind. Diese Mail wird jedem Empfänger nur einmal gesendet, unabhängig davon, wie viele Mails dieser Nutzer Ihnen während Ihrer Abwesenheit sendet.

So richten Sie eine Abwesenheitsmeldung ein

  1. Wählen Sie unter Einstellungen die Option Mail.

  2. Aktivieren Sie unter "Mails empfangen" die Option Automatische Antwort senden.

  3. Geben Sie im Textfeld die Nachricht ein, die gesendet werden soll, z. B. "Ich bin derzeit nicht im Büro. Meine Sprachnachrichten rufe ich ab, meine Mails jedoch nicht. Am 01.06.10 bin ich wieder zurück."

  4. Legen Sie Start- und Enddatum für die Nutzung dieser Nachricht fest.

  5. Klicken Sie auf Speichern. Die Abwesenheitsfunktion wird sofort aktiviert.
