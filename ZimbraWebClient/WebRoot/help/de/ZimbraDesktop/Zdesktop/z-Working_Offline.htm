Offline arbeiten

Mails, die von Ihren Accounts geladen wurden, stehen jederzeit in Zimbra Desktop zur Verfügung, auch wenn keine Internetverbindung besteht. Das Arbeiten in Zimbra Desktop ohne Internetverbindung wird "offline arbeiten" genannt.

Wenn Sie offline arbeiten, kann Zimbra Desktop nicht auf Ihre Mail-Accounts zugreifen, aber Sie können in Zimbra Desktop so arbeiten, als ob Sie mit dem Internet verbunden wären. Sie können Ihre Mails lesen, neue Mails verfassen, Ordner aufräumen und andere Änderungen vornehmen, die mit Ihrem Account synchronisiert werden, sobald wieder eine Internetverbindung hergestellt wird.

Sie erkennen an dem roten Quadrat neben Ihrem Accountnamen, dass Sie offline arbeiten.

 

Hinweis: Möglicherweise arbeiten Sie schon offline, bevor das Symbol rot dargestellt wird, weil der Netzwerkstatus immer dann aktualisiert wird, wenn Zimbra Desktop versucht, die Synchronisierung mit dem Server durchzuführen. Schlägt der Synchronisierungsversuch fehl, wird das Symbol rot dargestellt.

 

Sie können erzwingen, dass alle Accounts offline gehen, indem Sie im Übersichtsfenster auf das kleine Symbol neben den Accounts klicken.

Klicken Sie erneut auf das Symbol, um wieder online zu gehen.  

Möglichkeiten der Offline-Arbeit

  • Sie können Ihre Mails lesen, sie in Ordner verschieben und zur Nachverfolgung mit Tags versehen.

  • Sie können Mails beantworten und die Antworten absenden, Mails weiterleiten und neue Mails verfassen. Die Mails werden im Postausgang gespeichert und bei der nächsten Verbindung mit dem Internet automatisch gesendet.

  • Sie können Mails löschen. Sie werden in den Papierkorb verschoben. Bei einem Zimbra-Account werden die Mails bei der nächsten Verbindung mit dem Internet in den Papierkorb Ihres Accounts verschoben. Wenn Sie den Papierkorb leeren, werden die Mails gelöscht.

  • Sie können die Suche und die Erweiterte Suche nutzen, um Mails und Anhänge zu finden, die sich in Ihrem Zimbra Desktop-Account befinden.

  • Sie können eine Suche erstellen und speichern.

  • Sie können die Einstellungen inZimbra Desktop ändern. Diese Optionen beeinflussen Ihre Arbeit in Zimbra Desktop. Wenn Sie die Einstellungen im Zimbra-Web-Client-Desktop ändern, werden auch die Einstellungen Ihres Zimbra-Web-Clients geändert. Änderungen, die Sie in den Einstellungen für andere Mail-Accounts eingeben, wirken sich nur auf die Einstellungen in Zimbra Desktop aus. Ihre Account-Einstellungen werden dadurch nicht geändert.
