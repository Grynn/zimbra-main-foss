Aufgaben löschen

Sie können eine Aufgabe in einer Aufgabenliste auf eine der folgenden Arten löschen. Klicken Sie auf die Aufgabenliste, um die Aufgaben im Inhaltsfenster anzuzeigen.

  • Wählen Sie die Aufgabe aus, die Sie löschen möchten, und klicken Sie in der Symbolleiste auf das Papierkorb-Symbol.

  • Klicken Sie mit der rechten Maustaste auf die Aufgabe und wählen Sie Löschen. Klicken Sie auf Ja.

Die Aufgabe wird aus Ihrer Liste gelöscht.
