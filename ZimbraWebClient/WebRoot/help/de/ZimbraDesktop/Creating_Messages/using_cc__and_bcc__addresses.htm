}}}Die Abkürzungen "CC:"{{{ }}}und "BCC:"{{{ }}}stehen für "Carbon Copy" (Kopie) und "Blind Carbon Copy" (Blindkopie).{{{

  • }}}Über das Feld "CC:"{{{}}} können Sie einem interessierten Nutzer eine Kopie einer Mail senden, die an einen anderen Nutzer als Hauptempfänger gerichtet ist.{{{ }}}Alle Empfänger im Feld "CC:" sehen die vollständige Liste der Empfänger, wenn sie die Mail lesen.{{{

  • }}}Über das Feld "BCC:"{{{}}} können Sie einem Nutzer eine Kopie einer Mail senden, ohne dass die anderen Empfänger der Mail die Adresse dieses Nutzers sehen.{{{

}}}So geben Sie Adressen in das Feld "BCC:"{{{ }}}einer neuen Mail ein:{{{
