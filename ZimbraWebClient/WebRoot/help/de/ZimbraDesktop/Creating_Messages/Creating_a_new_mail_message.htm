Mails verfassen und senden

}}}Sie können eine Mail-Nachricht verfassen und senden, sobald Sie sie geschrieben haben, oder Sie können einen Entwurf anlegen und diesen zu einem späteren Zeitpunkt fertigstellen und versenden.{{{

}}}Um eine neue Mail zu verfassen, klicken Sie zunächst in der Symbolleiste auf , um eine leere Seite zum Verfassen einer Mail aufzurufen.{{{ }}}Sie können auch im Feld Von einer Mail mit der rechten Maustaste klicken und Neue Mail auswählen, um eine leere Seite zum Verfassen einer Mail zu öffnen.{{{

}}}Je nach Ihren Maileinstellungen wird entweder eine Registerkarte „Verfassen“ in der Symbolleiste „Anwendung“ angezeigt und eine Seite zum Verfassen einer Mail im selben Fenster geöffnet oder es wird ein neues Fenster zum Verfassen geöffnet.{{{ }}}Sie können gleichzeitig mehrere Registerkarten „Verfassen“ geöffnet haben und zwischen den jeweiligen Registerkarten in der Symbolleiste wechseln.{{{

}}}Geben Sie im Feld An der Seite „Verfassen“ die Mail-Adressen der Personen ein, an die Sie die Mail senden{{{ }}}möchten.{{{

  • }}}Sie können Mail-Adressen nachschlagen, indem Sie auf An klicken, um ggf. Ihre Kontakte oder die globale Adressliste zu durchsuchen.{{{

  • }}}Um BCC-Adressen hinzuzufügen, klicken Sie auf BCC-Feld anzeigen neben dem Textfeld Cc:{{{ }}}in der Nachrichten-Kopfzeile.{{{

}}}Wenn Sie verschiedene Persönlichkeiten angelegt haben, wird das Feld Von auf der Seite zum Verfassen einer Mail angezeigt.{{{ }}}Wählen Sie die Persönlichkeit aus, die Sie als Von-Adresse verwenden möchten.{{{

Sie können Ihre Mail im HTML-Format oder als reinen Text (Plain-Text) verfassen. }}}Das Standardformat richten Sie über die Registerkarte "Allgemeine Einstellungen" ein.{{{ Um das Format nur für eine Mail rasch zu ändern, klicken Sie in der Symbolleiste des Mail-Fensters auf Optionen und wählen HTML oder Plain-Text aus.

  • }}}Bei der Option Als HTML formatieren können Sie Ihre Mail mit verschiedenen Schriftarten formatieren, Farben verwenden, Tabellen erstellen und Hyperlinks hinzufügen.{{{

  • Bei der Option Als Plain-Text formatieren erzeugen Sie Text ohne Formatierung. Jeder Computer kann diese Art von Mails lesen.

}}}Wenn Sie Ihre Zimbra-Accounts verwenden, können Sie eine Lesebestätigung anfordern. Klicken Sie dazu in der Symbolleiste "Verfassen" auf Optionen und wählen Sie Lesebestätigung anfordern.{{{

}}}Sie können Ihrer Nachricht Dateien anhängen, wie z. B. Bilder, Dokumente, Tabellen und Audiodateien.{{{
