Neue Mail verfassen

So verfassen Sie eine neue Mail

  1. }}}Klicken Sie in der Symbolleiste auf .{{{ Die Seite zum Verfassen wird angezeigt.

  2. }}}Geben Sie wie gewünscht Adressen, Betreff und Textkörper ein.
    {{{

Hinweis: }}}Wenn Sie Vor- und Nachnamen eingeben, die nicht der Form einer gültigen Internet-Adresse entsprechen (name@domain.com), wird ein Dialogfeld angezeigt, in dem Sie darauf aufmerksam gemacht werden, dass die Adresse anscheinend nicht gültig ist.{{{ Sie können die Mail trotzdem senden, auch wenn sie möglicherweise nicht zugestellt werden kann.

  1. }}}Wenn Sie eine Lesebestätigung erhalten möchten, klicken Sie auf Optionen und wählen Sie Lesebestätigung anfordern.{{{

  2. Geben Sie den Text der Mail in das Textfeld unterhalb des Betreffs ein.

}}}Falsch geschriebene Wörter werden automatisch unterstrichen.{{{ }}}Klicken Sie mit der rechten Maustaste auf das unterstrichene Wort, um die richtige Schreibweise auszuwählen.{{{

  1. }}}Um einen Anhang hinzuzufügen, klicken Sie auf und navigieren Sie zu der Datei, die angehängt werden soll, oder verschieben Sie die Datei auf Ihrem Computer per Drag & Drop in die Mail-Kopfzeile.{{{

  2. }}}Fügen Sie Ihre Signatur hinzu.{{{ }}}Wenn Sie verschiedene Accounts eingerichtet haben, die im Dropdown-Menü Accounts angezeigt werden, wählen Sie die Mail-Adresse für diese Mail aus.{{{ }}}Wenn Sie einen Signaturstil für den Account definiert haben, klicken Sie in der Symbolleiste des Mail-Fensters auf Signatur, um diese Signatur hinzuzufügen.{{{

  3. Klicken Sie auf Senden, um die Mail zu senden.

Wenn Sie die Mail nicht sofort senden möchten, klicken Sie auf Entwurf speichern. Die Mail wird dann im Ordner Entwürfe gespeichert.
