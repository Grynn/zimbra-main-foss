Dateien an Mail anhängen

}}}Anhänge von Mail-Nachrichten können von Ihrem Computer und von Ordnern "Dateien", Kontaktlisten oder Mail-Nachrichten von ZWC stammen.{{{

Hinweis: Die Empfänger Ihrer Mail benötigen die entsprechende Software, um Ihre Dateien zu öffnen und zu lesen. Dateien, die Sie mit gängiger Software wie z. B. Programmen aus Microsoft Office erstellt haben, können im Allgemeinen von Nutzern, die mit typischen Desktop-Systemen arbeiten, geöffnet werden. }}}Häufig genutzte Dateiformate, wie Textdateien, HTML-Dateien und Bilder in den Formaten .GIF oder .JPG, können in einer Vielzahl von Programmen geöffnet werden.{{{ Ob eine Datei geöffnet werden kann, richtet sich jedoch immer nach dem System des Nutzers.

}}}So fügen Sie Anhänge mit dem Link "Anhang hinzufügen" hinzu:{{{

  1. Verfassen Sie die Mail wie unter Neue Mail verfassen beschrieben.

  2. }}}Klicken Sie auf .{{{ }}}Das Dialogfeld "Datei(en) anhängen" zeigt die Registerkarten "Mein Computer", "Aktenkoffer", "Mail anhängen" und "Kontakte".{{{ }}}Wählen Sie die Registerkarte für den Anhangstyp aus.{{{

    • }}}Mein Computer:{{{ }}}Navigieren Sie zum gewünschten Ordner auf Ihrem Computer und wählen Sie eine Datei.{{{ }}}Wählen Sie die Datei aus und klicken Sie auf Öffnen. Alternativ können Sie Dateien per Drag & Drop aus einem Ordner in die Kopfzeile der verfassten Nachricht verschieben.{{{

    • }}}Dateien:{{{ }}}Wählen Sie einen Ordner "Dateien".{{{ }}}Die Dateien dieses Ordners werden auf der rechten Seite angezeigt.{{{ }}}Markieren Sie die Dateien, die Sie anhängen wollen.{{{

    • }}}Mail anhängen:{{{ }}}Wenn sich die Mail nicht in Ihrem Posteingang befindet, wählen Sie den gewünschten Ordner aus.{{{ }}}Sie können im ausgewählten Ordner nach einer Nachricht suchen.{{{ }}}Mail-Nachrichten werden auf der rechten Seite angezeigt.{{{ }}}Wählen Sie die Nachricht aus, die Sie anhängen wollen.{{{

    • }}}Kontakte:{{{ }}}Wählen Sie die Kontaktliste aus.{{{ }}}Die Namen in der Liste werden auf der rechten Seite angezeigt.{{{ }}}Sie können Ihre Kontaktliste nach einem Namen durchsuchen.{{{ }}}Markieren Sie die Namen, die angehängt werden sollen.{{{ }}}Sie werden als .vcf-Dateien angehängt.{{{

Hinweis: }}}Halten Sie die Strg- oder Umschalt-Taste gedrückt, um mehrere Anhänge gleichzeitig hinzuzufügen.{{{

  1. Um die Bilder, die Sie hochladen, im Textkörper anzuzeigen, aktivieren Sie das Kontrollkästchen Bilder im Textkörper anzeigen.  

  2. }}}Wenn Sie alle Anhänge hinzugefügt haben, klicken Sie auf Anhängen.{{{ Die Anhänge werden unterhalb der Betreffzeile aufgeführt.

  3. Wenn die Mail sendebereit ist, klicken Sie auf Senden. Die Mail wird mit den Anhängen gesendet.

}}}So fügen Sie Anhänge auf Ihrem Computer per Drag & Drop hinzu:{{{

  1. }}}Wählen Sie eine oder mehrere Dateien aus, die angehängt werden sollen, und halten Sie dabei die Maustaste gedrückt.{{{

  2. }}}Ziehen Sie die Datei(en) in den Kopfzeilenbereich der Nachricht und lassen Sie die Maustaste los.{{{ }}}Die Dateinamen werden in der Kopfzeile angezeigt.{{{
