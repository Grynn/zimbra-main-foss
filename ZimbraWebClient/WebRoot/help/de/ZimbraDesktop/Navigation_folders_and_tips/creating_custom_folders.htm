Benutzerdefinierte Ordner anlegen

Neben den vordefinierten Systemordnern wie Posteingang, Gesendet und Papierkorb können Sie eigene Ordner anlegen. Sie können diese benutzerdefinierten Ordner auch in anderen Ordnern anlegen, auch in Systemordnern.

Sie können in einem übergeordneten Ordner keine zwei Ordner gleichen Namens anlegen. Es kann z. B. in Ihrem Posteingang nicht zwei Ordner mit dem Namen "Shopping" geben, aber Sie können einen Ordner "Shopping" in Ihrem Posteingang und einen anderen Ordner mit dem gleichen Namen im Ordner „Gesendet“ anlegen.

So legen Sie einen neuen Ordner an

  1. Klicken Sie auf der Registerkarte Mail in der Übersicht auf Ordner.  Das Dialogfeld Neuen Ordner erstellen wird geöffnet.

  2. Geben Sie einen Namen für den Ordner ein.

  3. Wählen Sie aus, wo der neue Ordner abgelegt werden soll. Klicken Sie auf Ordner, um den Ordner auf der obersten Ebene abzulegen, oder wählen Sie einen Ablageort in der Liste der vorhandenen Ordner aus. Der neue Ordner wird in dem ausgewählten Ordner abgelegt.

  4. Klicken Sie auf OK.

Um einen Ordner für einen RSS- oder ATOM-Feed anzulegen, aktivieren Sie das Kontrollkästchen RSS-/ATOM-Feed abonnieren und geben eine URL ein.
