Versionskontrolle der Dateien im Ordner "Dateien"

Bei der Versionskontrolle können Sie die Änderungen an den Dateien im Ordner "Dateien" verwalten. Die Versionsnummer wird in der Dokumentenliste angezeigt. Bei einer Ansicht nach Versionsverlauf können Sie die Vorgängerversionen der Datei und des Verfassers einsehen. Sie können eine Vorgängerversion der Datei wiederherstellen und Sie können Versionen löschen, die Sie nicht länger benötigen.

Hinweis: Jede Version einer Datei wird gegen Ihr Accountkontingent gewertet.  

Dateien, die Sie hochladen, können ausgecheckt werden, um zu verhindern, dass mehrere Personen an derselben Datei im Ordner "Dateien" arbeiten. Diese Dateien werden auf Ihren Desktop heruntergeladen. Wenn Sie die aktualisierte Datei hochladen, wird diese eingecheckt und die Version wird geändert.

So checken Sie eine Datei aus  

  1. Klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie dann Datei auschecken.

  2. Wählen Sie im Dialogfeld "Dateien Herunterladen" die Option Datei zum Bearbeiten auschecken und klicken Sie dann auf Herunterladen.  

Die Datei wird auf Ihren Computer heruntergeladen.
