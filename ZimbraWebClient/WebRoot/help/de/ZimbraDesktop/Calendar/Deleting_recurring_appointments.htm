Wiederkehrende Termine ändern

Wenn Sie wiederkehrende Besprechungen angelegt haben, können Sie einen Einzeltermin oder die Besprechungsserie ändern oder löschen. Die Teilnehmer werden per Mail benachrichtigt.

So ändern Sie eine Besprechung

  1. Klicken Sie mit der rechten Maustaste auf die Besprechung, die geändert werden soll.

  2. Wählen Sie aus, ob Sie nur das ausgewählte Datum, die Instanz, oder die gesamte Serie öffnen möchten. Die Seite Termindetails wird geöffnet.

  3. Geben Sie Ihre Änderungen ein. Sie können den Plan oder die Teilnehmer ändern, im Feld Text eine Erläuterung hinzufügen oder einen Anhang anhängen.

Wenn es nicht erforderlich ist, die Nutzer über die vorgenommenen Änderungen zu benachrichtigen, deaktivieren Sie Benachrichtigungsmail senden. Durch Deaktivieren von "Benachrichtigungsmail senden" haben Sie die Möglichkeit, eine Besprechung zu ändern, ohne alle Teilnehmer darüber zu informieren.

  1. Klicken Sie auf OK. Die Teilnehmer werden per Mail benachrichtigt.

So löschen Sie eine Besprechung aus einer Serie

  1. Klicken Sie mit der rechten Maustaste auf den Termin, um ihn zu löschen. Wählen Sie dann Instanz>Instanz löschen, um nur diese Besprechung zu löschen.

  2.  Klicken Sie auf OK.

So löschen Sie wiederkehrende Besprechungen aus einer Serie

  1. Klicken Sie mit der rechten Maustaste auf den Termin, um ihn zu löschen. Wählen Sie Serie>Serie löschen, um die wiederkehrende Besprechung zu löschen.  

  2. Wählen Sie entweder:

    • Gesamte Terminserie löschen, um alle Referenzen auf vergangene und zukünftige Termine der Besprechung zu entfernen.

    • Nur den ausgewählten Einzeltermin und alles Folgende löschen, um alle Referenzen auf die Besprechung ab diesem Datum bis zum Enddatum der wiederkehrenden Besprechung zu löschen.  Vorherige Besprechungen dieser wiederkehrenden Serie befinden sich weiterhin auf Ihrem Kalender.

  3. Klicken Sie auf OK.

  4. Die Teilnehmer werden per Mail benachrichtigt, und der Termin der wiederkehrenden Besprechung wird aus den Kalendern der Teilnehmer gelöscht.
