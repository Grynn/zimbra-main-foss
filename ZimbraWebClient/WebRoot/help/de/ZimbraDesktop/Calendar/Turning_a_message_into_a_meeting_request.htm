Mails und Kontakte in Terminanfragen umwandeln

Sie können rasch eine Terminanfrage erstellen, indem Sie einfach eine Mail, eine Unterhaltung oder einen Kontakt auf ein Datum im Mini-Kalender ziehen.  Wenn Sie eine Mail oder eine Unterhaltung mit der Maus ziehen, werden mithilfe der Informationen in der Mail die meisten Felder auf der Terminseite ausgefüllt.

  • Der Betreff der Mail wird in den Betreff des Termins übernommen.

  • Die Teilnehmer sind die Mail-Adressen in den Feldern An und CC: der Mail bzw. bei Unterhaltungen die letzte Mail in einer Unterhaltung. Wenn Sie einen Kontakt aus Ihrer Kontaktliste in den Mini-Kalender ziehen, wird die erste Mail-Adresse des Kontakts zum Feld "Teilnehmer" hinzugefügt.

  • Der Text einer Mail bzw. der Text aller Mails einer Unterhaltung wird zum Text der Einladung. Anhänge einer Mail werden nicht an die Terminanfrage angehängt.

Mails und Unterhaltungen in Terminanfragen umwandeln

  1. Klicken Sie auf eine bestimmte Mail oder Unterhaltung und ziehen Sie sie auf ein Datum im Mini-Kalender. Daraufhin wird die Terminseite angezeigt.

  2. Überprüfen Sie die Details der Terminanfrage und nehmen Sie bei Bedarf Änderungen vor.

  3. Stellen Sie die Uhrzeit für die Besprechung ein. Handelt es sich um eine wiederkehrende Besprechung, richten Sie die Wiederholungen ein.

  4. Klicken Sie auf OK, um die Einladung zu senden.

Kontakte in Terminanfragen umwandeln

  1. Klicken Sie auf einen Namen in Ihrer Kontaktliste und  ziehen Sie den Namen auf ein Datum im Mini-Kalender. Die Terminseite wird aufgerufen. Dabei steht der Name des Kontakts im Feld "Teilnehmer".

  2. Füllen Sie die Terminseite aus und klicken Sie auf OK, um die Einladung zu senden.
