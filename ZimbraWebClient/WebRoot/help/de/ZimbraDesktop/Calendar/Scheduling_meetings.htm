Zeitplanung für Termine

Sie können in jeder Ansicht, mit Ausnahme der Ansicht "Einstellungen", Besprechungen und Termine planen. Sie können einen Termin nur für sich selbst einrichten oder eine Besprechung planen und Teilnehmer einladen.

Einzelbesprechung auf der Terminseite planen

  1. Klicken Sie in einer beliebigen Ansicht, mit Ausnahme der Ansicht "Einstellungen", in der Symbolleiste auf den Pfeil von und wählen Sie Termin aus.

Hinweis: Wenn Sie einen Termin auf andere Art einrichten möchten, klicken Sie hier.

  1. Geben Sie in die Bereiche Details Folgendes ein:

Das Feld Von zeigt an, dass Sie mehr als eine Person für Ihren Account erstellt haben. Wählen Sie die Identität, die bei dem Anlegen dieses Termins verwendet werden soll.

Das Feld Betreff ist ein Pflichtfeld. Es wird als Beschreibung im Kalender angezeigt.

Das Feld Ort kann zur Beschreibung des Sandortes für die Besprechung dienen und liefert somit beispielsweise eine Adresse, eine Gebäudenummer eine Stockwerknummer. Öffnen Sie die Registerkarte "Ort suchen", um einen Raum zu finden und zu reservieren, der im Firmenverzeichnis aufgeführt ist.  Diese Orte wurden von Ihrem Systemadministrator als Ressourcen angelegt und können reserviert werden.

Unter Anzeigen als geben Sie an, ob dieser Termin auf der Seite zur Planung der Teilnehmer als frei, vorläufig, beschäftigt oder nicht mehr im Büro angezeigt wird.

Mit Markieren als können Sie einen Termin als privat oder öffentlich kennzeichnen.

Wenn Sie mehrere Kalender haben, wählen Sie im Menü Kalender aus, in welchem Kalender das Ereignis eingerichtet werden soll.

  1. Im Bereich Zeit geben Sie Datum und Uhrzeit der Besprechung ein. Geben Sie das Anfangsdatum ein oder klicken Sie auf den Pfeil, um einen Kalender aufzurufen, in dem Sie ein Datum auswählen können. Wählen Sie das Enddatum aus. Wählen Sie die Erinnerungsbenachrichtigung aus. Die Teilnehmer werden gemäß der Zeit, die Sie hier eingestellt haben, an die Besprechung erinnert.

  1. Geben Sie die Namen der Teilnehmer ein. Die Namen der Teilnehmer können Sie auf eine der folgenden Arten eingeben:

    • Geben Sie im Textfeld Teilnehmer die Mail-Adressen getrennt durch Semikolons (;) ein. In Ihrer Kontaktliste enthaltene Namen werden angezeigt, sobald sie Ihrer Eingabe entsprechen.

    • Um die Frei/Belegt-Zeiten der Teilnehmer anzuzeigen, öffnen Sie die Registerkarte Planen. Während Sie Namen und Mail-Adressen der Teilnehmer eingeben, wird, sofern die Planung der Teilnehmer bekannt ist, deren Verfügbarkeit in horizontalen Leisten neben den Namen angezeigt. Wenn die Liste der Teilnehmer vollständig ist, kehren Sie zur Registerkarte Termin zurück.  (nur Zimbra-Accounts)

Wenn Antworten auf Anfragen markiert ist, erhalten Sie eine Mail-Benachrichtigung, wenn Teilnehmer Ihre Besprechungsanfrage annehmen oder ablehnen.  Wenn Sie keine Antworten erhalten wollen, markieren Sie das Kontrollkästchen, um die Auswahl aufzuheben.

Wenn Benachrichtigungsmail senden markiert ist, wenn auf Speichern geklickt wird, wird eine Mail an die Liste der Teilnehmer gesendet. Durch Deaktivieren von "Benachrichtigungsmail senden" haben Sie die Möglichkeit, eine Besprechung zu ändern, ohne alle Teilnehmer darüber zu informieren.

  1. Im Feld Text können Sie weitere Informationen angeben, die in der Mail enthalten sein sollen. Um Anhänge hinzuzufügen, klicken Sie in der Symbolleiste auf Anhänge hinzufügen.

  2. Klicken Sie auf Speichern. Allen Teilnehmern wird per Mail eine Einladung gesendet, und der Termin wird in den Kalendern der Teilnehmer angezeigt.
