Wiederkehrende Termine anlegen

Wiederkehrende Termine sind Termine, die mit gewisser Regelmäßigkeit stattfinden. Sie können wiederkehrende Besprechungen für die tägliche, wöchentliche, monatliche oder jährliche Wiederholung planen. Außerdem können Sie wiederkehrende Termine benutzerdefiniert planen.

  1. Klicken Sie in einer beliebigen Ansicht, mit Ausnahme der Ansicht "Einstellungen", auf den Pfeil von und wählen Sie Termin aus.

  2. Geben Sie den Betreff für die Besprechung ein. Das Feld Betreff ist ein Pflichtfeld. Der Betreff wird als Beschreibung im Kalender angezeigt.

  3. Geben Sie die Namen der Teilnehmer ein. Die Namen der Teilnehmer können Sie auf eine der folgenden Arten eingeben:

    • Geben Sie im Textfeld Teilnehmer die Mail-Adressen getrennt durch Semikolons (;) ein. Während Sie tippen, werden übereinstimmende Namen in Ihren Kontaktlisten angezeigt. Der Planer wird geöffnet und zeigt alle Frei/Belegt-Informationen, die verfügbar sind.

    • Klicken Sie auf Teilnehmer, um nach Adressen zum Hinzufügen zu suchen.

    • Klicken Sie auf Planer, um Frei/Belegt-Informationen bei der Eingabe der Teilnehmernamen zu sehen. Wenn die Pläne der Teilnehmer bekannt sind, erscheint die Verfügbarkeit in den horizontalen Leisten neben den Namen.

Wählen Sie im Abschnitt Planer den Dropdown-Pfeil in der ersten Spalte aus, um einen Teilnehmer als optionalen Teilnehmer zuzuweisen.

  1. Klicken Sie auf Ort, um nach einem Ort zu suchen. Alternativ können Sie Ihre Eingabe in das Feld machen, um so einen Nicht-Ressourcen-Standort, wie eine Straßenanschrift oder einen Gebäudenamen einzugeben.

Wenn Sie eine Ressource reservieren müssen, klicken Sie auf Ressource anzeigen. Dann suchen Sie nach einer Ressource, wie beispielsweise einem Audio-/Videogerät.

  1. Wählen Sie Startdatum und eine Startuhrzeit aus. Geben Sie das Startdatum und die Startuhrzeit ein oder klicken sie auf den Abwärtspfeil, um ein Datum bzw. eine Uhrzeit auszuwählen. Wählen Sie das Enddatum und die Enduhrzeit aus.

Hinweis: Wenn eine Zeitzone angezeigt wird, ist dies die Zeitzone, in der Sie sich befinden. Normalerweise müssen Sie die Zeitzone nicht ändern. Wenn Sie Besprechungen planen, bei denen sich die Teilnehmer in verschiedenen Zeitzonen befinden, wird in der Einladung die Uhrzeit für die Besprechung in der Zeitzone des jeweiligen Teilnehmers angezeigt. Wenn Sie z. B. eine Besprechung mit Teilnehmern in Hamburg und London planen, zeigt die Einladung an die Teilnehmer in Hamburg die Uhrzeit als GMT+1 und die Einladung an die Teilnehmer in London die Uhrzeit als GMT an (eine Stunde Unterschied).

  1. Klicken Sie auf Wiederholung, um das Wiederholungsmuster anzuzeigen. täglich, wöchentlich, monatlich, jährlich oder benutzerdefiniert. Die Besprechung ist ab dem Anfangsdatum und der Uhrzeit mit dem Wiederholungsmuster geplant. Für benutzerspezifische Wiederholungen können Sie spezielle Muster planen.

    • Jeden Tag: Für tägliche Wiederholungen können Sie angeben, ob die Besprechung jeden Tag oder in einer anderen Folge, z. B. jeden zweiten oder vierten Tag, stattfinden soll.

    • Jede Woche: Für wöchentliche Wiederholungen können Sie angeben, ob die Besprechung jede Woche oder in einem anderen Muster, z. B. jede zweite Woche, stattfinden soll.  Sie können auswählen, an welchem Wochentag die Besprechung stattfinden soll.

    • Jeden Monat: Für monatliche Wiederholungen können Sie angeben, ob der Termin jeden Monat oder in einem anderen Muster, z. B. jeden zweiten Monat, stattfinden soll.  Es wird ein Kalender angezeigt, damit Sie auswählen, an welchem Tag bzw. an welchen Tagen die Besprechung stattfinden soll.

    • Jedes Jahr: Für jährliche Wiederholungen wählen Sie den Monat und das Datum aus.

Klicken Sie auf Anpassen, um ein Wiederholungsmuster auszuwählen.

  1. Unter Erinnerung können Sie festlegen, wie viele Minuten vor der Besprechung die Teilnehmer über die Besprechung informiert werden.

  2. Wenn Sie per Mail oder SMS benachrichtigt werden sollen, markieren Sie das entsprechende Kontrollkästchen.  Wenn Sie dies nicht auf Ihrer Seite "Benachrichtigungseinstellungen" eingerichtet haben, klicken Sie auf "Konfigurieren".

  3. Im Feld Text können Sie weitere Informationen angeben, die in der Mail enthalten sein sollen. Um Anhänge hinzuzufügen, klicken Sie in der Symbolleiste auf Anhänge hinzufügen. In der Symbolleiste unter Optionen können Sie das Versenden von Antworten auf Fragen und das Versenden einer Benachrichtigungsmail deaktivieren. Darüber hinaus können Sie den Termin als "Privat" kennzeichnen ...

  4. Klicken Sie auf Senden. Alle Teilnehmer werden per Mail eingeladen.

Wenn Sie die Einladung zu einem späteren Zeitpunkt versenden wollen, klicken Sie auf Speichern. Dadurch wird eine lokale Kopie des Termins gespeichert, es werden jedoch noch keine Einladungen an die Teilnehmer versendet.
