Neues Dokument erstellen

Sie können eine Seite im Aktenkoffer anlegen und derenInhaltslayout mithilfe der Funktionen für Rich-Text-Format gestalten. Sie können die Rechtschreibung auf der Seite überprüfen und über Suchbegriffe problemlos Links auf andere Seiten erstellen.

  1. Klicken Sie auf den Aktenkoffer-Ordner, in dem Sie arbeiten wollen. Der Fensterbereich Inhalt zeigt die Dateien an, die sich bereits im Aktenkoffer befinden.

  2. Wählen Sie in der Symbolleiste aus der Liste Neu die Option Dokument aus. Es wird eine leere Seite angezeigt. Geben Sie in die Überschriftszeile den Namen für das Dokument ein.

  3. Verfassen Sie den Inhalt der Seite. Standardmäßig arbeiten Sie im so genannten Rich-Text-Editor, mit dem Sie zu Ihrer Seite Stile, Farben und Tabellen hinzufügen können.

  1. Klicken Sie auf Speichern, um die Datei zu speichern. Zum Schließen des Dokuments klicken Sie auf das X in der oberen rechten Ecke.

Hinweis: Wenn Sie das Dokument im falschen Aktenkoffer-Ordner erstellt haben, können Sie die Datei in den richtigen Ordner verschieben. Dazu wählen Sie das Dokument aus und ziehen es in den richtigen Ordner.
