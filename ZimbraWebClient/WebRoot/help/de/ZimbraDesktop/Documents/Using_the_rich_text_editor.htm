Im Rich-Text-Editor arbeiten

}}}Sie können Ihre Mail-Nachrichten und Dokumente im Aktenkoffer verfassen - beispielsweise mit dem Rich-Text-Editor zur Formatierung des Seiteninhalts.{{{

}}}Wenn Sie im Rich-Text-Editor arbeiten, steht Ihnen eine Symbolleiste für die bequeme Formatierung Ihrer Seite zur Verfügung.{{{ Wählen Sie Schriftgrößen, Schriftarten, Textstile, Schriftfarben und Hintergrundfarben aus. Diese Formatierungswerkzeuge funktionieren genauso wie in vielen Textverarbeitungsprogrammen. }}}Außerdem können Sie Ihrer Seite Tabellen, Anhänge und Bilder hinzufügen.{{{

Absätze formatieren

}}}Das Symbol in der Symbolleiste zeigt die verfügbaren Absatzformatierungen an.{{{ }}}Diese Absatzformatierungen vereinfachen die Formatierung von Text und sorgen für einen einheitlichen Stil in Ihrem Dokument.{{{ }}}Es gibt u. a. folgende Absatzformatierungen: Normal für die meisten Absätze, sechs Ebenen von Überschriften zur Identifizierung von Abschnittstiteln, Adresse, um Text kursiv darzustellen und Vorformatiert, um Text in der Schriftart Courier darzustellen.{{{ }}}Diese Absatzformatierungen können nicht geändert werden. Sie können jedoch die Schriftart, Größe, Farbe usw. mithilfe der Bearbeitungsoptionen ändern.{{{

Tabellen auf einer Seite einfügen

Um eine Tabelle auf Ihrer Seite einzufügen, sind drei Schritte erforderlich. Sie fügen eine Tabelle ein, richten die Tabelleneigenschaften ein und füllen dann die Tabelle mit Inhalt. }}}Um eine Tabelle einzufügen, klicken Sie in der Symbolleiste auf das Tabellensymbol, wählen Tabelle einfügen aus und geben dann an, wie viele Zeilen und Spalten die Tabelle enthalten soll.{{{

Anhänge und Bilder zu Ihrer Seite hinzufügen

}}}Sie können Dateien mit Verknüpfungen aus der Dokumentendatei hochladen.{{{ }}}Klicken Sie auf die Büroklammer oder auf das Symbol für Bilder, , und suchen Sie die Datei, die hochgeladen werden soll.{{{ }}}Sie werden benachrichtigt, wenn Sie eine Datei hochladen, die bereits im Aktenkoffer vorhanden ist.{{{

}}}Mit einer Webseite verknüpfen{{{

}}}Statt die URL anzuzeigen, können Sie eine Verknüpfung zu einer externen Web-URL erstellen und dieser einen aussagekräftigen Titel verleihen.{{{ }}}Klicken Sie auf das Symbol und geben Sie die Seite oder die Web-URL ein, zu der die Verknüpfung erstellt werden soll.{{{
