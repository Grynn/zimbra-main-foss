Nach Mails suchen

Sie können nach Mails, Unterhaltungen oder Kontakten suchen.

  • Geben Sie dafür im Textfeld "Suchen" Ihren Suchbegriff ein. Dies wird unter Schnellsuche beschrieben.

  • }}}Mit der erweiterten Suche können Sie ausführliche Suchkriterien angeben.{{{

So suchen Sie über das Textfeld "Suchen" nach Mails

  1. Geben Sie im Feld Suchen Ihre Suchkriterien ein, z. B. has:attachment für Mails mit Anhängen oder auch einen Absendernamen.

  2. }}}Klicken Sie auf den Pfeil von „Suchen“ und wählen Sie die Art der Suche in der Liste aus.{{{

  3. Klicken Sie auf Suchen.

So suchen Sie mithilfe von ausführlichen Kriterien nach Mails

  1. Klicken Sie auf Erweitert, um den Bereich für die erweiterte Suche zu öffnen.

  2. Klicken Sie auf die Symbole in der Symbolleiste, um das Fenster für die Art von Kriterien aufzurufen, die Sie für Ihre Suche nutzen möchten.

  3. }}}Geben Sie die Suchkriterien ein.{{{

Sobald Sie in einem Suchfenster Kriterien eingeben oder in einem anderen Suchfenster eine Auswahl geändert haben, indem Sie z. B. ein anderes rundes Optionsfeld aktivieren oder eine Suchoption in einer Dropdown-Liste auswählen, wird die Suche ausgeführt.

Elemente, die den Suchkriterien entsprechen, werden unten rechts aufgeführt. Wenn die Unterhaltungsfunktion aktiviert ist, werden Unterhaltungen, die Mails enthalten, welche den Suchkriterien entsprechen, aufgelistet.

Hinweis: Um den Bereich der Suchergebnisse zu aktualisieren, klicken Sie auf Suchen.
