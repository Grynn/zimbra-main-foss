Suchabfragen speichern

Wenn Sie eine Suche erstellen, die Sie später wahrscheinlich noch einmal durchführen möchten, können Sie sie speichern.

  1. Erstellen Sie die Suchabfrage, indem Sie Text in die Leiste Suchen eingeben oder indem Sie über die erweiterte Suche eine Abfrage erstellen.

  2. Um die Suche zu speichern, klicken Sie auf in der Suchleiste. Daraufhin wird das Dialogfeld Suche speichern angezeigt.

  3. Geben Sie einen Namen für die gespeicherte Suche ein. Suchabfragen werden standardmäßig in dem Suchordner Suchvorgänge im Übersichtsfenster gespeichert. Sie können jedoch auch einen anderen Ordner angeben.

  4. Klicken Sie auf OK. Ihre Suche wird gespeichert.

Um eine gespeicherte Suche zu verwenden, klicken Sie im Suchordner darauf. Die Suchergebnisse werden sofort im Inhaltsfenster angezeigt.
