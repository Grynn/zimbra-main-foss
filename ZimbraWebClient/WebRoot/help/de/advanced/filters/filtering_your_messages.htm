Mails filtern

Beim Filtern werden eingehende Mails anhand von Übereinstimmungsregeln geprüft und entsprechende Maßnahmen eingeleitet.

Sie können eingehende Mails filtern, um sie in bestimmte Ordner zu verteilen, um sie automatisch zu kennzeichnen, weiterzuleiten oder zu löschen. Sie können z. B. folgende Regel erstellen:

  • Alle Mails von Ihrem Vorgesetzten werden im Ordner "Vorgaben vom Chef" abgelegt.

  • Alle Mails der Mailing-Liste "Firmenveranstaltungen" werden mit dem Tag "Veranstaltungen"gekennzeichnet.

So erstellen oder ändern Sie eine Filterregel

  1. Klicken Sie auf die Registerkarte Einstellungen .

  2. Öffnen Sie die Registerkarte Mailfilter und klicken Sie auf Neuer Filter.

  3. Daraufhin wird das Dialogfeld Filter hinzufügen angezeigt.

  4. Geben Sie unter Filtername einen Namen für die Regel ein.

  5. Wählen Sie unter Wenn eine/alle der folgenden Bedingungen erfüllt ist/sind eine Gruppierungseinstellung aus.  

  • Eine der folgenden Bedingungen bedeutet, dass die Aktion angewendet wird, wenn eine der Bedingungen im Filter erfüllt ist.

  • Alle der folgenden Bedingungen bedeutet, dass alle Bedingungen im Filter erfüllt sein müssen, damit die Filteraktion angewendet wird.  

Sie können die folgenden Schritte wiederholen, um mehrere Bedingungen und Aktionen in einem Filter einzurichten.

  • Wählen Sie in der ersten Dropdown-Liste aus, welchen Teil der Mail die Filterbedingung betreffen soll.  

  • Wählen Sie aus, welcher Vergleich vorgenommen werden soll. Die angebotenen Optionen richten sich nach Ihrer Auswahl in der vorherigen Dropdown-Liste. Es wird beispielsweise "enthält" oder "enthält nicht" angezeigt.

  • Geben Sie im Textfeld ein Wort oder einen Satzteil für den Vergleich ein.

Klicken Sie auf das wenn Sie weitere Bedingungen definieren möchten. Sie können weitere Bedingungen hinzufügen oder mit dem nächsten Bereich fortfahren, in dem Sie eine oder mehrere Aktionen hinzufügen.

  1. Wählen Sie unter Folgende Aktionen ausführen in der Dropdown-Liste eine Aktion aus.

  • Geben Sie ggf. einen Ordner bzw. ein Tag an. Was Sie angeben, richtet sich nach der ausgewählten Aktion.

Klicken Sie auf das wenn Sie weitere Bedingungen ergänzen möchten. Sie können weitere Aktionen hinzufügen oder Ihre Eingaben mit OK beenden.

  1. Aktivieren Sie das Kontrollkästchen Aktiv, um die Filterregel einzuschalten.

Der Filter wird automatisch auf alle neu empfangenen Mails angewendet, sobald diese eingehen.

  • Sie können einen neuen Filter auch aus einer Mail heraus definieren. Klicken Sie mit der rechten Maustaste auf die Mail und wählen Sie Neuer Filter. Das Dialogfeld Filter bearbeiten wird angezeigt. Dabei sind die Felder Von, An und Betreff mit den Daten aus dieser Mail ausgefüllt. Wenn die Filterbedingungen korrekt sind, geben Sie einen Namen für den Filter ein und bestätigen mit „OK“.
