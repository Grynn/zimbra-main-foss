Filterregeln einrichten

In Mailfiltern können Sie Regeln definieren, um eingehende Mails zu verwalten. Mailfilter definieren Sie auf der Registerkarte Einstellungen, Mailfilter.

Wenn eine eingehende Mail den Bedingungen einer Filterregel entspricht, werden die Aktionen, die in der Filterregel festgelegt sind, durchgeführt. Sie können beispielsweise eine Filterregel definieren, die alle Mails erkennt, die von Ihrem direkten Vorgesetzten gesendet werden. Diese Mails werden dann in dem Ordner "Vom Chef" abgelegt. Oder Sie lassen die Mails einer bestimmten Adresse direkt in den Papierkorb wandern.

Jede Filterregel besteht aus einer oder mehreren Bedingungen und einer oder mehreren Aktionen.

Filterbedingungen

Jede Filterregel kann mehrere Bedingungen enthalten. Wenn Ihre Vorgesetzte Ihnen beispielsweise Mails von mehreren Adressen sendet, z. B. von julia.schmidt@firma.de oder von julia-schmidt23@yahoo.com, können Sie den Filter "Chef" mit zwei Bedingungen, eine für jede Adresse, definieren.

Die Bedingungen sind u. a.:

  • Bestimmte Adressen in den Feldern Von, An, CC in der Kopfzeile einer Mail

  • Vorhandensein bzw. Nichtvorhandensein von Dateianhängen

  • Wörter bzw. Zeichenfolgen im Betreff oder Inhalt der Mail

  • Für alle Bedingungen können Sie auch angeben, dass diese "nicht" erfüllt sind. Sie können z. B. Mails filtern, die ein bestimmtes Wort nicht enthalten.

Sie können Bedingungen kombinieren, um anhand von komplexeren Kriterien nach Mails zu suchen.

Eine oder Alle

Bedingungen in einer Filterregel können mithilfe der Operatoren Eine oder Alle gruppiert werden. Diese Operatoren werden ähnlich wie die Suchfunktionen vom Typ "UND" und "ODER" verwendet, die bei der Suchfunktion beschrieben werden, wobei Eine der Funktion ODER und Alle der Funktion UND entspricht.

Wenn Sie bei der Definition der Bedingungen für eine neue Filterregel den Operator Eine angeben, dann gilt eine Mail, die eine der Bedingungen erfüllt, als Entsprechung. Wenn Sie jedoch Alle angeben, dann muss die Mail alle in der Filterregel definierten Bedingungen erfüllen, um dem Filter zu entsprechen.

Aktionen für Filterregeln

Jede Filterregel kann eine oder mehrere Aktionen angeben. Die Aktionen sind u. a.:

  • Mail im Posteingang belassen (keine Aktion)

  • Mail in einen anderen, angegebenen Ordner verschieben

  • Mail mit Tag markieren

  • Mail als gelesen oder als markiert kennzeichnen

  • Verwerfen: Diese Aktion löscht die Mail einfach. Die Aktion entspricht nicht der Löschfunktion im Menü. Wenn Sie eine Mail löschen, wird sie im Papierkorb abgelegt. Bei der Aktion „Verwerfen“ wird verhindert, dass die Mail Ihre Mailbox überhaupt erreicht.

  • Keine weiteren Filter anwenden: Diese Aktion verhindert, dass auf Mails, die der aktuellen Regel entsprechen, weitere Filterregeln angewendet werden.

Die Aktion „Keine weiteren Filter anwenden“ sollte in jeder Filterregel die letzte Aktion sein. Wird diese Aktion nicht angegeben, werden nachfolgende Filterregeln auf die Mail angewendet, auch wenn die Mail dem aktuellen Filter entspricht. Es ist unwahrscheinlich, dass sie möchten, dass eine Mail mehreren Filterregeln entspricht und mehreren, möglicherweise sogar gegensätzlichen Aktionen unterworfen wird.

Zeitpunkt der Filteranwendung

Jede eingehende Mail wird zum Zeitpunkt der Zustellung der Mail anhand der Filterregeln geprüft, und die Filteraktionen werden zu diesem Zeitpunkt auf entsprechende Mails angewendet.

Reihenfolge der Anwendung nach System

Die Filter werden in der Reihenfolge angewendet, in der sie auf der Registerkarte „Mailfilter“ aufgeführt sind. Diese Reihenfolge können Sie ändern.

Jede Mail wird anhand der Bedingungen in jedem Filter geprüft. Wenn die Mail die Bedingungen (entsprechend dem Operator Eine oder Alle der Filterregel) insgesamt erfüllt, gilt sie als Entsprechung. Die Reihenfolge der Bedingungen in einem Filter ist nicht relevant.

Wenn die Mail den Bedingungen entspricht, werden alle der Filterregel zugewiesenen Aktionen in der Reihenfolge ausgeführt, in der sie in dem Filter aufgeführt sind.

Aktive und nicht aktive Filter

Alle aktuell definierten Filter werden auf der Registerkarte „Einstellungen, Mailfilter“ aufgeführt. Über das Kontrollkästchen Aktiv können Sie Filter ein- bzw. ausschalten, ohne die Filterregeln zu löschen.
