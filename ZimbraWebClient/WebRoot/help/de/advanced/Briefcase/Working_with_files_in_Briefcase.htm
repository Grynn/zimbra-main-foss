Mit Dateien im Ordner "Dateien" arbeiten

Dateien im Ordner "Dateien" öffnen

Dateien, die sich in Ihrem Aktenkoffer befinden, können Sie auf jedem Computer öffnen. Sie benötigen die entsprechende Softwareanwendung, um die Datei zu öffnen, zu lesen und zu ändern. Häufig genutzte Dateiformate wie Textdateien, HTML-Dateien und Bilder in den Formaten .gif oder .jpg können mit einer Vielzahl von Programmen geöffnet werden.

Um eine Datei zu öffnen, klicken Sie mit der rechten Maustaste auf die Datei und wählen Öffnen aus. Wenn der Dateityp unbekannt ist, wird ein Dialogfeld angezeigt, in dem Sie gefragt werden, was Sie mit der Datei vorhaben, ob Sie sie auf Festplatte speichern oder in einer bestimmten Anwendung öffnen möchten.

Dateien verschieben

Sie können Dateien von einem Ordner "Dateien" in einen anderen verschieben.

  1. Wählen Sie die Datei(en), die verschoben werden soll(en), aus.

  2. Klicken Sie in der Symbolleiste auf .

  3. Wählen Sie den Zielordner oder erstellen Sie einen neuen Ordner unter "Dateien".

  4. Klicken Sie auf OK.

Dateien mit Tags markieren

Sie können Dateien mit Tags markieren.

  1. Klicken Sie auf die gewünschte Datei und dann in der Symbolleiste auf .

  2. Wählen Sie den Tag aus. Tags werden nur in der Detailansicht angezeigt.

  3. Klicken Sie auf OK.

Dateien sortieren

Sie können Dateien im Aktenkoffer nach Name, Größe oder Datum sortieren.
