Dateien aus Ihrem Ordner "Dateien" senden

}}}Während Sie in den Aktenkoffer-Ordnern arbeiten, können Sie per Mail einen Link auf die Dateien in diesem Ordner, aber auch eine Datei als Mailanhang versenden.{{{ }}}Die Empfänger benötigen die entsprechende Software, um die Dateien zu öffnen.{{{

Link auf eine Datei per Mail senden

Sie können Mails senden, die Links auf Dateien im Ordner "Dateien" enthalten. }}}Der Empfänger benötigt die entsprechende Freigabeberechtigung, um mit diesen Dateien zu arbeiten.{{{ }}}Alle Änderungen, die der Empfänger an einer Datei vornimmt, werden in Ihrer Datei im Aktenkoffer-Ordner übernommen.{{{

  1. }}}Öffnen Sie den Aktenkoffer-Ordner und klicken Sie mit der rechten Maustaste auf die Dateien, die Sie als Link senden möchten.{{{

  2. }}}Wählen Sie Link(s) senden.{{{ }}}Eine Bestätigungsmeldung wird eingeblendet und fordert Sie zur Bestätigung auf.{{{ Klicken Sie auf Ja. }}}Daraufhin wird die Seite zum Verfassen einer Mail geöffnet, wobei die Browser-Adresse für jeden Link im Textkörper der Mail angezeigt wird.{{{

  3. }}}Fügen Sie die Mail-Adresse des Empfängers ein und vervollständigen Sie den Text der Mail.{{{

  1. Klicken Sie auf Senden.

Datei aus dem Ordner "Dateien" per Mail senden

Sie können Dateien direkt aus dem Ordner "Dateien" heraus als Mail-Anhänge senden.  

  1. }}}Öffnen Sie den Aktenkoffer-Ordner und klicken Sie mit der rechten Maustaste auf die Dateien, die Sie senden möchten.{{{ }}}Wählen Sie Als Anhang senden aus.
    {{{ }}}Die Seite zum Verfassen einer Mail wird geöffnet{{{

  2. }}}Fügen Sie die Mail-Adresse des Empfängers ein und vervollständigen Sie den Text der Mail.{{{

  1. Klicken Sie auf Senden.
