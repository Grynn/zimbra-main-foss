Im Ordner "Dateien" arbeiten

}}}Im Aktenkoffer können Sie Dateien in Ihrem Account speichern, sodass Sie diese Dateien jederzeit nach dem Anmelden bei Ihrem Account von einem beliebigen Computer aus aufrufen können.{{{

}}}Sie können Dokumente, Kalkulationstabellen, Präsentationen, Bilder, PDF-Dateien von Ihrem Computer oder Ihrem Computer-Netzwerk hochladen, und Sie können mit Ihren Mails versendete Anhänge in einem Aktenkoffer speichern.{{{ }}}Sie haben die Möglichkeit, verschiedene Ordner im Aktenkoffer anzulegen, um Ihre Dateien zu organisieren, und Sie können Ihre Dateien für andere Nutzer freigeben.{{{

Die Dateien in diesem Ordner sind Kopien der ursprünglichen Datei. }}}Änderungen, die Sie an den Dateien in diesem Ordner vornehmen, haben keine Auswirkungen auf die ursprüngliche Datei.{{{

}}}Aktenkoffer-Ordner anlegen{{{

  1. }}}Klicken Sie auf die Registerkarte Aktenkoffer.{{{

  2. }}}Wählen Sie in der Symbolleiste unter Neu die Option Aktenkoffer.{{{


  1. Daraufhin wird das Dialogfeld Neuen Ordner in Dateien erstellen angezeigt. }}}Geben Sie einen Namen für den neuen Ordner ein.{{{

Dieser Name muss innerhalb der Hierarchie Ihrer Mailbox-Ordner eindeutig sein. }}}Aktenkoffer-Ordner dürfen nicht denselben Namen haben wie ein übergeordneter Ordner in der Mail, im Kalender oder im Adressbuch von @CompanyName@.{{{ }}}Wenn Sie beispielsweise einen Kalender mit dem Namen „Urlaub“ haben, dürfen Sie keinem übergeordneten Aktenkoffer-Ordner den Namen „Urlaub“ geben.{{{ Ein Ordner unter "Dateien", der sich in einem untergeordneten Ordner befindet, darf "Urlaub" heißen.

  1. }}}Wählen Sie die Ordnerhierarchie aus.{{{ Sie können einen neuen übergeordneten Ordner anlegen oder den Ordner in einen vorhandenen Ordner einfügen.

  2. }}}Klicken Sie auf OK.{{{ Der neue Ordner wird in der Übersicht angezeigt.

}}}Dateien hochladen{{{

}}}Beim Hochladen wird eine Datei von Ihrem Computer auf Ihren Mail-Account hochgeladen.{{{ Dadurch steht die Datei jederzeit zur Verfügung, wenn Sie sich an Ihrem Account anmelden. }}}Ihre Account-Kapazität bestimmt, wie viele Dateien unter „Dateien“ hochgeladen werden können.{{{

So laden Sie Dateien hoch

  1. }}}Öffnen Sie den Aktenkoffer und klicken Sie in der Symbolleiste auf Datei hochladen.{{{

  2. }}}Klicken Sie im Dialogfeld Neue Datei in Dateien hochladen auf Durchsuchen, um die Dateien, die hochgeladen werden sollen, zu finden.{{{ }}}Sie können mehrere Dateien gleichzeitig zum Hochladen auswählen, wenn sich diese Dateien in demselben Verzeichnis befinden.{{{ }}}Zum Hinzufügen anderer Dateien klicken Sie auf Hinzufügen und navigieren zur gewünschten Datei.{{{

Sie können keine Datei hochladen, die denselben Namen hat wie eine Datei, die bereits unter "Dateien" vorhanden ist. }}}Sie können im Dialogfeld „Hochladen“ angeben, wie beim Hochladen von Dateien mit demselben Namen vorgegangen werden soll.{{{ Wenn Sie diese Option ändern, wird die Änderung im Dialogfeld gespeichert.

    • Wenn Sie benachrichtigt werden möchten, wenn Sie eine Datei mit demselben Namen hochladen, wählen Sie Nachfragen aus. }}}Wenn eine Datei denselben Namen wie eine bereits hochgeladene Datei hat, wird das Dialogfeld Konflikt beim Hochladen angezeigt.{{{ }}}Sie können die vorhandene Datei behalten oder durch die neue Datei ersetzen.{{{ Sie können sich die neue Datei auch ansehen. Dies ist die Standardeinstellung.

    • }}}Wenn Sie möchten, dass die hochgeladene Datei immer die bereits vorhandene Datei gleichen Namens ersetzt, wählen Sie Durch meine Datei ersetzen.{{{ Die neue Datei wird hochgeladen und ersetzt die vorhandene Datei. Sie werden über diesen Vorgang dann nicht mehr informiert.

    • Wenn Sie nicht möchten, dass Ihre Dateien unter "Dateien" ersetzt werden, wählen Sie Vorhandene Datei behalten. Die Datei wird nicht hochgeladen und Sie werden dann nicht mehr informiert.

  1. Wenn alle Dateien hochgeladen sind, klicken Sie auf OK. Die Dateien werden unter "Dateien" angezeigt.  

  2. }}}Wählen Sie Ansicht, um festzulegen, wie die Dateien im Aktenkoffer angezeigt werden sollen.{{{

    • }}}Explorer-Ansicht.{{{ }}}Die Dateien werden nach Typ angezeigt (PDF, Word, PowerPoint usw.){{{

    • Detailansicht: }}}Die Dateien werden als Liste angezeigt, mit Namen, Dateityp, Dateigröße, Datum und Besitzer.{{{ }}}Klicken Sie in der Detailansicht auf die Überschrift, um die Dateien nach Namen, Größe oder Datum zu sortieren.{{{

    • Browser-Ansicht Spalten: }}}Die Dateinamen werden in Spalten angezeigt.{{{

Hinweis: Für Ihren Account ist möglicherweise eine maximale Größe für hochgeladene Dateien definiert. }}}Wenn eine Datei zu groß ist, wird eine Warnung angezeigt.
{{{ }}}Dateien, die Sie hochladen, wirken sich auf die Kapazität Ihres Accounts aus.{{{  

}}}Mail-Anhänge hinzufügen{{{

Sie können Mail-Anhänge rasch unter "Dateien" speichern.

  1. }}}Um einen Anhang hinzuzufügen, klicken Sie in der Mail-Kopfzeile neben dem Link des Anhangs auf den Link Zu Dateien hinzufügen.
    {{{

  2. Im Dialogfeld Zu Dateien hinzufügen wählen Sie den Zielordner aus oder legen einen neuen Ordner unter "Dateien" an.

  3. }}}Klicken Sie auf OK.{{{

}}}Dateien anzeigen{{{

Sie können auswählen, wie Sie die Dateien in Ihrem Ordner "Dateien" anzeigen möchten. Wählen Sie in der Symbolleiste auf der Registerkarte "Dateien" eine der folgenden Optionen im Menü Anzeigen.

  • Explorer-Ansicht: Die Dateien werden als Miniaturansichten angezeigt.

  • Detailansicht: }}}Die Dateien werden in einer Zeile mit Dateigröße, Hochladedatum, Besitzer, bei einer freigegebenen Datei sind möglicherweise nicht Sie der Besitzer, und dem Ordner, in dem sich die Datei befindet, angezeigt.{{{

  • Browser-Ansicht Spalten: Die Dateien werden in Spalten angezeigt. }}}In dieser Ansicht können Sie auf eine Datei klicken, um Angaben zu der Datei wie Größe, Erstellungsdatum, Ersteller, Änderungsdatum und ändernder Nutzer, anzuzeigen.{{{

}}}Dateien mit Tags markieren{{{

Sie können Dateien mit Tags markieren.

  1. }}}Klicken Sie auf die gewünschte Datei und dann in der Symbolleiste auf .{{{

  2. Wählen Sie den Tag aus. Tags werden nur in der Detailansicht angezeigt.

  3. }}}Klicken Sie auf OK.{{{

}}}Dateien sortieren{{{

}}}Wenn der Aktenkoffer in der Detailansicht angezeigt wird, können Sie die Dateien im Aktenkoffer nach Name, Größe oder Datum sortieren.{{{ }}}Die Sortierreihenfolge wird beim Wechsel der Ansichten beibehalten.{{{

}}}Aktenkoffer freigeben{{{

}}}Aktenkoffer-Ordner können für andere Personen in Ihrer Firma und auch für externe Nutzer freigegeben werden.{{{ }}}Sie können anderen Nutzern die Berechtigungen erteilen, im Aktenkoffer Inhalte zu erstellen und zu bearbeiten - oder Sie vergeben nur eine Leseberechtigung.

Weitere Informationen zum Freigeben von Dateien finden Sie unter Dateien freigeben.
