Aufgaben hinzufügen

Sie legen eine neue Aufgabe an, indem Sie einfach einen Namen für die Aufgabe in Ihre Aufgabenliste eingeben. Sie können auch die Aufgabenseite öffnen und ausführliche Informationen zu der Aufgabe ergänzen, u. a. Notizen und Anhänge.

  • Im Zimbra-Assistenten können Sie schnell Aufgaben anlegen, wenn Sie in einer Mail arbeiten, eine Besprechung im Kalender eintragen oder einen Kontakt anzeigen. Beim Anlegen der Aufgabe werden alle verfügbaren Informationen in den Textbereich „Aufgabe“ eingefügt.

So legen Sie rasch eine Aufgabe an

  1. Wählen Sie im Übersichtsfenster die Aufgabenliste aus.

  2. Klicken Sie im Inhaltsfenster auf die oberste Zeile, in der steht Klicken Sie hier, um eine neue Aufgabe zu erstellen.    

  3. Ein Textfeld wird angezeigt. Geben Sie den Namen der Aufgabe ein.

  4. Drücken Sie die Eingabetaste. Die Aufgabe wird in Ihre Aufgabenliste aufgenommen.

  5. Wenn Sie eine Aufgabe in der falschen Liste angelegt haben, öffnen Sie die Aufgabe und wählen in der Dropdown-Liste „Aufgabenlisten“ die richtige Liste aus. Sie können die Aufgabe auch mit der Maus auf die richtige Liste im Inhaltsfenster ziehen.

So legen Sie eine Aufgabe mit Details an

In Ihren Aufgabenlisten legen Sie Aufgaben an und verfolgen deren Fortschritt. Außerdem können Sie innerhalb Ihrer Aufgaben Notizen schreiben und Dateien anhängen, auf die Sie über die Aufgabe schnell zugreifen können. Es ist nützlich, einer Aufgabe alle relevanten Informationen beizufügen, wenn Sie Ihre Aufgabenlisten für andere Nutzer freigeben.

Sie können die Länge des Projekts schätzen, indem Sie das Anfangsdatum und das Fertigstellungsdatum eingeben und die Priorität festlegen. Wenn Sie die Aufgabe anlegen, können Sie den Status auswählen, z. B. Nicht begonnen oder In Bearbeitung. Außerdem können Sie den Prozentsatz der Fertigstellung auswählen.

  1. Wählen Sie im Navigationsfenster die Aufgabenliste aus.

  2. Klicken Sie in der Aufgaben-Symbolleiste auf Neu. Eine unbenannte Aufgabenseite wird geöffnet.

  3. Geben Sie im Feld Betreff einen Namen für die Aufgabe ein.

  4. (Optional) Legen Sie die Priorität fest, und wählen Sie unter Datum das Anfangsdatum, den Fälligkeitstag und den Status aus.

  5. Wenn Sie auf „Neu“ geklickt haben, bevor Sie die gewünschte Aufgabenliste geöffnet hatten, können Sie in der Dropdown-Liste Aufgabenliste eine Aufgabenliste auswählen.  

  1. Geben Sie im Textfeld zusätzliche Notizen ein und fügen Sie Anhänge hinzu, indem Sie in der Symbolleiste auf Anhänge hinzufügen klicken.

  2. Klicken Sie auf Speichern, um die Aufgabe zu speichern. Die neue Aufgabe wird in Ihre Aufgabenliste aufgenommen.
