Kontakte automatisch hinzufügen

Sie können Kontakte rasch in Ihre Adressbücher aufnehmen, indem Sie in einer Mail eine Adresse auswählen und mit der rechten Maustaste klicken. Wenn Sie die Adressbuchoption Automatisches Hinzufügen von Kontakten aktivieren aktiviert haben, wird eine neue Adresse, an die Sie eine Mail senden, automatisch in Ihr Adressbuch „Mailempfänger“ aufgenommen.

Kontakte aus der Mail-Kopfzeile übernehmen

Sie können rasch Mail-Adressen in Ihre Adressbücher aufnehmen. Sie können Mail-Adressen aus den Feldern Von, An, CC und BCC: übernehmen. Diese Möglichkeit haben Sie in jeder Mail- oder Unterhaltungsansicht.

Wenn Sie Ihren Mauszeiger auf einer Adresse positionieren, wird in einem Tooltip die vollständige Mail-Adresse des Namens angezeigt. Befindet sich der Name bereits in einem Adressbuch, werden Name und Mail-Adresse angezeigt. Befindet sich der Name noch nicht in einem Ihrer Adressbücher, wird nur die Mail-Adresse angezeigt.

 

Wenn Sie mit der rechten Maustaste auf die Adresse klicken, wird ein Kontextmenü mit der Option „Zu Kontakten hinzufügen“ angezeigt. Wählen Sie Zu Kontakten hinzufügen, um den Namen in Ihr Adressbuch aufzunehmen. Der Name wird im Kontaktformular angezeigt.

Sie können weitere Informationen zum Kontakt eingeben und auswählen, in welchem Adressbuch der Kontakt gespeichert werden soll.

Kontakte beim Senden einer Mail hinzufügen

Ihre Mailbox enthält automatisch das Adressbuch Mailempfänger. Wenn Sie die Adressbuchoption aktiviert haben, die dafür sorgt, dass die Adressen, an die Sie Mails senden, automatisch in Ihr Adressbuch aufgenommen werden (sofern sie noch nicht vorhanden sind), dann werden die Adressen automatisch zu Ihrem Adressbuch „Mailempfänger“ hinzugefügt.  

Das Adressbuch „Mailempfänger“ ist praktisch ein zusätzliches Adressbuch, in dem sich verschiedenste Adressen befinden.
