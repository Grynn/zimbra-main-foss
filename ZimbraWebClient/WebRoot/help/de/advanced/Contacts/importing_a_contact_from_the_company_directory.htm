Kontakte automatisch hinzufügen

Sie können Kontakte rasch in Ihre Adressbücher aufnehmen, indem Sie in einer Mail eine Adresse auswählen und mit der rechten Maustaste klicken. Wenn Sie das Adressbuch-Kontrollkästchen Automatisches Hinzufügen von Kontakten aktivieren" aktiviert haben, wird eine neue Adresse, an die Sie eine Mail senden, automatisch in die Liste der gemailten Kontakte aufgenommen.

Kontakte aus der Mail-Kopfzeile übernehmen

Sie können rasch Mail-Adressen in Ihre Adressbücher aufnehmen. Sie können Mail-Adressen aus den Feldern „Von“, „An“, „CC“ oder „BCC:“ übernehmen. Diese Möglichkeit haben Sie in jeder Mail- oder Unterhaltungsansicht.

  1. Wenn Sie Ihren Mauszeiger auf einer Adresse positionieren, wird in einem Tooltip die vollständige Mail-Adresse des Namens angezeigt. Befindet sich der Name bereits in einem Adressbuch, werden Name und Mail-Adresse angezeigt. Befindet sich der Name noch nicht in einem Ihrer Adressbücher, wird nur die Mail-Adresse angezeigt.

 

 

  1. Klicken Sie mit der rechten Maustaste auf die Adresse, und wählen Sie Zu Kontakten hinzufügen aus, um den Namen Ihrer Kontaktliste hinzuzufügen. Das Kontaktformular wird mit dem Namen eingeblendet.

Sie können weitere Informationen zum Kontakt eingeben und auswählen, in welchem Adressbuch der Kontakt gespeichert werden soll.

Kontakte beim Senden einer Mail hinzufügen

Ein Adressbuch mit gemailten Kontakten wird automatisch in Ihrer Mailbox eingerichtet. Wenn Sie unter „Einstellungen“, Registerkarte „Adressbuch“ die Einstellung Neue Kontakte zu "gemailten Kontakten hinzufügen" markiert haben, werden diesem Adressbuch automatisch neue Kontaktinformationen hinzugefügt.
