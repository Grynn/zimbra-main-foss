Neue Kontakte hinzufügen

Sie können neue Kontakte hinzufügen, indem Sie in der Symbolleiste auf den Pfeil von klicken oder indem Sie mit der rechten Maustaste auf einen Namen in der Kopfzeile einer Mail klicken und Kontakte auswählen. Wenn Sie einen neuen Kontakt hinzufügen, geben Sie die Informationen zu Name und Firma in getrennten Feldern ein. Sie können auswählen, wie Sie den Kontakt in Ihre Liste aufnehmen möchten. Sie können bis zu drei Mail-Adressen und drei Postadressen eingeben.

So fügen Sie einen neuen Kontakt hinzu

  1. Klicken Sie in der Symbolleiste auf den Pfeil neben Neu und wählen Sie Neuen Kontakt aus. Daraufhin wird das Formular Neuer Kontakt angezeigt.

  2. Geben Sie die Kontaktinformationen ein. Dazu gehören Vor- und Nachname, Mail-Adresse, Funktion, Geschäftsdaten, Details zu Mail und Instant Messenger und Notizen. Sie können auch ein Foto für den Kontakt hochladen.

  3. Wählen Sie unter Speichern als aus, wie der Name angezeigt werden soll. Standardmäßig wird der Kontakt in der Form „Nachname, Vorname“ angezeigt.

  4. Wählen Sie unter Adressbuch eines Ihrer persönlichen Adressbücher aus, in dem der Kontakt gespeichert werden soll.

  5. Klicken Sie auf Speichern.

So übernehmen Sie einen neuen Kontakt aus einer Mail

  1. Öffnen Sie die Mail. Die Kopfzeile wird am oberen Rand der Maildetails grau angezeigt. Sie können Mail-Adressen aus den Feldern Von, An, CC und BCC übernehmen.

  2. Klicken Sie mit der rechten Maustaste auf den Namen, den Sie in Ihr Adressbuch übernehmen möchten, und wählen Sie Zu Kontakten hinzufügen.

  3. Daraufhin wird das Formular Neuer Kontakt geöffnet. Die aus dem Mailkopf übernommenen Daten werden bereits angezeigt. Ergänzen Sie ggf. weitere Informationen und prüfen Sie auch die übernommenen Felder auf ihre Richtigkeit.

  4. Wählen Sie unter Speichern als aus, wie der Name angezeigt werden soll. Standardmäßig wird der Kontakt in der Form „Nachname, Vorname“ angezeigt.

  5. Wählen Sie unter Adressbuch eines Ihrer persönlichen Adressbücher aus, in dem der Kontakt gespeichert werden soll.

  6. Klicken Sie auf Speichern.

 

Sie können auch Kontaktgruppen anlegen.
