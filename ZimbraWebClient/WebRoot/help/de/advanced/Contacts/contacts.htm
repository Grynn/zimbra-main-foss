Ihr Adressbuch

Über die Funktion Adressbuch können Sie Kontaktdetails in persönlichen Adressbüchern speichern. Standardmäßig ist für Sie ein Adressbuch Kontakte und ein Adressbuch Mailempfänger vorhanden. Sie können weitere Adressbücher anlegen und diese auch für andere Nutzer freigeben.

Sie können zu jedem Ihrer Adressbücher Kontakte hinzufügen. Um einen Kontakt anzulegen, ist nur ein Name erforderlich. Sie können auch ausführliche Kontaktkarten anlegen, die den vollständigen Namen, mehrere Mail-Adressen, geschäftliche, private und andere Anschriften, Telefonnummern und Notizen zu dem Kontakt enthalten. Sie können auch Kontaktgruppen anlegen.

Standardmäßig wird das Adressbuch Mailempfänger erweitert, wenn Sie eine Mail an eine neue Adresse senden, die sich noch nicht in einem Ihrer Adressbücher befindet. Sie können diese Funktion über die Registerkarten „Einstellungen, Adressbuch“ abstellen. Deaktivieren Sie das Kontrollkästchen Automatisches Hinzufügen von Kontakten aktivieren.

Ihr Mail-Administrator kann eine Obergrenze für die Anzahl der Einträge in Ihren Adressbüchern einstellen. Wenn Sie diese maximale Anzahl erreichen, können Sie keine weiteren Kontakte aufnehmen. Sie haben jedoch die Möglichkeit, nicht benötigte Kontakte zu löschen, um Speicherplatz zu schaffen.

Beim Verfassen einer Mail zeigt die Autovervollständigungsfunktion eine Liste von Namen aus Ihren Adressbüchern an, die dem Text entsprechen, den Sie eingeben.

Kontaktinformationen anzeigen

Sie können sich die Kontaktnamen als Liste oder in detaillierten Karten ansehen. Die Kontaktliste ist eine vertikale Auflistung Ihrer gesamten Kontakte. Die Ansicht Karte zeigt alle Informationen an, die Sie für einen Kontakt in das Kontaktformular eingegeben haben.

Sie können die Ansicht für eine Sitzung über das Dropdown-Feld Anzeigen ändern. Sie können die Standardansicht des Adressbuchs auf der Registerkarte „Einstellungen, Adressbuch“ unter Standard-Kontaktansicht ändern.

Wenn Sie den Mauszeiger in einer Mail oder im Mailinhalt auf einem Namen positionieren, werden die vollständigen Kontaktinformationen als Tooltip angezeigt, sofern sich der Name in einem Ihrer Adressbücher befindet.

Ihre Adressbücher freigeben

Sie können jedes Ihrer Adressbücher freigeben. Dies ist nützlich, wenn Ihre Adressbücher bestimmte Kategorien von Kontakten enthalten, auf die andere Nutzer Zugriff erhalten sollen. Je nach der Rolle, die Sie bei der Freigabe für das Adressbuch zuordnen, hat der Nutzer nur Leseberechtigung oder er hat vollständige Zugriffsrechte und kann Kontakte ändern, ergänzen oder löschen.

Weitere Adressbuchfunktionen

Über weitere Funktionen haben Sie folgende Möglichkeiten:

  • Kontakte mithilfe von Tags markieren

  • Adressbücher importieren und exportieren
