Adressbuch

}}}Im Adressbuch können Sie Kontaktdetails speichern.{{{ Standardmäßig ist für Sie ein Adressbuch Kontakte und ein Adressbuch Mailempfänger vorhanden. }}}Sie können weitere Adressbücher anlegen und diese auch für andere Nutzer freigeben.{{{

Sie können zu jedem Ihrer Adressbücher Kontakte hinzufügen. }}}Um einen Kontakt anzulegen, ist nur ein Name erforderlich. Sie können auch ausführliche Kontaktformulare anlegen, die den vollständigen Namen, mehrere Mail-Adressen, geschäftliche, private und andere Anschriften, Telefonnummern und ein Bild enthalten.{{{ Sie können auch Kontaktgruppen anlegen.

Standardmäßig wird das Adressbuch Mailempfänger erweitert, wenn Sie eine Mail an eine neue Adresse senden, die sich noch nicht in einem Ihrer Adressbücher befindet. }}}Sie können diese Funktion unter „Einstellungen, Adressbuch“ deaktivieren.{{{ }}}Deaktivieren Sie die Option „Einstellungen“, Neue Kontakte zu "Mailempfänger" hinzufügen.{{{

Ihr Mail-Administrator kann eine Obergrenze für die Anzahl der Einträge in Ihren Adressbüchern einstellen. Wenn Sie diese maximale Anzahl erreichen, können Sie keine weiteren Kontakte aufnehmen. Sie haben jedoch die Möglichkeit, nicht benötigte Kontakte zu löschen, um Speicherplatz zu schaffen.

Beim Verfassen einer Mail zeigt die Autovervollständigungsfunktion eine Liste von Namen aus Ihren Adressbüchern an, die dem Text entsprechen, den Sie eingeben.

Ihre Adressbücher freigeben

Sie können jedes Ihrer Adressbücher freigeben. Dies ist nützlich, wenn Ihre Adressbücher bestimmte Kategorien von Kontakten enthalten, auf die andere Nutzer Zugriff erhalten sollen. Je nach der Rolle, die Sie bei der Freigabe für das Adressbuch zuordnen, hat der Nutzer nur Leseberechtigung oder er hat vollständige Zugriffsrechte und kann Kontakte ändern, ergänzen oder löschen.
