IM-Chat beginnen

Sobald Sie Freunde hinzugefügt haben, können Sie einen IM-Chat mit ihnen beginnen.

So beginnen Sie einen Chat:

  1. Doppelklicken Sie auf den Name des Kontakts, mit dem Sie chatten wollen. Es öffnet sich ein IM-Chat-Fenster mit dem Namen des Freundes in der Kopfzeile.

  2. Geben Sie unten im Chat-Fenster den Text ein, den Sie versenden wollen, und drücken Sie auf dieEingabetaste oder klicken Sie auf Senden. Die Nachricht wird unter Ihrem Namen im Chat-Fenster angezeigt.

  3. Wenn Ihr Freund Ihnen eine Antwortnachricht sendet, können Sie diese unter seinem Namen im IM-Chat-Fenster sehen.

  4. Zum Beenden eines Chats drücken Sie auf die Esc-Taste oder klicken Sie auf das rote X, um das Fenster zu schließen.

Ihr Chat-Transkript wird standardmäßig im Ordner „Chats“ gespeichert. Diesen Ordner finden Sie im Übersichtsfenster unter der Registerkarte „Mail“. Weitere Informationen zum Ändern der Speichereinstellungen finden Sie unter IM-Einstellungen einrichten.

Emoticons in Ihrem IM verwenden

Sie können auch Yahoo! Emoticons während Ihres IM-Chats verwenden. Klicken Sie auf den Pfeil der mit einem gelben Smiley versehenen Schaltfläche, um das Dropdown-Menü für die Emoticons zu öffnen.

Klicken Sie auf das gewünschte Emoticon, um es in das Textfeld einzufügen. Geben Sie Text ein, um das Emoticon einzubinden, oder drücken Sie einfach auf die Eingabetaste oder klicken Sie auf Senden, um nur das Emoticon zu senden. Das Emoticon wird im Chat-Fenster angezeigt.

Andere Möglichkeiten zum Starten eines IM-Chats

  • Klicken Sie mit der rechten Maustaste auf den Namen des Freundes und wählen Sie die Option Neuer Chat.

  • Doppelklicken Sie auf einen Kontakt in Ihrer Favoritenliste, um ein Chat-Fenster zu öffnen
