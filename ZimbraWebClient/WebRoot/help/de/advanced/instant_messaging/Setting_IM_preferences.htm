IM-Einstellungen einrichten

Mit den Optionen auf der Registerkarte Einstellungen>Instant Messaging haben Sie folgende Möglichkeiten: Denken Sie daran, auf „Speichern“ zu klicken, um alle an Ihren Einstellungen vorgenommenen Änderungen zu speichern.

  • Anmelden: Markieren Sie das Kontrollkästchen, wenn Sie wollen, dass Sie automatisch beim IM angemeldet werden, wenn Sie sich bei @ClientNameLong@ anmelden. Ist dieses Kontrollkästchen nicht ausgewählt, müssen Sie sich manuell anmelden, indem Sie Ihren Status im Statusmenü von „Offline“ in „Online“ ändern.

  • Benachrichtigungen: Wählen Sie Optionen zur Anzeige von IM-Benachrichtigungen aus

    • Bei Auswahl der Option Sofortnachrichten aktivieren werden kurze Popup-Meldungen angezeigt, um andere über Ihren Kontaktstatus zu informieren.

    • Bei Auswahl der Option Anwesenheitsbenachrichtigungen anzeigen wird eine kurze Popup-Meldung angezeigt, um Sie darüber zu informieren, ob ein Freund „Online“ oder „Offline“ ist.

    • Bei Auswahl der Option Benachrichtigung anzeigen, wenn sich der Status eines Freundes ändert wird eine kurze Popup-Meldung angezeigt, wenn sich der Status eines Freundes im @CompanyName@ IM ändert.

  • Wenn eine Nachricht eingeht: Wählen Sie die Einstellungen aus, die im IM angezeigt werden sollen.

    • Bei Auswahl der Option Sound abspielen wird ein Sound wiedergegeben, wenn eine Nachricht eingeht. Um die Sounds hören zu können, müssen Sie das Plug-in Quicktime oder Windows Media Player installiert haben.

    • Bei Auswahl der Option Browser-Titel blinkend darstellen blinkt der Browser-Titel, wenn Sie auf einer anderen Browser-Registerkarte oder in einem anderen Browser-Fenster arbeiten und eine IM-Nachricht erhalten.

    • Popup-Benachrichtigung anzeigen: Sie müssen Yahoo! BrowserPlus installiert haben.

  • Chat: Gestalten Sie Ihr Chaterlebnis individuell.

    • Bei Auswahl der Option Chatverlauf speichern werden Ihre Chat-Transkripte gespeichert. Sie können über den Ordner „Chats“ unter der Registerkarte „Mail“ auf die gespeicherten Chats zugreifen.

    • Bei Auswahl der Option Status „Nicht aktiv“ anzeigen werden Sie gegenüber Ihren Freunden als „Nicht aktiv“ angezeigt, wenn Sie @ClientNameShort@ für einen Zeitraum, der unter Zeitüberschreitung nach Leerlauf festgelegt wurde, nicht verwendet haben.

  • Unter der Option Zeitüberschreitung nach Leerlauf können Sie die Zeit auswählen, die bis zur Änderung Ihres Status verstreichen soll. Ihr Status ändert sich entsprechend Ihren Spezifikationen unter der Option Anwesenheit einstellen.

  • Unter der Option Anwesenheit einstellen können Sie den Status auswählen, der angezeigt wird, wenn Sie nicht auf den IM zugreifen. Der Status ändert sich entsprechend der unter Zeitüberschreitung nach Leerlauf festgelegten Zeitdauer.
