Neue Seite anlegen

Sie können eine Seite anlegen und den Inhalt mithilfe der Funktionen für Rich-Text-Format oder mithilfe von HTML-Tags gestalten. Sie können die Rechtschreibung auf der Seite überprüfen und über Suchbegriffe problemlos Links auf andere Seiten erstellen.

  1. Klicken Sie auf das Notizbuch, in dem Sie arbeiten möchten. Das Inhaltsverzeichnis wird im Fenster Inhalt angezeigt.

  2. Klicken Sie in der Symbolleiste auf Neu. Daraufhin wird eine leere Seite geöffnet. Geben Sie in der Kopfzeile im Feld Seite den Namen für die neue Seite ein. Dieser Name wird im Inhaltsverzeichnis angezeigt.

  3. Verfassen Sie den Inhalt der Seite. Standardmäßig arbeiten Sie im sogenannten Rich-Text-Editor, mit dem Sie zu Ihrer Seite Stile, Farben, Tabellen und Tabellenkalkulations-Arbeitsblätter hinzufügen können.

Wenn Sie sich mit dem HTML-Format auskennen und lieber HTML-Tags nutzen möchten, klicken Sie in der Symbolleiste auf Format und wählen HTML-Quelle aus. Daraufhin wird statt der Seite ein leeres Feld angezeigt, in dem Sie mithilfe von HTML-Tags die HTML-Standardsyntax eingeben und die Seite erstellen können.

  1. Klicken Sie auf Speichern, um die Datei zu speichern. Sie können nun weitere Details in die Seite aufnehmen oder auf Schließen klicken, um zum Inhaltsverzeichnis zurückzukehren.
