Im Rich-Text-Editor arbeiten

Sie können Ihre Seiten entweder im Rich-Text-Editor verfassen, um den Inhalt der Seite direkt zu formatieren, oder Sie können mit HTML-Quellcode arbeiten, um Ihre Seiten mithilfe von HTML-Tags zu formatieren. Sie können zwischen den beiden Formaten umschalten und die Funktionen im Rich-Text-Editor nutzen, um Tabellen und Tabellenkalkulations-Arbeitsblätter einzufügen.

Wenn Sie im Rich-Text-Editor arbeiten, steht Ihnen eine zweizeilige Symbolleiste für die bequeme Formatierung Ihrer Seite zur Verfügung. Wählen Sie Schriftgrößen, Schriftarten, Textstile, Schriftfarben und Hintergrundfarben aus. Diese Formatierungswerkzeuge funktionieren genauso wie in vielen Textverarbeitungsprogrammen.

Außerdem können Sie Tabellen, Tabellenkalkulations-Arbeitsblätter, Anhänge und Bilder in Ihre Seite einfügen.

Absätze formatieren

Über das Symbol in der Symbolleiste rufen Sie die verfügbaren Absatzformatierungen auf. Diese Absatzformatierungen vereinfachen die Formatierung von Text und sorgen für einen einheitlichen Stil Ihrer Notizbuchseiten. Es gibt u. a. folgende Absatzformatierungen: Normal für die meisten Absätze, sechs Ebenen von Überschriften, um Abschnittstitel zu formatieren, Adresse, um Text kursiv darzustellen und Vorformatiert, um Text in der Schriftart Courier darzustellen. Diese Absatzformatierungen können nicht geändert werden. Sie können ein Format jedoch mit den Bearbeitungsoptionen für Schriftart, Größe, Farbe usw. ändern.

Tabellen auf einer Seite einfügen

Um eine Tabelle auf Ihrer Seite einzufügen, sind drei Schritte erforderlich. Sie fügen eine Tabelle ein, richten die Tabelleneigenschaften ein und füllen dann die Tabelle mit Inhalt. Um eine Tabelle einzufügen, klicken Sie in der Symbolleiste auf das Tabellensymbol , wählen „Tabelle einfügen“ und geben dann an, wie viele Zeilen und Spalten die Tabelle enthalten soll.

Arbeitsblatt einer Tabellenkalkulation auf einer Seite einfügen 

Um ein Arbeitsblatt einer Tabellenkalkulation einzufügen, klicken Sie in der Symbolleiste auf das Arbeitsblatt-Symbol . Es wird ein Arbeitsblatt mit sechs Spalten auf der Seite eingefügt. Mit Arbeitsblättern können Sie Datenbanken anlegen, um Listen mit Aktivitäten, Planungen, Aufgaben usw. zu verfolgen. Zu den Arbeitsblättern gehören grundlegende Tabellenkalkulationsfunktionen und einige vordefinierte Funktionen.

Anhänge und Bilder zu Ihrer Seite hinzufügen

Sie können Dateien hochladen, auf die dann ein Link auf der Notizenseite eingefügt wird. Klicken Sie auf die Büroklammer, , oder auf das Symbol für Bilder, , und suchen Sie die Datei, die Sie hochladen möchten. Sie werden benachrichtigt, wenn Sie eine Datei hochladen, die bereits unter „Notizen“ oder „Dateien“ vorhanden ist.

Links auf Notizbuch- oder Webseiten einrichten

Sie können Links zu anderen Notizbuchseiten oder zu einer externen Web-URL unter „Notizen“ erstellen und dem Link einen aussagekräftigen Namen geben, damit nicht die URL angezeigt wird.
