Mails und andere Objekte mit Tags versehen

Die Zuordnung von Tags ähnelt der Markierung, außer dass Sie mehrere Tags erstellen und einem einzigen Objekt zuordnen können, um es in mehrere Kategorien gleichzeitig einzuordnen. Nachdem Sie einen Tag erstellt haben, können Sie damit Unterhaltungen, Mails und Kontakte markieren.

So markieren Sie ein Objekt mit der Maus mit einem Tag

Ziehen Sie den Tag mit der Maus aus dem Übersichtsfenster auf der linken Seite auf das Objekt im Inhaltsfenster.

So markieren Sie ein Objekt über die Menüs mit einem Tag

  1. Klicken Sie im Inhaltsfenster mit der rechten Maustaste auf das Objekt.

  2. Wählen Sie Mail mit Tag versehen (oderUnterhaltung mit Tag versehen oder Kontakt mit Tag versehen usw.).

  3. Wählen Sie in der daraufhin angezeigten Liste den Namen des Tags aus. Es werden nur Tags aufgeführt, die dem ausgewählten Objekt noch nicht zugewiesen wurden.

So erstellen Sie einen neuen Tag und weisen ihn einem Objekt zu

  1. Klicken Sie in der Liste mit der rechten Maustaste auf das Objekt und wählen Sie im Kontextmenü die Option zum Zuweisen eines Tags.

  2. Wählen Sie Neuer Tag.

  3. Geben Sie einen Namen für den Tag ein und klicken Sie auf OK.

So löschen Sie einen Tag an einem Objekt

  1. Klicken Sie in der Liste mit der rechten Maustaste auf das Objekt und wählen Sie im Kontextmenü die Option zum Zuweisen eines Tags.

  2. Wählen Sie Tag entfernen. Wenn das Objekt mit mehreren Tags versehen ist, können Sie einen Tag oder alle Tags entfernen.

So versehen Sie mehrere Objekte gleichzeitig mit Tags bzw. entfernen Tags

  1. Wählen Sie alle Objekte aus, die Sie mit Tags versehen möchten, indem Sie die Strg-Taste bzw. die Umschalttaste drücken und mit der Maus auf die gewünschten Objekte klicken.

  2. Klicken Sie auf einem der ausgewählten Objekte und wählen Sie im Kontextmenü die Option zum Zuweisen eines Tags.

  3. Wählen Sie den Tag aus, den Sie zuweisen oder entfernen möchten.

So finden Sie mit Tag versehene Mails schnell

Klicken Sie im Übersichtsfenster auf einen Tag.

Sie können auch nach Tags suchen. Geben Sie die Abfrage in das Suchfeld als Tag: <"Tagname"> ein, und klicken Sie auf Suchen. Die Suchabfrage tag:wichtig würde alle Objekte anzeigen, die mit dem Tag "wichtig" markiert sind.
