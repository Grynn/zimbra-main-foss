Termin über Schnelleingabe anlegen

Im Dialogfeld „Schnelleingabe“ können Sie rasch einen Termin anlegen. Wählen Sie in Ihrem Kalender eine Anfangsuhrzeit und doppelklicken Sie auf den Zeitraum bzw. ziehen Sie die Maus über den Zeitraum. Daraufhin wird das Dialogfeld „Schnelleingabe Termin“ geöffnet.

Die Schnelleingabe ist eine Option. Wenn Sie möchten, dass bei Doppelklick oder Ziehen im Kalender immer die Terminseite geöffnet wird, deaktivieren Sie die Schnelleingabe auf der Seite „Optionen“.

So legen Sie schnell einen Termin an

  1. Klicken Sie im gewünschten Kalender auf die Uhrzeit für den Termin, oder klicken und ziehen Sie mit der Maus von der Startzeit bis zur Endzeit des Termins. Daraufhin wird das Dialogfeld Schnelleingabe Termin angezeigt.

  2. Um in der Tages- oder Wochenansicht einen ganztägigen Termin anzulegen, klicken und ziehen Sie die Maus vom oberen Rand des Kalenders über einen oder mehrere Tage.

  1. Geben Sie Betreff und Ort ein. Wenn es sich um einen wiederkehrenden Termin handelt, geben Sie die Wiederholungsart ein.

  2. Klicken Sie auf OK. Der Termin wird im Kalender angezeigt.

Wenn Sie Teilnehmer einladen müssen oder die Optionen für die Wiederholung ändern möchten, klicken Sie auf Mehr Details, um die Terminseite zu öffnen.
