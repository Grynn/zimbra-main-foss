Termin anlegen

Sie können in jeder Ansicht, mit Ausnahme der Ansicht „Einstellungen“, Besprechungen und Termine planen. Sie können einen Termin nur für sich selbst einrichten oder eine Besprechung planen und Teilnehmer einladen.

Einzelbesprechung auf der Terminseite planen

  1. Klicken Sie in einer beliebigen Ansicht, mit Ausnahme der Ansicht „Einstellungen“, auf den Pfeil von und wählen Sie Termin aus.

  2. Wenn Sie einen Termin auf andere Art einrichten möchten, klicken Sie hier.

  1. Geben Sie unter „Details“ den Betreff, den Ort, den Frei/Belegt-Status und die Art der Besprechung (privat/öffentlich) an.

Der Betreff wird als Beschreibung im Kalender angezeigt und muss eingegeben werden.

 Orte, die von Ihrem Systemadministrator als Ressourcen angelegt wurden, können reserviert werden. Geben Sie den Namen des Ortes, sofern Sie ihn kennen, in das Feld „Ort“ ein. Wenn Sie den Namen nicht kennen, können Sie auf der Registerkarte Orte suchen nach dem Ort suchen. Wenn Sie den Ort auswählen, wird er in diesem Feld angezeigt.

Unter Anzeigen als geben Sie an, ob dieser Termin auf der Seite zur Planung der Teilnehmer als frei oder beschäftigt angezeigt wird.

Mit Markieren als können Sie einen Termin als privat, öffentlich oder vertraulich kennzeichnen.

Wenn Sie mehrere Kalender haben, wählen Sie im Menü Kalender aus, in welchem Kalender das Ereignis eingerichtet werden soll.

  1. Im Bereich Zeit geben Sie Datum und Uhrzeit der Besprechung ein. Geben Sie das Anfangsdatum ein oder klicken Sie auf den Pfeil, um einen Kalender aufzurufen, in dem Sie ein Datum auswählen können. Wählen Sie das Enddatum aus.

  1. Hinweis: Wenn eine Zeitzone angezeigt wird, (Einstellungen > Kalender), ist dies die Zeitzone, in der Sie sich befinden. Normalerweise müssen Sie die Zeitzone nicht ändern. Wenn Sie Besprechungen planen, bei denen sich die Teilnehmer in verschiedenen Zeitzonen befinden, wird in der Einladung die Uhrzeit für die Besprechung in der Zeitzone des jeweiligen Teilnehmers angezeigt. Wenn Sie z. B. eine Besprechung mit Teilnehmern in Hamburg und London planen, zeigt die Einladung an die Teilnehmer in Hamburg die Uhrzeit als GMT+1 und die Einladung an die Teilnehmer in London die Uhrzeit als GMT an (eine Stunde Unterschied).

  1. Fügen Sie Ressourcen hinzu. Wenn Sie den Namen der Ressource kennen, geben Sie ihn in das Feld „Ressource“ ein. Kennen Sie den Namen nicht, können Sie auf der Registerkarte Ressourcen suchen nach verfügbaren Ressourcen suchen.

  2. Geben Sie die Namen der Teilnehmer ein. Die Namen der Teilnehmer können Sie auf eine der folgenden Arten eingeben:

    • Öffnen Sie die Registerkarte Teilnehmer suchen. Geben Sie einen Namen ein und wählen Sie die zu verwendende Liste aus, entweder „Kontakte“ oder „Globale Adressliste“. Wählen Sie die Namen aus und klicken Sie auf Hinzufügen. Klicken Sie abschließend auf OK.

    • Geben Sie im Textfeld Teilnehmer die Mail-Adressen getrennt durch Semikolons (;) ein. In Ihrer Kontaktliste enthaltene Namen werden angezeigt, sobald sie Ihrer Eingabe entsprechen.

    • Um die Frei/Belegt-Zeiten der Teilnehmer anzuzeigen, öffnen Sie die Registerkarte Teilnehmer planen. Während Sie Namen und Mail-Adressen der Teilnehmer eingeben, wird, sofern die Planung der Teilnehmer bekannt ist, deren Verfügbarkeit in horizontalen Leisten neben den Namen angezeigt. Wenn die Liste der Teilnehmer vollständig ist, kehren Sie zur Registerkarte Termin zurück.  

  3. Im Feld Text können Sie weitere Informationen angeben, die in der Mail enthalten sein sollen. Um Anhänge hinzuzufügen, klicken Sie in der Symbolleiste auf Anhänge hinzufügen.

  4. Klicken Sie auf Speichern. Allen Teilnehmern wird per Mail eine Einladung gesendet, und der Termin wird in den Kalendern der Teilnehmer angezeigt.

Wenn Sie einen Termin zu einem freigegebenen Kalender hinzufügen, müssen Sie möglicherweise die Anzeige aktualisieren, damit der Termin angezeigt wird.
