Mögliche Kalenderansichten

Die Ansicht „Arbeitswoche“ wird beim ersten Aufruf des Kalenders angezeigt. Sie können die Ansicht jederzeit über die Symbolleiste „Kalender“ ändern. Wenn Sie nur in der Symbolleiste eine neue Kalenderansicht auswählen, gilt die Ansicht lediglich für die jeweilige Sitzung. Sie können einstellen, welche Ansicht beim Öffnen des Kalenders angezeigt werden soll. Wählen Sie hierzu die Registerkarten Einstellungen und dann „Kalender“.

Es gibt folgende Ansichten:

  • Die Ansicht Tag zeigt die Aktivitäten des jeweiligen Tages.

  • Die Ansicht Arbeitswoche zeigt die Aktivitäten von Montag bis Freitag.

  • Die Ansicht Woche zeigt die Aktivitäten aller sieben Wochentage an.  Über die Registerkarte „Einstellungen, Kalender“ können Sie auswählen, welcher Tag als erster Tag der Woche angezeigt werden soll.

  • Die Ansicht Monat zeigt die Aktivitäten eines Monats.

  • Die Ansicht Planung wird angezeigt, wenn Sie mehrere Kalender haben. Jeder Kalender wird in einer eigenen Spalte angezeigt.
