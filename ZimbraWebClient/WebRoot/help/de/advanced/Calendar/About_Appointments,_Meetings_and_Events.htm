Termine einrichten

Aktivitäten, die Sie planen, sind Termine, Besprechungen oder Ereignisse.

  • Termine: Ein Termin ist eine Aktivität, an der keine anderen Personen beteiligt sind. Wenn Sie einen Termin planen, wird keine Mail zur Bestätigung des Termins gesendet. Sie können wiederkehrende Termine einrichten. Über die Schnelleingabe können Sie rasch einen Termin anlegen.

  • Besprechungen: Besprechungen sind Termine, bei denen andere Personen beteiligt sind. Wenn Sie eine Besprechung anlegen, wählen Sie Teilnehmer aus und senden diesen per Mail eine Einladung. Sie können Besprechungen verlegen und wiederkehrende Besprechungen einrichten.

  • Ereignisse: Ein Ereignis ist eine Aktivität, die einen ganzen Tag in Anspruch nimmt. Ereignisse werden im Kalender nicht mithilfe von Uhrzeiten angezeigt, sie erscheinen als Banner im oberen Bereich des Kalenderplans.

Alle drei Arten von Aktivitäten können auf verschiedene Weise eingegeben werden. Sie können in jeder Ansicht, mit Ausnahme der Ansicht „Einstellungen“, einen Termin einrichten. Um einen Termin anzulegen, gehen Sie auf eine der folgenden Arten vor:

  • Klicken Sie in der Symbolleiste auf den Pfeil von und wählen Sie Termin. Diese Vorgehensweise ist in jeder Ansicht, mit Ausnahme der Ansicht „Einstellungen“, möglich.

  • Geben Sie Termine direkt im Kalender ein. Wählen Sie in jeder Ansicht, außer der Ansicht „Monat“, das Datum und eine Anfangsuhrzeit und doppelklicken Sie auf den Zeitraum bzw. ziehen Sie die Maus über den Zeitraum. Daraufhin wird das Dialogfeld Schnelleingabe Termin geöffnet.

  • Klicken Sie mit der rechten Maustaste entweder auf den Mini-Kalender oder die Anfangsuhrzeit in einer der Kalenderansichten. Sie können auswählen, welche Art von Termin Sie anlegen möchten: Neuer Termin oder Neuer ganztägiger Termin.

  • In einer Mail wird bestimmter Text als Datum interpretiert. Dann ist es möglich, mit der rechten Maustaste auf den Text zu klicken und einen Termin anzulegen. Ausdrücke wie „heute“, „morgen“ oder ein Wochentag bzw. ein genaues Datum werden in Mails hervorgehoben. Bewegen Sie die Maus auf einen solchen Text, um zu ermitteln, ob Sie einen Termin geplant haben. Klicken Sie auf den hervorgehobenen Text, um Ihren Kalender zu öffnen.
