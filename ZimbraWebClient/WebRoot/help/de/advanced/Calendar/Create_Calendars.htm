Kalender anlegen

Ein Kalender ist bereits vorhanden. Sie können weitere Kalender anlegen, um einen Überblick über verschiedene Arten von Ereignissen zu haben. Sie können z. B. einen Kalender für die Arbeit, einen privaten Kalender und einen für Aktivitäten der Firma anlegen. In der Planungsansicht werden die Kalender nebeneinander angezeigt.

So legen Sie einen Kalender an

  1. Öffnen Sie die Registerkarte „Kalender“.

  2. Klicken Sie in der Übersicht mit der rechten Maustaste auf Kalender. Oder klicken Sie in der Symbolleiste auf und wählen Sie Kalender aus.

  3. Geben Sie im Dialogfeld Neuen Kalender erstellen den Namen für den Kalender ein und wählen Sie eine Farbe aus, in der die Aktivitäten auf diesem Kalender angezeigt werden sollen.

  4. Klicken Sie auf OK. Der neue Kalender wird in der Kalenderliste angezeigt.

So legen Sie einen Kalender an und synchronisieren Termine mit einem externen Kalender

  1. Öffnen Sie die Registerkarte „Kalender“.

  2. Klicken Sie in der Übersicht mit der rechten Maustaste auf Kalender. Oder klicken Sie in der Symbolleiste auf und wählen Sie Kalender aus.

  3. Geben Sie im Dialogfeld Neuen Kalender erstellen den Namen für den Kalender ein und wählen Sie eine Farbe aus, in der die Aktivitäten auf diesem Kalender angezeigt werden sollen.

  4. Aktivieren Sie Termine von externem Kalender synchronisieren. Geben Sie die gültige URL einer iCal-Datei eines externen Kalenders ein.

  5. Klicken Sie auf OK.
