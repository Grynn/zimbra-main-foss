Wiederkehrende Termine anlegen

Wiederkehrende Termine sind Termine, die mit gewisser Regelmäßigkeit stattfinden. Sie können wiederkehrende Besprechungen für die tägliche, wöchentliche, monatliche oder jährliche Wiederholung planen. Außerdem können Sie wiederkehrende Termine benutzerdefiniert planen.

  1. Klicken Sie in einer beliebigen Ansicht, mit Ausnahme der Ansicht „Einstellungen“, auf den Pfeil von und wählen Sie Termin aus.

  2. Geben Sie den Betreff ein. Der Betreff wird als Beschreibung im Kalender angezeigt und muss eingegeben werden.

  3. Geben Sie einen Ort ein. Sie können jeden beliebigen Ort eingeben. Orte, die von Ihrem Systemadministrator als Ressourcen angelegt wurden, können reserviert werden. Geben Sie den Namen des Ortes, sofern Sie ihn kennen, in das Feld „Ort“ ein. Wenn Sie den Namen nicht kennen, können Sie auf der Registerkarte Orte suchen nach dem Ort suchen. Wenn Sie den Ort auswählen, wird er in diesem Feld angezeigt.

  4. Wählen Sie Datum und Uhrzeit für den Termin aus. Geben Sie das Datum ein oder klicken Sie auf den Pfeil, um einen Kalender aufzurufen, in dem Sie ein Datum auswählen können. Wählen Sie das Enddatum aus.

  1. Hinweis: Wenn eine Zeitzone angezeigt wird, ist dies die Zeitzone, in der Sie sich befinden. Normalerweise müssen Sie die Zeitzone nicht ändern. Wenn Sie Besprechungen planen, bei denen sich die Teilnehmer in verschiedenen Zeitzonen befinden, wird in der Einladung die Uhrzeit für die Besprechung in der Zeitzone des jeweiligen Teilnehmers angezeigt. Wenn Sie z. B. eine Besprechung mit Teilnehmern in Hamburg und London planen, zeigt die Einladung an die Teilnehmer in Hamburg die Uhrzeit als GMT+1 und die Einladung an die Teilnehmer in London die Uhrzeit als GMT an (eine Stunde Unterschied).

  1. Klicken Sie auf Wiederholung, um die Wiederholungsmuster anzuzeigen: täglich, wöchentlich, monatlich, jährlich oder benutzerdefiniert. Die Besprechung wird ab dem Anfangsdatum und der Uhrzeit mit dem Wiederholungsmuster geplant. Für benutzerspezifische Wiederholungen können Sie spezielle Muster planen.

  2. Wenn es für die wiederkehrende Besprechung ein Enddatum gibt, klicken Sie neben dem Feld „Wiederholung“ auf den Link.  

  3. Wählen Sie ein Wiederholungsmuster aus und klicken Sie bei Bedarf, wenn Sie das Muster weiter definieren möchten, auf den Link daneben.

    • Jeden Tag: Für tägliche Wiederholungen können Sie angeben, ob die Besprechung jeden Tag oder in einer anderen Folge, z. B. jeden zweiten oder vierten Tag, stattfinden soll.

    • Jede Woche: Für wöchentliche Wiederholungen können Sie angeben, ob die Besprechung jede Woche oder in einem anderen Muster, z. B. jede zweite Woche, stattfinden soll. Sie können auswählen, an welchem Wochentag die Besprechung stattfinden soll.

    • Jeden Monat: Für monatliche Wiederholungen können Sie angeben, ob der Termin jeden Monat oder in einem anderen Muster, z. B. jeden zweiten Monat, stattfinden soll. Es wird ein Kalender angezeigt, damit Sie auswählen, an welchem Tag bzw. an welchen Tagen die Besprechung stattfinden soll.

    • Jedes Jahr: Für jährliche Wiederholungen wählen Sie den Monat und das Datum aus.

  4. Fügen Sie Ressourcen hinzu. Wenn Sie den Namen der Ressource kennen, geben Sie ihn in das Feld „Ressource“ ein. Kennen Sie den Namen nicht, können Sie auf der Registerkarte Ressourcen suchen nach verfügbaren Ressourcen suchen.

  5. Geben Sie die Namen der Teilnehmer ein. Die Namen der Teilnehmer können Sie auf eine der folgenden Arten eingeben:

    • Öffnen Sie die Registerkarte Teilnehmer suchen. Geben Sie einen Namen ein und wählen Sie die zu verwendende Liste aus, entweder „Kontakte“ oder „Globale Adressliste“. Wählen Sie die Namen aus und klicken Sie auf Hinzufügen. Klicken Sie abschließend auf OK.

    • Geben Sie im Textfeld Teilnehmer die Mail-Adressen getrennt durch Semikolons (;) ein. Während Sie tippen, werden übereinstimmende Namen aus Ihrer Kontaktliste angezeigt.

    • Um die Frei/Belegt-Zeiten der Teilnehmer anzuzeigen, öffnen Sie die Registerkarte Planung. Während Sie Namen und Mail-Adressen der Teilnehmer eingeben, wird, sofern die Planung der Teilnehmer bekannt ist, deren Verfügbarkeit in horizontalen Leisten neben den Namen angezeigt. Wenn die Liste der Teilnehmer vollständig ist, kehren Sie zur Registerkarte Termin zurück.  

  6. Im Feld Text können Sie weitere Informationen angeben, die in der Mail enthalten sein sollen. Um Anhänge hinzuzufügen, klicken Sie in der Symbolleiste auf Anhänge hinzufügen.

  7. Klicken Sie auf Speichern. Allen Teilnehmern wird per Mail eine Einladung gesendet, und der Termin wird in den Kalendern der Teilnehmer angezeigt.
