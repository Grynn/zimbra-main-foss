}}}Inhalt Ihres Accounts sichern {{{

}}}Sie können alle Ihre Account-Daten – Mails und Anhänge, Kontakte, Kalender, Aufgaben, Notizbücher „Notizen“ und Aktenkoffer-Ordner – oder aber bestimmte Objekte aus Ihrem Account exportieren.{{{ }}}Sie können die Daten dann auf Ihrem Computer oder an einem anderen Speicherort als Sicherungskopie ablegen.{{{ }}}Die Account-Daten werden als Archivdatei in Form einer mit Gzip komprimierten TAR-Datei ( tgz) gespeichert, die importiert werden kann, um Ihren Account wieder herzustellen.{{{

 Hinweis:}}} Wenn Sie einzelne Kontaktordner exportieren, werden diese als .csv-Dateien gespeichert, einzelne Kalenderordner werden als .ics-Dateien gespeichert.{{{

Die Daten werden kopiert und nicht aus Ihrem Account entfernt.

}}}Sie exportieren die Datei aus Einstellungen > Importieren/Exportieren.{{{

So exportieren Sie Daten

  1. }}}Klicken Sie auf der Registerkarte „Einstellungen“ auf den Ordner Importieren/Exportieren.{{{

  2. }}}Exportieren Sie entweder alle Account-Daten oder wählen Sie einen bestimmten Ordner für den Export aus.{{{

    • }}}Um alle Daten für den Account zu exportieren, wählen Sie Account.{{{ Hinweis: }}}Der Papierkorb und Spam-Ordner werden nicht exportiert.{{{

    • }}}Um eine Kontaktliste oder einen bestimmten Kalender zu exportieren, wählen Sie die jeweilige Option.{{{ }}}Wenn Sie einen Kalender oder einen Kontakt für den Export auswählen, legen Sie das Exportformat fest.{{{ }}}Weitere Informationen finden Sie unter Adressbücher exportieren und iCal-Kalender exportieren.{{{

    • }}}Wenn Sie mehrere bestimmte Arten von Objekten für den gleichzeitigen Export auswählen möchten, klicken Sie auf Erweiterte Suche, um die Ansicht zu erweitern.{{{ }}}Aktivieren Sie die Kontrollkästchen der Datenobjekte, die Sie exportieren möchten.{{{ }}}Alle Objekte der ausgewählten Art aus allen Ordnern unterhalb des ausgewählten Ordners werden exportiert.{{{ }}}Wenn Sie bestimmte Daten benötigen, können Sie über den Suchfilter angeben, welche Arten von Daten exportiert werden sollen.{{{ }}}Weitere Informationen finden Sie unter Beschreibung der Sprache für Suchabfragen und unter Suchbeispiele.{{{  

  1. }}}Klicken Sie auf Exportieren und geben Sie an, wo die Datei gespeichert werden soll.{{{

  2. }}}Klicken Sie auf OK.{{{

}}}Hinweis:{{{ }}}Die Datei mit den exportierten Account-Daten kann in einem Archivierungsprogramm wie WinRAR angezeigt werden.{{{

So importieren Sie gespeicherte Account-Daten

Sie stellen einen Account mit Hilfe der exportierten Daten wieder her.  

  1. }}}Klicken Sie unter „Einstellungen“ auf den Ordner Importieren/Exportieren.{{{

  2. }}}Navigieren Sie im Abschnitt Importieren zu der Datei, die Sie importieren wollen.{{{ }}}Wenn Sie Kalender oder Kontakte importieren, geben Sie das Format an, das importiert werden soll.{{{ }}}Weitere Informationen finden Sie unter Kontakte importieren und iCal-Kalender importieren.{{{ }}}Wenn Sie „Account“ auswählen, können Sie nur die bereits exportierte TGZ-Datei in Ihren Account importieren.{{{  

}}}Wählen Sie unter Datentypen die spezifischen Objekttypen aus, die aus der gespeicherten Datei *.tgz importiert werden sollen.{{{ Wenn Sie auf "Importieren" klicken, werden nur diese Objekttypen importiert.

  1. }}}Klicken Sie auf Importieren.{{{
