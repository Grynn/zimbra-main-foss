}}}Objekte in Ihren Account importieren{{{

}}}Über Einstellungen > Importieren/Exportieren können Sie Ihre vollständigen Account-Daten, bestimmte Ordner, Kalender und Kontaktlisten importieren.{{{

Kontakte importieren

}}}Sie können Kontaktlisten importieren, die als Textdatei mit durch Komma getrenntem Text (Format .csv) gespeichert wurden.{{{ }}}Die Importfunktion steht im Ordner Einstellungen > Importieren/Exportieren zur Verfügung.{{{

  1. }}}Klicken Sie auf der Registerkarte Einstellungen auf den Ordner Importieren/Exportieren.{{{

  2. }}}Wählen Sie im Bereich „Importieren“ den Typ Kontakte.{{{

  3. }}}Wählen Sie in der Dropdown-Liste neben Kontakte den Typ der Kontaktdatei, die Sie importieren möchten:{{{

    • Zimbra-Kontakte

    • }}}Yahoo! Kontakte{{{

    • }}}GMail-Kontakte{{{

    • Outlook 2000-Kontakte

    • Outlook 2003-Kontakte

    • Outlook-Kontakte

    • }}}Thunderbird-Kontakte{{{

}}}Kontaktlisten anderer Accounts zeigen die Kontaktdaten möglicherweise nicht in derselben Reihenfolge an.{{{ }}}Indem Sie den richtigen Typ der zu importierenden Kontaktdatei auswählen, wird die Reihenfolge der Felder vorgegeben, damit die Daten korrekt in Ihrem Adressbuch angezeigt werden.{{{

  1. }}}Suchen Sie die zu importierende .csv-Datei.{{{

  2. }}}Klicken Sie auf Importieren.{{{ }}}Wählen Sie im Dialogfeld Adressbuch auswählen entweder ein vorhandenes Adressbuch, um die Kontakte hinzuzufügen, oder erstellen Sie ein neues Adressbuch, in das die Datei importiert werden soll.{{{

  3. }}}Klicken Sie auf OK.{{{ Der Inhalt der Kontaktdatei wird importiert.

iCal-Kalender importieren

}}}Sie können iCalendars-Kalender (iCal), die im Format .ics gespeichert wurden, importieren. iCalendar ist das Standardformat für den Austausch von Kalenderdaten über das Internet, das Dateiformat lautet .ics.{{{ }}}Die Importfunktion steht auf der Registerkarte Einstellungen > Importieren/Exportieren zur Verfügung.{{{

}}}Sie können Ihren iCal-Kalender in einen vorhandenen Kalender importieren oder einen neuen Kalender anlegen und die .ics-Datei in diese Datei importieren.{{{

So importieren Sie einen iCal-Kalender

  1. }}}Klicken Sie unter Einstellungen auf Importieren/Exportieren.{{{

  2. }}}Wählen Sie im Bereich „Importieren“ im Feld Typ die Option Kalender.{{{

  3. }}}Suchen Sie im Feld „Datei“ die zu importierende .ics-Datei.{{{

  4. }}}Klicken Sie auf Importieren.{{{ }}}Wählen Sie im Dialogfeld Ordner auswählen den Ordner für den Import des Kalenders aus. Wählen Sie hierzu entweder einen vorhandenen Kalender aus oder legen Sie einen neuen Kalender an.{{{

  5. }}}Klicken Sie auf OK.{{{ Der Kalender wird in Ihren Account importiert.

}}}iCal-Kalender werden regelmäßig entsprechend den vom Systemadministrator festgelegten Abfrageintervallen für dezentrale Kalender aktualisiert.{{{
