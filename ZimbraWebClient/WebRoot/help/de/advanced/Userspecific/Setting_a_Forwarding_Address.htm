Adresse für Weiterleitung einrichten

Sie können eine Mail-Adresse angeben, an die Ihre Mails weitergeleitet werden sollen. Dabei können Sie auswählen, ob die Mails in Ihrem Posteingang gespeichert oder nach der Weiterleitung automatisch gelöscht werden sollen.

So leiten Sie Ihre Mails weiter

  1. Öffnen Sie die Registerkarte Einstellungen und wählen Sie die Registerkarte Mail.

  2. Blättern Sie zum Bereich „Mails empfangen“ und geben Sie unter Wenn eine Mail eingeht: Eine Kopie weiterleiten an die vollständige Adresse für die Weiterleitung ein.

  3. Wenn die Mail automatisch in Ihrem Account gelöscht werden soll, aktivieren Sie die Option Keine lokale Kopie der Mail speichern.

  4. Klicken Sie auf Speichern.
