Passwort ändern

Der Administrator weist das erste Passwort zu. Je nach der Account-Konfiguration können Sie das Passwort möglicherweise selbst ändern. Ist dies der Fall, finden Sie die Option Passwort ändern auf der Registerkarte "Einstellungen, Allgemein".

So ändern Sie Ihr Passwort

  1. Klicken Sie auf die Registerkarte Einstellungen.

  2. Öffnen Sie die Registerkarte Allgemein und klicken Sie unter Anmeldeoptionen auf Passwort ändern.

  3. Geben Sie Ihr altes Passwort und Ihr neues Passwort ein und geben Sie Ihr neues Passwort zur Sicherheit noch einmal ein. Hinweis: Ihr Passwort darf keine Buchstaben mit Akzenten oder Umlauten enthalten, wie etwa folgende: ñ, é, ó usw.

  4. Klicken Sie auf Passwort ändern. Ihr Passwort wird nun geändert.

In Ihrer Firma gibt es möglicherweise bestimmte Regeln für Passwörter, es kann z. B. die Anzahl der Zeichen vorgegeben sein, wie häufig Sie Ihr Passwort ändern müssen und wie häufig Sie dasselbe Passwort verwenden dürfen. Wenn Sie beim Ändern Ihres Passworts Schwierigkeiten haben, wenden Sie sich an Ihren Systemadministrator.
