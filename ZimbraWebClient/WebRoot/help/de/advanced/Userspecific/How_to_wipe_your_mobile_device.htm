Wipe-Funktion für Ihr Mobilgerät ausführen

Hinweis: }}}Die Mobilfunktion ist nur bei der @CollaborationProductFullName@ Network Edition verfügbar{{{

}}}Wenn Sie Ihr Mobilgerät verloren haben oder es gestohlen wurde, können Sie den gesamten Inhalt des Geräts über den Link „Wipe-Funktion für Gerät ausführen“ auf der Seite Einstellungen > Mobilgeräte entfernen.{{{ }}}Dieser Befehl löscht alle Daten auf dem Gerät und setzt es auf die Werkseinstellungen zurück.{{{

  1. }}}Klicken Sie auf Einstellungen und wählen Sie Mobilgeräte.{{{

  2. }}}Wählen Sie das Gerät, für das Sie die Wipe-Funktion ausführen möchten, und klicken Sie auf Wipe-Funktion für Gerät ausführen.{{{  

Als Status wird Wipe-Funktion anstehend angezeigt. }}}Der Server wartet darauf, dass das Gerät eine Verbindung zum Server herstellt, um die Funktion zu starten.{{{ }}}Wenn das Gerät eine Verbindung zum Server hergestellt hat, wird der Inhalt des Geräts gelöscht.{{{ }}}Als Status wird nun Wipe-Funktion ausgeführt angezeigt.{{{

Hinweis: }}}Sie können die Ausführung der Wipe-Funktion für das Gerät abbrechen, solange sich das Gerät noch im Status „Wipe-Funktion anstehend“ befindet.{{{
