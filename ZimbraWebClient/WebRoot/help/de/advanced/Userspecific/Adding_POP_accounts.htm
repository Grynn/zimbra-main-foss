Externe Accounts hinzufügen

Statt sich bei jedem Ihrer externen Mail-Accounts anzumelden, können Sie diese Accounts direkt in diesem Zimbra-Account auf Mails abfragen. Siehe Auf externe Accounts zugreifen.

So fügen Sie externe Accounts zu Ihrem Web-Client-Account hinzu

  1. Öffnen Sie die Registerkarte Einstellungen, Accounts und klicken Sie auf Externen Account hinzufügen. Ein Name für einen neuen externen Account wird zu Ihren Accounts hinzugefügt. Nun müssen Sie diesen Account einrichten.

  2. Geben Sie unter Einstellungen für externen Account im Feld Accountnamen einen Namen für den Account ein. Beim Verfassen neuer Mails können Sie diesen Accountnamen in der Liste der Absenderadressen unter "Von" auswählen.

  3. Geben Sie im Feld Mail-Adresse die genaue Mail-Adresse des externen Accounts ein.

  4. Nun können Sie die Accounteinstellungen einrichten. Diese Daten müssen Sie ggf. bei Ihrem Mail-Provider erfragen.

    • Wählen Sie den Account-Typ, POP oder IMAP.

    • Nutzername des Accounts. Geben Sie den Namen dieses Mail-Accounts ein.

    • Mail-Server. Geben Sie den Hostnamen des Mail-Servers ein.

    • Passwort: Geben Sie das Passwort für die Anmeldung an diesem externen Account ein. Um zu überprüfen, ob das eingegebene Passwort korrekt ist, aktivieren Sie Passwort anzeigen.

    • Erweiterte Optionen Wenn Ihr Mailanbieter einen anderen Port als den Standardport nutzt, ändern Sie ihn. Wenn Sie eine sichere Verbindung nutzen, aktivieren Sie „Für den Zugriff auf diesen Server eine verschlüsselte Verbindung (SSL) nutzen“.

  5. Wählen Sie unter Mails herunterladen auf aus, wo die Mails gespeichert werden sollen. Standardmäßig legen Sie einen neuen Ordner an und geben diesen zum Ablegen der Mails an. Sie können sich die Mails aber auch in Ihrem Posteingang zustellen lassen.

  6. Aktivieren Sie Mails nach dem Herunterladen auf dem Server löschen, wenn die Mails in Ihrem externen Account gelöscht werden sollen, wenn sie an Ihren ZWC-Account gesendet werden.

  7. Geben Sie in das Textfeld unter Von: den Namen ein, der in Ihren ausgehenden Mails im Feld „Von“ angezeigt werden soll (z. B. Peter Meier). Dies ist der ausgeschriebene Name, der vor Ihrer Mail-Adresse angezeigt wird.

  8. Wenn Sie möchten, dass Antworten an einen anderen Namen und an eine andere Adresse und nicht an die im Feld "Von" angezeigte Adresse gesendet werden sollen, aktivieren Sie Antwortadresse. Geben Sie den Namen und die Adresse ein, an die die Antworten für diese Identität gesendet werden sollen. Beispiel: Antworten auf Schulungsmails, die Peter Meier sendet, sollen an Mitarbeiterschulung unter training@example.com gesendet werden.

  9. Für diese Persönlichkeit können Sie eine Signatur auswählen. Wenn Sie die entsprechende Signatur noch nicht angelegt haben, können Sie diesen Teil der Account-Angaben zu einem späteren Zeitpunkt bearbeiten. Wenn Sie nicht möchten, dass die Signatur automatisch eingefügt wird, wählen Sie Signatur nicht anhängen.

  10. Klicken Sie auf Speichern, um diesen externen Account zu speichern.

Wenn Sie die Signatur anlegen möchten, gehen Sie nach dem Speichern zur Registerkarte "Signatur". Siehe Signaturen verwenden.
