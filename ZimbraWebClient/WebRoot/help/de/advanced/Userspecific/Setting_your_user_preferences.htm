Benutzereinstellungen einrichten

Wenn Ihr Account angelegt wird, werden die Standardeinstellungen konfiguriert. Diese Optionen definieren die Funktionsweise für Ihre Mailbox, Adressbücher und Kalenderanwendungen. Sie können auch Ihre persönlichen Mail-Identitäten und POP-Accounts einrichten und einen Stil für Ihre Mailbox auswählen.

Diese Einstellungen können Sie auf der Registerkarte Einstellungen ändern. Wenn eine hier beschriebene Option in Ihrer Ansicht nicht verfügbar ist, ist die Funktion für Ihren Account nicht eingerichtet.

Die Einstellungen legen Sie auf den folgenden Registerkarten fest. Klicken Sie auf das Wort, um eine ausführliche Beschreibung der Registerkarte aufzurufen.

  • Allgemein

    • Sie sind am erweiterten Web-Client angemeldet. Wenn dies bei Ihrer Anmeldung der standardmäßig eingestellte Client-Typ sein soll, wählen Sie Erweitert aus.

    • Die Einstellung Erweitert bietet die vollständigen Webkommunikationsfunktionen. Dieser Web-Client funktioniert am besten mit neueren Browsern und schnelleren Internetverbindungen.

    • Die Einstellung Standard wird empfohlen, wenn Sie mit einer langsamen Internet-Verbindung oder mit einem älteren Browser arbeiten oder wenn Sie einen barrierefreien Zugang wünschen.

    • Unter Stil wählen Sie die Hintergrundfarbe für Ihre Mailbox-Oberfläche aus.

    • Wenn die angezeigte Zeitzone nicht korrekt ist, wählen Sie für Ihren Account die Standardzeitzone aus. Auf diese Weise legen Sie Datum und Uhrzeit für die von Ihnen gesendeten und empfangenen Mails fest, und Termine im Kalender werden korrekt angezeigt. Diese Zeitzone kann sich von der Zeitzone, die für Ihren Computer eingerichtet ist, unterscheiden.

    • Wenn Passwort ändern angezeigt wird, können Sie Ihr Passwort ändern. Wenn Sie beim Ändern Ihres Passworts Schwierigkeiten haben, erfragen Sie die Passwortregeln bei Ihrem Administrator.

      • Wenn diese Option nicht verfügbar ist und Sie Ihr Passwort ändern möchten, wenden Sie sich an Ihren Systemadministrator.

    • Unter Sucheinstellungen können Sie angeben, dass bei jeder Suche automatisch auch im Papierkorb und im Spam-Ordner gesucht wird. Standardmäßig werden diese Ordner nicht durchsucht.

    • Unter Sprache der Suche können Sie die Option Suchzeichenfolge immer anzeigen aktivieren, wenn die Details der Suchabfrage im Suchtextfeld angezeigt werden sollen.

    • Aktivieren Sie die Option Kontrollkästchen für die schnelle Auswahl von Objekten in Listen anzeigen,  um im Inhaltsfenster für jedes Objekt ein Kontrollkästchen anzuzeigen. Über die Kontrollkästchen können Sie eines oder mehrere Objekte für einen Bearbeitungsschritt auswählen, z. B. Löschen, Verschieben, als gelesen/ungelesen markieren.

  • Mail

    • Wählen Sie, wie viele Objekte (Mails oder Unterhaltungen) pro Seite angezeigt werden sollen. Standardmäßig werden 25 Objekte je Seite angezeigt, aber Sie können auch auswählen, dass 10, 50 oder 100 Objekte je Seite angezeigt werden.

    • Wählen Sie aus, wie Ihre Mails gruppiert werden sollen: nach Unterhaltungen oder nach einzelnen Mails. Wenn Sie die Ansichtsfunktion Unterhaltungen nutzen, ist standardmäßig die Ansicht nach Unterhaltungen eingestellt.

    • Das Abrufintervall gibt an, wie häufig Ihr Computer neue Mails abruft. Standardmäßig wird alle 5 Minuten auf neue Mails geprüft. Der Mindestzeitraum wird von Ihrem Zimbra-Administrator festgelegt.

    • Geben Sie an, ob Sie Mails, die im HTML-Format gesendet wurden, als HTML anzeigen möchten. Wenn Sie diese Option ausschalten, werden alle Mails als reiner Text (Plain-Text) angezeigt. Dabei sehen Sie die HTML-Tags an sich, die Formatierung wird nicht umgesetzt.

    • Optionen für die Mail-Vorschau

      • Legen Sie fest, ob beim Anzeigen von Mails das Lesefenster aufgerufen werden soll. Standardmäßig wird das Lesefenster angezeigt.

      • Sofern aktiviert, zeigt der Client Mailauszüge in der Mailliste an. Wenn Sie den Mauszeiger auf dem Betreff einer Mail positionieren, werden die ersten Wörter der Mail angezeigt.

      • Sofern aktiviert, werden die Mails durch Doppelklick in einem neuen Fenster geöffnet.

    • Aktivieren Sie Bilder, um Bilder in HTML-Mails automatisch herunterzuladen. Wenn diese Option aktiviert ist, müssen Sie nicht extra klicken, um das Bild in der Mail anzuzeigen.

    • Die Standardmailsuche definiert die Suche, die bei Ihrer Anmeldung durchgeführt werden soll. Standardmäßig wird nach Ihrem Posteingang gesucht. Wenn Sie sich anmelden, werden die Inhalte Ihres Posteingangs angezeigt. Sie können dieses Feld ändern und einen anderen Ordner, einen Tag oder eine gespeicherte Suche angeben, damit diese Suche durchgeführt wird, wenn Sie sich beim Zimbra-Web-Client anmelden. Informationen zur Syntax finden Sie unter Beschreibung der Sprache für Suchabfragen. Geben Sie eine Adresse an, an die eine Benachrichtigung gesendet werden soll, wenn Sie neue Mails erhalten.

    • Geben Sie eine Adresse an, an die Ihre Mails weitergeleitet werden. Die ursprüngliche Mail kann dabei aus Ihrer Zimbra-Mailbox gelöscht werden.

    • Geben Sie an, ob auf eingehende Mails eine automatische Antwort gesendet werden soll. Diese Funktion wird häufig als Abwesenheitsmeldung genutzt. Sie können den Text selbst eingeben.

    • Unter „Mails von mir...“ legen Sie fest, wie mit Mails verfahren werden soll, die von Ihnen gesendet werden, die Ihre Adresse als einen der Empfänger der Mail enthalten oder die an eine Adresse wie einen Verteiler gehen, in dem auch Sie enthalten sind.

     

  • Verfassen

    • Geben Sie an, welchen Texteditor Sie standardmäßig nutzen möchten: HTML oder Plain-Text.

    • Richten Sie die Standardschriftart ein, die im HTML-Format zum Verfassen von Mails, Notizen und Kalendereinträgen verwendet wird. Sie können die Schriftart, Größe und Farbe für Ihre Standardschrift definieren. Standardmäßig wird Arial, 12 Punkt, schwarz verwendet.

    • Die Option Mails immer in neuem Fenster verfassen bewirkt, dass beim Verfassen neuer Mails ein neues Fenster unabhängig von Ihrer Zimbra-Mailbox geöffnet wird, in dem Sie die Mail schreiben können. Sie können, während das Fenster zum Verfassen geöffnet ist, Ihre Mailbox anzeigen und in ihr navigieren.

    • Mails, die Sie verfassen, werden automatisch in Abständen von wenigen Minuten als Entwürfe gespeichert. Sie können diese Funktion ausschalten, wenn Sie nicht möchten, dass automatisch gespeichert wird.

    • Wählen Sie aus, ob der ursprüngliche Text im Textkörper Ihrer Antwortmail enthalten sein soll. Sie können angeben, dass der ursprüngliche Text enthalten sein soll, dass er nicht enthalten sein soll, Sie können den Text als Anhang oder im Textkörper der Mail mit einem von Ihnen ausgewählten Präfix beifügen, oder Sie können nur die letzte Mail einschließen. (Dann wird nur zitiert, was die Person, von der Sie die Mail erhalten haben, geschrieben hat. Frühere Mails sind nicht enthalten.)

    • Geben Sie an, ob dem zitierten Text der weitergeleiteten oder beantworteten Mail in jeder Zeile eines der Zeichen > oder | als Präfix vorangestellt werden soll.

    • Standardmäßig ist die Option Kopie im Ordner „Gesendet“ speichern aktiviert. Kopien der von Ihnen gesendeten Mails werden in Ihrem Ordner „Gesendet“ gespeichert.

    • Wählen Sie aus, wie der ursprüngliche Text einer Mail beigefügt werden soll, wenn Sie eine Mail weiterleiten: im Textkörper der Mail, im Textkörper der Mail mit Präfix oder als Anhang zu der Mail.

  • Signaturen

    Sie können für Ihre Mails Signaturen anlegen. Ihre Signatur kann Ihren Namen und weiteren Text mit der maximal für Ihren Account zulässigen Anzahl von Zeichen enthalten. Wenn Sie mehrere Identitäten anlegen, können Sie verschiedene Signaturen erstellen und diese den jeweiligen Adressen zuordnen.

    • Geben Sie die Signatur ein, die Sie in Ihre Mails einfügen möchten.

    • Geben Sie an, an welcher Stelle Ihre Signatur in Ihre Mails eingefügt werden soll: Die Signatur kann am Ende aller Mails oder am Ende Ihres Texts in einer neuen, beantworteten oder weitergeleiteten Mail eingefügt werden.

  • Adressbuch

    • Sie können die Funktion aktivieren, dass beim Senden von Mails die Adressen automatisch zu Ihrem Adressbuch hinzugefügt werden sollen. Wenn Sie diese Funktion aktiviert haben, werden die Adressen in Ihrem Ordner „Mailempfänger“ gespeichert.

    • Sie können aktivieren, dass die globale Adressliste für die Autovervollständigung von Adressen genutzt wird.

    • Geben Sie an, ob Kontakte in Form einer Liste oder in Form von Karten angezeigt werden.

    • Wählen Sie aus, wie viele Kontakte pro Seite angezeigt werden sollen. Die Standardeinstellung ist 25.

    • Sie haben die Möglichkeit, Kontakte in Ihr Adressbuch zu importieren. Die Datei muss sich dafür in einem Format mit durch Komma getrenntem Text befinden.

    • Sie können Ihre Adressbücher auch exportieren. Die Datei wird in ein Format mit durch Komma getrenntem Text exportiert.

     

  • Accounts

    Auf der Registerkarte „Accounts“ erstellen und verwalten Sie Ihre Persönlichkeiten. Eine Persönlichkeit ist die Mail-Identität, die in von Ihnen gesendeten Mails im Feld „Von“ verwendet wird.

    Der Standardstil wird in der primären Identität definiert. Wenn Sie neue Identitäten anlegen, können Sie dieselben Einstellungen wie bei der Standardidentität nutzen oder Sie können andere Einstellungen einrichten.

    Über die Schaltfläche Externen Account hinzufügen können Sie Ihre externen POP/IMAP-Accounts angeben, damit Sie in diesem Web-Client die Mails Ihrer anderen Accounts empfangen können. Sie können die Funktion auch so einrichten, dass Antworten auf diese empfangenen Mails die jeweilige POP/IMAP-Adresse nutzen.

  • Mailfilter

    Auf der Registerkarte „Mailfilter“ definieren Sie die Regeln für Mailfilter. Weitere Informationen dazu, wie Sie Regeln für Mailfilter einrichten, finden Sie unter Funktionsweise von Filtern.

  • Kalender

    • Wählen Sie aus, wie Ihr Zimbra-Kalender angezeigt werden soll. Standardmäßig wird die Arbeitswoche angezeigt.

    • Geben Sie an, welcher Wochentag in Ihrem Kalender der erste Tag der Woche sein soll.

    • Wenn Sie sich häufig in verschiedenen Zeitzonen aufhalten, können Sie die Option „Liste der Zeitzonen in Terminansicht“ aktivieren.

    • Wenn Sie viele Termine ohne Teilnehmer anlegen, aktivieren Sie die Option Schnelleingabe von Terminen verwenden.

    • Dann wird ein Mini-Kalender in Ihrem Kalender angezeigt. Den Mini-Kalender können Sie in jeder Ansicht aktivieren.

    • Legen Sie fest, wie viele Minuten vor einem Termin Sie daran erinnert werden möchten. Standardmäßig werden Sie fünf Minuten vorher erinnert.

    • Sie können Kalender anderer Programme importieren und exportieren.

  • Instant Messenger

    Sie können festlegen, wie der Instant Messenger (IM) in Ihren Account integriert wird.

    • Anmelden: Wenn Sie automatisch beim Instant Messenger angemeldet werden möchten, aktivieren Sie diese Option.

    • Benachrichtigungen: Hier geben Sie die Optionen für IM-Benachrichtigungen und die Anzeige Ihres Status für andere an.

    • Chat: Ihr Chatverlauf wird automatisch in einem Chatordner gespeichert. Die Nachrichten in diesem Ordner ähneln Mails, Sie können sie drucken, als Mail weiterleiten und aus dem Ordner löschen. Sie können auch die Zeitüberschreitung bei Leerlauf einrichten.

  • Tastenkürzel

    Mit Tastenkürzeln können Sie die Anwendung ohne Maus bedienen. Es steht eine Vielzahl von Tastenkürzeln zur Verfügung. Sie können auch benutzerspezifische Tastenkürzel definieren, indem Sie Ordnern, Tags und gespeicherten Suchvorgängen Zahlen zuordnen.

    • Die Liste der Tastenkürzel zeigt alle vorhandenen Tastenkürzel an.

    • Auf der Registerkarte Tastenkürzel für Mail-Ordner können Sie Tastenkürzel zum schnellen Aufrufen Ihrer Mail-Ordner und zum Verschieben von Mails in diese Ordner einrichten. Sie wählen einen Ordner aus und weisen ihm eine Zahl zu. Sie weisen z. B. Ihrem Posteingang die Zahl 3 zu: Dann können Sie an einer beliebigen Stelle in Ihrer Mailbox v3 eingeben und gelangen zum Posteingang. Wenn Sie ein Objekt auswählen und .3 eingeben, wird das Objekt in den Posteingang verschoben.

    • Auf der Registerkarte Tastenkürzel für gespeicherte Suchvorgänge erstellen Sie Tastenkürzel für den Schnellzugriff auf gespeicherte Suchvorgänge. Sie wählen eine gespeicherte Suche aus und weisen ihr eine Zahl zu. Wenn Sie z. B. der Suche, die Sie am häufigsten verwenden, die Zahl 3 zuweisen, können Sie an einer beliebigen Stelle in der Mailbox s3 eingeben, um diese Suche rasch durchzuführen.

    • Auf der Registerkarte Tastenkürzel für Tags erstellen Sie Tastenkürzel für den Schnellzugriff auf alle Objekte mit einem bestimmten Tag bzw. für das Markieren einer Mail oder eines Kontakts mit einem Tag. Sie wählen einen Tag aus und weisen ihm eine Zahl zu. Wenn Sie z. B. dem Tag Verfolgen die Zahl 3 zuweisen, können Sie an einer beliebigen Stelle in Ihrer Mailbox y3 eingeben, um alle Objekte anzuzeigen, die mit dem Tag Verfolgen markiert sind. Wählen Sie ein Objekt und geben Sie t3 ein, um das Objekt mit dem Tag Verfolgen zu markieren.

    Sie können jede beliebige Zahl zuordnen, die Zahl darf jedoch innerhalb jeder Registerkarte für Tastenkürzel nur einmal vorkommen.
