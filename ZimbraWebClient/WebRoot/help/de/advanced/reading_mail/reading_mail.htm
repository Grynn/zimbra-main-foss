Mails lesen

Standardmäßig werden alle neuen Mails in Ihrem Posteingang abgelegt. Wenn Sie Filterregeln für Mails eingerichtet haben, werden neue Mails möglicherweise an anderen Stellen und nicht in Ihrem Posteingang abgelegt. Die neben jedem Ordner in Klammern angezeigte Zahl ist die Anzahl ungelesener Mails in dem jeweiligen Ordner.

Wenn Sie die Unterhaltungsansicht nutzen, werden Mails in Unterhaltungen gruppiert. Wenn Sie den Mauszeiger auf einer Unterhaltung positionieren, zeigt ein Tooltip die erste Zeile des Inhalts der neuesten Mail an.

So lesen Sie eine Mail in der Unterhaltungsansicht

  1. Doppelklicken Sie auf die Unterhaltung, in der sich die Mail befindet.

  2. Im Inhaltsfenster werden jetzt die einzelnen Mails der Unterhaltung angezeigt. Die Spalte Fragment zeigt die erste Zeile des Mailinhalts an. Um eine Mail zu öffnen, klicken Sie auf die Zeile mit der Mail, die Sie lesen möchten.

Wenn Sie das Lesefenster geöffnet haben, wird der Mailinhalt im unteren Fenster angezeigt. Andernfalls doppelklicken Sie in der Liste der Mails auf die gewünschte Mail, um sie im Inhaltsfenster anzuzeigen.

So lesen Sie eine Mail in der Mailansicht

  • Doppelklicken Sie auf die Mail.

Der Mailinhalt wird im unteren Fenster angezeigt. Wird der Inhalt nicht angezeigt, doppelklicken Sie in der Liste der Mails auf die gewünschte Mail.
