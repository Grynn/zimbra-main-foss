Mails als ungelesen markieren

Der Zimbra-Web-Client betrachtet Mails, die Sie geöffnet haben, als gelesen. Unterhaltungen, die ungelesene Mails enthalten, sind fettgedruckt. Sie können den Status einer Mail von „gelesen“ in „ungelesen“ ändern, indem Sie mit der rechten Maustaste auf die Mail klicken und im Kontextmenü Als ungelesen markieren auswählen. Sie können auch eine Unterhaltung als ungelesen markieren.

  • Wenn Sie eine Unterhaltung als ungelesen markieren, werden alle Mails in der Unterhaltung als ungelesen markiert.

  • Wenn Sie eine einzelne Mail als ungelesen markieren, wird die Unterhaltung, in der sich die Mail befindet, als ungelesen markiert.

Auf die gleiche Weise können Sie ungelesene Mails als gelesen markieren.
