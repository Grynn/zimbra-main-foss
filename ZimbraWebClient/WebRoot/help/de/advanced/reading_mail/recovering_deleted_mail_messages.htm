Gelöschte Objekte wiederherstellen

Der Papierkorb ist für alle @CompanyName@-Anwendungen verfügbar, die Sie nutzen. Wenn Sie den Papierkorb aus der Anwendung öffnen, werden nur spezielle Objekte für die ausgewählte Anwendung angezeigt. Wenn Sie jedoch in einer beliebigen Anwendung Papierkorb leeren wählen, werden Objekte aus allen Anwendungen gelöscht.

Sie können Dateien aus dem Papierkorb vor dem Leeren des Papierkorbs wiederherstellen, indem Sie sie mit der Maus in einen anderen Ordner ziehen.

Wenn Gelöschte Objekte wiederherstellen angezeigt wird, nachdem Sie mit der rechten Maustaste auf den Papierkorb geklickt haben, können Sie gelöschte Mails aus dem Papierkorb wiederherstellen.

  1. Klicken Sie mit der rechten Maustaste auf den Mail-Papierkorb und wählen Sie die Option „Gelöschte Objekte wiederherstellen“ aus.

  2. Wählen Sie eine oder mehrere Mails aus, die wiederhergestellt werden sollen.

  3. Klicken Sie auf Wiederherstellen in, und wählen Sie dann den Zielordner aus, oder legen Sie einen neuen Ordner an.

  4. Klicken Sie auf „OK“.

  5. Schließen Sie den Papierkorb.

Aus dem Papierkorb gelöschte Objekte können innerhalb von 30 Tagen nach dem Löschen aus dem Papierkorb wiederhergestellt werden. Nach 30 Tagen sind die Objekte dauerhaft gelöscht.

Mails, die im -Papierkorb belassen werden, werden möglicherweise automatisch bereinigt, in diesem Fall können Sie sie nicht wiederherstellen.
