Mails im HTML-Format anzeigen

Sie können Mails in zwei Formaten anzeigen.

  • Plain-Text:. Mails im Plain-Text-Format enthalten keine HTML-Formatierungscodes. Plain-Text-Mails können in allen anderen Mail-Clients angezeigt werden. Sie können sich Mails auch dann im Plain-Text-Format anzeigen lassen, wenn sie HTML-Code enthalten. In der Plain-Text-Ansicht werden die HTML-Tags im Textkörper der Mail angezeigt.

  • HTML: HTML-Mails können formatierten Text, Nummerierungen, Aufzählungszeichen, farbigen Hintergrund und Links enthalten, wodurch der Text leichter lesbar wird. Einige Mail-Clients können keine HTML-Mails anzeigen.

Um Ihre Mails immer im HTML-Format anzuzeigen, aktivieren Sie auf der Registerkarte Einstellungen>Mail das Kontrollkästchen Mails als HTML anzeigen (falls möglich). Die Mails werden formatiert angezeigt, sofern sie als HTML-Mails gesendet wurden.
