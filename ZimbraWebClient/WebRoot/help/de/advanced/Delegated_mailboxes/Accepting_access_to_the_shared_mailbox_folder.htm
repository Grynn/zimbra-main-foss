Zugriff auf einen freigegebenen Mailbox-Ordner annehmen

Wenn Sie per Mail informiert werden, dass Ihnen der Zugriff auf den Mail-Ordner einer anderen Person gewährt wird, können Sie diese Freigabe annehmen oder ablehnen.

Wenn Sie die Freigabe annehmen, wird der freigegebene Mail-Ordner in der Übersicht in der Liste Ihrer Ordner angezeigt.

So nehmen Sie an

  1. Klicken Sie in der Mail auf Freigabe annehmen. Das Dialogfeld „Freigabe annehmen“ wird geöffnet. Es beschreibt die Ihnen zugeordnete Rolle.

  2. Bevor Sie annehmen, können Sie den Namen des Mail-Ordners ändern und eine Farbe für den Ordner auswählen.

  3. (Optional) Geben Sie an, ob Sie eine Bestätigung an den freigebenden Nutzer senden möchten.

  4. Klicken Sie auf Ja. Der Mail-Ordner wird in die Liste Ihrer Ordner aufgenommen. Die Mail wird in Ihren Papierkorb verschoben.
