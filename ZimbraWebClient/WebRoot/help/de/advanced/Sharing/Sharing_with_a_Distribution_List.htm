}}}Über eine Verteilerliste freigeben{{{

}}}Anstatt einzelne Freigabeanforderungen zu erstellen, können Sie Objekte auch über eine Verteilerliste freigeben.{{{ }}}Ihr Systemadministrator richtet Verteilerlisten ein und wenn Sie für diese Liste ein Objekt freigeben, benachrichtigen Sie den Administrator.{{{ }}}Der Administrator veröffentlicht das freigegebene Objekt in der Liste.{{{

}}}Wenn ein neues freigegebenes Objekt veröffentlicht wird, werden die bestehenden Mitglieder der Liste automatisch über das neue freigegebene Objekt benachrichtigt.{{{ }}}Jeder auf der Verteilerliste (DL) verfügt über dieselben Freigaberechte.{{{

}}}Wenn der Verteilerliste neue Mitglieder hinzugefügt werden, können Sie automatisch über die Objekte, die für die Liste freigegeben werden, benachrichtigt werden.{{{ }}}Ihnen werden dieselben Rechte zugestanden wie den anderen Mitgliedern der Verteilerliste.{{{ }}}Wenn ein Mitglied aus der Verteilerliste entfernt wird, werden die Freigaberechte widerrufen.{{{
