Übersichtsfenster anzeigen

Auf der linken Seite der Benutzeroberfläche Ihres Zimbra-Web-Clients finden Sie Ihre Ordner, Suchvorgänge, Tags und Zimlets (im erweiterten Web-Client).

 

Ordner

  • Im Posteingang werden standardmäßig die neuen Mails abgelegt.

  • Unter Gesendet finden Sie die Mails, die Sie gesendet haben.

  • In Entwürfe finden Sie verfasste Mails, die noch nicht gesendet wurden.

  • Spam enthält Mails, die von Ihnen oder vom System als Spam markiert wurden.

  • Trash enthält Objekte aus

  • Lokalen Ordnern, die Sie gelöscht haben. Dieser Ordner ist nur für ZWC-Accounts verfügbar.

  • Im Anschluss finden Sie die von Ihnen erstellten Ordner. Wenn Ihr Mail-Account aus einem anderen Mail-System übernommen wurde, z. B. von Microsoft® Outlook, gibt es hier möglicherweise weitere Ordner, die aus Ihrem Vorgängersystem stammen.

Anzahl ungelesener Mails. Die neben jedem Ordner in Klammern angezeigte Zahl ist die Anzahl ungelesener Mails in dem jeweiligen Ordner. Auch Tag-Ordner zeigen die Anzahl ungelesener Mails an. Alle Objekte, die ungelesene Mails enthalten, werden fett dargestellt.

Um die Anzahl der Mails und die Gesamtgröße eines Ordners zu ermitteln, positionieren Sie die Maus auf einem Ordner.

.

Suchvorgänge

Dieser Bereich enthält Suchvorgänge, die Sie erstellt und zur weiteren Verwendung gespeichert haben.

Tags

Dieser Bereich zeigt die von Ihnen erstellten Tags an. Klicken Sie auf einen Tag, um alle Mails mit diesem Tag anzuzeigen. Weitere Informationen hierzu finden Sie unter Mails mit Tags klassifizieren.

Zimlets

Zimlets dienen dazu, Daten und Inhalte von Drittanbietern in die Funktionen von @ClientNameLong@ zu integrieren. Über Zimlets können Sie verschiedene Arten von Inhalten in Ihren Mails handhaben. Je nach der Konfiguration, mit der Sie arbeiten, haben Sie im Übersichtsfenster möglicherweise keinen Zugriff auf Zimlets.
