Durch @ClientNameLong@ navigieren

Wenn Sie sich beim @ClientNameLong@-Client anmelden, wird eine Ansicht ähnlich der folgenden dargestellt.

Eine Seite des @ClientNameLong@-Clients umfasst die folgenden Bereiche:

  1. Suchleiste. In diesem Bereich werden Suche und erweiterte Suche angezeigt. Auf der rechten Seite finden Sie Links für die Hilfe und zum Abmelden.

  2. Registerkarten der Anwendungen: Dieser Bereich umfasst den Nutzernamen, mit dem Sie sich angemeldet haben, sowie Ihre Mailbox-Kapazität. Die Anwendungen, die Sie aufrufen können, sind auf den folgenden Registerkarten aufgeführt.

  3. Symbolleiste: Sie zeigt Befehle für die aktuelle Anwendung an. In diesem Beispiel wird die Symbolleiste der Anwendung Mail angezeigt.

  1. Übersichtsfenster: Hier finden Sie Ihre Ordner, auch die Systemordner wie Posteingang, Gesendet, Entwürfe, Spam und Papierkorb sowie alle von Ihnen angelegten Ordner, Suchvorgänge, die Sie gespeichert haben, von Ihnen erstellte Tags und Zimlet-Links, die für die Integration von Fremdanwendungen in Ihre Mailbox erstellt wurden. In der Anwendung "Kalender" zeigt das Übersichtsfenster Ihre Kalenderliste und Zimlets an.

  2. Unterhalb des Übersichtsfensters kann ein Mini-Kalender angezeigt werden. Die Anzeige des Mini-Kalenders ist optional und kann unter Kalender-Einstellungen aktiviert werden.

  3. Inhaltsfenster: Der Inhalt in diesem Bereich richtet sich nach der jeweils aufgerufenen Anwendung. Im Posteingang zeigt das Inhaltsfenster alle Mails und Unterhaltungen Ihres Posteingangs an.

 

Hinweis: Sie können Ihre Ansicht auch zu einer Drei-Fenster-Ansicht ändern, mit dem Lesefenster auf der rechten Seite, oder Sie können das Lesefenster vollständig aus der Ansicht in der Mail-Symbolleiste ausblenden.
