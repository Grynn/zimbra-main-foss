Suchabfrage erstellen und speichern

}}}Wenn Sie häufig dieselbe Suchabfrage durchführen, können Sie eine Suchabfrage erstellen und diese in Ihrem Suchordner speichern.{{{ }}}Wenn Sie die Suche erneut durchführen möchten, klicken Sie einfach auf den Ordner mit den gespeicherten Suchvorgängen, um die Suche zu starten. Die Ergebnisse werden im Inhaltsfenster angezeigt.{{{

So erstellen und speichern Sie eine Suchabfrage

  1. }}}Klicken Sie unter Übersichtsfenster > Suche auf Bearbeiten.{{{ }}}Das Übersichtsfenster wird aktualisiert und zeigt alle Ihre Ordner in alphabetischer Reihenfolge an. Im Inhaltsfenster wird der Posteingang angezeigt.{{{

  2. }}}Klicken Sie in der Symbolleiste auf Neue Suche.{{{ Im Inhaltsfenster wird das Formular für eine neue Suche angezeigt.

  3. }}}Geben Sie im Feld Name einen Namen für die Suche ein.{{{

  4. }}}Im Feld Übergeordneter Ordner wählen Sie aus, wo diese Suche gespeichert werden soll.{{{ }}}Standardmäßig wird der Stammordner angezeigt.{{{ }}}Der Ordner wird im Übersichtsfenster unter der Überschrift „Suche“ gespeichert.{{{ }}}Wenn Sie die Suche in einem anderen Ordner speichern möchten, wählen Sie den Ordner in der Dropdown-Liste aus.{{{ }}}Sie können den Suchordner anhand des Symbols mit dem Vergrößerungsglas erkennen.{{{

  5. }}}Erstellen Sie im Feld Suchabfrage Ihre Suchabfrage.{{{ }}}Informationen zur zulässigen Suchgrammatik finden Sie unter Beschreibung der Sprache für Suchabfragen.{{{

  6. }}}Geben Sie im Dialogfeld Suche speichern einen Namen für den Ordner der gespeicherten Suche ein und wählen Sie aus, wo der Suchordner abgelegt werden soll.{{{ }}}Wenn Sie den Ordner im Stammordner speichern, wird die Suche unter „Suchen“ angezeigt.{{{ }}}Wenn Sie einen anderen Ordner auswählen, wird die Suche in dem Ordner gespeichert.{{{

  7. }}}Klicken Sie auf OK.{{{

Zum Durchführen Ihrer Suche klicken Sie im Übersichtsfenster auf den Suchordner. Die Suchergebnisse werden im Inhaltsfenster angezeigt.
