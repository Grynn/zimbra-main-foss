Suche durchführen

Die Zimbra-Suche ist eine leistungsstarke Funktion. Sie können zwar lernen, wie Sie komplexe Textabfragen eingeben (dies wird beschrieben unter Beschreibung der Sprache für Suchabfragen), doch es ist viel schneller, einfach einen Namen oder ein Wort ohne Schlüsselwörter einzugeben. Wenn Sie z. B. alle Mails finden möchten, die das Wort "tim" in einer der Zeilen Betreff, An, Von, CC oder BCC, im Textkörper oder im Dateianhang enthalten, geben Sie einfach tim in das Suchtextfeld ein und klicken auf Suchen.

Sie geben an, wo gesucht werden soll. Sie können in Mails, persönlichen Kontakten, persönlichen Kalendern und persönlichen Aufgabenlisten suchen.

Sie können jeden Wert, z. B. ein Wort, einen Vornamen, Nachnamen, eine Telefonnummer oder einen Domainnamen eingeben. Zur größeren Flexibilität nutzen Sie das Sternchen * als Platzhalter.  

Platzhalter * bei der Suche nutzen

Das Sternchen * kann in einer Suche als Platzhalter eingesetzt werden, um Inhalte mit Wörtern ähnlicher Schreibweise zu finden.

Nutzen Sie das Sternchen * als Platzhalter im Anschluss an eine bestimmte Zeichenfolge. Beispiel: Die Suche nach Hu* findet Objekte mit den Wörtern Hund, Hut usw.

Um nach Kontakten in Ihrem Adressbuch zu suchen, müssen Sie für die Abfrage ein vollständiges Wort eingeben. Um z. B. die Kontakte nach der Telefonnummer zu durchsuchen, müssen Sie die vollständige Telefonnummer so eingeben, wie sie im Kontakteintrag steht. Sie können nicht z. B. nur nach der Vorwahl suchen.

So suchen Sie nach Kontakten in Ihrem Adressbuch

  1. Wählen Sie im Textfeld „Suchen“ in der Dropdown-Liste die Option Persönliche Kontakte aus. Damit werden alle Ihre Adressbücher durchsucht.

  2. Geben Sie einen Namen oder andere Kontaktdaten in das Textfeld Suchen ein.

  3. Klicken Sie auf Suchen, um die Suche durchzuführen.

So suchen Sie rasch nach Mails

  1. Wählen Sie im Textfeld „Suchen“ in der Dropdown-Liste Mails aus.

  2. Geben Sie ein Wort, einen Namen oder andere mailspezifische Daten in das Suchtextfeld ein.

  3. Klicken Sie auf Suchen, um die Suche durchzuführen.

Komplexe Suchabfrage erstellen

Wenn Sie eine Suchabfrage häufig durchführen oder nach bestimmten, ausführlichen Informationen suchen, können Sie eine Suchabfrage erstellen und diese in Ihrem Suchordner speichern. Wenn Sie die Suche erneut durchführen möchten, klicken Sie einfach auf die Suche.

So erstellen und speichern Sie eine Suchabfrage

  1. Klicken Sie in der Suchübersicht auf Bearbeiten. Es wird das Fenster Neue Suche erstellen angezeigt.

  2. Wählen Sie aus, wo der Suchordner abgelegt werden soll. Wenn Sie den Ordner im Stammordner speichern, wird die Suche unter „Suchen“ angezeigt. Wenn Sie einen anderen Ordner auswählen, wird die Suche in dem Ordner gespeichert.

  3. Geben Sie im Feld Name für die Suche den Namen für die Suche ein.

  4. Geben Sie im Feld Suchabfrage Ihre Abfrage mithilfe der unter Beschreibung der Sprache für Suchabfragen beschriebenen Sprache für Suchabfragen ein. Sie können auch die Operatoren AND oder OR in Ihrer Abfrage nutzen.

    Eine Suchabfrage mit AND oder OR eingeben

    Bei Suchabfragen mit mehreren Kriterien können Sie nach Elementen suchen, die einem angegebenen Kriterium oder allen angegebenen Kriterien entsprechen:

    • Wenn Sie nach Mails suchen, die einem der angegebenen Kriterien entsprechen, handelt es sich um eine OR-Suche (ODER), weil die Kriterien als erfüllt gelten, wenn die Mail entweder X oder Y enthält.

    • Wenn Sie nach Mails suchen, die X und Y enthalten, handelt es sich um eine AND-Suche (UND), weil die Mail alle angegebenen Kriterien erfüllen muss, um als Entsprechung zu gelten.

  5. Klicken Sie auf Suche erstellen, um die Suche zu erstellen und sie im angegebenen Ordner zu speichern.

Zum Durchführen Ihrer Suche klicken Sie im Übersichtsfenster auf den Namen der Suche. Die Suchergebnisse werden im Inhaltsfenster angezeigt.
