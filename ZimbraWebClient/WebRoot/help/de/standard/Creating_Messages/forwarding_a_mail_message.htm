Mail weiterleiten

Mit der Funktion Weiterleiten können Sie die Mail an einen anderen Nutzer weiterleiten. Beim Weiterleiten einer Mail werden alle Dateianhänge weitergeleitet, während beim Antworten nur der Textkörper erhalten bleibt, die Anhänge jedoch nicht.

Die Art und Weise, wie Ihre Mail-Nachricht weitergeleitet wird, können Sie unter "Einstellungen Verfassen" einrichten. Sie können festlegen, ob der Originaltext, die Kopfzeile "An/Von" und die Betreffzeile der Nachricht in Ihre weitergeleitete Mail eingefügt werden sollen. Alternativ können Sie auch festlegen, dass die Original-Nachricht als Anhang an Ihre Mail gehängt wird.

So leiten Sie eine Mail weiter

    1. Öffnen Sie die Mail, die Sie weiterleiten möchten.

    2. Klicken Sie in der Leiste oberhalb des Mail-Felds auf "Weiterleiten". . Ein Fenster zum Verfassen von Mails wird geöffnet. Der Text der weitergeleiteten Mail wird im Textkörper angezeigt.   

    3. Geben Sie die Adressen der Empfänger in die Felder An:, Cc: oder Bcc: ein.

    4. Geben Sie, wenn Sie möchten, einen kurzen Text im oberen Bereich ein, damit der Empfänger weiß, warum Sie die Mail weiterleiten.

    5. Klicken Sie auf Senden.
