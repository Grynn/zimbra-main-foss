Aufgaben ändern

So aktualisieren Sie eine Aufgabe

  1. Öffnen Sie die Aufgabenliste, die Ihre Aufgabe enthält.

  2. Klicken Sie im Fenster Inhalt auf die Aufgabe, die Sie ändern möchten. Daraufhin wird die Seite „Aufgabe“ angezeigt.

  3. Ändern Sie die Daten zu dieser Aufgabe.

  4. Klicken Sie auf Speichern, wenn Sie alle Änderungen eingegeben haben.

So markieren Sie eine Aufgabe als erledigt

Wenn Sie eine Aufgabe fertigstellen, aktivieren Sie im Inhaltsfenster neben der Aufgabe das Kontrollkästchen, um die Aufgabe umgehend als erledigt zu markieren. Der Prozentsatz der Fertigstellung wird nun als 100% angegeben.
