Mail-Systemordner

Die Systemordner sind Posteingang, Gesendet, Entwürfe, Spam und Papierkorb. Diese Ordner können Sie nicht verschieben, umbenennen oder löschen. Systemordner werden immer an oberster Stelle in Ihrer Ordnerliste angezeigt. Darunter folgen die von Ihnen angelegten Ordner.

Posteingang

Neue Mails werden im Posteingang abgelegt. Standardmäßig wird nach der Anmeldung der Posteingang angezeigt.

Gesendet

Eine Kopie der von Ihnen gesendeten Mails wird im Ordner Gesendet gespeichert.

Entwürfe

Mails, die Sie verfasst, aber noch nicht gesendet haben, können Sie im Ordner Entwürfe speichern. Wenn Sie eine Mail im Ordner "Entwürfe" öffnen, wird sie in der Ansicht zum Verfassen aufgerufen.

Spam

Das Filtern unerwünschter, automatisch versendeter Mails (werden auch "Spam-" oder "Junk-Mails" genannt) wird von einem Spam-Filter durchgeführt, bevor diese Mails Ihren Posteingang erreichen. Mails, bei denen es sich möglicherweise um Spam-Mails handelt, möglicherweise aber auch nicht, werden in Ihrem Spam-Ordner abgelegt. Sie können diese Mails überprüfen und sie entweder in einen anderen Ordner verschieben oder löschen. Wenn Sie die Mails nicht löschen, werden sie nach einem von Ihrem Mail-Administrator festgelegten Zeitraum automatisch gelöscht.

Hinweis:  Wenn Sie eine große Zahl unerwünschter Mails erhalten, wenden Sie sich an Ihren Mail-Systemadministrator. Indem Sie Spam melden, helfen Sie Ihrem Mail-Administrator, den Spam-Filter Ihrer Firma präziser einzustellen.

Wenn Sie eine Mail vorfinden, die eindeutig nicht relevant ist, können Sie sie auswählen und dann im Feld Weitere Aktionen die Option „Spam“ auswählen.

Papierkorb

Gelöschte Mails werden im Ordner Papierkorb abgelegt und bleiben dort vorhanden, bis Sie den Papierkorb manuell leeren oder der Papierkorb automatisch gelöscht wird. ails werden nach einem vom Administrator festgelegten Zeitraum automatisch gelöscht.
