Mails und Kontakte mit Tags versehen

Tags sind Ihr persönliches Klassifizierungssystem für Mails, Kontakte und Termine. Tags helfen Ihnen, diese Objekte zu finden. Sie können beispielsweise einen Tag „Sofort erledigen“ einrichten und einen anderen „Mittlere Priorität“. Die mit Tag markierten Mails können sich in unterschiedlichen Ordnern befinden. Mithilfe von Tags haben Sie die Möglichkeit, alle Mails mit einem bestimmten Tag anzuzeigen, ganz gleich, wo diese abgelegt sind.

Sie können mehrere Tags erstellen und einem einzigen Objekt zuordnen, um es in mehrere Kategorien gleichzeitig einzuordnen. Nachdem Sie einen Tag erstellt haben, können Sie damit Unterhaltungen, Mails und Kontakte markieren.

So erstellen Sie einen neuen Tag

  1. Klicken Sie in der Tagübersicht auf Bearbeiten.

  2. Klicken Sie im Inhaltsfenster auf Neuer Tag.

  3. Geben Sie einen Namen für den Tag ein und wählen Sie eine Farbe aus.

  4. Klicken Sie auf Neuen Tag erstellen. Der neue Tag wird in der Tagübersicht aufgeführt.

So versehen Sie mehrere Objekte gleichzeitig mit Tags bzw. entfernen Tags

  1. Aktivieren Sie die Kontrollkästchen der Objekte, die Sie mit einem Tag markieren möchten.

  2. Wählen Sie im Pulldown-Menü Weitere Aktionen den gewünschten Tag aus.

  3. Klicken Sie auf Los.

So löschen Sie einen Tag von mehreren Objekten

  1. Aktivieren Sie die Kontrollkästchen der Objekte, die mit dem zu löschenden Tag markiert sind.

  2. Wählen Sie im Pulldown-Menü Weitere Aktionen unter „Tag entfernen“ den gewünschten Tag aus.

  3. Klicken Sie auf Los.
