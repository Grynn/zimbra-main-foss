Dateianhänge öffnen

Mails können angehängte Dateien enthalten, z. B. Textverarbeitungsdateien, Arbeitsblätter aus Tabellenkalkulationen, Textdateien, ZIP-Dateien, Bilder, ausführbare Anwendungen oder andere Arten von Computerdateien, die sich im Computernetzwerk des Absenders befinden. In Mails mit Anhängen wird eine Büroklammer neben dem Betreff angezeigt.

}}}Einige Arten von Dateien, wie JPEG-Bilder oder Textdateien, können auf nahezu jedem Computersystem angezeigt werden.{{{ }}}Andere, wie Microsoft-Word-Dateien, müssen mit derselben Anwendung geöffnet werden, in der sie auch erstellt wurden.{{{

Wenn Sie eine Mail mit Anhängen erhalten, können Sie in der Kopfzeile auf den Link zum Anhang klicken, um zum Ende der Mail zu gelangen. Am Ende der Mail können Sie auswählen, ob Sie den Anhang anzeigen, den Anhang in HTML anzeigen oder den Anhang herunterladen möchten.

  • Wenn Sie den Anhang anzeigen möchten, müssen Sie auf Ihrer Workstation eine Anwendung haben, die diesen Dateityp lesen kann.

  • Wenn Sie angeben, dass Sie den Anhang herunterladen möchten, wird ein Dialogfeld zum Speichern angezeigt. Wählen Sie den gewünschten Speicherort aus und speichern Sie die Datei. Sie können die Datei zu einem späteren Zeitpunkt außerhalb des Mail-Clients öffnen.

Anhänge in HTML anzeigen

Möglicherweise verfügen Sie über die Funktion, Anhänge in HTML anzuzeigen. Dadurch können Sie einen Anhang öffnen, auch wenn die erforderliche Anwendung nicht installiert ist. }}}Klicken Sie auf Als HTML anzeigen, um den Inhalt der Datei anzuzeigen.{{{ Diese Funktion wird von Ihrem Systemadministrator eingerichtet.

Sicherheit vor Viren

Computerviren können sich über Mailanhänge verbreiten. Im Allgemeinen implementieren Mail-Administratoren Virenfilter für eingehende Mails sowie Virenprüfungen für PCs. Es besteht jedoch immer die Möglichkeit, dass sich trotzdem ein Virus verbreitet.

Mailviren werden größtenteils dadurch aktiviert, dass ein Dateianhang, der einen Virus enthält, geöffnet oder angezeigt wird. Von vielen dieser Viren ist bekannt, dass sie sich über die Einträge in den Adressbüchern der Nutzer verbreiten. Wir empfehlen Ihnen deshalb, beim Öffnen von Dateianhängen vorsichtig vorzugehen, auch wenn Sie den Absender scheinbar kennen.
