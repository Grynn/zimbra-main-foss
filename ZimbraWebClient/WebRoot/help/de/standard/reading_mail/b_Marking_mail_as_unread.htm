Mails als ungelesen markieren

Mails, die Sie geöffnet haben, werden als gelesen markiert und nicht mehr fett dargestellt.  .

So markieren Sie eine Mail als ungelesen

  1. Klicken Sie auf das Kontrollkästchen der Mail, die Sie als ungelesen markieren möchten.

  2. Wählen Sie im Pulldown-Menü Weitere Aktionen die Option Als ungelesen markieren.

  3. Klicken Sie auf Los. Die ausgewählte Mail wird fett dargestellt und das Häkchen wird entfernt.

Hinweis: Wenn Sie alle Mails in einer Unterhaltung als ungelesen markieren möchten, wählen Sie die Unterhaltung aus. Wenn Sie eine einzelne Mail in einer Unterhaltung als ungelesen markieren möchten, öffnen Sie die Unterhaltung und wählen die Mail aus.

Auf die gleiche Weise können Sie ungelesene Mails als gelesen markieren.
