開啟附件

郵件中可含附件，例如文字處理文件、試算表、文字檔、ZIP 檔、圖片、可執行應用程式、以及儲存在寄件人電腦網絡中的任何其他類型的電腦檔案。含附件的郵件通常會在主題旁邊後顯示迴形針圖示。

有些類型的檔案，如 JPEG 影像檔或文字檔，幾乎可在任何電腦系統上開啟。其他檔案，如Microsoft Word 文件，則必須使用建立時使用的程式才能開啟。

在收到含附件的郵件時，可按兩下郵件標題處的附件部分。一個對話方格隨即出現，詢問是打開還是儲存該檔案。

  • 如果選擇查看附件，工作站上則必須裝有可讀取該類型檔案的應用程式。

  • 如果選擇下載附件，則會顯示一個瀏覽對話方格，並顯示目前的檔案系統。選擇一個合適的位置並儲存檔案。此後，你便可在郵件用戶端開啟該檔案。

以 HTML 查看附件

你可能具備以 HTML 格式查看附件的功能。這樣，即便你沒有安裝相應的應用程式也可打開附件。  按下以 html格式查看便可查看檔案的內容。此功能由系統管理員設定。

防範病毒

電腦病毒可透過郵件附件傳播。郵件系統管理員可按標準作法對郵件進行病毒過濾，並對個人電腦進行病毒掃描。但是，總還是有一些病毒漏網。

這種郵件病毒，大多數是在用戶打開或查看附件（含病毒者）時發作的。眾所週知，這類病毒許多是利用用戶通訊錄上的條目進行傳播的。故建議用戶在開啟附件時要格外小心，即便寄件人是你認識的人。
