Zimbra Collaboration Suite 開源夥伴

 

歡迎使用 Zimbra™ Web Cient, Advanced Edition

Zimbra Collaboration Suite 是一功能齊全的通信及協作辦公應用程式，提供可靠和高效能的電郵、通訊錄、記事簿、事項清單以及網絡文件製作功能。  

  • 你的帳戶有可能不包括上述全部功能。若需瞭解你的帳戶都啟用了哪些功能，請與你的系統管理員聯繫。

從 Zimbra 網絡用戶端 (ZWC)，你可使用下列功能。

電子郵件

  • 寫信和傳送郵件

  • 讀取和回覆郵件

  • 透過對話功能追蹤來往郵件，也可轉換到傳統郵件視窗

  • 在郵件中加入附件

  • 將郵件轉寄給一個或多個收件人

  • 透過特徵或指定的文字搜尋郵件和附件

  • 建立你自己的資料夾用以整理郵件

  • 建立標記用以整理郵件

  • 建立過濾器以便將來信匯入指定的資料夾

  • 設定不同的帳戶身份和地址，用以管理你在工作和個人生活中的不同角色。

  • 設定你的帳戶，以便從 ZWC 郵箱直接收取 POP3 或 IMAP 帳戶的郵件。

  • 授權他人查看或管理你的郵件夾

通訊錄

  • 建立和管理多個通訊錄

  • 匯入和匯出聯絡人清單

  • 分享個人通訊錄

記事簿

  • 建立和管理多個記事簿

  • 建立預約、會議和事件

  • 授權他人查看或管理你的記事簿

  • 查看參會者日程的空閒/忙碌狀態

  • 匯入/匯出其他記事簿程式。

事項

  • 建立多個「事項清單」

  • 建立待辦事項

  • 將附件加入事項

  • 管理事項、設定優先級和追蹤進度

  • 授權他人查看或管理你的「事項清單」

文件

  • 建立多個記事本，用以整理你的網絡文件

  • 使用 HTML 編輯器設計頁面，建立表格和試算表，加入圖片，建立網頁連結

  • 授權給所在組織機構中的群組或個人建立和編輯你的文件記事本

  • 允許公眾查看你的文件記事本

公事包

  • 上傳任何類型的文件到你的公事包，以便隨時登入 ZWC 存取這些檔案

  • 建立資料夾，用以整理上傳的檔案

  • 授權他人查看或管理你的「公事包」資料夾

即時通信 (IM) Beta

  • 即時通信

  • 建立聊天群組或與好友聊天

  • 建立好友名單並查看他們的線上狀態

  • 選擇你的線上狀態或建立自訂狀態訊息

  • 用電子郵件快速傳送你的聊天內容

  • 聊天內容自動儲存到「聊天」資料夾

偏好設定

  • 設定郵箱的各項功能，使之與郵件、通訊錄和記事簿配合使用

  • 建立郵件過濾器

  • 設定快速鍵，用於快速存取郵件夾、儲存的搜尋項和標記
