使用QuickAdd建立預約

QuickAdd 對話方格可用來快速建立預約。在記事簿上，先選擇開始時間，按兩下或拖曳時間，「快速加入預約」對話方格隨即出現。

QuickAdd 功能是一個選項。如果你習慣以按兩下或在記事簿上拖曳方式開啟「預約」頁面，請在「選項」頁面關閉此功能。

若需快速建立一個預約

  1. 在「記事簿」上，按下預約的時間 或按下預約的開始時間並拖曳至結束時間。 快速加入預約對話方格隨即顯示。

  2. 若需從「日」或「週」視窗中建立全天預約，請在記事簿的頂端，按下並拖曳劃過一天或幾天。

  3. 輸入主題、地點，如果是重複預約，選擇重複模式。

  4. 按確定。 該預約即顯示在記事簿上。

若需邀請參會者或改變重複選項，可按下更多詳情，以打開「預約」頁面。
