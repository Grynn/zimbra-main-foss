用戶偏好設定

預設的用戶偏好設定是在建立你的帳戶時設定的。 這些選項界定了你的郵箱、通訊錄和記事簿應用程式的使用方法。你還可以設定個人郵件 ID和 POP 帳戶，並為你的郵箱選擇一個基調主題。

你可在偏好設定索引標籤中改變這些設定。如果此處描述的選項在你的視窗中看不到，則說明該選項對於你的帳戶不適用。

你可在下列索引標籤中設定偏好設定。按下相關字詞後，可獲得相關索引標籤的詳盡說明。

  • 一般

    • 你登入的是進階版網絡用戶端。要將此設定為登入時使用的預設用戶端類型，請選擇進階版。

    • 進階版具有全套的網上協同辦公功能。此種網絡用戶端最適合於較新版的瀏覽器和更快的互聯網連線速度。

    • 如果互聯網連接速度較慢、所用瀏覽器為早期版本或為了存取更簡易，可採用標準版。

    • 在主題處選擇郵箱的背景色。

    • 如果此處顯示的時區不正確，請為你的帳戶選擇預設時區。時區功能可確定你傳送和接收郵件的正確日期和時間，並正確顯示你「記事簿」中的預約時間。  這可能與所用電腦上所設置的時區不同。

    • 如果顯示變更密碼，你可修改你的密碼。如果在變更密碼時遇到問題，請向你的管理員詢問密碼規則。

      • 如果沒有此選項，但你需要變更密碼，請與系統管理員聯繫。

    • 在搜尋設定中，你可選擇自動在每次執行搜尋時包含雜件箱和垃圾桶。預設情況下，這兩個資料夾是不搜尋的。

    • 在搜尋語言中，剔選總顯示搜尋字串以便在搜尋字框中顯示搜尋字串的細節。

    • 啟用總顯示選項核取方格  ，以便「內容」窗格中的每個條目都顯示一個核取方格。透過此核取方格，你可以選擇一項或多項同時執行形同的操作，例如刪除、移動、標記為已讀/未讀。

  • 郵件

    • 選擇每頁顯示多少條目（郵件或對話）。預設值為每頁顯示 25 個條目，但你可選擇 10、50 或 100。

    • 選擇郵件如何分組，依對話或依郵件。如果有「對話」功能，則「對話」是預設的。

    • 輪詢間隔是指你的電腦檢查新郵件的頻率。 預設間隔時間為 5 分鐘。最短的時間長度由 Zimbra 管理員設定。

    • 對於已經格式化為 HTML 的郵件，可指定是否以 HTML 方式查看。若關閉此項，郵件則以純文字形式顯示，即只顯示 HTML 標記，但不顯示標記語言版本。

    • 郵件預覽選項。

      • 用於設定查看郵件時是否顯示讀取窗格。預設值為總顯示讀取窗格。

      • 若啟動此項，用戶端會在郵件清單中顯示郵件片段。當游標經過郵件主題時，郵件開頭的部分文字會顯示出來。

      • 若啟用此項，按兩下後可在新視窗中開啟郵件

    • 啟用圖片後，HTML 郵件便可自動下載圖片。若啟用，你無需按下「顯示圖片郵件」就可以看見圖片。

    • 預設的「郵件搜尋」為登入時執行搜尋。預設的搜尋目標是你的收件箱。  在你登入時，「收件箱」的搜尋結果會即刻顯示。 你可修改此欄位，以使在剛開啟 Zimbra 網絡用戶端時顯示的是其他的資料夾、標記或儲存的搜尋。有關句法，請見查詢語言說明。啟用並指定一個地址，以便系統傳送關於收到新郵件的通知。

    • 指定一個郵件轉寄地址。你可指定將原始郵件從 Zimbra 郵箱中刪除。

    • 指定是否對收到的郵件傳送一個自動回覆訊息。 自動回覆訊息經常被用作外出不在訊息，或休假訊息，以表明你不在辦公室。

    • 設定「我傳送的郵件」部分，用以處理以下兩種情況：你傳送的郵件，但你自己的地址是郵件的收件人之一；或所含地址為包括你自己在內的郵件清單。

     

  • 寫信

    • 設定預設的文字編輯器，HTML 或純文字。

    • 設定以 HTML 格式撰寫郵件、文件和記事簿時所使用的預設字型。 你可確定字型的樣式、大小和顏色，用以作為預設的樣式。 預設字型為 Arial、12 point black。

    • 總在新視窗中撰寫，這一選項可使你在撰寫郵件時開啟一個與 Zimbra 郵箱分開的新寫信視窗。在寫信視窗開啟的情況下，你可查看和瀏覽你的郵箱。

    • 每隔幾分鐘，你所撰寫的郵件都會自動儲存到草稿箱。如果不需要此自動儲存功能，可關閉此功能。

    • 選擇是否在你的回覆郵件正文中包含原始郵件。你可選擇包含原始郵件，不包含原始郵件，以附件形式包含原始郵件，在正文中包含但加上你選的字首，或只包含最近的郵件。 （也就是說，只引用寄件人寫下的內容，而不是前面的文字。）

    • 如果選擇引用原始文字，請指定是否需要為轉寄或回覆的前一封郵件的每一行加上字首 > 或 |。

    • 預設情況下，在寄件箱儲存副本為啟用狀態。傳送郵件的副本在寄件箱中儲存。

    • 指定在轉寄郵件時如何包含郵件的原始文字：放入郵件正文、加上字首放入郵件正文或作為附件一併轉寄。

  • 署名

    你可為你的郵件建立各種署名。署名中可包括姓名和其他結束語句，帳戶設定中對語句的長短和字元數量上限均有限制。如果你有多個身份，則可建立不同的署名，並分別指派給具體的地址。

    • 輸入你的署名。

    • 指定署名在郵件中的位置：將署名加入郵件的最下面，或加入你撰寫、回覆或轉寄的文字的結束處。

  • 通訊錄

    • 啟用此功能後，你可在傳送郵件時自動將地址加入通訊錄。如果啟動此功能，地址將儲存到你的「郵件聯絡人」資料夾中。

    • 啟用後，可在自動完成地址時使用「全網域地址清單」。

    • 選擇以何種形式查看聯絡人，清單或名片。

    • 選擇每頁顯示的聯絡人的人數。預設值為 25。

    • 匯入聯絡人到你的「聯絡人」清單。匯入的檔案必須是逗號分隔格式。

    • 匯出聯絡人清單。匯出的檔案採用逗號分隔格式。

     

  • 帳戶

    帳戶索引標籤用於建立和管理你的身份。身份是一種郵件 ID，在你傳送郵件的「寄件人」欄位中使用。

    預設的樣式在主身份中定義。在建立新的身份時，可使用與「預設身份」相同的設定，也可設定不同的偏好設定。

    加入外部帳戶按鈕用於識別你的外部 POP/IMAP 帳戶，從而使網絡用戶端能從你的其他帳戶收取郵件。 你可透過設定，使回信也使用相應的 POP/IMAP 地址。

  • 郵件過濾器

    使用「郵件過濾器」索引標籤定義郵件過濾規則。若需瞭解如何設定郵件過濾規則，請參見過濾工作原理。

  • 記事簿

    • 用於選擇如何查看 Zimbra 記事簿。預設值為按照工作週。

    • 指定在記事簿中把哪一天作為一週的第一天。

    • 如果經常往返於不同的時區，應該選擇在預約視窗中顯示時區清單。

    • 若需建立許多沒有參會者的預約，請剔選使用「快速加入」對話方格。

    • 「記事簿」中會顯示一個小日曆。 你可在任何視窗中啟用小日曆。

    • 設定在預約前多少分鐘進行提醒。預設值為五分鐘。

    • 匯入或匯出其他記事簿程式。

  • 即時通信

    你可設定如何將即時通信（IM）整合到你的帳戶。

    • 登入。 如果你需要自動登入即時通信服務，須啟用此項。

    • 通知。 此選項可用來設定如何接收IM通知，以及你希望別人如何看見你的線上狀態。

    • 聊天。 你的聊天記錄會自動儲存到「聊天」資料夾。此資料夾中的郵件與任何電子郵件一樣，可以列印、作為郵件轉寄或從資料夾中刪除。你還可以設定空閒愈時。

  • 快速鍵

    快速鍵可使你在使用應用程式時不用滑鼠。有大量的快速鍵可用。你也可以建立自訂的快速鍵，方法是：將數字別名指派給資料夾、標記和儲存的搜尋。

    • 快速鍵清單顯示了全部現有的快速鍵。

    • 郵件夾快速鍵索引標籤用於建立能快速前往你的郵件夾並將郵件移至該資料夾的快速鍵。你須選擇一個資料夾，然後便可指定一個數字作為快捷方式。例如，將 3 指派給「收件箱」：在郵箱的任何地方，輸入v3可使你轉移到「收件箱」。選擇一個條目然後輸入 .3，將把該條目移至「收件箱」。

    • 已存搜尋項快捷方式索引標籤用於建立一個快速鍵，用以快速使用已儲存的搜尋項。你須選擇一個儲存的搜尋項，然後便可指定一個數字作為快捷方式。例如，如果把 3 指派給你常用的一個搜尋項，則從你的郵箱的任何地方，可透過輸入s3快速執行該搜尋。

    • 用標記快速鍵索引標籤建立的快速鍵，可用於快速存取具有指定標記的所有條目，或將一個標記套用於郵件或聯絡人。你須選擇一個標記，然後便可指定一個數字作為快捷方式。例如，將 Follow up 標記賦予數字 3 後，你可從郵箱的任何地方輸入 y3 即可看見所有標記為 Follow up的項目。選擇一個條目並輸入 t3 ，便可將該條目標記為 Follow up。

    你可指定任何數字，但不能在同一快速鍵索引標籤內重複。
