建立新頁面

你可建立新頁面並透過 Rich-text 格式工具或輸入 HTML 索引標籤來設計內容版面。你可檢查拼字，並用關鍵字建立前往其他頁面的連結。

  1. 按下要在其中工作的記事本。目錄頁面隨即在內容窗格中顯示。

  2. 按下工具列中的新建。一個空白頁面隨即開啟。在標題頁面欄位中，輸入新頁面的名稱。名稱隨即在目錄中顯示。

  3. 撰寫頁面內容。你可使用預設的 rich text 編輯器在頁面中加入樣式、顏色、表格和試算表。

如果你熟悉 HTML 並想使用 HTML 索引標籤，按下工具列上的格式，然後選擇 HTML 源碼。頁面隨即變為一空白欄位，此時你就可以使用標準的 HTML 標記語法建立頁面了。

  1. 按下儲存儲存該檔案。你現在可向頁面加入細節，也可按下關閉回到目錄頁面。
