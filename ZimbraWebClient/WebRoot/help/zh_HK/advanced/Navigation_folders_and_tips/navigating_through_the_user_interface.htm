瀏覽 Zimbra Web Client

如下圖所示，登入 Zimbra Web Client 後，你會看見一個全屏視窗。

Zimbra Web Client 頁面由以下區域組成：

  1. 瀏覽器控件。可顯示標準的瀏覽器按鈕，如首頁、目前地址及其他功能，具體內容取決於所用的瀏覽器軟件和設定。瀏覽器工具列上的按鈕不屬於 Zimbra Web Client，你存取的每個網頁都會顯示這些按鈕。

  2. 搜尋工具列、用戶名稱以及限額顯示。搜尋、進階搜尋、你登入時使用的用戶名稱以及你的郵箱容量限額顯示在此區域。

  3. 應用程式索引標籤。可使用的應用程式列在頂端的各個索引標籤中。右邊是常見問題和登出。

  1. 工具列。 顯示目前使用的應用程式的可用操作項。在此例中顯示的是郵件應用程式工具列。

  2. 窗格。顯示你的資料夾，包括系統資料夾收件箱、 寄件箱, 草稿 雜件箱和垃圾桶，以及你所建立的任何自訂資料夾、已儲存的搜尋項和你建立的標記、從郵箱建立的用於與第三方應用程式整合的Zimlet 連結。如果你在「記事簿」中，窗格會顯示你的記事簿清單和 Zimlet。

  3. 在窗格下可顯示一個小日曆。小日曆是一選項，可在偏好設定的記事簿索引標籤中啟用。

  4. 內容窗格。此區域中的內容取決於所使用的應用程式。在「收件箱」視窗中，內容窗格顯示的是收件箱中所有的郵件和對話。
