xxxx用標記分類郵件和聯絡人xxxx

xxxx標記是你個人對郵件、聯絡人、事項和預約的分類系統。xxxx 使用標記有助於尋找相關條目。xxxx例如，你可能有兩個標記，分別是 xxxxxxxxImmediate Turnaroundxxxxxxxx 和 xxxxxxxxMedium Priorityxxxxxxxx。xxxxxxxx被標記的郵件可位於不同的資料夾中。xxxxxxxx你可搜尋某個標記，所有具備該標記的郵件都會顯示出來，不論它們位於哪個資料夾。xxxx

  • xxxx你可以建立許多標記，沒有數量限制。xxxx
