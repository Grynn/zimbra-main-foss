支援的附件類型

用瀏覽按鈕在檔案系統中找到的任何檔案，都可作為附件加入。

在 Zimbra 網絡用戶端，你可直接開啟附件，條件是：所用工作站須有合適的應用程式，並且副檔名沒有被系統管理員封鎖。 例如，要開啟用 Microsoft Word 建立的文件，就需要能夠在你的電腦上執行 Microsoft Word。

對於大多數電腦，檔案類型由其副檔名指定，而副檔名通常就是一個英文句點，後跟三字母代碼。具體例子如expenses.xls 或 New-Items.doc.。如果檔案類型是你電腦上安裝的軟件所支援的類型，通常可以按兩下檔案，你的電腦就會自動啟動相應的應用程式讀取該檔案。

透過各種外挂程式，許多檔案類型可在瀏覽器上直接開啟。通常情況下，當你在頁面上按下一個檔案連結時，瀏覽器會自動檢測是否有適用的外挂程式，必要時提示你下載該外挂程式。請注意，系統管理方面的規定可能不允許終端用戶下載和安裝任何類型的軟件。

  • 備註：
    對於 Windows 電腦而言，機器假定檔案名稱中的檔案類型符合實際格式。即便檔案本身格式正確，但如果檔案名稱中的檔案類型副檔名不對，電腦則不能識別，亦不能使用正確的應用程式。

    越來越多的電腦病毒都是透過附件傳播的。故在某些環境下，系統管理員已採取了一定的措施，阻止有特定類型附件的郵件進入系統，特別是具有 EXE 或 .ZIP 副檔名的檔案。如果將郵件發到這樣的系統，即被設定為阻止某些特定類型的附件之系統，則你有可能會收到被阻止的通知，或根本沒有通知。

為便於查閱，這裏列出了一些廣為使用的檔案副檔名。

  • .TXT - 表明是文字檔，即只包含文字資訊的檔案。文字檔可用各種文字編輯器開啟和讀取，其中包括 Notepad、WordPad、Word、指令行編輯器、瀏覽器、甚至試算表程式。

  • .HTM - 指 HTML 檔案，通常可用瀏覽器打開，這是一種特殊類型的文字檔。

  • .ZIP – 指用 ZIP 壓縮程式壓縮的檔案。若需讀取和展開 ZIP 檔案，需要WinZip或其他壓縮程式。

  • .EXE - 指執行檔或程式檔案。按兩下這樣的檔案通常可啟動相關的應用程式。

  • .DOC – 指 Microsoft Word 文件檔案。

  • .JPG – 指 JPEG 格式的影像檔。你可在瀏覽器中開啟 JPEG 檔案，也可使用 Paint 或 Adobe Photoshop 一類的圖片處理程式。

  • .GIF – 指 GIF 格式的影像檔。你可用瀏覽器或圖片處理程式開啟 GIF 檔案。

  • .MPG – 指 MPEG 格式的電影片段或動畫檔案。通常可用瀏覽器外挂程式或 Windows Media Player 等播放器播放。

  • .WMV - 指 WMV 格式（Windows 特有格式）的電影片段檔案。

  • .PDF - 指 Acrobat Reader 格式的文件。通常可在瀏覽器中用外挂程式開啟 PDF 檔。
