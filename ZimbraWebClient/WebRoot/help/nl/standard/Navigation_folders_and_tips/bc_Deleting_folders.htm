xxxxMappen verwijderenxxxx

xxxxJe kunt zelfgemaakte mappen verwijderen.xxxx xxxxSysteemmappen kunnen niet worden verwijderd.xxxx

xxxxWanneer je een map verwijdert, wordt de inhoud ervan verplaatst naar de Prullenbak.xxxx xxxxDe inhoud van de Prullenbak kan worden hersteld door de inhoud naar een andere map te verslepen.xxxx

xxxx xxxxxxxxLet opxxxxxxxx:xxxx xxxxDe Prullenbak wordt regelmatig leeggemaakt; dus items in de Prullenbak gaan op een gegeven moment verloren.xxxx

xxxxGa als volgt te werk om een map te verwijderen:xxxx

  1. xxxxSelecteer de map die je wilt verwijderen en klik in de kop van de map op xxxxxxxxBewerkenxxxxxxxx.xxxx

  2. xxxxKruis xxxxxxxxDeze map verwijderenxxxxxxxx aan.xxxx

  3. xxxxKlik op xxxxxxxxMap verwijderenxxxxxxxx.xxxx

xxxxAls er in de map berichten zijn die je wilt bewaren, verplaats je deze vóór het verwijderen naar een andere map.xxxx

 

xxxxDe inhoud van een map verwijderenxxxx

xxxxAls je alles in een van je mappen wilt verwijderen, maar niet de map zelfxxxx

  1. Selecteer de map en klik in de kop van de map op Bewerken.

  2. Schakel de optie Alle items in deze map permanent verwijderen in.

  3. xxxxKlik op xxxxxxxxAlle items verwijderenxxxxxxxx.xxxx xxxxDe inhoud van de map wordt onmiddellijk verwijderd.xxxx xxxxOngedaan maken is niet mogelijk.xxxx
