xxxxAls je naar contacten in je adresboeken wilt zoeken, moet de query een volledig woord zijn.xxxx xxxxAls je bijvoorbeeld naar contacten op telefoonnummer wilt zoeken, moet je het volledige telefoonnummer opgeven zoals het vermeld staat in het contact. Je kunt dus niet alleen maar op netnummer zoeken.xxxx
