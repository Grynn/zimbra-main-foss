  1. Maak de zoekquery door tekst in te voeren op de balk Zoeken of door een query te maken via Geavanceerd zoeken.

  2. Als je de zoekopdracht wilt opslaan, klik je op op de balk Zoeken. xxxxHet dialoogvensterxxxxxxxxZoekopdracht opslaanxxxxxxxx wordt geopend.xxxx

  4. xxxxKlik op xxxxxxxxOKxxxxxxxx.xxxx xxxxJe zoekopdracht wordt opgeslagen.xxxx
