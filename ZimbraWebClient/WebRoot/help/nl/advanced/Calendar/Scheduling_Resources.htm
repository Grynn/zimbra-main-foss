Scheduling resources

Resources are considered a location (meeting room, building, or other location) or equipment (audio/visual) that can be scheduled for a meeting. When you set up a meeting, to schedule a resource, you select the resource and "invite" it to the meeting.  The resource receives the invite and if it is free, accepts the meeting.  

Administrators set up resource accounts and configures the resource's scheduling policy. A meeting can be declined if the resource already has a meeting scheduled for that time.  In come cases, the resource scheduling policy is set up to allows a resource to accept dates in recurring meeting even if there are conflicts.  If you invite a resource to a recurring meeting and some of the dates are already scheduled a Conflicting Resources dialog is displayed shows the conflicting date and time.

If a recurring meeting has scheduled a resource, but has some dates are already scheduled by another meeting, you also receive an email showing which dates conflict.
