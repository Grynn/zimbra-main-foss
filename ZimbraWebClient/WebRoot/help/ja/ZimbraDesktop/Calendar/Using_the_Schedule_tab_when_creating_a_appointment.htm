[提案]パネルで参加者および場所の空き状態を表示する

会議を予定する場合、参加者の予定なし/予定あり状態を確認することができるため、すべての参加者に良い時間を見つけることができます。参加者に予約を追加すると、[概要]ペインの[提案]パネルにその参加者の30分間のタイムスロットと、少なくとも1つの場所が示されます。

ミニカレンダーは、参加者の空き状態を色つきで表示します。

  • 緑色はそれらの日にすべての参加者が参加でき、少なくとも1つの場所が使用可能であることを示します。

  • オレンジは、全員は参加できないが、少なくとも半数は参加でき、少なくとも1つの場所が使用できることを示します。日付の上にマウスポインタを置くと、参加できない人の名前を最大10人まで表示することができます。

  • 赤はそれらの日に半数以上の参加者が参加できないことを示します。

右の数はその時に使用できる場所の数を示します。クリックして、提案される場所のリストを表示して、場所を選択できます。

以下の例では、すべての参加者が1月中は1月18日以外は参加可能であることを示します。1月18日は半数以下の人しか参加できません。

 

提案時間にどの種類の場所を含めるのか、プリファレンスを設定できます。[概要]ペインの見出しで[提案時刻]をクリックして、[提案のプリファレンス]ダイアログを開きます。

  • [名前]、[最小収容]、[説明]、[場所]、[ビル]、[階]： 予約する部屋リソース、名前、収容人数、特定の場所などについて、具体的に指定できます。備考： この情報は、検索するために部屋に対してあらかじめ設定しておく必要があります。

  • [次を含む]：参加者の空き状態の検索に含まれる時間を指定します。

    • [自分の勤務時間のみ]：設定した勤務時間のみを検索します。カレンダーのプリファレンスで設定した勤務日と時間です。

    • [すべての参加者の勤務時間]：場所を検索するとき、すべての人の勤務時間を考慮に入れます。備考： 異なるタイムゾーンの参加者を含める場合、特に離れたタイムゾーンの人が何人もいる場合は、すべての参加者が来れる場所を見つけることはできません。 この場合、提案パネルには場所のリソースを示しません。

    • [非勤務時間]：この検索では、勤務時間としてマークしていない時間を含めます。夜や週末などです。
