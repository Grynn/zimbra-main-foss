}}}Zimbra Desktopについて{{{

}}}Zimbra Desktopは、1つのデスクトップツール内でさまざまなメールアカウントを管理できるオフラインクライアントです。{{{ }}}Zimbra Desktopでは、Zimbraメール、Yahoo!メール、Gmail、Outlook、およびPOPやIMAPを使用するその他の仕事用または個人用のメールアカウントを簡単に設定できます。{{{ }}}各アカウントはそれぞれ個別に管理されます。{{{

}}}アカウントを設定すると、メールメッセージが自分のコンピュータにダウンロードされるようになるため、インターネットに接続しているかどうかに関係なく、いつでもそれらのメールにアクセスできます。{{{}}}「オフラインで作業する」を参照してください。{{{}}}すべてのアカウントのメールメッセージをZimbra Desktopで読み、さらに返信、転送、削除することができます。{{{}}}保存できるメールの量はお使いのコンピュータのディスク容量によって決まります。あらかじめ定義されたアカウントの割り当て容量には関係ありません。{{{

}}}自分のアカウントのいずれかに新しいメールメッセージが届くと、そのメッセージはZimbra Desktopに自動的にダウンロードされます。{{{このとき、Desktopを開く必要はありません。 }}}WindowsのトレイまたはMacのDockにエンベロープアイコンが表示され、すべてのアカウントにおける未読メッセージの合計数が示されます。{{{このアイコンをクリックすると、Desktopアプリケーションが開きます。

}}}Zimbra Desktopで使用できる主な機能{{{

}}}Zimbra Desktopでは、1箇所から複数のメールアカウントにアクセスできるだけでなく、連絡先を管理することもできます。そのほか、カレンダー、タスクリストアプリケーション、添付ファイルを保存するためのブリーフケースツールなどもあります。{{{

}}}主な機能のリストについては、「Zimbra Desktopの機能の概要」を参照してください。{{{

設定して管理できるアカウントの種類

以下のメールアカウントを設定できます。

  • }}}Zimbraアカウント。{{{}}}Zimbraを実行している場合、Zimbra Desktopを使用してメールアカウントにアクセスできます。{{{}}}これには、メールメッセージ、フォルダ、アドレスブック、カレンダー、タスクリスト、ブリーフケースの文書が含まれます。{{{}}}ブラウザ経由でZimbraアカウントにアクセスするには、Zimbra Web Clientを使用します。{{{}}}ZimbraアカウントのZWCで設定したユーザー設定は、Zimbra Desktopからアクセスできます。{{{}}}備考：Zimbra DesktopからZimbraのすべての機能にアクセスするには、ZimbraのバージョンがZCS 5.0以降でなければなりません。{{{}}}Zimbraのバージョン番号については、Zimbraのシステム管理者にお問い合わせください。{{{

  • }}}Yahoo!メール。{{{}}}Yahoo!メール、Yahoo!メールセキュリティーパック、Yahoo! Small Business、Ymail、Rocketmailのアカウントを設定できます。{{{ }}}これらのアカウントのメールメッセージ、メールフォルダ、およびアドレスブックは、Zimbra Desktopからアクセスできます。{{{

  • }}}Gmail。{{{}}}Gmailアカウントは、IMAPアクセスを許可するように設定する必要があります。{{{}}}IMAPを有効にするために、Gmailアカウントにログインしなければならない場合があります。{{{}}}メールメッセージと連絡先は、Zimbra Desktopからアクセスできます。{{{

  • }}}Microsoft Exchange IMAPアカウント。{{{}}}Exchange IMAPアカウントのメールメッセージは、Zimbra Desktopからアクセスできます。{{{

}}}お使いのアカウントが上記の種類以外のものでも、POP3アカウントまたはIMAP4アカウントであれば、Zimbra Desktopでそのアカウントを設定できます。{{{}}}受信メールサーバーや送信メールサーバーの詳細を含め、アカウントに関する設定情報を記入する必要があります。{{{}}}これらの情報については、システム管理者またはインターネットサービスプロバイダにお問い合わせください。{{{
