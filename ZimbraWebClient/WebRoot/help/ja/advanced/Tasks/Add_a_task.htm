タスクを追加する

新しいタスクは、単にタスクリストにタスク名を入力して作成するか、タスクページを開き、備考や添付ファイルなどを含め、タスクに関する詳細情報を追加して作成できます。

  • Zimbraアシスタントを使用すれば、メールメッセージやカレンダー内の会議を操作しているとき、あるいは連絡先を表示しているときでも、タスクをすばやく追加できます。 タスクが作成されると、入力したすべての情報がタスクのテキスト領域に追加されます。

タスクをすばやく追加するには

  1. 概要ペインでタスクリストを選択します。

  2. 内容ペインの「新しいタスクを追加するには、ここをクリック」と表示されている先頭の行をクリックします。    

  3. テキストフィールドが表示されます。タスクの名前を入力します。

  4. Enterキーを押します。そのタスクがタスクリストに追加されます。

  5. 間違ったリストにタスクを作成した場合は、そのタスクを開き、[タスクリスト]ドロップダウンから正しいリストを選択するか、内容ペインから正しいリストにそのタスクをドラッグアンドドロップします。

タスクを詳細情報とともに追加するには

タスクリストを使用して、タスクを作成したり、タスクの進行状況を管理したりできます。また、タスク内に備考を記入することや、ファイルを添付してタスク内から簡単にアクセスできるようにすることも可能です。このようにタスクに関連するすべて情報を一箇所に配置しておけば、他のユーザーとタスクリストを共有するときにとても便利です。

 開始日と期限を入力してプロジェクトの期間を予測したり、優先度を設定したりできます。また、タスクを開始するときには、[開始前]や[進行中]などのステータスを選択したり、パーセント値による完了度を指定したりできます。

  1. ナビゲーションペインでタスクリストを選択します。

  2. タスクツールバーの[新規]をクリックします。 無題のタスクページが開きます。

  3. [件名]フィールドに、タスクの名前を入力します。

  4. （オプション）[日付]領域で、[優先度]、[開始日]、[期限]、およびステータスを指定します。

  5. 目的のタスクリストを開く前に[新規]をクリックした場合でも、[タスクリスト]ドロップダウンリストから正しいタスクリストを選択できます。  

  6. テキストフィールドに備考を記入します。また、添付ファイルを追加する場合は、ツールバーの[添付ファイルを追加]をクリックします。

  7. [保存]をクリックしてタスクを保存します。その新しいタスクがタスクリストに追加されます。
