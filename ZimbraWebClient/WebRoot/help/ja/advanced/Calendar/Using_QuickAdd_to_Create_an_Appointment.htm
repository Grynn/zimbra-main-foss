簡易追加機能を使用して予約を作成する

[予約を簡易追加]ダイアログを使用すると、予約をすばやく作成できます。カレンダーで、開始時刻を指定し、その時刻をダブルクリックするか、開始時刻から終了時刻までドラッグすると、[予約を簡易追加]ダイアログが開きます。

簡易追加はオプションです。カレンダー上をダブルクリックするかドラッグしたときに[予約]ページが開くようにしたい場合は、[オプション]ページで簡易追加機能を無効にします。

クイック予約を作成するには

  1. 予約を作成するカレンダーで、予約の時間をクリックするか、予約の開始時刻から終了時刻までをドラッグします。  [予約を簡易追加]ダイアログが表示されます。

  2. 全日予約を作成するには、日ビューまたは週ビューを開き、カレンダーの上部で1日またはそれ以上の日をドラッグして選択します。

  3. 件名と場所を入力します。繰り返し予約の場合は、繰り返しモードも選択します。

  4. [ＯＫ]をクリックします。 この予約がカレンダーに表示されます。

参加者を招待する必要がある場合や、繰り返しオプションを変更する必要がある場合は、[詳細]をクリックして[予約]ページを開きます。
