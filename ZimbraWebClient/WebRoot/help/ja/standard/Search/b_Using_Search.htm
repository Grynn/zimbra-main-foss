検索を使用する

Zimbraには、パワフルな検索ツールが備わっています。「クエリ言語の記述」で説明しているように、複雑なテキストクエリの作成方法を学ぶこともできますが、キーワードを使わずに名前や語句を入力するだけでも検索できます。たとえば、件名欄、To:欄、From:欄、Cc:欄、Bcc:欄、メッセージ本文、または添付ファイルのどこかに「会議」という語句を含んだメールをすべて検索するには、[検索]テキストボックスに「会議」と入力し、[検索]をクリックするだけです。

検索する場所を選択します。検索は、メッセージ、個人連絡先、個人カレンダー、および個人タスクリスト内で行うことができます。

単語、姓、名、電話番号、ドメイン名など、どのような値でも入力できます。  さらに、アスタリスク（*）をワイルドカードとして使用することも可能です。   

検索でアスタリスクワイルドカード（*）を使用する

検索でアスタリスク（*）をワイルドカードとして使用すると、類似した語句を含む内容を検索できます。  

何らかの文字の後にアスタリスク（*）をワイルドカードとして使用します。たとえば、｢テレ*｣と指定すると、テレビやテレフォンといった語句を含む項目が返されます。

アドレスブック内の連絡先を検索する場合は、検索する語句全体をクエリに使用する必要があります。たとえば、電話番号で連絡先を検索するには、市外局番だけではなく、目的の連絡先エントリに登録されている電話番号全体を入力する必要があります。

アドレスブック内の連絡先を検索するには

  1. [検索]テキスト領域のドロップダウンリストから[個人連絡先]を選択します。これで、すべてのアドレスブックが検索の範囲となります。

  2. [検索]テキストボックスに、名前またはその他の連絡先データを入力します。

  3. [検索]をクリックして検索を実行します。

メールメッセージをすばやく検索するには

  1. [検索]テキスト領域のドロップダウンリストから[メールメッセージ]を選択します。

  2. [検索]テキストボックスに、語句、名前、またはメールに関連したその他のデータを入力します。

  3. [検索]をクリックして検索を実行します。

複雑な検索クエリを作成する

同じ検索クエリを頻繁に実行する場合や、特定の詳細情報を検索する場合は、検索クエリを作成し、それを[検索]フォルダに保存しておきます。そうすれば、その検索クエリをクリックするだけで、同じ検索をいつでも実行できます。

検索クエリを作成して保存するには

  1. 概要ペインの[検索]の横にある[編集]をクリックします。[新しい検索を作成]ペインが開きます。

  2. 検索クエリの保存場所を選択します。検索クエリをルートフォルダに保存した場合は、その検索クエリが[検索]領域に表示されます。別のフォルダを選択した場合は、その検索がそのフォルダ内に保存されます。

  3. [検索名]フィールドに、検索の名前を入力します。

  4. [検索クエリ]フィールドに、「クエリ言語の記述」で説明しているクエリ言語を使用してクエリを入力します。クエリには、And検索やOr検索を含めることもできます。

    検索クエリを作成する際にAND検索またはOR検索を使用する

    複数の条件を使用する検索では、指定した条件のいずれかに一致する項目を探すことも、指定した条件のすべてに一致する項目を探すこともできます。 :

    • 指定した条件のいずれかに一致するメッセージを検索する場合、これをOR検索と言います。なぜなら、メッセージがいずれかの項目（つまり、X OR Y）を含んでいる場合、そのメッセージは一致するとみなされるからです。

    • XとYの両方を含むメッセージを検索する場合、これをAND検索と言います。指定したすべての条件を満たすメッセージが一致項目とみなされます。

  5. [検索を作成]をクリックして、検索を作成し、それを指定のフォルダに保存します。

作成済みの検索を実行するには、概要ペインでその検索名をクリックします。内容ペインに検索結果が表示されます。
